## Aspose.PDF for Node.js via C++에 오신 것을 환영합니다.

Node.js는 JavaScript로 작성된 서버 측 및 네트워킹 애플리케이션을 위한 크로스 플랫폼 런타임 환경입니다. 기본적으로 브라우저 컨텍스트 밖에서 JavaScript 애플리케이션을 실행할 수 있게 해줍니다.

Aspose.PDF for Node.js via C++는 JavaScript를 실행할 수 있게 해 주고 개발자들이 Node.js 환경에서 PDF와 관련된 다양한 작업을 수행하도록 돕는 강력한 툴킷입니다.

Aspose.PDF for Node.js는 개발자들이 PDF 파일을 직접 조작할 수 있게 합니다. Aspose.PDF for Node.js via C++는 WebAssembly 기술을 사용하여 구축되었으며 Aspose.PDF for .NET을 기반으로 합니다.
