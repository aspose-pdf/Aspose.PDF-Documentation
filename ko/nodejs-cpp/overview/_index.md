_이 페이지는 Aspose.PDF for Node.js via C++ 기능에 대한 개요입니다._

**Aspose.PDF for Node.js via C++**는 개발자가 Node.js 환경에서 PDF 파일을 직접 조작할 수 있게 합니다. 이 Node.js 솔루션은 WebAssembly 기술을 사용하여 구축되었으며 Aspose.PDF for .NET을 기반으로 합니다. aspose-pdf for Node.js via C++를 쉽게 사용하고 설치 지침을 따라 주세요.

**CJS 및 MJS 파일 확장자에 대해**

Node.js는 코드를 조직하고 공유하기 위한 두 가지 모듈 시스템을 지원합니다. 다음과 같습니다:

- CommonJS 모듈
- ECMAScript 모듈

CommonJS 모듈은 Node.js에서 고전적인 종속성 처리 방식이었으며, ECMAScript 모듈은 최신 추가 기능으로 현대 브라우저에서 사용되는 표준 형식을 따릅니다. 'cjs', 'mjs'는 JavaScript 파일에서 사용되는 모듈 형식(CommonJS 및 ECMAScript 모듈)을 지정하기 위해 사용되는 확장자입니다. 

코드 스니펫에서 AsposePDFforNode.js 모듈을 사용할 때 차이를 확인할 수 있습니다. 'require'와 'import'의 주요 차이점은 사용 사례에 있습니다. **Require**는 전체 모듈을 가져오는 것에만 제한되는 반면, **Import**는 전체 모듈뿐만 아니라 해당 모듈의 특정 내보내기도 가져오는 유연성을 제공합니다.

**두 가지 다른 접근 방식**

JavaScript는 비동기 작업을 처리하기 위해 'then'과 'await' 두 가지 접근 방식을 제공합니다.

'then' 메서드는 프로미스를 사용하여 성공적인 결과와 오류를 순차적으로 관리합니다. 반면에, 'await'는 JavaScript가 대기하도록 만들며, 실제로 해당 프로미스가 해결되고 결과를 반환할 때까지 함수 실행을 일시 중지합니다.

Aspose.PDF for Node.js는 다음과 같은 다양한 기능을 지원합니다:

- PDF 파일 정보 설정, 가져오기 및 제거
- 북마크 삭제
- 주석 삭제
- PDF에서 텍스트를 추출
- PDF 문서 최적화
- 리소스 최적화 
- PDF를 두 개의 파일로 분할합니다.
- 두 파일을 PDF로 병합
- PDF 복구
- 변환 기능

또한, Aspose.PDF for Node.js via C\u002B\u002B는 높은 성능과 완벽한 품질로 PDF 문서를 다양한 형식으로 쉽게 변환하는 데 적용할 수 있습니다. 섹션을 참조하십시오 [Aspose.PDF 지원 파일 형식](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). 또한, 이 툴킷은 PDF 문서의 암호화 및 복호화를 지원합니다.
