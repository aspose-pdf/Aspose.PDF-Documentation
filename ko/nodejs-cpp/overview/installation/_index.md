# 설치

**Aspose.PDF for Node.js via C++**는 PDF를 Node.js 환경에서 직접 작업하기 위해 사용되는 쉽고 안전한 툴킷입니다. 

Aspose.PDF for Node.js via C++를 *-ZIP 아카이브에서 설치하고 사용하려면, 다음 지침을 따르세요:

- Aspose.PDF 사이트에서 아카이브를 다운로드합니다
- ZIP 압축 파일에서 파일을 추출합니다
- 존재한다면 *.lic 파일을 'aspose.lic'으로 이름을 바꾸세요

다음에 대한 링크를 따라가세요 [직접 다운로드](https://releases.aspose.com/pdf/nodejscpp/)

NPMJS에서 Aspose.PDF for Node.js via C++를 설치하고 사용하려면, 다음 지침을 따르세요:

- npm i asposepdfnodejs
- 존재하는 경우 *.lic 파일을 ‘aspose.lic’으로 이름을 바꾸세요
