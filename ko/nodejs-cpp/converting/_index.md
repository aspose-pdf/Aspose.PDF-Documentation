Aspose.PDF for Node.js via C\u002B\u002B는 Node.js 환경에서 PDF 문서를 다양한 형식으로 변환할 수 있도록 지원합니다. 또한 Aspose.PDF 변환 품질을 확인하고 Aspose.PDF 변환기 앱을 사용하여 온라인으로 결과를 확인할 수 있습니다. 코드 샘플을 통해 문서 변환 섹션을 학습하십시오.

## C++를 통해 Node.js를 사용하여 변환하는 방법

Microsoft Word 문서는 가장 다재다능하고 편집이 가능한 문서입니다. 이 기사에서는 Aspose.PDF를 사용하여 Node.js 환경에서 PDF를 Word로 변환하는 방법을 배웁니다.

- [PDF를 DOC로 변환](/pdf/ko/nodejs-cpp/convert-pdf-to-doc/) - Node.js를 사용하여 PDF 문서를 Word 형식으로 변환할 수 있습니다.

숫자 형식은 표의 데이터를 읽기 쉽게 만들 뿐만 아니라 표를 보다 쉽게 사용할 수 있도록 필요합니다. 물론, 이러한 데이터를 PDF 문서에서 Excel 형식으로 변환해야 한다면 당사의 Aspose.PDF for Node.js를 사용하십시오.

- [PDF를 Microsoft XLSX로 변환](/pdf/ko/nodejs-cpp/convert-pdf-to-xlsx/) - 이 섹션에서는 PDF 문서를 XLSX로 변환하는 방법을 설명합니다.

PowerPoint 형식은 다양한 프레젠테이션을 만드는 데 사용됩니다. PPT 파일은 다양한 정보를 포함하는 많은 슬라이드 또는 페이지를 포함합니다.

- [PDF를 Microsoft PowerPoint로 변환](/pdf/ko/nodejs-cpp/convert-pdf-to-powerpoint/) - 여기서는 변환 프로세스를 추적하면서 PDF를 PowerPoint로 변환하는 것에 대해 이야기합니다.

HyperText Markup Language는 하이퍼텍스트 문서 기술 언어이며, 웹 페이지를 만들기 위한 표준 언어입니다. Aspose.PDF for Node.js를 사용하면 HTML 문서를 쉽게 변환하고 그 반대도 가능합니다.

다양한 목적을 위해 PDF로 변환해야 하는 이미지 형식이 많이 있습니다. Aspose.PDF for Node.js는 가장 널리 사용되는 이미지 형식을 지원합니다.

- [PDF를 다양한 이미지 형식으로 변환](/pdf/ko/nodejs-cpp/convert-pdf-to-images-format/) - PDF 페이지를 JPEG, PNG, SVG 등 다른 형식의 이미지로 변환합니다

이 섹션에는 다음과 같은 형식이 포함됩니다: EPUB, XPS, TeX, Text, 및 Grayscale PDF.

- [PDF 파일을 다른 형식으로 변환하기](/pdf/ko/nodejs-cpp/convert-pdf-to-other-files/) - 이 항목은 PDF 문서를 다양한 형식으로 변환하는 방법을 설명합니다

PDF/A는 전자 문서의 장기 보관을 위해 설계된 PDF 버전입니다.
솔직히 외부에서는 PDF인지 PDF/A인지 판단하기가 매우 어렵습니다. 이 파일을 확인하려면 검증자를 사용합니다. PDF를 PDF/A로, 그리고 그 반대로 변환하는 품질에 관한 다음 기사들을 확인하십시오.

- [PDF를 PDF/A 형식으로 변환](/pdf/ko/nodejs-cpp/convert-pdf-to-pdfa/) - Aspose.PDF의 Node.js 툴킷은 PDF를 PDF/A로 변환하는 쉬운 방법을 제공합니다.

- [PDF/A를 PDF 형식으로 변환](/pdf/ko/nodejs-cpp/convert-pdfa-to-pdf/) - Node.js를 사용하여 PDF/A를 PDF 형식으로 쉽고 빠르게, 높은 품질로 변환합니다.
