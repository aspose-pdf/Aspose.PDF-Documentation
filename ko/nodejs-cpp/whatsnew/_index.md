Aspose.PDF for Node.js via C++의 첫 번째 공개 릴리스를 발표하게 되어 기쁩니다.

**Aspose.PDF for Node.js via C++** 는 개발자가 PDF 파일을 Web에서 직접 조작할 수 있게 합니다. 이 Node.js 솔루션은 WebAssembly 기술을 활용하여 구축되었으며 Aspose.PDF for .NET을 기반으로 합니다. aspose-pdf for Node.js via C++ 를 쉽게 사용할 수 있으며, 그 후 설치 안내를 따라 주세요.

방문해 주세요 [제품 페이지](https://products.aspose.com/pdf/nodejs-cpp/) 자세한 내용을 보시고 첫 번째 릴리스를 다운로드할 수 있는 기회를 얻으세요 [Aspose.PDF for Node.js via C++]().

## **주요 지원 기능**

- PDF 파일 정보 설정, 가져오기 및 제거
- 북마크 삭제
- 주석 삭제
- PDF에서 텍스트를 추출
- PDF 문서 최적화
- 리소스 최적화 
- PDF를 두 개 파일로 분할
- 두 파일을 PDF로 병합
- PDF 복구
