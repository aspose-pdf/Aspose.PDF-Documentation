PDF는 Portable Document Format의 약자로, 소프트웨어, 하드웨어 또는 운영 체제와 무관하게 전자 형태로 문서를 표시하는 데 사용됩니다.

PDF는 현재 국제 표준화 기구(ISO)에서 관리하는 개방형 표준입니다.

원래 목표는 문서의 내용과 레이아웃을 보존하고 보호하는 것이었으며, 어느 플랫폼이나 컴퓨터 프로그램에서 보든 상관없었습니다. 이 때문에 PDF는 편집이 어렵고 때로는 정보를 추출하는 것조차도 도전이 됩니다.

하지만 **Aspose.PDF for Node.js via C++**는 PDF 문서를 작업할 때 발생하는 대부분의 작업을 처리하는 데 도움이 될 수 있습니다.

다음 작업을 수행할 수 있습니다:

- [PDF 최적화](/pdf/ko/nodejs-cpp/optimize-pdf/) - 페이지 콘텐츠 최적화, 파일 크기 최적화, 사용되지 않은 객체 제거, 모든 이미지를 압축하여 문서를 성공적으로 최적화합니다.
- [PDF 리소스 최적화](/pdf/ko/nodejs-cpp/optimize-pdf-resources/) - 페이지 콘텐츠 최적화, 파일 크기 최적화, 사용되지 않은 객체 제거, 모든 이미지를 압축하여 문서를 성공적으로 최적화합니다.
- [PDF 병합](/pdf/ko/nodejs-cpp/merge-pdf/) - Node.js를 사용하여 여러 PDF 파일을 하나의 PDF 문서로 병합합니다.
- [PDF 분할](/pdf/ko/nodejs-cpp/split-pdf/) - Node.js via C++를 사용하여 PDF 페이지를 개별 PDF 파일로 분할합니다.
