PDF는 Portable Document Format의 약자로, 소프트웨어, 하드웨어 또는 운영 체제와 무관하게 전자 형태로 문서를 표시하는 데 사용됩니다.

PDF는 오늘날 국제 표준화 기구(ISO)가 유지 관리하는 개방형 표준입니다.

원래 목표는 문서의 내용과 레이아웃을 보존하고 보호하는 것이었으며, 어떤 플랫폼이나 컴퓨터 프로그램에서 보든 상관없었습니다. 그래서 PDF는 편집이 어려우며 때로는 정보 추출 자체가 도전이 됩니다.

하지만 **Aspose.PDF for Go via C++**는 PDF 문서 작업 시 발생하는 대부분의 작업을 처리하는 데 도움을 줄 수 있습니다.

다음과 같은 작업을 수행할 수 있습니다:

- [PDF 최적화](/pdf/ko/go-cpp/optimize-pdf/) - 페이지 내용 최적화, 파일 크기 최적화, 사용되지 않는 개체 제거, 모든 이미지 압축을 통해 문서 최적화를 성공적으로 수행합니다.
- [PDF 리소스 최적화](/pdf/ko/go-cpp/optimize-pdf-resources/) - 페이지 내용 최적화, 파일 크기 최적화, 사용되지 않는 개체 제거, 모든 이미지 압축을 통해 문서 최적화를 성공적으로 수행합니다.
