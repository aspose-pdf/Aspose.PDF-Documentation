## C++를 통한 Aspose.PDF for Go

_이 페이지는 Aspose.PDF for Go 기능에 대한 개요입니다._

Go는 종종 Golang이라고도 불리며, 구글이 확장 가능하고 효율적인 소프트웨어 개발에서 직면한 과제를 해결하기 위해 개발한 현대적인 프로그래밍 언어입니다. 2009년 Robert Griesemer, Rob Pike, Ken Thompson에 의해 만들어진 Go는 단순함, 속도, 신뢰성을 결합하여 개발자들 사이에서 인기 있는 언어가 되었습니다.

Go 언어는 가비지 컬렉션, 강력한 타입 지정, 그리고 goroutine과 채널과 같은 동시성 메커니즘과 같은 유익한 기능들을 포함합니다. 이러한 기능들은 개발자가 견고한 애플리케이션을 작성하여 여러 작업을 동시에 효율적으로 처리할 수 있게 하며, 이는 클라우드 기반 서비스와 웹 서버에 매우 중요합니다.

Go의 인기는 그 쉬움, 최소주의 디자인, 그리고 원활한 확장성에서 비롯됩니다. 컴파일 속도가 뛰어나 고도로 최적화된 바이너리를 생성하여 런타임 오버헤드를 줄입니다. 도구와 라이브러리를 포함한 성장하는 생태계와 함께 Go는 마이크로서비스, DevOps, 빅데이터와 같은 분야에서 널리 채택되고 있습니다.

이 언어는 또한 커뮤니티 중심 개발을 강조하여 개발자들 간의 협업을 촉진합니다. Go의 실용적인 코딩 접근 방식과 성능에 대한 집중은 현대의 고성능 애플리케이션에 이상적입니다.

Go는 2018년 이후 인기가 높아졌으며, 개발자들이 스택에 추가하고 싶어 하는 가장 인기 있는 언어입니다.

이 라이브러리를 사용하면 Adobe Acrobat을 설치하지 않고도 PDF 파일을 처음부터 만들거나 기존 PDF 문서를 완전히 처리하는 풍부한 기능을 구현할 수 있습니다. Aspose.PDF for Go는 **다양한 기능**을 지원합니다.

### 문서 최적화

**Aspose.PDF for Go**는 PDF를 최적화하는 다양한 방법을 제공합니다, 예:

- PDF 최적화.
- PDF 리소스 최적화.

자세한 내용은 Docs 페이지를 확인해 보세요: [PDF 최적화](/pdf/ko/go-cpp/optimize-pdf/)

### 페이지 추가, 삭제 또는 회전

그 [PDF 페이지 작업](/pdf/ko/go-cpp/working-with-pages/) 섹션에서는 PDF에서 페이지를 추가, 삭제, 회전하는 방법을 설명합니다.

PDF 페이지를 추가, 삭제 및 회전하고, 페이지 속성을 가져오고 설정하며, 배경 색을 지정하여 문서의 구성과 접근성을 향상시킬 수 있습니다.

### PDF 페이지를 다른 형식으로 변환합니다

PDF 페이지를 이미지로 변환하면 PDF 문서를 직접 보거나 상호작용하는 데 어려움을 겪는 사용자에게 콘텐츠 접근성을 높일 수 있습니다. Aspose.PDF는 Go를 사용하여 PDF를 다양한 형식으로 변환할 수 있도록 합니다. 다음과 같습니다:

- TIFF
- BMP
- JPG
- PNG
- DICOM
- SVG
- DOC
- DOCX
- EPUB
- PPTX
- TEX
- TXT
- XLSX
- XPS

당신은 방문할 수 있습니다 [PDF 형식 변환](/pdf/ko/go-cpp/conversion/) 자세한 내용 페이지.
