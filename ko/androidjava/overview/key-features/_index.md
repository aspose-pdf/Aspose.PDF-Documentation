## 일반 기능

- Aspose.PDF for Android via Java는 Java 런타임을 실행하는 모든 모바일 장치를 지원합니다, 포함
- API는 40비트 또는 128비트 암호화를 제공합니다.

## 지원되는 PDF 버전

Aspose.PDF for Android via Java은 PDF 버전 1.2, 1.3, 1.4, 1.5, 1.6 및 1.7을 지원합니다.

## 글꼴

- 핵심 글꼴 14개.
- PostScript Type1 글꼴.
- TrueType 글꼴.
- Type 3 글꼴.
- PDF에 글꼴 정보 삽입
- CJK 글꼴.
- Unicode 지원.

## 텍스트

- 텍스트 교체.
- 텍스트 추출 (전체 PDF, 특정 페이지 또는 페이지 범위에서).
- 텍스트 추가.

## 그래프

- 선, 원, 호, 곡선과 같은 다양한 그래프 모양을 지원합니다.
- 그래프는 좌표로 지정된 페이지의 임의 위치에 그릴 수 있습니다.
- 그래프는 회전, 확대/축소, 채우기가 가능하며, 대시 및 폴리대시 선 스타일을 정의할 수 있습니다.
- 그래프는 좌표를 기반으로 그려지며 상대적인 위치가 아니라 구식 위치를 가지고 있습니다.
- CMYK 및 Gray 색 공간이 지원됩니다.
