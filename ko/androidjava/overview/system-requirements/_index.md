## Overview

PDF 문서를 생성하기 위해 [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/)를 실행하는 기계에는 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html)이 설치되어 있을 필요가 없습니다. Aspose.PDF for Java 자체가 PDF 문서 생성 엔진이기 때문입니다. 그러나 Android용 Java를 통해 Aspose.PDF가 생성한 PDF 문서를 보기 위해서는 기계에 적어도 [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html)가 필요합니다. 이는 [여기서](https://www.adobe.com/acrobat/pdf-reader.html) 무료로 다운로드할 수 있습니다.

## Supported Operating Systems

Android용 Java를 통해 Aspose.PDF는 Java 런타임을 실행하는 모든 모바일 장치를 지원합니다. 그러나 이에 국한되지 않습니다:

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## 개발 환경

Java를 통해 Android에서 Java 플랫폼을 대상으로 하는 개발 환경에서 Aspose.PDF for Android를 사용하여 애플리케이션을 개발할 수 있지만, 다음 환경은 명시적으로 지원됩니다:

- Android Studio 및 SDK 도구
