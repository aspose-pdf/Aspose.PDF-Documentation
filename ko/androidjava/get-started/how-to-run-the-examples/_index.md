## GitHub에서 다운로드

Aspose.PDF for Android via Java의 모든 예제가 호스팅되는 곳은 [Github](https://github.com/aspose-pdf/Aspose.PDF-for-Java). 저장소를 좋아하는 Github 클라이언트를 사용해 복제하거나 ZIP 파일을 다운로드할 수 있습니다. [여기](https://github.com/aspose-pdf/Aspose.PDF-for-Java/archive/master.zip).

ZIP 파일의 내용을 컴퓨터의 임의 폴더에 추출합니다. 모든 예제는 **Examples** 폴더에 있습니다.

![todo:image_alt_text](how-to-run-the-examples_1.png)

이 프로젝트는 Maven 빌드 시스템을 사용합니다. 최신 IDE라면 프로젝트와 종속성을 쉽게 열거나 가져올 수 있습니다. 아래에서는 인기 있는 IDE를 사용하여 예제를 빌드하고 실행하는 방법을 보여줍니다.

### IntelliJ IDEA

**File** 메뉴를 클릭하고 **Open**을 선택합니다. 프로젝트 폴더로 이동하여 **pom.xml** 파일을 선택합니다.

![todo:image_alt_text](how-to-run-the-examples_2.png)

프로젝트가 열리고 종속성이 자동으로 다운로드됩니다. Project 탭에서 **src/main/java** 폴더의 예제들을 탐색합니다. 예제를 실행하려면 파일을 오른쪽 클릭하고 "Run .."을 선택하면 예제가 실행되고 출력이 내장 콘솔 출력 창에 표시됩니다.

![todo:image_alt_text](how-to-run-the-examples_3.png)

### Eclipse

**File** 메뉴를 클릭하고 **Import**를 선택합니다. **Maven** - Existing Maven Projects를 선택합니다.

![todo:image_alt_text](how-to-run-the-examples_4.png)

GitHub에서 복제하거나 다운로드한 폴더로 이동하여 **pom.xml** 파일을 선택합니다.

![todo:image_alt_text](how-to-run-the-examples_5.png)

프로젝트가 열리고 종속성이 자동으로 다운로드됩니다. Package Explorer 탭에서 **src/main/java** 폴더의 예제들을 탐색합니다. 예제를 실행하려면 파일을 오른쪽 클릭하고 **Run As** - **Java Application**을 선택하면 예제가 실행되고 출력이 내장 콘솔 출력 창에 표시됩니다.

![todo:image_alt_text](how-to-run-the-examples_6.png)

### NetBeans

**File** 메뉴를 클릭하고 **Open Project**를 선택합니다. GitHub에서 복제하거나 다운로드한 폴더를 찾아갑니다. **Examples** 폴더의 아이콘이 Maven 프로젝트임을 표시합니다. Examples를 선택하고 엽니다.

![todo:image_alt_text](how-to-run-the-examples_7.png)

프로젝트가 열리며 의존성이 자동으로 다운로드됩니다. Projects 탭에서 **source packages**에 있는 예제를 찾아볼 수 있습니다. 예제를 실행하려면 파일을 마우스 오른쪽 클릭하고 **Run File**을 선택하면 예제가 실행되고 출력이 내장 콘솔 출력 창에 표시됩니다.

![todo:image_alt_text](how-to-run-the-examples_8.png)

### 기여

예제를 추가하거나 개선하고 싶다면, 프로젝트에 기여해 주시길 권장합니다. 이 저장소의 모든 예제와 쇼케이스 프로젝트는 오픈 소스이며 귀하의 애플리케이션에서 자유롭게 사용할 수 있습니다.

기여하려면 저장소를 포크하고, 소스 코드를 편집한 뒤 풀 리퀘스트를 생성하면 됩니다. 변경 사항을 검토한 후 유용하다고 판단되면 저장소에 포함시킬 것입니다.
