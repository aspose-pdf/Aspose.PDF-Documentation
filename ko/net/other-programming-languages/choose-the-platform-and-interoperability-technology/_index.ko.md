{{% alert color="primary" %}}

[Aspose.PDF for .NET](/pdf/net/)은 처음부터 PDF 문서를 생성하는 기능을 제공하며 기존 PDF 문서를 편집/조작할 수 있는 클래스 라이브러리입니다. Aspose.PDF는 [Aspose.PDF for .NET](/pdf/net/) 및 [Aspose.PDF for Java](/pdf/java/) 두 가지 독립적인 제품으로 제공됩니다. 우리는 Aspose.PDF가 "기본적으로" 다음을 지원한다고 말할 수 있습니다:

- .NET 프레임워크 및 C# 또는 VB.NET과 같은 .NET 플랫폼을 대상으로 하는 모든 프로그래밍 언어.
- Java 플랫폼 및 Java 프로그래밍 언어.

{{% /alert %}}

[Aspose.PDF for .NET](/pdf/net/)은 ASP, C++, ColdFusion, Delphi, JavaScript, Perl, PHP, PowerBuilder, Python, VBScript 등 많은 프로그래밍 언어에서 사용할 수 있다는 점을 유의하십시오.

## 다른 언어 지원

{{% alert color="primary" %}}

이 섹션에는 다양한 프로그래밍 언어로 Aspose.PDF를 사용하는 방법을 보여주는 기사와 예제가 포함되어 있으며 다음 주제를 다룹니다:
이 섹션에는 다양한 프로그래밍 언어에서 Aspose.PDF를 사용하는 방법을 보여주는 기사와 예제가 포함되어 있으며 다음 주제를 다룹니다:

- “[Aspose.PDF for .NET](/pdf/net/) 또는 [Aspose.PDF for Java](/pdf/java/)” 중 어느 것을 사용할지 결정하는 방법은 무엇입니까?
- Aspose.PDF 객체의 객체를 생성하고, 메서드를 호출하고, 속성에 접근하는 방법.

이 섹션의 원칙을 따르면 프로젝트에서 Aspose.PDF의 많은 기능을 활용할 수 있습니다.

{{% /alert %}}

**Aspose.PDF 클래스 라이브러리의 두 가지 버전이 있습니다:**

- .NET 플랫폼을 대상으로 하는 모든 프로그래밍 언어와 함께 사용할 [Aspose.PDF for .NET](/pdf/net/).
- Java 프로그래밍 언어와 함께 사용할 [Aspose.PDF for Java](/pdf/java/).

다른 프로그래밍 언어에서 Aspose.PDF를 사용하려면 *상호 운용성 기술* 의 사용이 필요합니다. 상호 운용성 기술은 프로그래밍 언어가 .NET 또는 Java용으로 작성된 클래스 라이브러리를 호출할 수 있도록 해야 합니다.

{{% alert color="primary" %}}

다양한 프로그래밍 언어에 사용하기 위한 권장 상호 운용성 기술 목록은 다음과 같습니다:
{{% /alert %}}

|**프로그래밍 언어** |**사용할 Aspose.PDF** |**상호 운용 기술** |
| :- | :- | :- |
|ASP |.NET |COM Interop |
|Delphi |.NET |COM Interop |
|Jscript |.NET |COM Interop |
|Perl |.NET |COM Interop |
| |Java |Inline::Java |
|PHP |.NET |COM Interop |
| |Java |Java Bridge |
|PowerBuilder |.NET |COM Interop |
|Python |.NET |COM Interop |
| |Java |JPype |
|VBScript |.NET |COM Interop |
|Visual Basic |.NET |COM Interop |
