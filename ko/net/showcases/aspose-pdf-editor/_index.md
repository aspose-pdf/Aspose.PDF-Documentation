## Aspose.PDF for .NET의 Html5 PDF 편집기란 무엇인가요?

Aspose.PDF for .NET의 HTML5 PDF 편집기는 사용자가 온라인에서 PDF 파일을 생성, 편집 및 변환할 수 있도록 하는 오픈 소스 웹 기반 PDF 편집기입니다. 사용자는 PDF 파일을 보기 및 편집하기 위해 자신의 웹 애플리케이션에 편집기를 쉽게 삽입할 수 있습니다. HTML5 PDF 편집기는 HTML5, jQuery Ajax, ASP.NET, Bootstrap을 사용하여 개발되었으며 백엔드는 Aspose.PDF for .NET에 의해 지원됩니다. 편집기의 UI는 사용자 요구에 따라 기능을 쉽게 이해하고 향상시킬 수 있도록 매우 간단하게 유지됩니다.

## PDF 편집기 개발자 가이드

### 새로운 PDF 파일 생성

기존 PDF 문서를 편집하는 것 외에도, Html5 PDF 편집기는 메뉴 바에서 "새 파일 생성" 옵션을 사용하여 처음부터 새로운 PDF 파일을 생성하는 것도 지원합니다. 이 기능을 사용하여 편집기에서 빈 PDF를 생성하고, 텍스트/이미지를 추가한 후 원하는 형식으로 저장할 수 있습니다. 다음 섹션에서는 이 기능의 기술적 세부 사항에 대해 논의하겠습니다.

**HTML - "새 파일 생성" 메뉴 항목이 페이지에서 클릭됨**

"새 파일 생성" 메뉴 항목을 클릭하면 Editor.js 파일에서 onNewFileClicked 메서드가 호출됩니다.

**jQuery - CreateNewFile 메서드에 대한 Ajax 요청을 서버로 전송**

Editor.js 탭 아래에서 onNewFileClicked 메서드의 소스 코드를 확인하십시오. 이 메서드는 CanvasSave.aspx.cs 파일에서 CreateNewFile 메서드를 호출합니다.

**ASP.NET 웹 메서드가 서버 요청을 처리합니다**

Canvas.aspx.cs 탭 아래에서 CreateNewFile 메서드의 소스 코드를 확인하십시오. 이 메서드는 ResetData 메서드를 사용하여 이전에 로드된 파일에 대한 기존 데이터를 지웁니다.

**Aspose.PDF for .NET를 사용한 새로운 PDF 파일 생성**

ResetData 메서드를 사용하여 데이터를 지운 후, CreateNewFile 메서드는 Aspose.PDF for .NET의 Document 클래스를 사용하여 새로운 PDF 파일을 생성합니다. 아래 탭에서 볼 수 있듯이 Document 객체는 빈 페이지가 하나 있는 파일을 생성합니다. 서버에서 파일이 생성된 후, 파일은 캔버스에 로드되기 위해 ImageConverter 메서드를 사용하여 이미지로 변환됩니다.

**결과 파일을 캔버스에 로드합니다.**

파일이 서버 측에서 이미지로 변환된 후, 제어는 Editor.js의 onNewFileClicked 메서드로 반환됩니다. onNewFileClicked 메서드는 모든 클라이언트 측 컬렉션을 재설정하고 생성된 이미지 파일을 캔버스에 로드합니다.

### PDF를 다양한 파일 형식으로 내보내기

HTML5 PDF 편집기는 메뉴 바에서 "파일 내보내기" 옵션을 사용하여 PDF 파일을 다양한 파일 형식으로 내보내는 것을 지원합니다. 이 기능을 사용하여 원하는 형식으로 PDF 파일을 내보낼 수 있습니다. 다음 섹션에서는 이 기능의 기술적 세부 사항에 대해 논의하겠습니다.

**HTML - "파일 내보내기" 메뉴 항목이 페이지에서 클릭됨**

"파일 내보내기" 메뉴 항목을 클릭하면 목록에서 내보낼 형식을 선택할 수 있습니다. 내보낼 형식을 선택한 후, Editor.js 파일에서 "ExportFile" 메서드가 호출됩니다.

**jQuery - btnFileExport_Click 메서드에 대한 Ajax 서버 요청 전송**

Editor.js 탭 아래에서 "ExportFile" 메서드의 소스 코드를 확인하십시오. 이 메서드는 CanvasSave.aspx.cs 파일에서 파일 형식 매개변수를 사용하여 서버 메서드 btnFileExport_Click에 요청을 보냅니다.

**ASP.NET 웹 메서드가 서버 요청을 처리합니다**

Canvas.aspx.cs 탭 아래를 확인하십시오. btnFileExport_Click에 전달된 파일 형식 매개변수를 기반으로 PDF 파일은 Aspose.PDF Document 객체의 Save 메서드를 사용하여 변환됩니다.

**클라이언트 브라우저에서 다운로드를 위한 파일 내보내기**

서버에서 파일이 생성된 후, 제어는 Editor.js의 ExportFile 메서드로 반환되어 파일이 사용자가 다운로드할 수 있도록 브라우저로 전송됩니다.

### PDF 파일 추가 또는 병합

Html5 PDF 편집기는 메뉴 바에서 "파일 추가" 옵션을 사용하여 PDF 파일을 추가하거나 병합하는 것을 지원합니다. 이 기능을 사용하여 입력 파일에 PDF 파일을 추가할 수 있습니다. 다음 섹션에서는 이 기능의 기술적 세부 사항에 대해 논의하겠습니다.

**HTML - "파일 추가" 메뉴 항목이 페이지에서 클릭됨**

"파일 추가" 메뉴 항목을 클릭하면 파일 업로드 대화 상자를 사용하여 파일을 업로드할 수 있습니다. 파일이 업로드된 후, Editor.js 파일에서 "fileSelected" 메서드가 호출됩니다.

**jQuery - ProcessRequest 메서드에 대한 서버 요청 전송**

Editor.js 탭 아래에서 "fileSelected" 메서드의 소스 코드를 확인하십시오. 파일이 서버에 게시되고 CanvasSave.aspx.cs 파일에서 "ProcessRequest" 메서드가 호출됩니다.

**ASP.NET 웹 메서드가 서버 요청을 처리합니다**

Canvas.aspx.cs 탭 아래를 확인하십시오. 전달된 양식 매개변수를 기반으로 추가할 파일이 서버에 저장되고 "AppendFile" 메서드가 호출됩니다. AppendFile 메서드는 Aspose.PDF for .NET를 사용하여 업로드된 파일을 추가하고 결과 파일을 이미지로 변환한 후 제어를 Editor.js의 "fileSelected" 메서드로 반환합니다.

**파일 추가 후 캔버스 내용 업데이트**

서버에서 파일이 생성된 후, 제어는 Editor.js의 "fileSelected" 메서드로 반환되어 편집기의 내용을 업데이트합니다.
