## 소개

Aspose .NET PDF 가져오기 DNN 모듈을 사용하면 Adobe Acrobat이나 PDF 리더와 같은 다른 소프트웨어 없이도 PDF 문서의 내용을 읽을 수 있습니다.

### 모듈 기능

이 모듈은 [Aspose.PDF](http://www.aspose.com/pdf-component-suite.aspx)가 제공하는 강력한 가져오기 기능을 보여줍니다. 간단한 파일 브라우저 컨트롤, 콘텐츠 창 선택, 그리고 모듈이 추가된 페이지에 **PDF에서 가져오기** 버튼을 추가합니다. 버튼을 클릭하면 문서의 내용이 즉시 화면에 표시됩니다.

## 설치

다운로드한 후, 다음 단계에 따라 모듈을 귀하의 DNN 웹사이트에 설치하세요:

1. **Host** 또는 다른 Supersuser 레벨 계정으로 사이트에 로그인하세요.
1. Host 메뉴로 이동하여 **Extensions**을 선택하세요.
1.
1. 다운로드한 zip 파일의 위치로 이동한 후 선택하고 **열기**를 클릭하세요.
1. **다음**을 클릭하고, 라이선스에 동의한 후 설치를 계속 진행하세요. 완료되면 **반환** 버튼을 클릭하세요.

DNN에서 제공하는 [이 모듈 설치 비디오](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)를 확인하세요

**참고:** 모듈을 업로드하는 동안 오류가 발생한다면, 이는 DNN 설치의 web.config에서 maxRequestLength 제한 때문입니다. web.config를 열고 maxRequestLength를 **maxRequestLength=”20480″**로 설정하여 20MB로 업데이트한 후 모듈을 다시 업로드하세요.

## 사용법

Aspose .NET PDF Import for the DNN 모듈을 설치한 후에는 웹사이트에서 사용하기 매우 간단합니다. 시작하기 위해 다음 단계를 따르세요

1. DNN에서 Host 또는 Admin 레벨 계정으로 로그인되어 있는지 확인하세요
1. Import 모듈을 추가하고 싶은 페이지로 이동하세요
1.
1. 목록에서 **Aspose .NET PDF Import for DNN**을 선택하고 원하는 페이지 위치로 드래그하세요.

Aspose .NET PDF Import for DNN 모듈을 페이지에 성공적으로 추가했습니다. 파일 브라우저와 **PDF에서 가져오기**라는 버튼이 페이지에 함께 나타납니다. 누구나 PDF 파일을 선택하고 **PDF에서 가져오기** 버튼을 클릭하여 선택한 문서의 내용을 페이지에 나열할 수 있습니다.

#### 소스 코드 구성 방법

소스 코드를 열고 확장하려면 다음이 설치되어 있어야 합니다.

- Visual Studio 2010 이상
- [DNN 개발 템플릿](http://www.aspose.com/docs/display/totalnet/1.2.1+Downloading)

아래 간단한 단계를 따라 시작하세요.

1. 소스 코드를 다운로드/복제합니다.
1. Visual Studio 2010을 열고 **파일** > **프로젝트 열기**를 선택합니다.
1. 다운로드한 최신 소스 코드로 이동하여 **AsposeDNNPdfImport.csproj**를 엽니다.
