## 소개

Aspose .NET PDF Import for DNN 모듈은 개발자가 Adobe Acrobat 또는 PDF 리더와 같은 다른 소프트웨어 없이 PDF 문서의 내용을 가져오거나 읽을 수 있도록 합니다.

### 모듈 기능

이 모듈은 [Aspose.PDF](https://products.aspose.com/pdf/net/)에서 제공하는 강력한 가져오기 기능을 보여줍니다. 모듈이 추가된 페이지에 간단한 파일 브라우저 컨트롤, 콘텐츠 창 선택 및 **PDF에서 가져오기** 버튼을 추가합니다. 버튼을 클릭하면 문서 내용이 즉시 화면에 표시됩니다.

## 시스템 요구 사항 및 지원 플랫폼

### 시스템 요구 사항

Aspose .NET Pdf Import for DNN 모듈을 설정하려면 다음 요구 사항을 충족해야 합니다:

- DNN 7.0 이상.

다른 버전의 DNN에 이 모듈을 설치하고 싶으시면 언제든지 문의해 주세요.

### 지원 플랫폼

Aspose .NET Pdf Import for DNN 모듈은 현재 다음을 지원합니다:

- DNN 7.0 이상.

다른 버전의 DNN에 이 모듈을 설치하고 싶으시면 언제든지 문의해 주세요.

## 설치

다운로드가 완료되면 DNN 웹사이트에 모듈을 설치하기 위해 다음 단계를 따르세요:

1. **Host** 또는 다른 슈퍼 사용자 계정으로 사이트에 로그인합니다.
1. 호스트 메뉴로 이동하여 **Extensions**를 선택합니다.
1. **Install Extension Wizard**를 클릭합니다.
1. 다운로드한 zip 파일의 위치로 이동하여 선택한 후 **Open**을 클릭합니다.
1. **Next**를 클릭하고 라이센스에 동의한 후 설치를 계속합니다. 완료되면 **Return** 버튼을 클릭합니다.

자세한 내용은 DNN의 [이 모듈 설치 비디오](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)를 확인하세요.

**참고:** 모듈을 업로드하는 동안 오류가 발생하면 이는 DNN 설치의 web.config에서 maxRequestLength 제한 때문입니다. web.config를 열고 maxRequestLength를 20MB로 업데이트하여 **maxRequestLength=”20480″**로 설정한 후 모듈을 다시 업로드해 보세요.

## 사용하기

Aspose .NET PDF Import for DNN 모듈을 설치한 후 웹사이트에서 사용하기가 정말 간단합니다. 시작하기 위해 다음 간단한 단계를 따르세요:

1. DNN에 Host 또는 Admin 수준 계정으로 로그인했는지 확인합니다.
1. 가져오기 모듈을 추가할 페이지로 이동합니다.
1. 상단 리본에서 **Modules** -> **Add New Module**을 선택합니다.

1. 목록에서 **Aspose .NET PDF Import for DNN**을 선택하고 페이지의 원하는 위치로 드래그합니다.

Aspose .NET PDF Import for DNN 모듈을 페이지에 성공적으로 추가했습니다. 이제 파일 브라우저와 **PDF에서 가져오기**라는 제목의 버튼이 콘텐츠 창 선택 드롭다운과 함께 페이지에 나타납니다. 누구나 PDF 파일을 선택하고 **PDF에서 가져오기** 버튼을 클릭하여 선택한 문서의 내용을 페이지에 나열할 수 있습니다.

소스 코드를 열고 확장하려면 다음이 설치되어 있어야 합니다:

- Visual Studio 2010 이상.
- [DNN 개발 템플릿](https://docs.aspose.com/total/net/aspose-dnn-module-development-template/#downloading).

시작하기 위해 다음 간단한 단계를 따르세요:

1. 소스 코드를 다운로드/클론합니다.
1. Visual Studio 2010을 열고 **File** > **Open Project**를 선택합니다.
1. 다운로드한 최신 소스 코드로 이동하여 **AsposeDNNPdfImport.csproj**를 엽니다.
