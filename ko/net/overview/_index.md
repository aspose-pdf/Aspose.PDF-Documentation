## Aspose.PDF for .NET 라이브러리
_이 페이지는 Aspose.PDF for .NET 기능에 대한 개요입니다._

**Aspose for .NET**은 .NET 애플리케이션에서 외부 종속성 없이 PDF 파일을 생성, 읽기-쓰기, 수정 및 기타 작업을 수행하는 데 사용되는 현대적이고 전문적인 PDF API입니다. .NET 또는 .NET Core용 프로그래밍 언어에서 Aspose for .NET을 사용할 수 있습니다.
이 구성 요소는 관리되는 C#로 작성되었으며, 개발자가 Microsoft .NET 애플리케이션(WinForms, WPF, ASP.NET 및 .NET Compact Framework)에 PDF 생성 및 조작 기능을 추가할 수 있도록 합니다.

이 라이브러리를 사용하면 처음부터 PDF 파일을 생성하거나 Adobe Acrobat을 설치하지 않고도 기존 PDF 문서를 완전히 처리할 수 있는 풍부한 기능을 구현할 수 있습니다. Aspose.PDF for .NET은 **다양한 기능**을 지원합니다.

### 문서 압축

PDF 문서를 압축하면 파일 크기를 줄여 저장, 공유 및 네트워크를 통한 전송이 더 쉬워집니다. 이는 고해상도 이미지나 그래픽이 포함된 대형 PDF 파일을 처리할 때 특히 중요합니다. 파일 크기가 작아지면 PDF 문서를 온라인이나 이메일 첨부 파일로 접근할 때 로딩 시간이 빨라집니다. 압축된 PDF는 더 빠르게 로드되어 사용자 경험과 생산성을 향상시킵니다.
**Aspose.PDF for .NET 라이브러리**는 다음과 같은 다양한 방법으로 PDF 크기를 줄일 수 있습니다:

- 모든 이미지를 축소하거나 압축합니다.
- 사용하지 않는 PDF 객체를 제거합니다.
- 사용하지 않는 PDF 스트림을 제거합니다.
- 중복 PDF 스트림을 연결합니다.
- 글꼴을 언바인딩합니다.
- 주석을 제거하거나 평면화합니다.
- AcroForm 필드를 제거합니다.
- flateDecode 압축을 사용합니다.
- XImageCollection에 이미지를 저장합니다(재사용 가능한 이미지).
- PDF를 RGB 색상 공간에서 그레이스케일로 변환합니다.

자세한 내용은 Docs 페이지를 확인하세요: [PDF 최적화](/pdf/ko/net/optimize-pdf/)

### 테이블 생성 및 조작

테이블은 PDF 문서에서 정보를 제시하기 위한 구조적이고 조직적인 형식을 제공합니다. 사용자는 데이터를 행과 열로 분류하고 배열할 수 있어 복잡한 데이터 세트를 이해하기 쉽게 만듭니다.

**Aspose.PDF for .NET**을 사용하여 PDF 문서에서 테이블을 생성하고 조작하면 데이터 조직 개선, 가독성 향상, 분석 능력 향상 및 정보를 효과적으로 제시하는 유연성이 증가하는 등 많은 이점이 있습니다.

자세한 내용은 다음 섹션을 확인하세요: [테이블 작업하기](/pdf/ko/net/working-with-tables/)

### 그래프 객체 지원

**Aspose.PDF for .NET**은 PDF 문서에 그래프를 추가하는 것을 지원합니다. 이 작업을 위해 Graph 클래스가 제공됩니다.  
PDF 문서에 그래프를 추가하면 데이터 시각화 개선, 명확성과 이해도 향상, 더 나은 의사 결정 지원 및 독자와의 참여 증가가 이루어집니다.

자세한 내용은 Docs 페이지를 확인하세요: [그래프 작업하기](/pdf/ko/net/working-with-graphs/)

### 광범위한 하이퍼링크 기능

하이퍼링크는 사용자가 PDF 문서 내에서 빠르고 효율적으로 탐색할 수 있도록 합니다. 하이퍼링크를 클릭하면 독자는 문서를 수동으로 스크롤하지 않고도 다른 섹션, 장 또는 페이지로 이동할 수 있습니다.

**Aspose.PDF for .NET**은 탐색 개선, 교차 참조, 외부 리소스 접근, 상호작용 및 커뮤니케이션 향상 등 많은 이점을 제공합니다. 이는 사용자에게 더 매력적이고 유익하며 접근 가능한 읽기 경험에 기여합니다.

자세한 내용은 다음 섹션을 확인하세요: [작업](/pdf/ko/net/actions/)

### 확장된 보안 제어

디지털 서명과 같은 보안 기능은 PDF 문서의 무결성을 보장하는 데 도움이 됩니다. 문서에 디지털 서명을 추가하면 발신자는 서명 이후 내용이 변조되거나 변경되지 않았음을 확인할 수 있어 수신자에게 문서의 진위에 대한 확신을 제공합니다.

**Aspose.PDF for .NET**을 사용하면 스마트 카드에서 PDF에 서명하고, 파일을 암호화 및 복호화하며, 이미지 및 서명 정보를 추출하는 등의 작업이 가능합니다.

자세한 내용은 Docs 페이지를 확인하세요: [보안 및 서명](/pdf/ko/net/securing-and-signing/)

### 북마크 추가 또는 제거

북마크는 사용자가 크거나 복잡한 PDF 문서를 탐색하는 데 편리한 방법을 제공합니다. 특정 섹션, 장 또는 제목에 대한 북마크를 생성하면 사용자는 클릭 한 번으로 관련 콘텐츠로 쉽게 이동할 수 있어 시간과 노력을 절약할 수 있습니다.

**Aspose.PDF for .NET** 라이브러리를 사용하면 PDF 파일에 북마크를 추가하고, 자식 북마크를 추가하거나 문서에서 모든 또는 특정 북마크를 삭제할 수 있습니다.

자세한 내용은 다음 섹션을 확인하세요: [북마크](/pdf/ko/net/add-and-delete-bookmark/)

### 목차 생성

TOC는 사용자가 PDF 문서의 내용을 빠르고 효율적으로 탐색할 수 있는 방법을 제공합니다. 주요 섹션, 장 또는 주제를 나열하고 해당 페이지 번호를 함께 제공함으로써 사용자는 문서 내에서 특정 정보를 쉽게 찾고 접근할 수 있습니다.
**Aspose.PDF for .NET** 라이브러리는 기존 PDF 파일에 TOC를 추가하는 방법을 설명하며, Aspose.Pdf 네임스페이스의 Heading 클래스를 사용합니다.

자세한 내용은 Docs 페이지를 확인하세요: [TOC 작업하기](/pdf/ko/net/manipulate-pdf-document/)

### 첨부 파일 및 주석 추가, 업데이트, 삭제

첨부 파일 및 주석을 추가하면 PDF 문서가 독자에게 더 상호작용적이고 매력적으로 만들어집니다. 사용자는 문서 내에 포함된 추가 콘텐츠(예: 멀티미디어 파일, 링크 또는 댓글)와 상호작용하여 읽기 경험을 향상시킬 수 있습니다.

Aspose.PDF for .NET 라이브러리는 [주석](/pdf/ko/net/annotations/)에 대한 다양한 조작을 제공합니다. 다음과 같은 유형을 추가할 수 있습니다:

- 텍스트 주석.
- 하이라이트 주석.
- 도형 주석.
- 멀티미디어 주석.
- 스티키 주석.
- 추가 주석.

또한, 우리 라이브러리는 PDF에서 첨부 파일을 추가, 추출 및 제거할 수 있습니다. 
자세한 내용은 다음 [문서 페이지](/pdf/ko/net/attachments/)를 확인하세요.

### PDF 파일 메타데이터

메타데이터는 PDF 문서에 대한 필수 정보를 제공합니다. 제목, 저자, 생성 날짜 및 주제와 같은 정보는 사용자가 문서를 빠르게 식별하고 분류하는 데 도움이 되어 대량의 PDF 파일을 관리하고 조직하는 데 용이합니다.

PDF 파일 메타데이터는 문서 식별, 저작권 표시, 버전 추적, 설명, 검색 가능성, 접근성, 보안, 상호 운용성, 보존 및 준수 등 여러 목적을 수행합니다.

**Aspose.PDF for .NET** 라이브러리를 사용하여 PDF 문서에서 [메타데이터](/pdf/ko/net/pdf-file-metadata/)를 쉽게 관리할 수 있습니다.

### 텍스트 및 이미지 추가, 교체 또는 제거

텍스트 및 이미지를 추가, 교체 또는 제거하면 사용자가 PDF 문서의 내용을 편집할 수 있어 필요에 따라 업데이트, 수정 또는 개선할 수 있습니다. 이러한 유연성은 사용자가 문서의 내용을 변경 사항이나 수정 사항을 정확하게 반영하도록 수정할 수 있게 합니다.

Aspose.PDF를 사용하면 PDF 내에서 텍스트를 편집하고, 교체하거나 회전시키고, PDF에서 텍스트를 검색하거나 가져올 수 있습니다.

또한, 이미지를 추가하고 위치를 변경하며 삭제하고 PDF에서 [복잡한 작업](/pdf/ko/net/working-with-images/)을 수행할 수 있습니다.

### 페이지 분할, 연결, 추출 또는 삽입

Docs의 [PDF 페이지 작업하기](/pdf/ko/net/working-with-pages/) 섹션에서는 PDF에서 페이지를 추가, 이동, 삭제, 회전 또는 분할하는 방법을 설명합니다.

페이지를 분할, 연결, 추출 또는 삽입하면 사용자가 PDF 문서를 보다 효과적으로 관리할 수 있습니다. 사용자는 페이지를 조직하고 재배열하거나 결합하여 새로운 문서를 생성하여 문서 조직 및 접근성을 개선할 수 있습니다.

### 페이지를 이미지로 변환

PDF 페이지를 이미지로 변환하면 PDF 문서를 직접 보기 어렵거나 상호작용하기 어려운 사용자에게 콘텐츠를 더 쉽게 접근할 수 있게 합니다. 이미지는 스마트폰, 태블릿 및 컴퓨터를 포함한 다양한 장치에서 쉽게 볼 수 있으며, 전문 소프트웨어가 필요하지 않습니다.

Aspose.PDF는 C#을 사용하여 PDF를 다양한 이미지 형식으로 변환할 수 있습니다. 다음과 같은 형식이 있습니다:

- TIFF.
- BMP.
- EMF.
- JPG.
- PNG.
- GIF.
- SVG.

Aspose.PDF 라이브러리는 PDF를 이미지로 변환하기 위해 DocumentDevice 또는 ImageDevice 클래스를 사용합니다. [PDF를 다양한 이미지 형식으로 변환하기](/pdf/ko/net/convert-pdf-to-images-format/)를 확인하세요.

### PDF 문서 인쇄

PDF 문서를 인쇄하면 사용자가 전자 장치 없이 쉽게 운반하고 접근할 수 있는 물리적 사본을 만들 수 있습니다.

Aspose.PDF for .NET은 PDF 파일을 인쇄하는 다양한 방법을 제공합니다:

- .NET Framework 4.x를 사용하여 PDF 인쇄.
- .NET Core / .NET에서 PDF 인쇄.
- Facades(콘솔 앱)를 사용하여 XPS 프린터에 PDF 인쇄.
- WPF 애플리케이션에서 PDF 인쇄.
- PDF를 PostScript로 변환, 인쇄 작업 상태 확인 등 다양한 추가 작업.

자세한 내용은 다음 섹션을 확인하세요: [PDF 문서 인쇄하기](/pdf/ko/net/printing-document/).

또한, Aspose.PDF for .NET은 텍스트, 이미지, SVG, HTML을 PDF로 쉽게 변환하고 PDF를 다양한 문서 형식으로 변환하여 우수한 성능과 품질을 제공합니다.
