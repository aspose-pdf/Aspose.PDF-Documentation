**고급 작업**은 Aspose.PDF에서 논의된 [기본 작업](/pdf/ko/net/basic-operations/)을 통해 생성된 문서나 Adobe Acrobat, Google Docs, Microsoft Office, Open Office 또는 기타 PDF 생산자가 만든 PDF 파일을 프로그래밍 방식으로 다루는 방법에 대한 섹션입니다.

다양한 방법을 배울 수 있습니다:

- [문서 작업](/pdf/ko/net/working-with-documents/) - 문서 압축, 분할, 병합 및 전체 문서와 관련된 기타 작업 수행
- [페이지 작업](/pdf/ko/net/working-with-pages/) - 페이지 추가, 이동 또는 제거, 페이지 자르기, 워터마크, 도장 추가
- [텍스트 작업](/pdf/ko/net/working-with-text/) - PDF에서 텍스트 추가, 형식 지정, 검색 및 교체
- [이미지 작업](/pdf/ko/net/working-with-images/) - 문서에서 이미지 삽입, 제거, 추출
- [표 작업](/pdf/ko/net/working-with-tables/) - PDF에 표 삽입, 꾸미기, 표 데이터 추출
- [양식 작업](/pdf/ko/net/working-with-forms/) - 상호 작용 PDF 문서 다루기, 양식 필드 추가, 데이터 추출
- [폼 작업하기](/pdf/ko/net/working-with-forms/) - PDF 문서와 상호 작용하고, 폼 필드 추가, 데이터 추출
- [그래프 작업하기](/pdf/ko/net/graphs/) - 페이지에서 도형 조작
- [XML 작업하기](/pdf/ko/net/working-with-xml) - XML 구조를 기반으로 PDF 문서 구성
- [PDF 문서 비교하기](/pdf/ko/net/compare-pdf-documents/) - PDF 문서의 내용을 비교할 수 있음
- [탐색 및 상호 작용](/pdf/ko/net/navigation-and-interaction/) - 작업, 북마크, 페이지 탐색
- [주석](/pdf/ko/net/annotations/) - 사용자가 PDF 페이지에 사용자 정의 내용을 추가할 수 있습니다. PDF 문서에서 주석을 추가, 삭제 및 수정할 수 있습니다.
- [아티팩트](/pdf/ko/net/artifacts/) - PDF에서 워터마크 및 기타 특수 객체 처리
- [접근성. 태그 지정된 PDF](/pdf/ko/net/accessibility-tagged-pdf/) - 태그 지정은 PDF 접근성에 필수적입니다. Aspose.PDF는 PDF에 태그를 추가하고 논리적인 읽기 순서를 설정하고 구조와 유형을 나타내는 수단을 제공할 수 있습니다.
- [첨부 파일](/pdf/ko/net/attachments/) - PDF 문서에는 파일 첨부가 포함될 수 있습니다.

- [첨부 파일](/pdf/ko/net/attachments/) - PDF 문서는 파일 첨부를 포함할 수 있습니다.
- [PDFs의 메타데이터](/pdf/ko/net/pdf-file-metadata/) - 문서에서 메타 데이터를 가져오거나 설정, XMP 데이터 처리.
- [보안 및 서명](/pdf/ko/net/securing-and-signing/) - 프로그래밍 방식으로 PDF 문서를 보호하고 서명합니다.
- [문서 인쇄](/pdf/ko/net/printing-document/) - 다양한 유형의 애플리케이션(WinForms, WPF 등)에서 PDF를 인쇄합니다.
- [연산자](/pdf/ko/net/operators/) - PDF에서 저수준 작업을 수행합니다.
