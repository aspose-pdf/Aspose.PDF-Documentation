이 주제는 기존 PDF 파일에서 페이지 방향을 프로그래밍 방식으로 업데이트하거나 변경하는 방법에 대해 설명합니다.

다음 코드 스니펫은 [Aspose.PDF.Drawing](/pdf/net/drawing/) 라이브러리에서도 작동합니다.

## 페이지 방향 변경

Aspose.PDF for .NET 9.6.0 릴리스에서, 우리는 풍경에서 초상화로, 그 반대로 페이지 방향을 변경하는 것과 같은 훌륭한 새 기능을 추가했습니다. 페이지 방향을 변경하려면 다음 코드 스니펫을 사용하여 페이지의 MediaBox를 설정하세요. Rotate() 메소드를 사용하여 회전 각도를 설정함으로써 페이지 방향을 변경할 수도 있습니다.
