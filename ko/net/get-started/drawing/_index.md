## 소개

**Aspose.PDF.Drawing for .NET**은 System.Drawing 대신 Aspose.Drawing 라이브러리를 제공하는 Aspose.PDF의 변형입니다.

이 구성 요소는 관리되는 C#으로 작성되어 개발자가 Microsoft 애플리케이션에 PDF 파일 생성 및 관리 기능을 추가할 수 있도록 합니다. NET (WinForms, ASP.NET 및 .NET Compact Framework).

이는 비 Windows 플랫폼에서 System.Drawing.Common을 사용할 때 발생하는 수많은 문제와 버그를 피하는 데 도움이 됩니다.

또한, Aspose.PDF.Drawing 라이브러리는 애플리케이션을 컨테이너에 패키징할 수 있는 Docker 플랫폼에서 성공적으로 작동한다고 말해야 합니다. 자세한 내용은 [Docker에서 Aspose.PDF 실행 방법](https://docs.aspose.com/pdf/net/docker/) 페이지를 확인하십시오. 'ttf-mscorefonts-installer' 및 'fontconfig' 패키지가 Docker에서 Aspose.PDF.Drawing for .NET 라이브러리가 성공적으로 작동하는 데 필요합니다.

**Aspose.PDF.Drawing for .NET**은 Aspose.PDF와 동일한 기능을 제공합니다. PDF 문서와 상호 작용할 수 있으며, 페이지, 텍스트, 이미지, 첨부 파일, 글꼴, 북마크, 주석, 양식, 연산자, 스탬프, 워터마크, 링크, 보안 기능, 서명 및 인쇄와 같은 요소를 포함합니다.

**Aspose.PDF.Drawing for .NET**은 PDF 압축, 테이블 생성 및 조작, 그래프 객체 지원, 포괄적인 하이퍼링크 기능, 고급 보안 제어 구현, 사용자 정의 글꼴 관리, 데이터 소스와의 원활한 통합, 북마크 추가 또는 제거와 같은 다양한 기능을 자랑합니다. 사용자는 목차를 생성하고, 첨부 파일 및 주석을 추가, 업데이트 또는 삭제하며, PDF 양식 데이터를 가져오거나 내보내고, 텍스트 및 이미지를 추가, 교체 또는 제거할 수 있습니다. 또한 소프트웨어는 페이지 분할, 연결, 추출 또는 삽입, 페이지를 이미지로 변환, PDF 문서 인쇄 등과 같은 작업을 허용합니다.

## 장점

Aspose.Drawing은 System.Drawing.Common 라이브러리에 비해 여러 가지 장점을 제공합니다:

1. 크로스 플랫폼 지원. Aspose.PDF.Drawing은 Windows, Linux, macOS, Android 및 iOS를 포함한 다양한 플랫폼에서 드로잉 애플리케이션 개발을 가능하게 합니다.

1. Aspose.PDF.Drawing for .NET 라이브러리의 성능은 2%를 초과하지 않습니다.

1. 렌더링 품질. Aspose.PDF.Drawing을 사용하면 안티 앨리어싱 및 힌팅이 적용된 텍스트의 최고 품질 렌더링을 달성할 수 있습니다. 이는 고르지 않은 아티팩트 없이 부드러운 글꼴 가장자리를 보장하여 다양한 디스플레이와 해상도에서 텍스트를 읽을 수 있게 합니다. A4 형식의 문서에서의 차이는 페이지당 최대 10픽셀입니다.

1. 포괄적인 기능 세트. Aspose.PDF.Drawing은 도형, 선, 텍스트, 이미지 및 차트를 포함한 2D 그래픽을 위한 광범위한 기능을 자랑합니다.

1. API 호환성. System.Drawing 네임스페이스를 활용하여 Aspose.PDF.Drawing은 System.Drawing.Common을 원활하게 대체하여 기존 소스 코드를 변경할 필요 없이 대체할 수 있습니다.

## 기능

1. 벡터 그래픽 지원. Aspose.PDF.Drawing은 SVG 벡터 이미지 형식과 호환되어 개발자가 어떤 해상도에서도 품질을 유지하는 확장 가능한 그래픽을 제작할 수 있도록 합니다.

1. 텍스트 렌더링 기능. Aspose.PDF.Drawing은 고품질 텍스트 렌더링에 뛰어납니다. 이는 부드러운 글꼴 가장자리를 보장하여 고르지 않은 아티팩트를 제거하고 다양한 디스플레이와 해상도에서 가독성을 보장합니다.

1. 도형 서식 지원. Aspose.PDF.Drawing은 채우기 색상, 선 색상, 선 두께 및 선 스타일과 같은 도형 서식을 위한 다양한 옵션을 제공합니다.

1. 이미지 처리 기능. Aspose.PDF.Drawing for .NET은 추가, 교체 또는 제거를 포함한 다양한 이미지 처리 기능을 제공합니다.

Aspose.PDF.Drawing for .NET은 다음과 같은 다양한 기능을 지원합니다:

- 문서 압축.
- 테이블 생성 및 조작.
- 그래프 객체 지원.
- 광범위한 하이퍼링크 기능.
- 확장된 보안 제어.
- 사용자 정의 글꼴 처리.
- 데이터 소스와의 통합.
- 북마크 추가 또는 제거.
- 목차 생성.
- 첨부 파일 및 주석 추가, 업데이트, 삭제.
- PDF 양식 데이터 가져오기 또는 내보내기.
- 텍스트 및 이미지 추가, 교체 또는 제거.
- 페이지 분할, 연결, 추출 또는 삽입.
- 페이지를 이미지로 변환.
- PDF 문서 인쇄 등.

PDF 문서를 인쇄할 때는 System.Drawing.Common이 사용되므로 Windows 플랫폼에서만 작동합니다.

Aspose.PDF.Drawing에 대한 추가 라이센스는 없으며, Aspose.PDF와 동일하게 사용합니다. [Aspose PDF 라이센스](/pdf/net/licensing/) 섹션을 확인하십시오.

## 설치

**Aspose.PDF.Drawing for .NET**을 사용해 보려면 Visual Studio의 패키지 관리자 콘솔에서 `Install-Package Aspose.PDF.Drawing`을 실행하여 NuGet 패키지를 가져오십시오. 이미 Aspose.PDF for .NET이 있고 버전을 업그레이드하려면 `Update-Package Aspose.PDF.Drawing`을 실행하여 최신 버전을 가져오십시오. 자세한 정보는 [NuGet 패키지](https://www.nuget.org/packages/Aspose.PDF.Drawing/) 링크에서 확인할 수 있습니다.
