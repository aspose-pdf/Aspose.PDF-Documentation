## 개요
API는 .NET Standard 2.0 및 이후 버전 또는 .NET Framework 4.0을 지원하는 모든 운영 체제에서 애플리케이션을 구축하는 데 사용할 수 있습니다.

PDF 문서를 생성하고 조작하기 위해 Aspose.PDF for .NET가 실행되는 머신에 Adobe Acrobat이 설치되어 있을 필요는 없습니다: Aspose.PDF for .NET는 PDF 문서 생성 엔진입니다. 그러나 Aspose.PDF for .NET가 생성한 PDF 문서를 보려면 머신에 최소한 Adobe Acrobat Reader가 필요합니다. 이는 무료로 다운로드할 수 있습니다.

## 개발 환경
Aspose.PDF for .NET를 사용하여 .NET 플랫폼을 대상으로 하는 모든 개발 환경에서 애플리케이션을 개발할 수 있지만, 다음 환경은 명시적으로 지원됩니다:

- Microsoft Visual Studio 2010[^1].
- Microsoft Visual Studio 2011[^1].
- Microsoft Visual Studio 2012[^1].
- Microsoft Visual Studio 2013[^1].
- Microsoft Visual Studio 2015[^1].
- Microsoft Visual Studio 2017[^1].
- Microsoft Visual Studio 2019.
- Microsoft Visual Studio 2022.
- Microsoft Visual Studio Code.
