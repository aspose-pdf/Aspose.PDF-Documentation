PDF는 응용 프로그램 소프트웨어, 하드웨어 및 운영 체제에 독립적인 형식으로 문서 및 기타 참조 자료를 제공하기 위한 표준을 제공하기 위해 개발되었습니다. PDF 파일의 내용은 텍스트에 국한되지 않으며, 하이퍼링크, 이미지, 클릭 가능한 버튼과 양식, 전자 서명, 워터마크 등도 포함될 수 있습니다. 따라서 PDF 파일의 내용을 편집하거나 변경하기 위해 다른 형식으로 변환해야 하는 경우가 종종 있습니다. **Aspose.PDF for JavaScript via C++** 도구를 사용하면 PDF 문서를 가장 인기 있는 형식으로 성공적이고, 빠르고, 쉽게 변환할 수 있습니다.
 Aspose.PDF 지원 파일 형식의 전체 목록은 [Aspose.PDF 지원 파일 형식](https://docs.aspose.com/pdf/javascript-cpp/supported-file-formats/) 섹션을 참조하세요.

Aspose.PDF for JavaScript via C++*는 PDF 문서를 다양한 형식으로 변환할 수 있습니다. 또한 Aspose.PDF 변환의 품질을 확인하고 Aspose.PDF 변환기 앱을 통해 온라인으로 결과를 볼 수 있습니다. 코드 조각과 함께 문서를 변환하는 방법을 알아보세요.

## 변환을 위해 C++를 통한 JavaScript 사용 방법

Word 문서는 가장 다재다능하고 편집 가능한 형식입니다. PDF를 DOC로 수동 변환하는 것은 매우 시간이 많이 걸리는 작업입니다. 이 기사에서는 Javascript에서 PDF를 DOC 및 DOCX로 변환하는 방법을 배우게 됩니다.

- [PDF를 DOC로 변환](/pdf/ko/javascript-cpp/convert-pdf-to-doc/) - Javascript를 사용하여 PDF 문서를 Word 형식으로 변환할 수 있습니다.

숫자 형식은 테이블의 데이터를 더 쉽게 읽을 수 있도록 할 뿐만 아니라 테이블을 더 쉽게 사용할 수 있도록 하는 데 필요합니다. 물론, PDF 문서를 Excel 형식으로 변환해야 하는 경우, Aspose.PDF for Javascript를 사용하세요.

- [PDF를 Microsoft XLSX로 변환](/pdf/ko/javascript-cpp/convert-pdf-to-xlsx/) - 이 섹션에서는 PDF 문서를 XLSX 및 CSV로 변환하는 방법을 설명합니다.

PowerPoint 형식은 다양한 프레젠테이션을 만들 때 사용됩니다. PPT 파일에는 다양한 정보를 포함한 많은 슬라이드나 페이지가 포함되어 있습니다.

- [PDF를 Microsoft PowerPoint로 변환](/pdf/ko/javascript-cpp/convert-pdf-to-powerpoint/) - 여기에서는 변환 프로세스를 추적하여 PDF를 PowerPoint로 변환하는 방법에 대해 설명합니다.

하이퍼텍스트 마크업 언어는 하이퍼텍스트 문서 설명 언어이며, 웹 페이지를 만드는 표준 언어입니다. Aspose.PDF for Javascript를 사용하면 HTML 문서를 쉽게 변환할 수 있습니다.

다양한 목적을 위해 PDF로 변환해야 하는 이미지 형식이 많이 있습니다. Aspose.PDF for JavaScript는 가장 인기 있는 이미지 형식을 지원합니다.

- [PDF를 다양한 이미지 형식으로 변환](/pdf/ko/javascript-cpp/convert-pdf-to-images-format/) - JPEG, PNG, SVG 및 기타 형식으로 PDF 페이지를 이미지로 변환합니다.
이 섹션에는 EPUB, XPS, TeX, 텍스트 및 그레이스케일 PDF와 같은 형식이 포함됩니다.

- [PDF 파일을 다른 형식으로 변환하기](/pdf/ko/javascript-cpp/convert-pdf-to-other-files/) - 이 주제는 PDF 문서를 다양한 형식으로 변환하는 방법을 설명합니다.

PDF/A는 전자 문서의 장기 보관을 위해 설계된 PDF의 버전입니다.
솔직히 외관상으로는 PDF인지 PDF/A인지 구별하기가 매우 어렵습니다. 이 파일을 확인하려면 유효성 검사기를 사용합니다. PDF를 PDF/A로 및 그 반대로 고품질 변환하는 방법에 대한 다음 기사를 확인하세요.

- [PDF를 PDF/A 형식으로 변환하기](/pdf/ko/javascript-cpp/convert-pdf-to-pdfa/) - Aspose.PDF의 자바스크립트 툴킷은 PDF를 PDF/A로 변환하는 쉬운 방법을 제공합니다.

- [PDF/A를 PDF 형식으로 변환하기](/pdf/ko/javascript-cpp/convert-pdfa-to-pdf/) - 자바스크립트를 사용하여 PDF/A를 PDF 형식으로 쉽고 빠르게 고품질로 변환합니다.
