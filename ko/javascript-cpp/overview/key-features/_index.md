## 일반 기능

- **Aspose.PDF for JavaScript via C++**는 WebAssembly 기술을 사용하여 구축되었으며 Aspose.PDF for .NET을 기반으로 합니다.
- PDF 파일 정보 설정, 가져오기 및 제거
- 책갈피 삭제
- 주석 삭제
- JavaScripts 삭제
- PDF에서 텍스트 추출
- PDF 문서 최적화
- 리소스 최적화
- PDF를 두 개의 파일로 분할
- 두 개의 파일을 PDF로 병합
- PDF 복구

## 지원되는 PDF 버전

Aspose.PDF for JavaScript는 PDF 버전 1.2, 1.3, 1.4, 1.5, 1.6 및 1.7을 지원합니다.

## 글꼴

- 14개의 기본 글꼴.
- PostScript Type1 글꼴.
- TrueType 글꼴.
- Type 3 글꼴.

- PDF로 글꼴 정보 임베딩
- CJK 글꼴.
- 유니코드 지원

## 텍스트

- PDF에서 텍스트 추출

## 하이퍼링크 및 첨부 파일 기능

툴킷을 사용하여 다음을 추가할 수 있습니다:

- PDF에서 첨부 파일 제거
- PDF 문서 내 다른 페이지로의 하이퍼링크 단축키
- 다른 PDF 문서의 페이지로 하이퍼링크

- 비-PDF 파일로의 하이퍼링크
- 웹 위치에 대한 하이퍼링크.

하이퍼링크 생성과 함께 PDF 문서에서 하이퍼링크를 추출할 수 있는 기능도 얻을 수 있습니다.

## PDF 보안 기능

API는 40비트 또는 128비트 암호화를 제공합니다. 더 높은 보안을 위해 사용자 및 관리자 암호를 설정할 수도 있습니다. 이 라이브러리를 사용하여 내용 복사, 인쇄, 저하된 인쇄, 주석 수정, 양식 필드 완성 및 PDF 파일 조립과 같은 파일의 내용 조작을 모두 제한할 수 있습니다.

- PDF 파일 암호화
- PDF 파일 암호 해독
- 비밀번호 변경
- PDF 서명
