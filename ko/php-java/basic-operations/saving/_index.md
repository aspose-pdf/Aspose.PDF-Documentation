## 파일 시스템에 PDF 문서 저장

[Document](https://reference.aspose.com/pdf/java/com.aspose.pdf/Document) 클래스의 save 메서드를 사용하여 생성되거나 조작된 PDF 문서를 파일 시스템에 저장할 수 있습니다.

```php

    $document = new Document($inputFile);        
    $document->save($outputFile);
```
