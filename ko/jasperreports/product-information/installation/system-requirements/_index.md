{{% alert color="primary" %}}

Aspose.PDF for JasperReports는 다음 소프트웨어가 필요합니다:

- Java 2 Platform Standard Edition 5.0 이상.

- JasperReports 6.7.0 이상.

- JasperServer 6.7.0 이상 (JasperServer와 함께 Aspose.PDF for JasperReports를 사용하려는 경우).

{{% /alert %}}
