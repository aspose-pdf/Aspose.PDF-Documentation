## 일반 기능

- 대부분의 확립된 PDF 표준 및 PDF 사양을 지원합니다.
- BMP, GIF, JPEG 및 PNG를 포함한 다양한 이미지 형식으로 PDF를 읽고 내보낼 수 있습니다.
- PDF 문서의 기본 정보(예: 저자, 제작자)를 설정합니다.
- PDF 페이지 속성(예: 너비, 높이, cropbox, bleedbox 등)을 구성합니다.
- 페이지 번호 매기기, 북마크 레벨, 페이지 크기 등을 설정합니다.
- 텍스트, 단락, 머리글, 하이퍼링크, 그래프, 첨부 파일 등을 다룰 수 있습니다.

## 지원되는 PDF 표준

- ISO 32000-1 (PDF 1.2, PDF 1.3, PDF 1.4, PDF 1.5, PDF 1.6, PDF 1.7).
- ISO 32000-2 (PDF 2.0).
- ISO 19005 (PDF/A), ISO 15930 (PDF/X), ISO 24517 (PDF/E), ISO 14289 (PDF/UA).

## 변환 기능

Aspose.PDF for Python via .NET 라이브러리를 사용하면 PDF 문서를 가장 널리 사용되는 형식으로 그리고 그 반대로도 성공적이고 빠르고 쉽게 변환할 수 있습니다.

- PDF를 Word, Excel 및 PowerPoint로 변환합니다.
- PDF를 이미지 형식으로 변환합니다.
- PDF 파일을 HTML 형식으로 변환하고 그 반대도 가능합니다.
- PDF를 EPUB, 텍스트, XPS 등으로 변환합니다.
- EPUB, Markdown, 텍스트, XPS, PostScript, XML, LaTeX를 PDF로 변환합니다.
