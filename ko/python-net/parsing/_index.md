**PDF 파싱** 문서는 PDF 파일에서 다양한 종류의 정보를 추출하는 것과 관련된 용어입니다. 이 섹션에서는 다음과 같은 방법을 다룹니다:

- [PDF에서 텍스트 추출](/pdf/python-net/extract-text-from-pdf/). 텍스트 파싱 또는 추출은 준비된 PDF에서 가장 일반적인 작업입니다. 전체 문서, 특정 페이지, 또는 페이지 내 특정 영역에서 텍스트 파싱에 대해 배우게 됩니다.
- [PDF에서 이미지 추출](/pdf/python-net/extract-images-from-the-pdf-file/). 이미지 추출은 위의 텍스트 작업과 동일하게 이미지에 대해 수행됩니다.
- [PDF에서 글꼴 추출](/pdf/python-net/extract-fonts-from-pdf/). 글꼴 추출은 PDF 내 글꼴에 대한 특정 작업입니다.
- [폼에서 데이터 추출](/pdf/python-net/extract-data-from-acroform/). 폼이 포함된 PDF 문서가 여러 개 있다면, 해당 폼의 데이터를 가져와야 할 가능성이 높습니다. 이 문서는 Aspose.PDF for Python via .NET를 사용하여 AcroForms 데이터를 추출하는 방법을 이해하는 데 도움이 됩니다.
- [표에서 데이터 추출](/pdf/python-net/extract-data-from-table-in-pdf/). PDF 문서의 표에서 데이터를 가져옵니다.
- [PDF에서 벡터 데이터 추출](/pdf/python-net/extract-vector-data-from-pdf/). 위치, 색상, 선 두께 등과 같은 벡터 데이터(경로, 다각형, 폴리라인)를 얻을 수 있습니다.
