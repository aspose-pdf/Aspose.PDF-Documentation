**.NET을 통한 파이썬용 Aspose.pdf**는 PDF 파일의 테이블 작업을 위한 완벽한 도구를 제공합니다.테이블을 생성 및 추가하고, 테이블 레이아웃과 페이지 흐름을 제어하고, 표 형식 데이터를 추출하고, 기존 테이블을 업데이트하고, 기존 PDF에서 하나 이상의 테이블을 제거할 수 있습니다.

표 기반 보고서를 작성하거나, PDF에서 구조화된 데이터를 추출하거나, Python으로 기존 문서의 표 형식 콘텐츠를 업데이트해야 할 때 이 섹션을 사용하세요.

## 테이블 작업 다루기

다음과 같은 작업을 수행할 수 있습니다.

- [테이블 추가](/pdf/ko/python-net/adding-tables/) - 병합된 행/열, 테두리, 여백 및 패딩을 포함하여 PDF 파일에 표를 만들 수 있습니다.
- [테이블 추출](/pdf/ko/python-net/extracting-table/) - 기존 PDF 파일에서 테이블 데이터를 추출합니다.
- [테이블을 데이터 소스와 통합](/pdf/ko/python-net/integrate-table/) - 파이썬으로 데이터베이스와 판다 DataFrame 데이터에서 테이블을 빌드합니다.
- [테이블 조작](/pdf/ko/python-net/manipulating-tables/) - PDF 문서 내의 기존 테이블을 검사하고 업데이트합니다.
- [테이블 제거](/pdf/ko/python-net/removing-tables/) - PDF 문서에서 하나 이상의 테이블을 제거합니다.
