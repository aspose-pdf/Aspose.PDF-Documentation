.NET을 통한 파이썬용 Aspose.PDF 는 기존 PDF 문서의 이미지 작업을 위한 완전한 도구 세트를 제공합니다.

이 섹션을 사용하여 새 이미지를 삽입하고, 포함된 이미지를 제거 또는 교체하고, 이미지 콘텐츠를 추출하고, 페이지의 이미지 배치 세부 정보를 검사할 수 있습니다.

## 이미지 작업 다루기

다루는 작업은 다음과 같습니다.

- [기존 PDF 파일에 이미지 추가](/pdf/ko/python-net/add-image-to-existing-pdf-file/) - 고정된 좌표에 이미지를 배치하고 다양한 옵션으로 이미지를 삽입할 수 있습니다.
- [PDF 파일에서 이미지 삭제](/pdf/ko/python-net/delete-images-from-pdf-file/) - 선택한 페이지에서 하나의 이미지 리소스 또는 모든 이미지를 제거합니다.
- [PDF 파일에서 이미지 추출](/pdf/ko/python-net/extract-images-from-pdf-file/) - PDF 페이지에서 포함된 이미지를 추출합니다.
- [기존 PDF 파일의 이미지 바꾸기](/pdf/ko/python-net/replace-image-in-existing-pdf-file/) - 포함된 이미지를 리소스 색인 또는 배치로 대체합니다.
- [PDF 문서에서 이미지 검색 및 가져오기](/pdf/ko/python-net/search-and-get-images-from-pdf-document/) - 문서에서 이미지를 찾고 이미지 배치를 검사합니다.
