PDF 문서의 스탬프는 종이 문서에 고무 스탬프를 적용하는 것과 유사합니다.
PDF 파일의 스탬프는 PDF 파일에 추가 정보를 제공하며, 다른 사람이 사용하지 못하도록 보호하고 파일 내용의 보안을 확인하는 데 도움을 줍니다. **Aspose.PDF for Python via .NET**을 사용하면 PDF 문서에 이미지 또는 텍스트 스탬프를 추가할 수 있습니다.

Python으로 스탬프를 추가하는 방법을 배우려면 다음 섹션을 확인하세요:

- [PDF 페이지에 이미지 스탬프 추가](/pdf/python-net/image-stamps-in-pdf-page/) - 이미지 스탬프를 추가하고, 이미지 품질을 제어하며, 이미지 스탬프를 PDF 파일의 배경으로 사용합니다.
- [PDF 파일에 텍스트 스탬프 추가](/pdf/python-net/text-stamps-in-the-pdf-file/) - 텍스트 스탬프를 추가하고, TextStamp 객체의 정렬을 정의하며, PDF 파일에 스트로크 텍스트를 스탬프 형태로 채웁니다.
- [PDF 파일에 페이지 스탬프 추가](/pdf/python-net/page-stamps-in-the-pdf-file/) - 전체 페이지 또는 섹션에 오버레이하기 위해 페이지 스탬프를 추가하며, 워터마크, 문서 버전 관리, 혹은 PDF 파일에 맞춤 배경을 추가하는 데 유용합니다.
- [PDF 파일에 페이지 번호 스탬프 추가](/pdf/python-net/add-page-number/) - PDF 문서에 페이지 번호를 추가하고, 외관을 커스터마이즈하며, 배치를 제어합니다.
