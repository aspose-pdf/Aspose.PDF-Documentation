**Aspose.PDF for Python via .NET**은 파일의 어느 위치에든 페이지를 삽입하고 PDF 파일의 끝에 페이지를 추가할 수 있게 해줍니다. 이 섹션에서는 Acrobat Reader 없이 PDF에 페이지를 추가하는 방법을 보여줍니다.
Aspose의 Python 라이브러리를 사용하여 PDF 파일의 헤더와 푸터에 텍스트 또는 이미지를 추가하고, 문서에서 다른 헤더를 선택할 수 있습니다.
또한 Python을 사용하여 PDF 문서의 페이지를 프로그래밍 방식으로 자르는 것을 시도해 보세요.

이 섹션에서는 Artifact 클래스를 사용하여 PDF 파일에 워터마크를 추가하는 방법을 배웁니다. 해당 작업에 대한 프로그래밍 예제를 확인하게 됩니다.
PageNumberStamp 클래스([PageNumberStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/pagenumberstamp/))를 사용하여 페이지 번호를 추가합니다. 문서에 스탬프를 추가하려면 [ImageStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/imagestamp/) 및 [TextStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/textstamp/) 클래스를 사용하십시오. **Aspose.PDF for Python via .NET**을 사용하여 PDF 파일에 배경 이미지를 만들기 위해 워터마크를 추가합니다.

다음 작업을 수행할 수 있습니다:

- [Adding Pages](/pdf/python-net/add-pages/) - 원하는 위치에 페이지를 추가하거나 PDF 파일의 끝에 페이지를 추가하고 문서에서 페이지를 삭제합니다.
- [Moving Pages](/pdf/python-net/move-pages/) - 한 문서에서 다른 문서로 페이지를 이동합니다.
- [Deleting Pages](/pdf/python-net/delete-pages/) - PageCollection 컬렉션을 사용하여 PDF 파일에서 페이지를 삭제합니다.
- [Extracting Pages](/pdf/python-net/extract-pages/) - Aspose.PDF for Python을 사용하여 PDF 파일에서 페이지를 추출합니다.
- [Rotating Pages](/pdf/python-net/rotate-pages/) - 기존 PDF 파일의 페이지 방향을 변경할 수 있습니다.
- [Changing Page Size](/pdf/python-net/change-page-size/) - PDF 문서의 페이지 크기를 변경합니다.
- [Adding Headers and Footers](/pdf/python-net/add-headers-and-footers-of-pdf-file/) - PDF 파일의 헤더와 푸터에 텍스트 또는 이미지를 추가합니다.
- [Cropping PDF Pages](/pdf/python-net/crop-pages/) - 다양한 페이지 속성을 사용하여 PDF 문서의 페이지를 프로그래밍 방식으로 자를 수 있습니다.
- [Getting and Setting Page Properties](/pdf/python-net/get-and-set-page-properties/) - 크기, 회전, 색상 속성 등 PDF 페이지의 다양한 속성에 접근하고 수정합니다.
- [Stamping](/pdf/python-net/stamping/) - Python을 사용하여 PDF 페이지에 이미지 스탬프, 페이지 스탬프, 페이지 번호 스탬프 및 텍스트 스탬프를 추가합니다.
