Python에서 페이지 수준 PDF 작업을 수행해야 할 때 이 섹션을 사용하십시오.

**Aspose.pdffor Python.NET**을 사용하면 페이지를 삽입, 삭제, 이동, 추출, 회전, 크기 조정 및 잘라낸 다음 레이아웃 및 검토 워크플로에 머리글, 바닥글, 페이지 번호, 워터마크 및 스탬프를 적용할 수 있습니다.

## 페이지 작업 내용

다루는 작업은 다음과 같습니다.

- [페이지 추가](/pdf/ko/python-net/add-pages/) - 특정 위치에 페이지를 삽입하거나 문서 끝에 페이지를 추가합니다.
- [페이지 이동](/pdf/ko/python-net/move-pages/) - 문서 내에서 또는 문서 간에 페이지를 이동합니다.
- [페이지 삭제](/pdf/ko/python-net/delete-pages/) - PDF 문서에서 페이지를 제거합니다.
- [페이지 추출](/pdf/ko/python-net/extract-pages/) - 선택한 페이지를 새 PDF 파일로 추출합니다.
- [페이지 회전](/pdf/ko/python-net/rotate-pages/) - 기존 PDF 파일의 페이지 방향을 변경합니다.
- [페이지 크기 변경](/pdf/ko/python-net/change-page-size/) - PDF 문서의 페이지 크기를 조정합니다.
- [머리말 및 꼬리말 추가](/pdf/ko/python-net/add-headers-and-footers-of-pdf-file/) - 반복되는 텍스트, 이미지 또는 구조화된 머리말/꼬리말 콘텐츠를 추가합니다.
- [PDF 페이지 자르기](/pdf/ko/python-net/crop-pages/) - 페이지 지오메트리 설정을 사용하여 페이지 콘텐츠를 자릅니다.
- [페이지 속성 가져오기 및 설정](/pdf/ko/python-net/get-and-set-page-properties/) - 크기, 회전 및 색상 속성과 같은 PDF 페이지의 다양한 속성에 액세스하고 수정합니다.
- [스탬핑](/pdf/ko/python-net/stamping/) - 텍스트, 이미지, 페이지 및 페이지 번호 스탬프를 PDF 페이지에 적용합니다.
