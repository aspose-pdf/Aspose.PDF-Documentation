## 아크로폼의 기초

AcroForms는 Adobe에서 도입한 대화형 페이지 중심의 PDF 양식이며 버전 1.2부터 PDF 사양에 포함되었습니다.널리 지원되며 플랫폼 간 워크플로우에서 데이터 캡처, 검증 및 교환에 사용할 수 있습니다.

AcroForm 기반 워크플로우에서는 양식 데이터를 FDF 및 XFDF와 같은 형식으로 교환할 수 있습니다.따라서 AcroForms는 외부 시스템의 필드를 미리 채우거나, 전송된 값을 추출하거나, 문서 간에 양식 데이터를 동기화해야 하는 시나리오에 적합합니다.

.NET을 통한 Python용 Aspose.PDF 는 필드 채우기, 데이터 내보내기, 양식 구조 수정, 제출 작업 추가 등 프로그래밍 방식으로 AcroForms를 만들고 관리할 수 있는 API를 제공합니다.

실제 예는 다음 문서를 참조하십시오.

- [아크로폼 만들기](/pdf/ko/python-net/create-form/) - 처음부터 양식을 작성하십시오.
- [아크로폼 채우기](/pdf/ko/python-net/fill-form/) - PDF 문서의 양식 필드를 채웁니다.
- [아크로폼 추출](/pdf/ko/python-net/extract-form/) - 모든 필드 또는 특정 필드에서 값을 추출합니다.
- [양식 데이터 가져오기 및 내보내기](/pdf/ko/python-net/import-export-form-data/) - 양식 데이터를 일반적인 형식으로 가져오고 내보냅니다.
- [아크로폼 수정](/pdf/ko/python-net/modifying-form/) - AcroForm 필드 속성을 수정합니다.
- [PDF에서 양식 삭제](/pdf/ko/python-net/remove-form/) - PDF 문서에서 양식 필드를 제거합니다.
- [게시 양식](/pdf/ko/python-net/posting-form/) - PDF 양식에 제출 기능을 추가합니다.
