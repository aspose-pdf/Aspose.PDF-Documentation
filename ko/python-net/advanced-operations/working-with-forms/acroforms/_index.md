## AcroForms 기본 사항

**AcroForms** - Adobe에서 제공하는 독특한 PDF 양식 기술. AcroForms는 페이지 지향형 양식입니다. 1998년에 처음 등장했습니다. 데이터 형식 또는 FDF와 XML 양식 데이터 형식인 xFDF 형태로 입력을 받습니다. 서드파티 제공업체가 AcroForms를 지원합니다. Adobe가 AcroForms를 도입했을 때, 이를 "PDF 양식"이라고 명명했으며, 이는 Adobe Acrobat Pro/Standard의 작성자이며 특별한 정적 또는 동적 XFA 양식 유형이 아닙니다. AcroForms는 휴대성이 뛰어나 모든 플랫폼에서 작동합니다.```

AcroForms를 사용하면 PDF 양식 문서에 추가 페이지를 넣을 수 있습니다. 템플릿 개념 덕분에 AcroForms를 사용해 여러 데이터베이스 레코드로 양식을 채울 수 있습니다.

PDF 1.7은 데이터와 PDF 양식을 통합하는 두 가지 다른 방법을 지원합니다.

*AcroForms (또는 Acrobat forms라고도 함)*, PDF 1.2 형식 사양에 도입 및 포함되었습니다.
