우리는 종종 PDF 파일에 텍스트를 추가해야 합니다. 예를 들어, 본문 아래에 번역을 추가하거나 이미지 옆에 캡션을 넣거나, 단순히 신청서를 작성할 때가 있습니다. 모든 텍스트 요소를 원하는 스타일대로 서식 지정할 수 있다면 더욱 유용합니다. PDF 파일에서 가장 흔히 활용되는 텍스트 조작은 PDF에 텍스트 추가, PDF 내부 텍스트 서식 지정, 문서 내 텍스트 교체 및 회전입니다. **Aspose.PDF for Python via .NET**은 PDF 콘텐츠와 상호 작용하는 데 필요한 모든 기능을 제공하는 최적의 솔루션입니다.

다음 작업을 수행할 수 있습니다:

- [PDF 파일에 텍스트 추가](/pdf/python-net/add-text-to-pdf-file/) - PDF에 텍스트를 추가하고, 스트림 및 파일에서 글꼴을 사용하며, HTML 문자열을 추가하고, 하이퍼링크를 추가하는 등.
- [PDF 툴팁](/pdf/python-net/pdf-tooltip/) - Python을 사용해 보이지 않는 버튼을 추가하여 검색된 텍스트에 툴팁을 달 수 있습니다.
- [PDF 내부 텍스트 서식 지정](/pdf/python-net/text-formatting-inside-pdf/) - 텍스트를 서식 지정할 때 문서에 적용할 수 있는 다양한 기능이 있습니다. 줄 들여쓰기, 텍스트 테두리, 밑줄 텍스트 추가, Aspose.PDF 라이브러리를 사용한 새 줄 삽입 등을 할 수 있습니다.
- [PDF 텍스트 교체](/pdf/python-net/replace-text-in-pdf/) - PDF 문서의 모든 페이지에서 텍스트를 교체합니다. 먼저 TextFragmentAbsorber를 사용해야 합니다.
- [PDF 내부 텍스트 회전](/pdf/python-net/rotate-text-inside-pdf/) - TextFragment 클래스의 회전 속성을 사용하여 PDF 내부 텍스트를 회전합니다.
- [PDF 문서 페이지에서 텍스트 검색 및 가져오기](/pdf/python-net/search-and-get-text-from-pdf/) - TextFragmentAbsorber 클래스를 사용하여 페이지에서 텍스트를 검색하고 가져올 수 있습니다.
- [FloatingBox 사용](/pdf/python-net/floating-box/) - 이 페이지에서는 플로팅 박스 내부의 텍스트를 서식 지정하는 방법을 설명합니다.
