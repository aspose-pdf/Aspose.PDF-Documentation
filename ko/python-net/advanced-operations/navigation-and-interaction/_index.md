## 이 섹션의 주제

- [액션](/pdf/ko/python-net/actions/) - 이 기사에서는 다양한 유형의 작업을 수행하는 방법을 배웁니다.
- [책갈피](/pdf/ko/python-net/bookmarks/)- 대형 출판물에는 일반적으로 북마크 창에서 쉽게 보고 선택할 수 있는 북마크 프레임워크가 포함되어 있으므로 북마크를 클릭하여 해당 페이지나 장으로 이동할 수 있습니다.북마크 창은 내용 인식 요소이며 열려 있는 PDF 문서에 북마크 구조가 포함된 경우에만 사이드 바에 표시됩니다.
- [링크](/pdf/ko/python-net/links/)- Python 라이브러리를 사용하여 PDF 문서의 링크 작업을 할 수 있습니다.
