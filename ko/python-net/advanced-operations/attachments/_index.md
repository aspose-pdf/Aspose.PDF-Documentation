이 섹션에서는 Aspose.PDF for Python via .NET을 사용하여 PDF에서 첨부 파일을 작업하는 방법을 설명합니다.
첨부 파일은 상위 문서에 추가되는 추가 파일이며, pdf, 워드, 이미지 또는 기타 파일 등 다양한 파일 유형이 될 수 있습니다.
PDF에 첨부 파일을 추가하고, 첨부 파일의 정보를 얻어 파일로 저장하며, Python으로 PDF에서 첨부 파일을 프로그래밍 방식으로 삭제하는 방법을 배울 수 있습니다.
