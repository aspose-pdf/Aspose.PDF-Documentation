이 섹션에서는.NET을 통해 파이썬용 Aspose.PDF 파일을 사용하여 PDF의 첨부 파일을 처리하는 방법을 설명합니다.
첨부 파일은 상위 문서에 첨부되는 추가 파일로, pdf, Word, 이미지 또는 기타 파일과 같은 다양한 파일 유형일 수 있습니다.
PDF에 첨부 파일을 추가하고, 첨부 파일의 정보를 가져와, 파일에 저장하고, Python을 사용하여 프로그래밍 방식으로 PDF에서 첨부 파일을 삭제하는 방법을 배웁니다.

PDF에 지원 파일을 포함하거나, 기존 문서에서 포함된 파일을 제거하거나, 관련 자료를 PDF 포트폴리오로 함께 패키징해야 할 때 이 문서를 사용하십시오.
