## PDF 주석 편집기의 용도

PDF 워크플로우에 주석, 검토 주석 또는 프로그래밍 방식으로 관리해야 하는 기타 마크업 요소가 포함된 경우 이 파사드를 사용하십시오.이 섹션이 확장되면 이 섹션을 중심으로 구축된 주석 관련 자습서의 시작 페이지 역할을 할 것입니다. `PdfAnnotationEditor` 수업.
