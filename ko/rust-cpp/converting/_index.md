PDF는 애플리케이션 소프트웨어, 하드웨어 및 운영 체제와 무관한 형식으로 문서 및 기타 참고 자료를 제공하기 위한 표준으로 개발되었습니다. PDF 파일의 내용은 텍스트에만 국한되지 않으며, 하이퍼링크, 이미지, 클릭 가능한 버튼 및 양식, 전자 서명, 워터마크 등도 포함될 수 있습니다. 따라서 콘텐츠를 편집하거나 변경하기 위해 PDF 파일을 다른 형식으로 변환해야 하는 경우가 종종 있습니다.
**Our Aspose.PDF for Rust via C++** 도구를 사용하면 PDF 문서를 가장 많이 사용되는 형식으로 성공적이고 빠르게 쉽게 변환할 수 있습니다. 지원되는 형식의 전체 목록은 해당 섹션을 참조하십시오. [Aspose.PDF 지원 파일 형식](https://docs.aspose.com/pdf/rust-cpp/supported-file-formats/).

**Aspose.PDF for Rust via C++**는 PDF 문서를 다양한 형식으로 변환하도록 허용합니다. 또한 Aspose.PDF 변환 품질을 확인하고 Aspose.PDF 변환기 앱으로 온라인에서 결과를 볼 수 있습니다. 코드 스니펫을 사용한 문서 변환 섹션을 알아보세요.

## Rust를 사용하여 변환하는 방법

Word 문서는 가장 다재다능하고 편집 가능한 형식입니다. PDF를 DOC로 수동 변환하는 것은 매우 시간이 많이 소요되는 작업입니다. 이 기사에서는 Rust에서 PDF를 DOC 및 DOCX로 변환하는 방법을 배우게 됩니다.

- [PDF를 DOC로 변환](/pdf/ko/rust-cpp/convert-pdf-to-doc/) - Rust를 사용하여 PDF 문서를 Word 형식으로 변환할 수 있습니다.

숫자 형식은 표의 데이터를 읽기 쉽게 만들 뿐만 아니라 표를 보다 쉽게 사용할 수 있게 하기 위해 필요합니다. 물론 이러한 데이터를 PDF 문서에서 Excel 형식으로 변환해야 할 경우, 저희 Aspose.PDF for Rust를 사용하십시오.

- [PDF를 Microsoft XLSX로 변환](/pdf/ko/rust-cpp/convert-pdf-to-xlsx/) - 이 섹션에서는 PDF 문서를 XLSX 및 CSV로 변환하는 방법을 설명합니다.

PowerPoint 형식은 다양한 프레젠테이션을 만들 때 사용됩니다. PPT 파일은 다양한 정보를 포함하는 많은 슬라이드 또는 페이지를 포함하고 있습니다.

- [PDF를 Microsoft PowerPoint로 변환](/pdf/ko/rust-cpp/convert-pdf-to-powerpoint/) - 여기에서는 변환 프로세스를 추적하면서 PDF를 PowerPoint로 변환하는 것에 대해 이야기합니다

HyperText Markup Language는 하이퍼텍스트 문서 기술 언어로, 웹 페이지를 만드는 표준 언어입니다. Aspose.PDF for Rust를 사용하면 HTML 문서를 쉽게 변환하고 그 반대도 가능합니다.

다양한 목적을 위해 PDF로 변환해야 하는 이미지 형식이 많이 있습니다. Aspose.PDF for Rust는 가장 인기 있는 이미지 형식을 지원합니다.

- [PDF를 다양한 이미지 형식으로 변환](/pdf/ko/rust-cpp/convert-pdf-to-images-format/) - PDF 페이지를 JPEG, PNG, SVG 등 다양한 형식의 이미지로 변환

이 섹션에는 EPUB, XPS, TeX, Text 및 Grayscale PDF와 같은 형식이 포함됩니다.

- [PDF 파일을 다른 형식으로 변환](/pdf/ko/rust-cpp/convert-pdf-to-other-files/) - 이 주제에서는 PDF 문서를 다양한 형식으로 변환하는 방법을 설명합니다
