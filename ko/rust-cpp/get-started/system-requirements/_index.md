## C++를 통한 Aspose.PDF for Rust 지원 플랫폼

asposepdf 패키지는 개발자가 PDF 파일을 직접 조작할 수 있게 해주는 강력한 툴킷이며, PDF와 관련된 다양한 작업을 수행하는 데 도움이 됩니다.
PDF를 다른 형식으로 변환하기 위한 고유한 기능을 포함하고 있습니다.

Linux x64, macOS x86_64, macOS arm64 및 Windows x64 플랫폼에 대한 지원이 구현되었습니다.

패키지 루트 디렉터리의 ‘lib’ 폴더에 있는 플랫폼별 동적 라이브러리 버전이 결과 애플리케이션을 배포하는 데 필요합니다: 

- Linux x64 플랫폼용 libAsposePDFforRust_linux_amd64.so 
- macOS arm64 플랫폼용 libAsposePDFforRust_darwin_arm64.dylib 
- macOS x86_64 플랫폼용 libAsposePDFforRust_darwin_amd64.dylib 
- AsposePDFforRust_windows_amd64.dll for Windows x64 플랫폼용.
