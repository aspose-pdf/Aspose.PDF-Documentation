## 일반 기능

다양한 목록에서 로드 및 저장을 지원합니다 [파일 형식](/pdf/ko/rust-cpp/supported-file-formats/).

## 지원되는 PDF 표준

Aspose.PDF for Rust는 PDF 버전 1.2, 1.3, 1.4, 1.5, 1.6 및 1.7을 지원합니다.
