**Aspose.PDF for C++** 라이브러리는 PDF 문서를 다양한 형식으로 변환할 수 있는 기능을 제공하며, 다른 형식에서 PDF로 변환할 수도 있습니다.

우리의 **Aspose.PDF for C++** 라이브러리는 PDF 문서를 가장 인기 있는 형식으로 성공적이고 빠르며 쉽게 변환할 수 있도록 합니다.

## Aspose.PDF를 변환에 사용하는 방법

Aspose.PDF는 로드 및 저장 모두에서 가장 인기 있는 문서 형식을 지원합니다.

현재 섹션은 인기 있는 변환만 설명한다는 점에 주목하십시오.
지원되는 형식의 전체 목록은 [지원되는 Aspose.PDF 파일 형식](https://docs.aspose.com/pdf/cpp/supported-file-formats/)을 참조하십시오.

코드 스니펫과 함께 문서 변환에 대한 섹션을 탐색하십시오.

Word 문서는 가능한 한 다재다능하고 편집할 수 있습니다. PDF를 Word로 수동 변환하는 것은 매우 시간이 많이 걸리는 작업입니다. 이 문서에서는 C++에서 PDF를 Word로 프로그래밍 방식으로 변환하는 방법을 보여줍니다.

- [PDF를 Microsoft Word로 변환하기](/pdf/ko/cpp/convert-pdf-to-word/) - PDF 문서를 C++로 Word 형식으로 변환할 수 있습니다.

숫자 형식은 표의 데이터를 더 쉽게 읽을 수 있도록 할 뿐만 아니라 표를 더 쉽게 사용할 수 있도록 하는 데 필요합니다. 물론 PDF 문서에서 이러한 데이터를 Excel 형식으로 변환해야 하는 경우 Aspose.PDF 라이브러리를 사용하세요.

- [PDF를 Microsoft Excel로 변환하기](/pdf/ko/cpp/convert-pdf-to-excel/) - 이 섹션에서는 PDF 문서를 XLSX, ODS, CSV 및 SpreadSheetML로 변환하는 방법을 설명합니다.

PowerPoint 형식은 다양한 프레젠테이션을 만드는 데 사용됩니다. PPT 파일에는 다양한 정보를 포함하는 많은 슬라이드나 페이지가 포함되어 있습니다.

- [PDF를 Microsoft PowerPoint로 변환하기](/pdf/ko/cpp/convert-pdf-to-powerpoint/) - 여기서는 변환 프로세스를 추적하여 PDF를 PowerPoint로 변환하는 것에 대해 설명합니다.

HyperText Markup Language은 웹 페이지를 만드는 표준 언어로, 하이퍼텍스트 문서 설명 언어입니다. Aspose.PDF for C++를 사용하면 HTML 문서를 쉽게 변환할 수 있습니다.

- [PDF 파일을 HTML 형식으로 변환하기](/pdf/ko/cpp/convert-pdf-to-html/) - PDF 문서를 HTML 파일로 별도의 페이지 또는 단일 페이지로 변환

다양한 이미지 형식을 다양한 목적으로 PDF로 변환해야 하는 경우가 많습니다. Aspose.PDF는 가장 인기 있는 이미지 형식과 그 반대도 지원합니다.

- [이미지 형식을 PDF 파일로 변환하기](/pdf/ko/cpp/convert-images-format-to-pdf/) - 다양한 형식의 이미지를 PDF 파일로 변환
- [PDF를 다양한 이미지 형식으로 변환하기](/pdf/ko/cpp/convert-pdf-to-images-format/) - PDF 페이지를 JPEG, PNG 및 기타 형식의 이미지로 변환

이 섹션에는 EPUB, Markdown, PCL, XPS, LATex/TeX, 텍스트, PostScript와 같은 형식이 포함됩니다.

- [다른 파일 형식을 PDF로 변환하기](/pdf/ko/cpp/convert-other-files-to-pdf/) - 이 주제는 EPUB, XPS, Postscript, 텍스트 등 다양한 형식으로의 변환을 설명합니다.
- [PDF 파일을 다른 형식으로 변환](/pdf/ko/cpp/convert-pdf-to-other-files/) - 이 주제는 PDF 문서를 다양한 형식으로 변환하는 방법을 설명합니다.

PDF/A는 전자 문서의 장기 보관을 위해 설계된 PDF의 버전입니다. 솔직히 말해서 외부에서 PDF인지 PDF/A인지 판단하기가 매우 어렵습니다. 이 파일을 확인하려면 검증기가 사용됩니다. PDF를 PDF/A로, 반대로 PDF/A를 PDF로 고품질 변환하는 방법에 대한 다음 기사를 확인하세요.

- [PDF를 PDF/A 형식으로 변환](/pdf/ko/cpp/convert-pdf-to-pdfa/) - Aspose.PDF의 C++ 라이브러리는 PDF를 PDF/A로 변환하는 간편한 방법을 제공합니다.
- [PDF/A를 PDF 형식으로 변환](/pdf/ko/cpp/convert-pdfa-to-pdf/) - C++를 사용하여 PDF/A를 PDF 형식으로 쉽고 빠르며 높은 품질로 변환합니다.
