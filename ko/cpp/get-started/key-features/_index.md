## 일반 기능

- 완전한 C++로 작성됨
- С++ 환경 필요.
- Windows Forms 및 ASP.C++ 애플리케이션 지원.
- HTML 및 Microsoft Help 형식의 API 참조.
- 32비트 OS
- 64비트 OS

## 지원되는 PDF 버전

Aspose.PDF for С++는 PDF 버전 1.2, 1.3, 1.4, 1.5, 1.6 및 1.7을 지원합니다.
