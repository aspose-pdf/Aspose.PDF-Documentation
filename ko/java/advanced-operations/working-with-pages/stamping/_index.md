PDF 문서의 스탬프는 종이 문서에 고무 도장을 찍는 것과 유사합니다. PDF 파일의 스탬프는 PDF 파일이 다른 사람에 의해 사용되는 것을 보호하고 PDF 파일의 내용의 보안을 확인하는 등의 추가 정보를 제공합니다. **Aspose.PDF for Java**를 사용하면 PDF 문서에 이미지 또는 텍스트 스탬프를 추가할 수 있습니다.

Java로 스탬프를 추가하는 방법을 배우려면 다음 섹션을 확인하십시오:

- [PDF 페이지에 이미지 스탬프 추가](/pdf/ko/java/image-stamps-in-pdf-page/) - 이미지 스탬프 추가, 이미지 품질 제어, PDF 파일의 배경으로 이미지를 스탬프로 추가.
- [PDF 파일에 텍스트 스탬프 추가](/pdf/ko/java/text-stamps-in-the-pdf-file/) - 텍스트 스탬프 추가, TextStamp 객체의 정렬 정의, PDF에 스탬프로 텍스트 채우기.
- [PDF에 페이지 스탬프 추가](/pdf/ko/java/page-stamps-in-the-pdf-file/) - 다른 PDF 페이지를 기반으로 복잡한 스탬프 추가.
