## Aspose.PDF - PDF를 DOC 또는 DOCX로 변환

**Aspose.PDF Java for PHP**를 사용하여 PDF 문서를 DOC 또는 DOCX 형식으로 변환하려면, **PdfToDoc** 모듈을 호출하세요.

PHP 코드

```php

# 대상 문서 열기
$pdf = new Document($dataDir . 'input1.pdf');

# 연결된 출력 파일 저장 (대상 문서)
$pdf->save($dataDir . "output.doc");

print "문서가 성공적으로 변환되었습니다";

```
