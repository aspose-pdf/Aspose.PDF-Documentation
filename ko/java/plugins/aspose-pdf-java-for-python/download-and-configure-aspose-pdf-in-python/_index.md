## 소스 코드 구성 방법

소스 코드를 열고 확장하기 위해 다음 간단한 단계를 따르십시오:

1. 다음에서 Python 2.7.x.x만 설치하세요: <https://www.python.org/downloads/>
2. 다음에서 JPype를 설치하세요: <http://sourceforge.net/projects/jpype/files/JPype/>
3. 다음에서 Java JDK를 설치하세요: <http://www.oracle.com/technetwork/java/javase/downloads>
4. Python 프로젝트에서 사용하려는 Aspose.PDF for Java API를 다음에서 다운로드하세요: <http://www.aspose.com/community/files/72/java-components/default.aspx> 그리고 다운로드한 API의 .jar 파일을 %Python%/lib 폴더에 복사하세요.
5. GitHub에서 [Aspose.PDF Java for Python Examples](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Plugins/Aspose_Pdf_Java_for_Python)를 클론하세요.
6. CodePlex에서 [Aspose.PDF Java for Python Examples](http://asposepdfjavapython.codeplex.com/)를 클론하세요.
