## 일반 기능

- 완전히 Java로 작성되었으며, JDK 1.8 이상에서 작동
- 모든 Java 애플리케이션 지원
- JavaDoc 형식의 API 참조
- 32비트 OS 지원
- 64비트 OS 지원

## 지원되는 PDF 버전

Aspose.PDF for Java는 PDF 버전 1.2, 1.3, 1.4, 1.5, 1.6 및 1.7을 지원합니다.

## 글꼴

- 14개의 기본 글꼴
- PostScript Type1 글꼴
- TrueType 글꼴
- Type 3 글꼴
- 글꼴 정보 PDF에 포함
- CJK 글꼴
- 유니코드 지원

## 텍스트

- 텍스트 교체
- 텍스트 추출 (전체 PDF, 특정 페이지 또는 페이지 범위에서)
- 텍스트 추가

## 그래프

- 선, 원, 호, 곡선과 같은 다양한 형태의 그래프 지원

- 그래프는 좌표로 지정된 페이지의 어느 위치에나 그릴 수 있습니다.
- 그래프는 회전, 확대 또는 채우기할 수 있으며, 대시 및 복합 대시 선 스타일을 정의할 수 있습니다.
- 그래프는 좌표를 기준으로 그려지며 상대적인 위치가 아닌 절대적인 위치를 가집니다.
- CMYK 및 회색 색상 공간이 지원됩니다.
