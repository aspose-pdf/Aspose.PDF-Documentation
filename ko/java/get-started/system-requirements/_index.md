## Overview

PDF 문서를 생성하기 위해 [Aspose.PDF for Java](https://products.aspose.com/pdf/java/)를 실행하는 기계에는 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html)이 설치되어 있을 필요는 없습니다. Aspose.PDF for Java 자체가 PDF 문서 생성 엔진이기 때문입니다. 그러나 Aspose.PDF for Java로 생성된 PDF 문서를 보기 위해서는 적어도 [Adobe Acrobat Reader](https://www.adobe.com/acrobat/pdf-reader.html)가 필요합니다. 이는 무료로 다운로드할 수 있습니다.

## Supported Operating Systems

Aspose.PDF for Java는 Java 런타임을 실행하는 모든 32비트 또는 64비트 운영 체제를 지원합니다. 여기에는 다음이 포함되지만 이에 국한되지는 않습니다:

### Windows

- Microsoft Windows 2000 Professional (x64, x86)
- Microsoft Windows 2000 Server (x64, x86)
- Microsoft Windows 2003 Server (x64, x86)
- Microsoft Windows 2008 Server (x64, x86)
- Microsoft Windows 2012 Server (x64, x86)
- Microsoft Windows 2012 R2 Server (x64, x86)
- Microsoft Windows 2016 Server (x64, x86)
- Microsoft Windows 2019 Server (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Azure

### 리눅스

- 리눅스 (우분투, CentOS, 기타)

### 맥

- 맥 OS X

## 지원되는 자바 버전

Aspose.PDF for Java는 다음의 자바 버전을 지원합니다:

- J2SE 8.0 (1.8) 이상 (*예를 들어 Java 10*)
- IBM i (Iseries 또는 As/400)에서의 Aspose.Pdf for Java 지원
