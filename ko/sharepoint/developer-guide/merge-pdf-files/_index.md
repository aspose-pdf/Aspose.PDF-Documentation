{{% alert color="primary" %}}

여러 PDF 파일을 단일 PDF 파일로 병합/연결하는 기능은 PDF 파일 처리 애플리케이션에서 매우 인기 있고 필요성이 높은 기능입니다. 우리는 이 중요한 기능을 [Aspose.PDF for SharePoint 2.2.0](https://releases.aspose.com/pdf/sharepoint/new-releases/aspose.pdf-for-sharepoint-2.2.0/) 버전. 두 파일을 병합한다는 것은 두 번째 파일을 첫 번째 파일 끝에 추가하여 단일 파일을 만드는 것입니다.

{{% /alert %}}

## **PDF 파일 병합**

다음과 같이 SharePoint Document library에서 여러 PDF 파일을 단일 PDF로 병합합니다:

1.  병합할 PDF 파일을 SharePoint Document library에서 선택합니다.

![todo:image_alt_text](merge-pdf-files_1.png)

2.  Library Tools에서 Aspose Tools 탭을 클릭합니다.

![todo:image_alt_text](merge-pdf-files_2.png)

3.  Library Tools에서 "Merge to PDF" 옵션을 클릭하여 선택한 모든 PDF 파일을 결과 PDF로 병합합니다.

![todo:image_alt_text](merge-pdf-files_3.png)

4.  적절한 이름으로 결과 PDF 파일을 다운로드/저장하도록 프롬프트가 표시됩니다.

![todo:image_alt_text](merge-pdf-files_4.png)
