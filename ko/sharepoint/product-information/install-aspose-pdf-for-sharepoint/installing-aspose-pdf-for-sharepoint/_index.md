{{% alert color="primary" %}}

Aspose.PDF for SharePoint는 Aspose.PDF.SharePoint.zip 아카이브로 다운로드할 수 있습니다.

{{% /alert %}}

**이 아카이브에는:**

- Aspose.PDF.SharePoint.wsp
  SharePoint 솔루션 파일. Aspose.PDF for SharePoint는 서버 팜 전반에 걸쳐 배포/철수 및 기능 활성화/비활성화를 용이하게 하기 위해 SharePoint 솔루션으로 패키징됩니다.
- Aspose_LicenseAgreement.rtf

**최종 사용자 라이선스 계약:**

- Aspose.PDF for SharePoint.pdf

**사용자 문서:**

- Aspose.PDF for SharePoint Documentation.chm

**공개 API 참조가 포함된 사용자 문서:**

- setup.exe

**설치 프로그램:**

- setup.exe.config

**설치 구성 파일:**

설치 프로그램은 진행하기 전에 다음 조건을 확인합니다:

- SharePoint 2010이 설치되어 있습니다.
- 사용자는 SharePoint 솔루션을 설치할 권한이 있습니다.
- SharePoint 데이터베이스가 온라인 상태입니다.
- SharePoint 관리 서비스가 시작되었습니다.
- SharePoint 타이머 서비스가 시작되었습니다. SharePoint 관리 서비스와 타이머 서비스는 일부 설정 작업이 서버 팜의 모든 서버에 전파되는 타이머 작업에 의존하기 때문에 필요합니다.

**SharePoint용 Aspose.PDF 설치 방법:**

- Aspose.PDF.SharePoint zip 파일을 로컬 드라이브에 압축 해제하십시오.
- setup.exe를 실행하고 화면에 표시되는 지침을 따르세요.

**설치 프로그램은 다음 작업을 수행합니다:**

- 설치 전제 조건을 확인합니다. 검사 중 하나라도 실패하면 설치가 진행되지 않습니다.

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_1.png)



- 최종 사용자 라이선스 계약을 표시합니다. 진행하려면 사용자가 계약에 동의해야 합니다.

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_2.png)



- 배포 대상 선택 대화 상자를 표시합니다. 사용자는 기능이 활성화될 웹 애플리케이션 및 사이트 컬렉션을 선택합니다. 아래 그림을 참고하십시오.

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_3.png)



- 기능을 서버 팜에 배포합니다.

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_4.png)



- 선택한 사이트 컬렉션에 대해 기능을 활성화하고 해당 상위 웹 애플리케이션을 구성합니다.
- 기능이 배포되고 활성화된 웹 애플리케이션 및 사이트 컬렉션 목록을 표시합니다.
