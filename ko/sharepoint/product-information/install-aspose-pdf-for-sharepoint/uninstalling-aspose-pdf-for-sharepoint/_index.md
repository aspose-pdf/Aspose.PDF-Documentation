{{% alert color="primary" %}}

Aspose.PDF for SharePoint를 제거하려면 설정 프로그램을 실행하십시오. Aspose.PDF for SharePoint가 이미 설치된 경우, 설정 프로그램이 제거를 제안합니다.

![todo:image_alt_text](uninstalling-aspose-pdf-for-sharepoint_1.png)

제거 과정에서 설정 프로그램은 모든 사이트 컬렉션에 대해 Aspose.PDF for SharePoint 기능을 비활성화하고 서버 팜에서 솔루션을 회수합니다.

{{% /alert %}}
