## **서버 요구 사항**

Aspose.PDF for SharePoint를 설치하려면 Microsoft SharePoint Server 2010을 실행하는 컴퓨터가 필요합니다.

## **클라이언트 요구 사항**

SharePoint 사이트에 접근하려면 브라우저가 필요합니다. Aspose.PDF for SharePoint는 다음 브라우저에서 테스트되었습니다:

- Microsoft Internet Explorer 8
- Mozilla FireFox 3.5
- Google Chrome 3
- Safari 4
