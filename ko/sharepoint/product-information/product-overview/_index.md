### **Aspose.PDF for SharePoint 문서에 오신 것을 환영합니다!**
Aspose.PDF for SharePoint는 사용자가 목록, 목록 항목 및 SharePoint 위키 페이지를 PDF 파일 형식으로 내보낼 수 있게 해 주는 솔루션입니다.

{{% /alert %}}


{{% alert color="primary" %}}
Aspose.PDF for SharePoint는 Microsoft SharePoint Server 2010과 함께 사용하도록 설계되었습니다. Microsoft SharePoint Server 2010 외에 추가 시스템 요구 사항은 없습니다.

이 문서는 Aspose.PDF for SharePoint의 [기능](/pdf/ko/sharepoint/features/), [설치](/pdf/ko/sharepoint/install-aspose-pdf-for-sharepoint/), [평가 제한 사항](/pdf/ko/sharepoint/evaluate-aspose-pdf/), [라이선스](/pdf/ko/sharepoint/license-aspose-pdf-for-sharepoint/), 일반적인 사용 사례 및 설정.
