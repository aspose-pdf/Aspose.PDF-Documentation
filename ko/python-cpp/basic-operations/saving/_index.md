## 파일 시스템에 PDF 문서 저장

```python

    import AsposePDFPythonWrappers as ap

    document = ap.Document("sample.pdf")  # PDF 문서를 엽니다
    document.pages.add()  # 페이지를 추가합니다
    document.save("sample_mod.pdf")  # 변경된 문서를 저장합니다
```
