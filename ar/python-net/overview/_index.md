_هذه الصفحة هي نظرة عامة على ميزات Aspose.PDF for Python._

**Aspose.PDF for Python عبر .NET**، API لمعالجة PDF تتيح للمطورين العمل مع مستندات PDF دون الحاجة إلى Microsoft Office® أو Adobe Acrobat Automation. تحقق من صفحات الهبوط الخاصة بـ [Aspose.PDF for Python](https://products.aspose.com/pdf/python-net/) للحصول على وصف أكثر تفصيلاً للميزات والإمكانيات الخاصة بالمكتبة.

يدعم Aspose.PDF for Python مجموعة واسعة من الوظائف مثل:

- يدعم معظم معايير PDF المعروفة ومواصفات PDF.
- القدرة على قراءة وتصدير ملفات PDF في صيغ صور متعددة بما في ذلك BMP و GIF و JPEG و PNG.
- تعيين المعلومات الأساسية (مثل
 المؤلف، المنشئ) لوثيقة PDF.
- تكوين خصائص صفحة PDF (مثل العرض، الارتفاع، مربع القص، مربع النزيف إلخ.).
- إعداد ترقيم الصفحات، مستوى الإشارة المرجعية، أحجام الصفحات إلخ.
- القدرة على العمل مع النصوص، الفقرات، العناوين، الروابط التشعبية، الرسوم البيانية، المرفقات إلخ.

بالإضافة إلى ذلك، يمكن تطبيق Aspose.PDF for Python عبر .NET بسهولة لتحويل EPUB، Markdown، نص، XPS، PostScript، XML، LaTex إلى PDF وتحويل PDF إلى تنسيقات مستندات متعددة بأداء ممتاز وجودة جيدة.

جرب [تطبيقاتنا المجانية عبر الإنترنت](https://products.aspose.app/pdf/applications) التي تعرض بعض من أكثر وظائف Aspose.PDF شيوعًا.
