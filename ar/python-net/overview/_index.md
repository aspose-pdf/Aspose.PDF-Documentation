_هذه الصفحة نظرة عامة على ميزات Aspose.PDF for Python._

**Aspose.PDF for Python عبر .NET**, واجهة برمجة تطبيقات معالجة PDF تسمح للمطورين بالعمل مع مستندات PDF دون الحاجة إلى Microsoft Office® أو أتمتة Adobe Acrobat. تحقق من صفحات الهبوط لـ [Aspose.PDF for Python](https://products.aspose.com/pdf/python-net/) للحصول على وصف أكثر تفصيلًا للميزات وإمكانات المكتبة.

يدعم Aspose.PDF for Python مجموعة واسعة من الوظائف مثل:

- يدعم معظم معايير PDF المعتمدة ومواصفات PDF.
- القدرة على قراءة وتصدير ملفات PDF بصيغ صور متعددة تشمل BMP و GIF و JPEG و PNG.
- تعيين معلومات أساسية (مثل المؤلف، المنشئ) لمستند PDF.
- تكوين خصائص صفحة PDF (مثل العرض، الارتفاع، cropbox، bleedbox إلخ).
- تعيين ترقيم الصفحات، مستوى العلامات المرجعية، أحجام الصفحات إلخ.
- القدرة على التعامل مع النصوص والفقرات والعناوين والروابط الفائقة والرسوم البيانية والمرفقات إلخ.

بالإضافة إلى ذلك، يمكن تطبيق Aspose.PDF for Python عبر .NET بسهولة لتحويل EPUB و Markdown و Text و XPS و PostScript و XML و LaTex إلى PDF وتحويل PDF إلى صيغ مستندات مختلفة بأداء ممتاز وجودة جيدة.

جرّب تطبيقاتنا [المجانية عبر الإنترنت](https://products.aspose.app/pdf/applications) التي تعرض بعضًا من أكثر وظائف Aspose.PDF شهرة.
