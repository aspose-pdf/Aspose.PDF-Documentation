_هذه الصفحة هي نظرة عامة على Aspose.PDF لميزات بايثون. _

**Aspose.pdf لبايثون عبر .NET**، واجهة برمجة تطبيقات معالجة ملفات PDF التي تسمح للمطورين بالعمل مع مستندات PDF دون الحاجة إلى ميكروسوفت أوفيس® أو أدوبي أكروبات أوتوميشن. تحقق من الصفحات المقصودة لـ [Aspose.PDF لبيثون](https://products.aspose.com/pdf/python-net/) للحصول على وصف أكثر تفصيلاً لميزات وإمكانيات المكتبة.

يدعم Aspose.PDF لـ Python مجموعة متنوعة من الوظائف مثل:

- يدعم معظم معايير PDF الراسخة ومواصفات PDF.
- القدرة على قراءة وتصدير ملفات PDF بتنسيقات صور متعددة بما في ذلك BMP و GIF و JPEG و PNG.
- قم بتعيين المعلومات الأساسية (مثل المؤلف والمنشئ) لوثيقة PDF.
- قم بتكوين خصائص صفحة PDF (مثل العرض والارتفاع وصندوق الاقتصاص ومربع النزيف وما إلى ذلك).
- تعيين ترقيم الصفحات ومستوى الإشارة المرجعية وأحجام الصفحات وما إلى ذلك.
- القدرة على العمل مع النصوص والفقرات والعناوين والارتباطات التشعبية والرسوم البيانية والمرفقات وما إلى ذلك.

بالإضافة إلى ذلك، يمكن تطبيق Aspose.PDF لـ Python عبر .NET لتحويل EPUB و Markdown و Text و XPS و PostScript و XML و LaTeX إلى PDF وتحويل PDF إلى تنسيقات مستندات مختلفة بأداء ممتاز وجودة جيدة.

جرب لدينا [تطبيقات عبر الإنترنت](https://products.aspose.app/pdf/applications) التي توضح بعض وظائف Aspose.PDF الأكثر شيوعًا.
