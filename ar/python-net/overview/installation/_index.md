{{% alert color="primary" %}}

**Aspose.PDF for Python via .NET** هي مكتبة تتيح للمطورين إضافة قدرات معالجة PDF إلى تطبيقاتهم. يمكن استخدام الواجهة البرمجية لبناء تطبيقات 32-بت و64-بت لتوليد وقراءة وتحويل ومعالجة ملفات PDF دون الحاجة إلى Adobe Acrobat.

{{% /alert %}}

## وصف المنتج

**Aspose.PDF for Python** يسمح بأداء مجموعة من مهام معالجة المستندات مثل معالجة النماذج، الحصول على معلومات البيانات الوصفية وتعيينها، معالجة النص والصفحات، إدارة التعليقات التوضيحية، إضافة أو إزالة العلامات المرجعية والعلامات المائية، المرفقات، التعامل مع الخطوط المخصصة والمزيد.

**Aspose.PDF for Python** يمكن استخدامها لتطوير تطبيقات بايثون 32-بت و64-بت لأنظمة تشغيل مختلفة (مثل Windows وLinux) حيث يتم تثبيت بايثون 3.5 أو أحدث.

## تثبيت Aspose.PDF للبايثون عبر .NET

يمكنك بسهولة استخدام aspose-pdf للبايثون من الرابط للتنزيل المباشر [pip](https://pypi.org/project/aspose-pdf/).
قم بتشغيل 'pip install aspose-pdf' لجلب الحزمة. إذا كان لديك بالفعل Aspose.PDF للبايثون وتريد الحصول على أحدث نسخة، يرجى تشغيل 'pip install --upgrade aspose-pdf'.
