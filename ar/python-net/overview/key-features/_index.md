## الميزات العامة

- يدعم معظم معايير PDF المحددة ومواصفات PDF.
- القدرة على قراءة وتصدير ملفات PDF بعدة تنسيقات صور بما في ذلك BMP و GIF و JPEG و PNG.
- تعيين المعلومات الأساسية (مثل المؤلف، المُنشئ) لمستند PDF.
- تكوين خصائص صفحة PDF (مثل العرض، الارتفاع، cropbox، bleedbox وغيرها).
- تعيين ترقيم الصفحات، مستوى الإشارات المرجعية، أحجام الصفحات إلخ.
- القدرة على العمل مع النصوص والفقرة والعناوين والروابط التشعبية والرسوم البيانية والمرفقات إلخ.

## معايير PDF المدعومة

- ISO 32000-1 (PDF 1.2, PDF 1.3, PDF 1.4, PDF 1.5, PDF 1.6, PDF 1.7).
- ISO 32000-2 (PDF 2.0).
- ISO 19005 (PDF/A)، ISO 15930 (PDF/X)، ISO 24517 (PDF/E)، ISO 14289 (PDF/UA).

## ميزات التحويل

تتيح لك مكتبة Aspose.PDF لبايثون عبر .NET تحويل مستندات PDF الخاصة بك بنجاح وبسرعة وسهولة إلى أكثر الصيغ شيوعًا والعكس بالعكس.

- تحويل PDF إلى Word و Excel و PowerPoint.
- تحويل PDF إلى صيغ صور.
- تحويل ملف PDF إلى صيغة HTML والعكس.
- تحويل PDF إلى EPUB والنص و XPS وغيرها.
- تحويل EPUB و Markdown والنص و XPS و PostScript و XML و LaTex إلى PDF.

## إصدارات PDF المدعومة

يدعم Aspose.PDF لبايثون إصدارات PDF 1.2 و 1.3 و 1.4 و 1.5 و 1.6 و 1.7 و 2.0.
