## الميزات العامة

- يدعم معظم معايير PDF الراسخة ومواصفات PDF.
- القدرة على قراءة وتصدير ملفات PDF بتنسيقات صور متعددة بما في ذلك BMP و GIF و JPEG و PNG.
- قم بتعيين المعلومات الأساسية (مثل المؤلف والمنشئ) لوثيقة PDF.
- قم بتكوين خصائص صفحة PDF (مثل العرض والارتفاع وصندوق الاقتصاص ومربع النزيف وما إلى ذلك).
- تعيين ترقيم الصفحات ومستوى الإشارة المرجعية وأحجام الصفحات وما إلى ذلك.
- القدرة على العمل مع النصوص والفقرات والعناوين والارتباطات التشعبية والرسوم البيانية والمرفقات وما إلى ذلك.

## معايير PDF المدعومة

- أيزو 32000-1 (PDF 1.2، PDF 1.3، PDF 1.4، PDF 1.5، PDF 1.6، PDF 1.7).
- أيزو 32000-2 (بي دي إف 2.0).
- أيزو 19005 (PDF F/A)، أيزو 15930 (PDF\ X)، أيزو 24517 (PDF/E)، أيزو 14289 (PDF/UA).

## ميزات التحويل

يتيح لك Aspose.PDF for Python عبر مكتبة.NET تحويل مستندات PDF الخاصة بك بنجاح وسرعة وسهولة إلى التنسيقات الأكثر شيوعًا والعكس صحيح.

- قم بتحويل ملفات PDF إلى وورد وإكسل وبوربوينت.
- تحويل PDF إلى تنسيقات صور.
- قم بتحويل ملف PDF إلى تنسيق HTML والعكس صحيح.
- تحويل ملفات PDF إلى EPUB والنص وXPS وما إلى ذلك.
- تحويل ملفات EPUB، تخفيض السعر، النص، XPS، بوستسكريبت، XML، LaTeX إلى PDF.

## إصدارات PDF المدعومة
