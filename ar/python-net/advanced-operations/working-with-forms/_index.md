يصف هذا القسم طريقة سريعة وبسيطة لإكمال مستند PDF برمجيًا من خلال استخدام Aspose.PDF لـ Python عبر مكتبة.NET. يناقش القسم أيضًا كيف يمكن للمرء استخدام Aspose.PDF لـ Python لاكتشاف وتعيين الحقول المتاحة داخل ملف PDF موجود باستخدام AcroForms.

يسمح AcroForms للمستخدمين بملء النماذج وإرسالها وتخزينها إلكترونيًا، مما يجعلها طريقة مريحة وفعالة لجمع المعلومات.

- [اكروفورمز](/pdf/ar/python-net/acroforms/) - إنشاء نموذج وملء حقل النموذج واستخراج البيانات من النموذج وإضافة وإزالة الحقول في ملف PDF الخاص بك باستخدام مكتبة Python.
- [نماذج XFA](/pdf/ar/python-net/xfa-forms/) - تحويل XFA، واستخدام التجاهل يحتاج إلى العرض.
