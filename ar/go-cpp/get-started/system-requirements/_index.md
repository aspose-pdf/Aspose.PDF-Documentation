## المنصات المدعومة لـ Aspose.PDF for Go via C++

حزمة asposepdf هي مجموعة أدوات قوية تتيح للمطورين تعديل ملفات PDF مباشرة وتساعد على أداء مهام مختلفة للـ PDF.
تحتوي على ميزات فريدة لتحويل PDF إلى صيغ أخرى.

تم تنفيذ الدعم لمنصات Linux x64 و macOS x86_64 و macOS arm64 و Windows x64. تم استخدام [cgo](https://go.dev/wiki/cgo).

الإصدار الخاص بالمنصة من المكتبة الديناميكية الموجود في مجلد ’lib’ في الدليل الجذر لحزمة البرنامج مطلوب لتوزيع التطبيق الناتج:

- **libAsposePDFforGo_linux_amd64.so** لمنصة Linux x64.
- **libAsposePDFforGo_darwin_arm64.dylib** لمنصة macOS arm64.
- **libAsposePDFforGo_darwin_amd64.dylib** لنظام macOS بمعمارية x86_64.
- **AsposePDFforGo_windows_amd64.dll** لنظام Windows بمعمارية x64.

منصة Windows x64 تتطلب [MinGW-W64](https://www.mingw-w64.org/) مُثبت.
