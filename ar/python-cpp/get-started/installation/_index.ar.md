# التثبيت

## تقييم Aspose.PDF لـ Python عبر C++

يمكنك بسهولة تنزيل Aspose.PDF لـ Python للتقييم. التنزيل للتقييم هو نفسه التنزيل الذي يتم شراؤه. يصبح الإصدار التجريبي مرخصًا ببساطة عند إضافة بضع سطور من التعليمات البرمجية لتطبيق الترخيص.

يمكنك تنزيل [Aspose.PDF لـ Python](https://releases.aspose.com/pdf/pythoncpp/) للتقييم. التنزيل للتقييم هو نفسه التنزيل الذي يتم شراؤه.

يوفر الإصدار التجريبي من Aspose.PDF (بدون تحديد ترخيص) وظيفة المنتج الكاملة، ولكنه يحتوي على قيودين: يقوم بإدراج علامة مائية للتقييم، ويمكن عرض/تعديل أربعة عناصر فقط من أي مجموعة.

## تثبيت Aspose.PDF لـ Python عبر C++

يمكنك بسهولة استخدام aspose-pdf لـ Python من الرابط للتنزيل المباشر [pip](https://pypi.org/project/aspose-pdf-cpp-for-python/).

لتثبيت واستخدام Aspose.PDF لـ Python عبر C++، اتبع التعليمات التالية:

- إنشاء بيئة افتراضية: `python -m venv .venv`
- تثبيت الحزمة `pip install aspose-pdf-cpp-for-python`
