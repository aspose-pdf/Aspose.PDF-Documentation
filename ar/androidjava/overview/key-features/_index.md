## الميزات العامة

- يدعم Aspose.PDF for Android عبر Java أي جهاز محمول يشغل بيئة تشغيل Java بما في ذلك
- توفر API تشفيرًا 40 بت أو 128 بت

## الإصدارات المدعومة من PDF

يُدعم Aspose.PDF for Android عبر Java إصدارات PDF 1.2 و 1.3 و 1.4 و 1.5 و 1.6 و 1.7.

## مخطط

- يتم دعم أشكال مختلفة من المخططات، مثل الخط، والدائرة، والقوس، والمنحنى.
- يمكن رسم المخططات في أي موقع على الصفحة يتم تحديده بالإحداثيات.
- يمكن تدوير المخططات أو تحجيمها أو تعبئتها، ويمكن تعريف أنماط الخط المتقطّع والمتعدد المتقطّعات.
- يتم رسم الرسوم البيانية بناءً على الإحداثيات وتكون موقعها قديمًا بدلاً من أن تكون نسبية.
- يتم دعم مساحات اللون CMYK والرمادي.
