## نظرة عامة

لإنتاج مستندات PDF، الجهاز لتشغيل [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) ليس بحاجة إلى [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) تم تثبيتها كـ Aspose.PDF for Java نفسها هي محرك إنشاء مستندات PDF. ومع ذلك، من أجل عرض مستندات PDF التي ينتجها Aspose.PDF for Android عبر Java تحتاج الآلة على الأقل إلى [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). يمكن أن يكون [تم تنزيله](https://www.adobe.com/acrobat/pdf-reader.html) مجانيًا.

## أنظمة التشغيل المدعومة

يدعم Aspose.PDF لنظام Android عبر Java أي جهاز محمول يعمل على بيئة تشغيل Java بما في ذلك، لكن ليس مقتصرًا على:

### أندرويد

- أندرويد API 32
- أندرويد API 31
- أندرويد 4.0
- أندرويد 5.0
- أندرويد 6.0
- أندرويد 7.0
- أندرويد 8.0
- أندرويد 9.0
- أندرويد 10.0
- Android 11.0


## بيئات التطوير

يمكنك استخدام Aspose.PDF for Android عبر Java لتطوير التطبيقات في أي بيئة تطوير تستهدف منصة Java تحت Android، ولكن البيئات التالية مدعومة صراحةً:

- Android Studio وأدوات SDK
