_توفر هذه الصفحة نظرة عامة على ميزات Aspose.PDF لـ Node.js عبر C++._

يتيح **Aspose.PDF لـ Node.js عبر C++** للمطورين تعديل ملفات PDF مباشرة في بيئة Node.js. وقد تم بناء هذا الحل باستخدام تقنية WebAssembly ويستند إلى Aspose.PDF for .NET. يمكنك استخدام `asposepdfnodejs` بسهولة ثم اتباع تعليمات التثبيت.

**حول امتدادات الملفات CJS و MJS**

يدعم Node.js نظامين وحدويين لتنظيم ومشاركة الشيفرة. وهما:

- وحدات CommonJS
- وحدات ECMAScript

كانت وحدات CommonJS الطريقة التقليدية لمعالجة الاعتماديات في Node.js، بينما تُعد وحدات ECMAScript الإضافة الأحدث وتتبع الصيغة القياسية المستخدمة في المتصفحات الحديثة. ويُستخدم الامتدادان `cjs` و`mjs` لتحديد تنسيقات الوحدات (CommonJS وECMAScript Modules) المستخدمة في ملفات JavaScript.

في أمثلة الشيفرة، ستلاحظ اختلافًا في طريقة العمل مع الوحدة `asposepdfnodejs`. ويكمن الفرق الأساسي بين `require` و`import` في حالات الاستخدام الخاصة بكل منهما. إذ يقتصر `require` على استيراد الوحدات بالكامل، بينما يتيح `import` مرونة أكبر لاستيراد الوحدات الكاملة أو الصادرات المحددة منها.

**نهجين مختلفين**

تقدم JavaScript نهجين للتعامل مع العمليات غير المتزامنة: `then` و`await`.

تعمل الطريقة `then` مع الوعود لإدارة النتائج الناجحة والأخطاء بشكل متسلسل. أما `await` فتمكّن JavaScript من انتظار اكتمال العملية غير المتزامنة قبل متابعة تنفيذ الدالة وإرجاع النتيجة.

يدعم Aspose.PDF for Node.js مجموعة واسعة من الوظائف مثل:

- تعيين معلومات ملف PDF والحصول عليها وإزالتها
- حذف العلامات المرجعية
- حذف التعليقات التوضيحية
- استخراج نص من PDF
- تحسين مستند PDF
- تحسين الموارد
- تقسيم PDF إلى ملفين.
- دمج ملفين إلى PDF
- إصلاح PDF
- ميزات التحويل

بالإضافة إلى ذلك، يمكن استخدام Aspose.PDF لـ Node.js عبر C++ بسهولة لتحويل مستندات PDF إلى صيغ مختلفة بأداء عالٍ وجودة ممتازة. راجع قسم [صيغ الملفات المدعومة من Aspose.PDF](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). كما تدعم هذه الأداة تشفير مستندات PDF وفك تشفيرها.
