## ميزات أمان PDF

يوفر الـ API تشفيرًا بمستوى 40-bit أو 128-bit. ويمكن أيضًا تعيين كلمات مرور المستخدم والمالك لمزيد من الأمان. كما يمكن تقييد عمليات التعامل مع محتوى الملفات مثل نسخ المحتوى والطباعة منخفضة الدقة وتعديل التعليقات وإكمال حقول النماذج، إلى جانب تجميع ملفات PDF باستخدام هذه المكتبة.

- تشفير ملف PDF
- فك تشفير ملف PDF
- تغيير كلمة المرور
- توقيع PDF
