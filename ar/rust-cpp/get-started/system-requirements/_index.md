## المنصات المدعومة لـ Aspose.PDF for Rust عبر C++

الحزمة asposepdf هي مجموعة أدوات قوية تتيح للمطورين تعديل ملفات PDF مباشرة وتساعد في تنفيذ مهام متنوعة للـ PDF.
يحتوي على ميزات فريدة لتحويل PDF إلى صيغ أخرى.

تم تنفيذ الدعم لمنصات Linux x64، macOS x86_64، macOS arm64، وWindows x64.

الإصدار الخاص بالمنصة من المكتبة الديناميكية الموجودة في مجلد ‘lib’ في الدليل الجذر للحزمة مطلوب لتوزيع التطبيق الناتج: 

- libAsposePDFforRust_linux_amd64.so لمنصة Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib لمنصة macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib لمنصة macOS x86_64 
- AsposePDFforRust_windows_amd64.dll لمنصة Windows x64.
