**Advanced Operations** هو قسم يشرح كيفية التعامل مع ملفات PDF الموجودة برمجياً، سواء كانت مستندات تم إنشاؤها باستخدام Aspose.PDF كما نوقشت في [العمليات الأساسية](/pdf/ar/rust-cpp/basic-operations/), أو ملفات PDF تم إنشاؤها باستخدام Adobe Acrobat أو Google Docs أو Microsoft Office أو Open Office أو أي مُنتج PDF آخر.

ستتعلّم طرقًا مختلفة لـ:

- [العمل مع المستندات](/pdf/ar/rust-cpp/working-with-documents/) - ضغط، وتقسيم، ودمج المستندات وإجراء عمليات أخرى على المستند بالكامل.
- [العمل مع الصفحات](/pdf/ar/rust-cpp/working-with-pages/) - إضافة أو نقل أو حذف، تدوير الصفحات.
- [العمل مع النص](/pdf/ar/rust-cpp/working-with-text/) - إضافة نص إلى PDF باستخدام Aspose.PDF و Rust.
- [التأمين والتوقيع](/pdf/ar/rust-cpp/securing-and-signing/) - حماية وتوقيع مستند PDF الخاص بك برمجيًا
