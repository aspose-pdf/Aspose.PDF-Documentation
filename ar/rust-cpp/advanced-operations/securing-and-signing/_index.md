يوفر هذا القسم دليلاً شاملاً للعمل مع مستندات PDF المؤمنة باستخدام Aspose.PDF for Rust عبر C++. يوضح كيفية حماية ملفات PDF باستخدام كلمات مرور، وإدارة أذونات الوصول، وفتح أو فك تشفير المستندات المشفرة بأمان في Rust.

يتناول المقال مهام PDF المتعلقة بالأمان الشائعة، بما في ذلك تشفير ملفات PDF باستخدام خوارزميات تشفير حديثة، وفك تشفير الملفات المحمية بكلمة مرور، والتحكم في وصول المستخدم عبر علامات الأذونات. ستتعلم أيضًا كيفية فحص إعدادات الأذونات الحالية وفتح المستندات المؤمنة باستخدام بيانات اعتماد المالك لمزيد من المعالجة.

ستتعلم كيفية إنشاء مستندات PDF، وتطبيق حماية تشفير حديثة باستخدام تشفير قائم على AES، والتحكم في قدرات المستخدم مثل الطباعة، وتعديل المحتوى، وملء النماذج. كما يوضح المقال كيفية فتح ملفات PDF المحمية بكلمة مرور باستخدام بيانات اعتماد المالك وفك تشفيرها لإنتاج مستندات غير مقيدة مناسبة للمعالجة الإضافية.

للعمل على تعيين الأذونات والحصول على الأذونات، يرجى الرجوع إلى جدول مرجع أذونات PDF.  والذي يسرد علامات الأذونات المتاحة التي يمكن تطبيقها على مستند PDF للتحكم في كيفية تفاعل المستخدمين النهائيين معه.

## مرجع أذونات PDF

| **الإذن** | **الوصف** |
| :- | :- |
| Permissions::PRINT_DOCUMENT | السماح بالطباعة |
| Permissions::MODIFY_CONTENT | السماح بتعديل المحتوى (باستثناء النماذج/التعليقات التوضيحية) |
| Permissions::EXTRACT_CONTENT | السماح بنسخ/استخراج النص والرسومات |
| Permissions::MODIFY_TEXT_ANNOTATIONS | السماح بإضافة/تعديل التعليقات التوضيحية النصية |
| Permissions::FILL_FORM | السماح بملء النماذج التفاعلية |
| Permissions::EXTRACT_CONTENT_WITH_DISABILITIES | السماح باستخراج المحتوى لتسهيل الوصول |
| Permissions::ASSEMBLE_DOCUMENT | السماح بإدراج/تدوير/حذف الصفحات أو تغيير الهيكل |
| Permissions::PRINTING_QUALITY | السماح بطباعة عالية الجودة / دقيقة |
