**Aspose.Pdf for Reporting Services** هو امتداد تصيير مخصص لـ Microsoft SQL Server 2016/2017/2019/2022 Reporting Services و Power BI Report Server. يتم توفير Aspose.Pdf for Reporting Services كحزمة MSI واحدة يمكن تثبيتها على الحواسيب التي تعمل بأحد الأنظمة التالية:

- Microsoft SQL Server 2016 Reporting Services  
- Microsoft SQL Server 2017 Reporting Services  
- Microsoft SQL Server 2019 Reporting Services  
- مايكروسوفت SQL Server 2022 لخدمات التقارير  
- مايكروسوفت Power BI خادم التقارير
 
Aspose.Pdf لخدمات التقارير سهل النشر والإدارة، لأنه يتألف من تجميع .NET واحد فقط وهو Aspose.Pdf.ReportingServices.dll، مكتوب بالكامل بلغة C#، متوافق مع CLS ويحتوي فقط على شفرة مُدارة آمنة. يلزم وجود .NET Framework 4.8.1 مثبتًا على الخادم.

يجب نسخ Aspose.Pdf.ReportingServices.dll إلى المجلد ReportServer\bin ويجب تحديث ملف التكوين حتى تكون خدمات التقارير على علم بملحق العرض الجديد. يتم تنفيذ هذه الخطوات بواسطة برنامج تثبيت Aspose.Pdf لخدمات التقارير، ولكن يمكنك أيضًا تنفيذها يدويًا كما هو موضح لاحقًا في هذه الوثيقة.
