## مرحبًا بكم في Aspose.PDF لخدمات التقارير!

تفي خدمات تقارير Microsoft SQL Server بحاجة تواجهها العديد من المنظمات - الحاجة إلى بناء حلول ذكاء الأعمال والتقارير. حتى الآن، كان يتعين على المطورين تضمين التقارير في تطبيقاتهم، أو كان يتعين على المنظمات شراء حلول تقارير من جهات خارجية مكلفة وأحيانًا تكون إشكالية. الآن، تقدم خدمات تقارير Microsoft SQL Server حلاً كاملاً لتوزيع التقارير عبر المؤسسة؛ مما يمكّن الشركات من اتخاذ القرارات بشكل أفضل وأسرع.

**Aspose.Pdf لخدمات التقارير** هو حل فريد آخر من Aspose، الذي يجعل من الممكن توليد تقارير PDF في Microsoft SQL Server 2016/2017/2019/2022 لخدمات التقارير وخادم تقارير Power BI. كل ميزات تقرير RDL، بما في ذلك الجداول والمصفوفات والمخططات والصور، تُحوّل بدقة عالية إلى PDF.

تحتوي خدمات تقارير Microsoft SQL Server على قدرات مدمجة لتصدير التقارير كوثائق PDF، لكنها تفتقر إلى تقديم الدعم الفني اللازم للمستخدم النهائي. تسعى Aspose.Pdf لتوفير دعم فني رفيع المستوى وفعال.

تنشئ Aspose.Pdf for Reporting Services الوثائق على الخادم دون استخدام Adobe.Pdf SDK. تستخدم Aspose.Pdf for Reporting Services داخليًا Aspose.Pdf for .NET – المكون ذو المستوى العالمي لمعالجة الوثائق وتحويلها على جانب الخادم.
