## نظرة عامة

لإنتاج مستندات PDF، لا تحتاج الآلة لتشغيل [Aspose.PDF لـ JavaScript](https://products.aspose.com/pdf/javascript-cpp/) إلى تثبيت [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) حيث أن Aspose.PDF لـ JavaScript نفسه هو محرك إنشاء مستندات PDF. ومع ذلك، من أجل عرض مستندات PDF التي ينتجها Aspose.PDF لـ JavaScript، تحتاج الآلة على الأقل إلى [Adobe Acrobat Reader](https://www.adobe.com/acrobat/pdf-reader.html). يمكن تنزيله مجانًا.

## أنظمة التشغيل المدعومة

استخدم خادم ويب أو احصل على مثبت من الصفحة الرئيسية لـ Python (لتشغيل خادم ويب محلي).
 Aspose.PDF لـ JavaScript عبر C++ يدعم متصفحات الويب الحديثة:

- كروم
- إيدج
- فايرفوكس
- ويب كيت/سفاري
