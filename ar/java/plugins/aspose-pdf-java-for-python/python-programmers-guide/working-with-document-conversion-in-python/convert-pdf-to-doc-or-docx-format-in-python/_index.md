لتحويل مستند PDF إلى تنسيق DOC أو DOCX باستخدام **Aspose.PDF Java for Python**، ببساطة استدعِ وحدة **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "تم تحويل المستند بنجاح"
```
