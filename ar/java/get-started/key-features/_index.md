## الميزات العامة

- مكتوب بالكامل بلغة Java، ويعمل مع JDK 1.8 وما فوق
- يدعم جميع تطبيقات Java
- مرجع API بتنسيق JavaDoc
- دعم نظام تشغيل 32 بت.
- دعم نظام تشغيل 64 بت.

## إصدارات PDF المدعومة

يدعم Aspose.PDF for Java إصدارات PDF 1.2، 1.3، 1.4، 1.5، 1.6 و 1.7.

## الرسم

- تدعم أشكال مختلفة من الرسوم البيانية، مثل الخط والدائرة والقوس والمنحنى.

- يمكن رسم الرسوم البيانية في أي موقع على الصفحة يحدده الإحداثيات.
- يمكن تدوير الرسوم البيانية وتغيير حجمها أو ملؤها، ويمكن تعريف أنماط الخطوط المتقطعة والمتعددة.
- يتم رسم الرسوم البيانية بناءً على الإحداثيات وتحتوي على موضع قديم بدلاً من النسبي.
- تدعم مساحات الألوان CMYK وGray.
