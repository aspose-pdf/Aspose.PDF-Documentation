{{% alert color="primary" %}}

تسرد هذه الصفحة تغييرات واجهة برمجة التطبيقات العامة التي تم تقديمها في Aspose.PDF لـ Java 4.5.0. لا يتضمن ذلك الأساليب العامة الجديدة والمهملة فقط، بل يشتمل أيضًا على وصف لأي تغييرات في السلوك وراء الكواليس في Aspose.PDF لـ Java والتي قد تؤثر على التعليمات البرمجية الحالية. أي سلوك تم تقديمه يمكن اعتباره تراجعًا ويعدل السلوك الحالي هو مهم بشكل خاص ويتم توثيقه هنا.

{{% /alert %}}

تم نقل الفئات التالية من الحزمة **com.aspose.pdf** إلى حزمة **com.aspose.pdf.devices**.

- BmpDevice
- ColorDepth
- CompressionType
- Device
- DocumentDevice
- EmfDevice
- FormPresentationMode
- GifDevice
- GraphicsDevice
- ImageDevice
- ImageGenerator
- JpegDevice
- Margins
- PageDevice
- PngDevice
- Resolution
- ShapeType
- TextDevice
- TiffDevice
- TiffSettings
