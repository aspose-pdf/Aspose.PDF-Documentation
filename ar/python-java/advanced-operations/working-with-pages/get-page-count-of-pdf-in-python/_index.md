<ins>للحصول على عدد صفحات مستند Pdf باستخدام **Aspose.PDF Java for Python**، قم فقط باستدعاء فئة **GetNumberOfPages**.

**كود بايثون**
```
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "عدد الصفحات:" . page_count

```
