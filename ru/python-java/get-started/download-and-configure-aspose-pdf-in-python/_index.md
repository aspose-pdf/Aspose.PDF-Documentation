## Загрузка необходимых библиотек

Загрузите необходимые библиотеки, указанные ниже. Они требуются для выполнения примеров Aspose.PDF для Python через Java.

- Aspose: [Aspose.PDF для Python через Java Компонент](https://releases.aspose.com/pdf/pythonjava/)
- или PyPI: [Индекс пакетов Python](https://pypi.org/project/aspose-pdf-for-python-via-java/)
- [JPype](https://pypi.python.org/pypi/JPype1)
- JDK 1.8+

## Загрузка примеров с сайтов социального кодирования

Следующие выпуски выполняемых примеров доступны для загрузки на указанных ниже сайтах социального кодирования:

[Aspose.PDF для Python через Java](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Plugins/Aspose_Pdf_Java_for_Python)

### Как настроить исходный код

Пожалуйста, следуйте этим простым шагам, чтобы открыть и расширить исходный код при использовании:

1. Установите только Python 3.7+ с [Официального сайта](https://www.python.org/downloads)
2. Установите JPype 1.5.0+ с [Sourceforge](http://sourceforge.net/projects/jpype/files/JPype/)
3. Установите Java JDK 1.8+ с [Java Downloads](https://www.oracle.com/java/technologies/downloads/)
4. Загрузите Aspose.PDF для Python через Java API: [Aspose.PDF для Python через Java Component](https://releases.aspose.com/pdf/pythonjava/)
5. Или установите Aspose.PDF для Python через Java API с помощью pip: [PyPI](https://pypi.org/project/aspose-pdf-for-python-via-java/)
