## Установка Aspose.PDF для C++ через NuGet

NuGet — это бесплатная, с открытым исходным кодом система управления пакетами, ориентированная на разработчиков, для платформы C++, предназначенная для упрощения процесса включения сторонних библиотек в приложение C++ во время разработки. Это расширение для Visual Studio, которое упрощает добавление, удаление и обновление библиотек и инструментов в проектах Visual Studio, использующих C++ Framework. Библиотека или инструмент могут быть легко переданы другим разработчикам путем создания пакета NuGet и его хранения в [репозитории NuGet](https://www.nuget.org/packages/Aspose.PDF.Cpp/). Когда вы устанавливаете пакет, NuGet копирует файлы в ваше решение и автоматически вносит необходимые изменения, такие как добавление ссылок и изменение ваших файлов app.config или web.config. Если вы решите удалить библиотеку, NuGet удаляет файлы и отменяет все изменения, внесенные в ваш проект, чтобы не оставалось мусора.

### Ссылка на Aspose.PDF для C++

#### Установка пакета с использованием консоли диспетчера пакетов

- Откройте ваше C++ приложение в Visual Studio.
- В меню Инструменты выберите **Диспетчер пакетов NuGet**, а затем **Консоль диспетчера пакетов**.

- Введите команду `Install-Package Aspose.PDF` для установки последнего полного релиза, или введите команду `Install-Package Aspose.PDF -prerelease` для установки последнего релиза, включая исправления.- Нажмите `Enter`

#### Обновление пакета с помощью консоли диспетчера пакетов

Если вы уже добавили компонент через NuGet, выполните следующие шаги, чтобы обновить ссылку на последнюю версию:

- Откройте ваше C++ приложение в Visual Studio.
- В меню Инструменты выберите **Диспетчер пакетов NuGet**, а затем **Консоль диспетчера пакетов**.
- Введите команду `Update-Package Aspose.PDF` для ссылки на последний полный выпуск или введите команду `Update-Package Aspose.PDF -prerelease` для установки последнего выпуска, включая исправления.

#### Установка пакета с помощью графического интерфейса диспетчера пакетов

Следуйте этим шагам, чтобы добавить компонент с помощью графического интерфейса диспетчера пакетов:

- Откройте ваше C++ приложение в Visual Studio.
- В меню Проект выберите **Управление пакетами NuGet**.
- Выберите вкладку **Обзор**.
- Введите Aspose.PDF в поле поиска, чтобы найти Aspose.PDF для C++.
- Нажмите Установить/Обновить рядом с последней версией Aspose.PDF для C++.
