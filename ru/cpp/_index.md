<h2>Добро пожаловать в раздел документации Aspose.PDF для C++</h2>

Aspose.PDF for C++ — это нативная библиотека C++, которая позволяет разработчикам создавать, читать и обрабатывать PDF-документы без использования Adobe Acrobat. Aspose.PDF for C++ позволяет разработчикам создавать формы, добавлять/редактировать текст, управлять страницами PDF, добавлять аннотации, обрабатывать пользовательские шрифты и многое другое. Here in the documentation section of Aspose.PDF for C++, you can explore all the features of the API along with the basic examples to demonstrate the implementation of those features in your C++ application.
