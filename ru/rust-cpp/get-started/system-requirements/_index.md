## Поддерживаемые платформы Aspose.PDF for Rust via C++

Пакет asposepdf — это мощный набор инструментов, который позволяет разработчикам напрямую работать с PDF‑файлами и выполнять различные задачи с PDF.
Содержит уникальные функции для преобразования PDF в другие форматы.

Реализована поддержка платформ Linux x64, macOS x86_64, macOS arm64 и Windows x64.

Требуется версия динамической библиотеки, специфичная для платформы, из папки ‘lib’ в корневом каталоге пакета, для распространения полученного приложения: 

- libAsposePDFforRust_linux_amd64.so для платформы Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib для платформы macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib для платформы macOS x86_64 
- AsposePDFforRust_windows_amd64.dll для платформы Windows x64.
