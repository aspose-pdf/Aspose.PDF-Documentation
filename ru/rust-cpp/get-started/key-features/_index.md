## Кроссплатформенное программное обеспечение

Поддерживает различные платформы и версии Rust. См. [Системные требования](/pdf/ru/rust-cpp/system-requirements/) для подробностей.

## Общие функции

Поддерживает загрузку из и сохранение в широкий список [форматов файлов](/pdf/ru/rust-cpp/supported-file-formats/).

## Поддерживаемые стандарты PDF

Aspose.PDF for Rust поддерживает версии PDF 1.2, 1.3, 1.4, 1.5, 1.6 и 1.7.

## Функции по типу

### Основные операции

- new PDF
- open PDF
- save PDF
- save_as
- page_insert
- page_add
- page_count
- page_delete
- extract_text
- set_license

### Организация

- optimize
- optimize_resource
- grayscale
- rotate
- set_background
- repair
- page_add_text
- page_set_size
- page_rotate
- page_grayscale

### Преобразование и сохранение

- PDF, DOC, DOCX, XLSX, PPTX, JPEG, PNG, BMP, TIFF, EPUB, DICOM, SVG, XPS, TEX, TXT
