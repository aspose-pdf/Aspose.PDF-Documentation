Aspose.PDF for Rust via C++ предоставляет встроенный функционал для программного изменения PDF‑документов. В этом примере приложение открывает существующий PDF‑файл, применяет автоматическую нумерацию страниц ко всем страницам и сохраняет изменённый документ под новым именем.

Этот подход полезен при создании окончательных документов для распространения, печати или архивных целей. Процесс требует всего несколько строк кода и не изменяет исходный файл, если только его явно не перезаписать.

Нумерация страниц является распространённым требованием для отчетов, счетов‑фактур, контрактов, руководств и других многостраничных документов. The [add_page_num()](https://reference.aspose.com/pdf/rust-cpp/organize/add_page_num/) метод автоматически вставляет номера страниц во все страницы документа, обеспечивая согласованную пагинацию по всему файлу.

После добавления номеров страниц обновлённый документ сохраняется как новый PDF‑файл.

1. Откройте существующий PDF‑документ.
1. Добавьте нумерацию страниц с [add_page_num()](https://reference.aspose.com/pdf/rust-cpp/organize/add_page_num/) метод.
1. Сохраните обновлённый документ.

```rs

    use asposepdf::Document;

    fn main() -> Result<(), Box<dyn std::error::Error>> {
        // Open a PDF-document with filename
        let pdf = Document::open("sample.pdf")?;

        // Add page number to a PDF-document
        pdf.add_page_num()?;

        // Save the previously opened PDF-document with new filename
        pdf.save_as("sample_add_page_num.pdf")?;

        Ok(())
    }
```
