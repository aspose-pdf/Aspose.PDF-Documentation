PDF был разработан как стандарт для представления документов и других справочных материалов в формате, независимом от прикладного программного обеспечения, аппаратного обеспечения и операционной системы. Содержание файлов PDF не ограничивается только текстом; он может включать гиперссылки, изображения, интерактивные кнопки и формы, электронные подписи, водяные знаки и многое другое. Поэтому часто необходимо конвертировать файлы PDF в другой формат для редактирования или изменения их содержимого.

Полный список поддерживаемых форматов смотрите в разделе [Поддерживаемые форматы файлов Aspose.PDF](https://docs.aspose.com/pdf/rust-cpp/supported-file-formats/).

**Aspose.PDF for Rust via C++** позволяет конвертировать PDF‑документы в различные форматы. Кроме того, вы можете проверить качество конвертации Aspose.PDF и просмотреть результаты онлайн с помощью приложения Aspose.PDF converter. Ознакомьтесь с разделами конвертации документов с примерами кода.

## Как использовать Rust для конвертации

Документы Word являются самыми универсальными и редактируемыми. Конвертация PDF в DOC вручную — очень трудоемкая задача. В этой статье вы узнаете, как конвертировать PDF в DOC и DOCX с помощью Rust.

- [Конвертировать PDF в DOC](/pdf/ru/rust-cpp/convert-pdf-to-doc/) - вы можете конвертировать ваш PDF‑документ в формат Word с помощью Rust.

Форматы чисел нужны не только для того, чтобы данные в таблице было легче читать, но и чтобы таблицу было проще использовать. Конечно, если вам необходимо конвертировать такие данные из PDF‑документа в формат Excel, используйте наш Aspose.PDF for Rust.

- [Конвертировать PDF в Microsoft XLSX](/pdf/ru/rust-cpp/convert-pdf-to-xlsx/) - этот раздел описывает, как конвертировать PDF‑документ в XLSX и CSV.

Формат PowerPoint используется для создания различных презентаций. Файлы PPT содержат большое количество слайдов или страниц с различной информацией.

- [Конвертировать PDF в Microsoft PowerPoint](/pdf/ru/rust-cpp/convert-pdf-to-powerpoint/) - здесь речь идет о конвертации PDF в PowerPoint с отслеживанием процесса преобразования

HyperText Markup Language — это язык описания гипертекстовых документов, стандартный язык для создания веб-страниц. С помощью Aspose.PDF for Rust вы можете легко конвертировать HTML‑документы и наоборот.

Существует множество форматов изображений, которые необходимо конвертировать в PDF для различных целей. Aspose.PDF for Rust поддерживает наиболее популярные форматы изображений.

- [Конвертировать PDF в различные форматы изображений](/pdf/ru/rust-cpp/convert-pdf-to-images-format/) - конвертировать страницы PDF в изображения в форматах JPEG, PNG, SVG и других форматах

Этот раздел включает такие форматы, как: EPUB, XPS, TeX, Text и Grayscale PDF.

- [Конвертировать PDF‑файл в другие форматы](/pdf/ru/rust-cpp/convert-pdf-to-other-files/) - эта тема описывает способ конвертации PDF‑документа в различные форматы
