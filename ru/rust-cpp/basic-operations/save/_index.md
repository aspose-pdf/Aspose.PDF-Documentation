## Сохранить PDF-документ в файловой системе

Пример демонстрирует [новый](https://reference.aspose.com/pdf/rust-cpp/core/new/) метод создания нового PDF‑документа, который является фундаментальной функцией для динамического формирования документов, таких как отчёты или счета‑фактуры.

Код простой и может служить шаблоном для более продвинутых функций, таких как добавление текста, изображений или аннотаций в PDF.

Этот пример демонстрирует простое использование библиотеки Aspose.PDF Rust, показывая её потенциал для создания, изменения и сохранения документов.

```rs

    use asposepdf::Document;

    fn main() -> Result<(), Box<dyn std::error::Error>> {
        // Create a new PDF-document
        let pdf = Document::new()?;

        // Save the previously opened PDF-document with new filename
        pdf.save_as("sample_new.pdf")?;

        Ok(())
    }
```
