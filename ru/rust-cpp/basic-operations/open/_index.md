## Открыть существующий PDF-документ

Этот фрагмент кода демонстрирует основные операции по работе с PDF с использованием Aspose.PDF for Rust. Это открытие файла, сохранение изменений и правильное закрытие ресурсов. Это делает его базовым примером для разработчиков, работающих с PDF‑документами.

Пример прост и понятен, его легко понять и изменить. Он идеален для начинающих или в качестве шаблона для более сложных приложений.

Возможность открывать и сохранять PDF‑документы является основной требованием во многих сценариях, например генерации отчётов, изменения документов или создания автоматизированных рабочих процессов.

Этот пример демонстрирует простоту использования API для простой работы с PDF, которую можно расширить, включив продвинутые функции, такие как извлечение текста, аннотации или заполнение форм.

```rs

    use asposepdf::Document;

    fn main() -> Result<(), Box<dyn std::error::Error>> {
        // Open a PDF-document named "sample.pdf"
        let pdf = Document::open("sample.pdf")?;

        // Save the previously opened PDF-document with new filename
        pdf.save_as("sample_open.pdf")?;

        Ok(())
    }
```
