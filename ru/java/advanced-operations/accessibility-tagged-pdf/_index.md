Тегированный PDF документ может быть доступен для людей с ограниченными возможностями. Добавление тегов доступности в PDF файлы облегчает чтение и навигацию по документу для экранных читалок и других вспомогательных технологий, с оглавлениями, гиперссылками, закладками, альтернативным текстом и так далее.

Aspose.PDF предоставляет функциональность для создания PDF документов, которые совместимы со стандартом PDF/UA (также известным как "Раздел 508" или "Стандарт WCAG"), а также для проверки совместимости с этими стандартами.
 These standards describe requirements for document contents to make it more friendly and useful for people with disabilities and also for users with highly limited devices. When you make your PDF documents accessible or create Tagged PDFs, it means adding tags, bookmarks, alt text, and other information that makes the files readable and navigable to users who are visually or mobility impaired. Using Aspose.PDF, it is now achievable to create valid, accessible PDF documents. Following section shows how to add tagged content to create Tagged PDF.

Эти стандарты описывают требования к содержимому документов, чтобы сделать их более дружественными и полезными для людей с ограниченными возможностями, а также для пользователей с сильно ограниченными устройствами. Когда вы делаете ваши PDF-документы доступными или создаете тегированные PDF-файлы, это означает добавление тегов, закладок, альтернативного текста и другой информации, которая делает файлы читаемыми и удобными для навигации для пользователей с нарушениями зрения или подвижности. С использованием Aspose.PDF теперь возможно создавать действительные, доступные PDF-документы. Следующий раздел показывает, как добавить тегированное содержимое для создания тегированного PDF.
