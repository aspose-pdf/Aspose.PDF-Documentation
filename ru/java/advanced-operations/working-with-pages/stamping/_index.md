Штамп в PDF-документе аналогичен нанесению резинового штампа на бумажный документ. Штамп в PDF-файле предоставляет дополнительную информацию для PDF-файла, такую как защита PDF-файла от использования другими и подтверждение безопасности содержимого PDF-файла. **Aspose.PDF for Java** позволяет добавлять изображения или текстовые штампы в ваш PDF-документ.

Изучите следующие разделы, чтобы узнать, как добавить штамп с помощью Java:

- [Добавить штампы изображений на страницу PDF](/pdf/ru/java/image-stamps-in-pdf-page/) - добавить штамп изображения, контролировать качество изображения, штамп изображения как фон вашего PDF-файла.
- [Добавить текстовые штампы в PDF-файл](/pdf/ru/java/text-stamps-in-the-pdf-file/) - добавить текстовый штамп, определить выравнивание для объекта TextStamp, заполнить текст штампа в PDF.
- [Добавить штампы страниц в PDF](/pdf/ru/java/page-stamps-in-the-pdf-file/) - добавить сложный штамп на основе страницы из другого PDF.
