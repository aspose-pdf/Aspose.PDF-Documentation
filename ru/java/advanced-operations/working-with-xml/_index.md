XML (Extensible Markup Language) — это способ повторного использования данных в файле или автоматизации процесса замены данных в одном файле данными из другого файла. Создание XML-схемы сложно выполнить правильно; создание нескольких взаимосвязанных схем, которые можно расширять, еще сложнее. Давайте узнаем, как Aspose.PDF справляется с задачами работы с XML.

- [Поддерживаемая XML-схема](/pdf/ru/java/supported-xml-schema/) - используйте следующую XML-схему для работы с XML-документами.
- [Создание документа PDF "Hello World" через XML и XSLT](/pdf/ru/java/create-a-hello-world-pdf-document-through-xml-and-xslt/) - используйте XSLT для преобразования вашего существующего XML-документа в PDF.

- [Создание PDF из XML](/pdf/ru/java/generate-pdf-from-xml) - Aspose.PDF предлагает несколько способов создания PDF на основе XML-документа.
