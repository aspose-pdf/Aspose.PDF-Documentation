# Введение в Aspose.PDF Java для Python

## JPype

JPype — это попытка предоставить программам на Python полный доступ к библиотекам классов Java. Это достигается не через повторную реализацию Python, как это сделано в Jython/JPython, а через интерфейс на родном уровне в обеих виртуальных машинах.

Читать далее на [readthedocs.org](http://jpype.readthedocs.org/en/latest/userguide.html)

## Aspose.PDF для Java

Aspose.PDF для Java — это продвинутая библиотека классов для Java, которая позволяет выполнять широкий спектр задач по обработке документов непосредственно в ваших Java приложениях.

Aspose.PDF для Java поддерживает обработку текстовых документов (DOC, DOCX, OOXML, RTF), HTML, OpenDocument, PDF, EPUB, XPS, SWF и всех форматов изображений. С помощью Aspose.PDF вы можете генерировать, изменять и конвертировать документы без использования Microsoft Word.

Читать далее на [aspose.com](https://products.aspose.com/words/java)

## Aspose.PDF Java для Python

Проект Aspose.PDF для Python показывает, как различные задачи могут быть выполнены с использованием Aspose.PDF Java API в Python. Этот проект направлен на предоставление полезных примеров для разработчиков на Python, которые хотят использовать Aspose.PDF для Java в своих проектах на Python с использованием [JPype](http://jpype.readthedocs.org/en/latest/userguide.html).

Ниже приведены системные требования для использования Aspose.PDF Java для Python:
