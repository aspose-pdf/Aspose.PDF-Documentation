Чтобы преобразовать PDF-документ в формат DOC или DOCX с использованием **Aspose.PDF Java for Python**, просто вызовите модуль **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "Документ был успешно преобразован"
```
