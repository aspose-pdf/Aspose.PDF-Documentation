Чтобы получить количество страниц в Pdf документе, используя **Aspose.PDF Java для Python**, просто вызовите класс **GetNumberOfPages**.

```Python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "Количество страниц:" . page_count

```
