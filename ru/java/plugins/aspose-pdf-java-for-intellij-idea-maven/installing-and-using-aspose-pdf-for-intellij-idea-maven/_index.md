## Installing

### How to Install Aspose.PDF Java for IntelliJ IDEA (Maven)

Steps below demonstrates how to install **Aspose.PDF Java for IntelliJ IDEA (Maven)** plugin.

Шаги ниже показывают, как установить плагин **Aspose.PDF Java for IntelliJ IDEA (Maven)**.

The  plugin is listed on IntelliJ IDEA's CE [official plugin website](https://plugins.jetbrains.com/plugin/7974-aspose-pdf-java-for-intellij-idea-maven/) - plugin repository as **Aspose.PDF Java for IntelliJ IDEA Maven**, so you don’t need to download it manually and install it from disk, but this extra option still exists in IntelliJ IDEA.

Плагин указан на [официальном сайте плагинов](https://plugins.jetbrains.com/plugin/7974-aspose-pdf-java-for-intellij-idea-maven/) IntelliJ IDEA CE - репозиторий плагинов как **Aspose.PDF Java for IntelliJ IDEA Maven**, поэтому вам не нужно загружать его вручную и устанавливать с диска, но эта дополнительная опция все еще существует в IntelliJ IDEA.

**Aspose.PDF Java for IntelliJ IDEA (Maven)** plugin can be easily installed (without manual downloading) from within IntelliJ IDEA. To do it:

Плагин **Aspose.PDF Java for IntelliJ IDEA (Maven)** может быть легко установлен (без ручной загрузки) из IntelliJ IDEA. Для этого:

1. Select **Settings** from the **File** menu.

1. Выберите **Settings** из меню **File**.

2. Click **Plugins**, then browse the repositories in IntelliJ IDEA.

2. Нажмите **Plugins**, затем просмотрите репозитории в IntelliJ IDEA.

## Использование

### Мастер проекта Aspose.PDF Maven

Чтобы создать проект Aspose.PDF Maven для использования [Aspose.PDF для Java API](https://products.aspose.com/pdf/java/) в IntelliJ IDEA, вам необходимо подключение к интернету.

Затем выполните следующие шаги:

1. Выберите **New Project**.
2. Выберите **Aspose.PDF Maven Project**.
3. Нажмите **Next**.

![todo:image_alt_text](https://i.imgur.com/hd2mYnW.jpg)


4. Укажите **GroupId, ArtifactId** и Версию для вашего maven проекта и нажмите Далее.

![todo:image_alt_text](https://i.imgur.com/aNlx0Qe.jpg)

5. Вы можете выбрать флажок "Также скачать исходный код примеров", чтобы загрузить примеры использования API (как показано на приведенном выше снимке экрана).

6. Укажите **Имя проекта** и **Расположение** так же, как при создании любого Java приложения.

![todo:image_alt_text](https://i.imgur.com/8VEeFun.jpg)

7. Нажмите **Готово**.

Это извлечет последнюю ссылку на maven зависимость [Aspose.PDF for Java API](https://products.aspose.com/pdf/java/) из Aspose Cloud Maven Repository и настроит её в pom.xml. Если вы выбрали флажок "Также скачать исходный код примеров", начнется загрузка примеров из [Aspose.PDF for Java API GitHub Repository](https://goo.gl/cPAbJM).

![todo:image_alt_text](https://i.imgur.com/bG2mdam.jpg)

![todo:image_alt_text](https://i.imgur.com/aNYhLl5.jpg)

Поздравляем!
 Вы успешно создали проект Aspose.PDF Maven для использования [Aspose.PDF for Java API](https://products.aspose.com/pdf/java/).

![todo:image_alt_text](https://i.imgur.com/UJlSbuM.jpg)

Созданный проект Aspose.PDF for Java API Maven готов к модификации в соответствии с требованиями вашего проекта.

Если вы выбрали загрузку [Примеров Исходного Кода](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples).

### Мастер Примеров Aspose.PDF

Чтобы импортировать/скопировать [Примеры Исходного Кода](https://goo.gl/cPAbJM) в проект с помощью **Мастера Примеров Aspose.PDF**, вам сначала нужно загрузить примеры, выбрав "Также Загрузить Исходный Код Примеров" в Мастере Проекта Aspose.PDF Maven.

Затем выполните следующие шаги в IntelliJ IDEA:

1. Выберите **Файл** > **Новый -** Как показано ниже:

![todo:image_alt_text](https://i.imgur.com/N8tT9Q0.jpg)

-ИЛИ- **Правая кнопка мыши** (в окне проекта) => **Новый**  - Как показано ниже:

![todo:image_alt_text](https://i.imgur.com/aUBWkhp.jpg)
2. Выберите пункт **Пример Aspose.PDF** из меню - как показано ниже:

![todo:image_alt_text](https://i.imgur.com/hvH2V0m.jpg)

Мастер обновит / проверит наличие последних доступных [Примеры Aspose.PDF для Java API](https://goo.gl/cPAbJM)

![todo:image_alt_text](https://i.imgur.com/5PZwsuq.jpg)

3. Появится диалоговое окно **Мастер примеров Aspose.PDF**, где вы можете выбрать любую категорию примеров, чтобы создать / скопировать примеры в ваш проект. Выберите любую категорию примеров и нажмите **Создать**:

![todo:image_alt_text](https://i.imgur.com/l2oGUBW.jpg)

Это создаст / скопирует выбранные примеры категории в проект в пакет **com.aspose.pdf.examples**.
Также все необходимые ресурсы для выполнения примеров будут скопированы в папку **src/main/resources** - как показано ниже:

![todo:image_alt_text](https://i.imgur.com/8GQL3Yb.jpg)

Поздравляем! Вы успешно выполнили следующие действия:

1. Вы создали Maven-проект Aspose.PDF, используя Мастер проектов Maven Aspose.PDF,

2. Вы загрузили [Aspose.PDF для Java API - Примеры исходного кода](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)

3. Вы создали / скопировали исходные коды примеров в ваш проект, используя мастер примеров Aspose.PDF.
