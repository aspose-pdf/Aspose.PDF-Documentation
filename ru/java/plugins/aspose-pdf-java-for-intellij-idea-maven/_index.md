## Введение

### IntelliJ IDEA

IntelliJ IDEA - это интегрированная среда разработки (IDE) для Java для разработчиков программного обеспечения. Она разработана компанией JetBrains. Изначально созданная для Java, в настоящее время поддерживает широкий спектр языков, включая JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure и Kotlin.

В отчете Infoworld за 2010 год IntelliJ получила наивысшую оценку тестового центра из четырех лучших инструментов программирования на Java: Eclipse, IntelliJ IDEA, NetBeans и Oracle JDeveloper.

### Aspose.PDF для Java

Aspose.PDF для Java - это компонент для создания PDF-документов, который позволяет вашим Java-приложениям читать, записывать и обрабатывать PDF-документы без использования Adobe Acrobat.

### Aspose.PDF Java для IntelliJ IDEA - Maven

- Aspose.PDF Java для IntelliJ IDEA (Maven) - это плагин для IntelliJ IDEA, который позволяет создавать проекты maven (т.е.
 созданный проект содержит зависимость maven для Aspose.PDF для Java API) и скачивает примеры исходного кода API для использования Aspose.PDF для Java API.
- Этот плагин помогает вам работать с Aspose.PDF для Java API в вашей любимой IntelliJ IDEA.

МАСТЕРА:
Плагин содержит два мастера

Мастер 1: Aspose.PDF Maven Project

- Мастер Aspose Maven Project плагина позволяет разработчикам создавать Maven проект для Aspose.PDF для Java API.
- Ссылка на зависимость maven для Aspose.PDF для Java API автоматически извлекается из [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) и добавляется в pom.xml.
- Созданный проект всегда будет содержать самую последнюю доступную версию зависимости Maven для Aspose.PDF для Java API.
- Шаги мастера также предоставляют возможность загрузки [Примеров Исходного Кода для использования Aspose.PDF для Java API](https://github.com/aspose-pdf/Aspose.PDF-for-Java)

Мастер 2: Примеры Aspose.PDF
- Мастер примеров Aspose.PDF позволяет создавать/копировать загруженные примеры исходного кода Aspose.PDF для Java API в ваш проект. Все примеры для выбранной категории будут скопированы в папку пакета проекта "com.aspose.pdf.examples" вместе с необходимыми ресурсами в папке "src/main/resources", которые нужны для запуска примеров.
- Примеры исходного кода Aspose.PDF для Java API предназначены для демонстрации различных способов использования API.

## Системные требования и поддерживаемые платформы

### Системные требования

- Microsoft Windows 8/7/Vista/2003/XP (включая 64-разрядные) / Linux GNOME или KDE / Mac OS X 10.5 или выше
- Минимум 1 ГБ ОЗУ, рекомендуется 2 ГБ ОЗУ
- 300 МБ места на жестком диске + минимум 1 ГБ для кэшей
- Минимальное разрешение экрана 1024x768
- Рекомендуется JDK 1.8 или выше
- Интернет-соединение (рекомендуется 2 МБ или быстрее)

### Поддерживаемые платформы

Плагин поддерживает следующие версии IntelliJ IDEA:

- 13.0.1 (Community / Ultimate Edition)
- 13.1.2 (CE / UE)
- 14.0.2 (CE / UE)

- 14.1.5 (CE/ UE)

## Загрузка

### Загрузка IntelliJ IDEA

Сначала необходимо установить IntelliJ IDEA CE / UE перед загрузкой плагина Aspose.PDF Java для IntelliJ IDEA (Maven).

Чтобы скачать IntelliJ IDEA

1. Перейдите на <https://www.jetbrains.com/idea/download/>
1. Выберите вашу операционную систему
1. Нажмите на загрузку UE или CE.

### Загрузка плагина Aspose.PDF Java для IntelliJ IDEA (Maven)

- Плагин можно загрузить напрямую с [URL хостинга плагинов JetBrains](https://goo.gl/z06gC0) - на репозитории плагинов IntelliJ IDEA
  и позже установить с помощью опции IDE:
  - Перейдите в Файл > Настройки...>Плагины
  - Выберите "Установить плагин с диска"
- Или он может быть установлен напрямую (без ручной загрузки с URL хостинга плагинов JetBrains) с использованием опций IDE:
  - Перейдите в Файл > Настройки...>Плагины
  - Выберите "Обзор репозиториев"
  - В поисковую строку введите: "Aspose"
  - Выберите плагин "Aspose.PDF Java for IntelliJ IDEA Maven" для установки.
