## Введение

### Eclipse IDE

Eclipse IDE — это известная интегрированная среда разработки на Java (IDE). IDE, безусловно, является самым известным продуктом проекта Eclipse Open Source. Сегодня это ведущая среда разработки для Java с долей рынка около 60%.

Среду Eclipse IDE можно расширять с помощью дополнительных программных компонентов. Eclipse называет эти программные компоненты Плагинами. Несколько проектов с открытым исходным кодом и компаний расширили Eclipse IDE или создали автономные приложения (Eclipse RCP) на основе фреймворка Eclipse.

### Aspose.PDF для Java

[Aspose.PDF для Java](https://products.aspose.com/pdf/java/) — это мощный API для создания PDF-документов, который позволяет вашим Java-приложениям читать, записывать и обрабатывать PDF-документы без использования Adobe Acrobat.

Aspose.PDF для Java предлагает невероятное множество функций, включая параметры сжатия PDF, создание и обработку таблиц, поддержку графиков, функции работы с изображениями, расширенные функции гиперссылок, расширенные средства управления безопасностью и обработку пользовательских шрифтов.

### Aspose.PDF Java (Maven) для Eclipse

- Aspose.PDF Java (Maven) для Eclipse — это плагин для **Eclipse IDE** от **Aspose.** Этот плагин предназначен для разработчиков, использующих платформу Maven для разработки на Java и желающих использовать Aspose.PDF для Java в своих проектах. Плагин позволяет создавать maven-проекты для использования Aspose.PDF для Java API, а также загружать [Примеры кода](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) API.

**Aspose.PDF Maven Project (мастер)**

- Этот мастер нового проекта позволяет разработчикам создавать **Maven** проект для использования Aspose.PDF для Java через New -> Project -> Maven-> Aspose.PDF Maven Project.

- Ссылка на зависимость maven Aspose.PDF для Java API автоматически загружается из [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) и добавляется в pom.xml.
- Созданный проект всегда будет содержать самую последнюю доступную версию зависимости **Maven** для Aspose.PDF для Java API.
- Шаги мастера также предлагают возможность загрузки [Примеры кода](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) для использования Aspose.PDF для Java API.
Пример кода Aspose.PDF (мастер)

- Этот мастер создания нового файла позволяет скопировать загруженные [Примеры кода](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) в ваш проект для использования Aspose.PDF для Java из New -> Other -> Java -> Aspose.PDF Code Example.
- Доступные образцы отображаются в формате дерева, из которого пользователь может выбирать их по категориям.
- Все примеры в выбранной категории будут скопированы в папку пакета проекта "**com.aspose.pdf.examples**" вместе с необходимыми ресурсами в папке "**src/main/resources**", необходимыми для запуска примеров.
- Примеры кода Aspose.PDF для Java API предназначены для демонстрации различных функций API.

- Мастер также будет искать и обновлять недавно доступные Примеры кода) из репозитория примеров Aspose.PDF для Java.

## Системные требования и поддерживаемые платформы

### Системные требования

- **Оперативная память системы:** 2 ГБ или больше (рекомендуется)
- **ОС:** Любая операционная система, поддерживающая Java VM (виртуальная машина)
- **Интернет-соединение:** 2 МБ или быстрее (рекомендуется)

### Поддерживаемые платформы

- Eclipse Mars.1 (4.5.1) - рекомендуется
- Eclipse Juno или более поздняя версия.

## Загрузка

### Загрузка Eclipse IDE

Сначала необходимо установить Eclipse IDE, прежде чем загружать Aspose.PDF Java (Maven) для плагина Eclipse.

Чтобы загрузить Eclipse IDE

1. Перейдите на [https://eclipse.org](https://eclipse.org/).
1. Загрузите и установите рекомендованную Eclipse IDE для разработчиков Java SE / EE.

### Загрузка Aspose.PDF Java (Maven) для Eclipse

Ниже приведены три рекомендованных метода для успешной загрузки и установки Aspose.PDF Java (Maven) для плагина Eclipse:

- Установка методом перетаскивания из [Eclipse Marketplace](https://marketplace.eclipse.org/content/asposepdf-java-maven-eclipse) в ваше рабочее пространство Eclipse.

- Или перейдите в **Помощь** > **Установить новое программное обеспечение...** > Введите следующий URL обновления в **Работа с**
Затем выберите "Aspose.PDF Java (Maven) for Eclipse" и нажмите **Finish**. Примите Лицензионное соглашение и установите плагин.

### Как применить лицензию Aspose?

Этот плагин использует пробную версию Aspose.PDF. Когда вы будете довольны своей оценкой, вы можете приобрести лицензию на [веб-сайте Aspose](https://purchase.aspose.com/buy).
Чтобы удалить сообщение об оценке и ограничения функций, необходимо применить лицензию продукта. Вы получите файл лицензии после покупки продукта. Пожалуйста, выполните следующие шаги для применения лицензии:

- Убедитесь, что файл лицензии называется Aspose.PDF.Java.lic
- Поместите файл **Aspose.PDF.Java.lic** в папку, содержащую Aspose.PDF.jar
- Используйте следующий код для активации лицензии:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.PDF.Java.lic");

### Как настроить исходный код Aspose.PDF Java (Maven) для Eclipse

Ниже приведены простые шаги, которые плавно приведут к успешной настройке исходного кода плагина **"Aspose.PDF Java (Maven) для Eclipse"** в среде разработки Eclipse IDE.

1. Загрузите / Клонируйте исходный код.
1. Выберите **Файл** > Импортировать > Общие > Существующие проекты в рабочее пространство
1. Перейдите к последнему загруженному проекту
1. Выберите проект Eclipse, который вы хотите импортировать
1. Нажмите Готово
1. Плагин Aspose.PDF Java для Eclipse теперь готов к улучшению.
