## Введение

### NetBeans IDE

NetBeans — это полнофункциональная Java IDE, полностью написанная на Java. Она в первую очередь предназначена для разработки на Java, но также поддерживает другие языки, в частности, PHP, C/C++ и HTML5.

NetBeans IDE позволяет быстро и легко разрабатывать Java настольные, мобильные и веб-приложения, а также HTML5 приложения с использованием HTML, JavaScript и CSS. Это бесплатное и с открытым исходным кодом программное обеспечение, которое имеет большую аудиторию пользователей и разработчиков по всему миру.

Команда NetBeans активно поддерживает продукт и принимает предложения по функциям от широкой общественности. Каждому выпуску предшествует время для тестирования и обратной связи от сообщества.

### Aspose.PDF для Java

[Aspose.PDF для Java](https://products.aspose.com/pdf/java/) — это надежный API для создания PDF-документов, который позволяет вашим Java-программам читать, записывать и обрабатывать PDF-документы без использования Adobe Acrobat.

Aspose.PDF для Java предлагает невероятное множество функций, включая варианты сжатия PDF, создание и обработку таблиц, поддержку графиков, функции обработки изображений, обширную функциональность гиперссылок, расширенные средства безопасности и работу с пользовательскими шрифтами.

### Aspose.PDF Java для NetBeans (Maven)

- Aspose.PDF Java для NetBeans (Maven) — это плагин для **NetBeans IDE** от **Aspose.** Плагин предназначен для разработчиков, использующих платформу Maven для разработки на Java, и желающих использовать Aspose.PDF для Java в своих проектах. Плагин позволяет создавать maven-проекты для использования Aspose.PDF для Java API, а также загружать [Примеры Кода](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) API.

## МАСТЕРА

Плагин содержит два мастера

### Aspose.PDF Maven Project (мастер)

- Этот мастер нового проекта позволяет разработчикам создавать **Maven** проект для использования Aspose.PDF для Java через Новый проект -> Maven -> Aspose.PDF Maven Project.
- Ссылка на зависимость Aspose.PDF для Java API maven автоматически извлекается из [Aspose Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf) и добавляется в pom.xml.
- Созданный проект всегда будет содержать самую последнюю доступную версию зависимости **Maven** для Aspose.PDF для Java API.
- Шаги мастера также включают опцию для загрузки примеров кода для использования Aspose.PDF для Java API.

### Пример кода Aspose.PDF (мастер)

- Этот мастер создания нового файла позволяет вам скопировать загруженные [примеры кода](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) в ваш проект для использования Aspose.PDF для Java из **Новый файл -> Java -> Пример кода Aspose.PDF.**
- Доступные образцы отображаются в формате дерева, откуда пользователь может выбирать их по категориям.

- Все примеры в выбранной категории будут скопированы в папку пакета проекта **com.aspose.pdf.examples** вместе с необходимыми ресурсами в папке **src/main/resources**, которые нужны для запуска примеров.
- Примеры кода Aspose.PDF для Java API предназначены для демонстрации разнообразия API.
- Мастер также будет искать и обновлять вновь доступные примеры из [репозитория примеров Aspose.PDF для Java.](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)

## Системные требования и поддерживаемые платформы

### Системные требования

- **Системная память:** 2 ГБ или больше (рекомендуется)
- **ОС:** Любая операционная система, поддерживающая Java VM (виртуальная машина)
- **Интернет-соединение:** 2 МБ или быстрее (рекомендуется)

### Поддерживаемые платформы

- NetBeans 8.1 (рекомендуется)
- NetBeans 8.0

## Загрузка

### Загрузка NetBeans

У вас должна быть предварительно установлена IDE NetBeans перед загрузкой **плагина Aspose.PDF Java для NetBeans (Maven)**.

Плагин **Aspose.PDF Java для NetBeans (Maven)** поддерживает **NetBeans IDE8.0 и 8.1**. Мы рекомендуем использовать только последнюю версию NetBeans IDE.

Чтобы скачать NetBeans IDE,

1. Перейдите по ссылке: <https://netbeans.apache.org/download/index.html>

1. Нажмите Загрузить, чтобы скачать соответствующий пакет NetBeans IDE.

Скачайте только поддерживаемый Java EE / SE пакет NetBeans последнего текущего релиза, т.е. **NetBeans IDE 8.1**.

### Скачать Aspose.PDF Java для NetBeans (Maven)

- Плагин можно напрямую скачать с URL размещения плагинов NetBeans: <http://plugins.netbeans.org/plugin/63218> и позже установить с помощью опции IDE:
  - Перейдите в **Инструменты**>**Плагины**>**Загруженные**
  - Нажмите **Добавить плагины...** и найдите скачанный файл плагина с расширением .nbm для **Aspose.PDF Java для NetBeans (Maven)**.
  - Нажмите **"Установить"**, чтобы установить плагин
- Или его можно напрямую установить (без ручной загрузки с URL размещения плагинов NetBeans) с помощью опций IDE:
  - Перейдите в **Инструменты**>**Плагины**>**Доступные плагины**
  - В поле поиска введите: "Aspose"
  - Выберите плагин **Aspose.PDF Java NetBeans (Maven)**
  - Нажмите **"Установить"**

### Как применить лицензию Aspose?

Этот плагин использует пробную версию Aspose.PDF. После того как вы оцените продукт, вы можете приобрести лицензию на [веб-сайте Aspose](https://purchase.aspose.com/buy). Чтобы убрать сообщение об оценке и ограничения на функции, необходимо применить лицензию на продукт. Вы получите лицензионный файл после покупки продукта. Пожалуйста, следуйте шагам ниже, чтобы применить лицензию.

- Убедитесь, что файл лицензии назван как **Aspose.PDF.Java.lic
- Поместите файл **Aspose.PDF.Java.lic** в папку, содержащую Aspose.PDF.jar
- Используйте следующий код для активации лицензии:

```java
License license = new License();
license.setLicense("Aspose.PDF.Java.lic");
```

### #Как настроить исходный код Aspose.PDF Java для NetBeans (Maven)

Ниже приведены простые шаги, которые плавно приведут к успешной настройке исходного кода плагина **"Aspose.PDF Java for NetBeans (Maven)"** в NetBeans IDE
