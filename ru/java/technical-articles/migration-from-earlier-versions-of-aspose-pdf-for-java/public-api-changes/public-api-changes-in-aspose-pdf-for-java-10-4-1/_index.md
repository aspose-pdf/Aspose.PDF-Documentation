{{% alert color="primary" %}}

На этой странице перечислены изменения публичного API, введенные в [Aspose.PDF для Java 10.4.1](http://www.aspose.com/community/files/72/java-components/aspose.pdf-for-java/entry641525.aspx). Она включает не только новые и устаревшие публичные методы, но и описание любых изменений в поведении за кулисами в Aspose.PDF для Java, которые могут повлиять на существующий код. Любое поведение, введенное как регрессия и изменяющее существующее поведение, особенно важно и документируется здесь.

{{% /alert %}}

**Изменения публичного API и несовместимые изменения**

Изменения в классе **com.aspose.pdf.IconFit**:

- public boolean getSpreadOnBorder() -> public boolean isSpreadOnBorder() 

Изменения в классе **com.aspose.pdf.Measure**:

- public boolean getForceDenominator() -> public boolean isForceDenominator()

Изменения в классе **com.aspose.pdf.RedactionAnnotation**:

- public boolean getRepeat() -> public boolean isRepeat()

Изменения в классе **com.aspose.pdf.SaveOptions**:

- public boolean getCloseResponse() -> public boolean isCloseResponse()

Изменения в классе **com.aspose.pdf.Signature**:

- public boolean getShowProperties() -> public boolean isShowProperties()

Изменения в классе **com.aspose.pdf.TextParagraph**:

- public boolean getJustify() -> public boolean isJustify()

Изменения в классе **com.aspose.pdf.TextStamp**:

- public boolean getWordWrap() -> public boolean isWordWrap()
- public boolean getJustify() -> public boolean isJustify()
- public boolean getScale() -> public boolean isScale()

Изменения в классе **com.aspose.pdf.TextState**:

- public boolean getUnderline() -> public boolean isUnderline()
