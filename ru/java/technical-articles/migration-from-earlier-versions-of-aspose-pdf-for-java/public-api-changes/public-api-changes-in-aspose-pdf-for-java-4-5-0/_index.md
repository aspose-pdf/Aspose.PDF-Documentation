{{% alert color="primary" %}}

На этой странице перечислены изменения в публичном API, которые были введены в Aspose.PDF для Java 4.5.0. Это включает не только новые и устаревшие публичные методы, но и описание любых изменений в поведении за кулисами в Aspose.PDF для Java, которые могут повлиять на существующий код. Любое поведение, введенное, которое может быть воспринято как регрессия и изменяет существующее поведение, особенно важно и документируется здесь.

{{% /alert %}}

Следующие классы были перемещены из пакета **com.aspose.pdf** в пакет **com.aspose.pdf.devices**.

- BmpDevice
- ColorDepth
- CompressionType
- Device
- DocumentDevice
- EmfDevice
- FormPresentationMode
- GifDevice
- GraphicsDevice
- ImageDevice
- ImageGenerator
- JpegDevice
- Margins
- PageDevice
- PngDevice
- Resolution
- ShapeType
- TextDevice
- TiffDevice
- TiffSettings
