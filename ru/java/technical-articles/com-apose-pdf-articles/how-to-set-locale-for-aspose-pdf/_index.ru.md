Библиотека Aspose.PDF for Java не поддерживает локали с определенной комбинацией язык-страна (например, "en-KR"). Однако функция установки классической локали для Aspose.PDF присутствует в API и может быть использована путем вызова метода com.aspose.pdf.LocaleOptions.setLocale().
