Инструмент Configuring Tool для Aspose.Pdf for Reporting Services может помочь вам настроить расширение Aspose.Pdf for Reporting Services для любой из поддерживаемых версий Report Server (RS). В настоящее время он поддерживает RS2016, RS2017, RS2019, RS2022 и Power BI Report Server. Инструмент Configuring Tool требует .NET Framework 4.8.

Если вы хотите установить расширение и зарегистрировать его на Report Server, выберите тип действия 'Register'. Для отмены регистрации и удаления расширения выберите тип действия 'Unregister'.

![todo:image_alt_text](install-with-configuring-tool_1.png)

**Следующие шаги описывают, как использовать его подробно:**

1. Введите или найдите путь к файлу DLL для расширения Aspose.Pdf for Reporting Services;
1. Выберите соответствующий тип действия: Register или Unregister;
1. Выберите вкладку, соответствующую версии Report Server, которую вы хотите настроить. Убедитесь, что вы выбрали файл DLL, предназначенный для вашей версии RS. Если требуемая версия продукта не установлена на машине, инструмент настройки сообщит вам об этом с подсказками. Если вы настраиваете расширение для указанного экземпляра RS2016 (не для экземпляра по умолчанию 'MSSQLSERVER'), введите пользовательское имя экземпляра, затем нажмите кнопку 'Refresh'.
1. Убедитесь, что пути и имена файлов конфигурации, отображаемые в нижних текстовых полях, правильные. Если они неверны, вы можете нажать кнопку 'Refresh', чтобы попытаться снова найти экземпляр RS, или можете найти их вручную.
1. Нажмите кнопку 'Config'. Инструмент теперь попытается выполнить запрошенную настройку и сообщит вам, успешна ли она.
