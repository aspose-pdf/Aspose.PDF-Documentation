{{% alert color="primary" %}}

Вы можете добавить пользовательские свойства для некоторых элементов отчета, чтобы расширить их использование, таких как оглавление, стрелки линий и так далее. В этом разделе описывается этот процесс.

Чтобы добавить пользовательские свойства, вам нужно отредактировать файл кода документа RDL следующими шагами:

1. Как показано на следующем рисунке, откройте ваш проект, перейдите к обозревателю решений и щелкните правой кнопкой мыши по выбранному файлу отчета, затем выберите пункт меню 'Просмотр кода'.

![todo:image_alt_text](adding-custom-properties_1.png)

2. Отредактируйте файл кода XML. Например, если вы хотите добавить пользовательское свойство для элемента отчета диаграммы, вам нужно добавить код, аналогичный красному тексту в следующем примере.

**Example**

{{< highlight csharp >}}

<chart Name="chart1">
    <Left>5.5cm</Left>
    <Top>0.5cm</Top>
      ......
    <Style>
      ......
    </style>     
    <CustomProperties>
      <CustomProperty>
        <Name>IsInList</Name>
        <Value>True</Value>
      </CustomProperty>
    </CustomProperties>
</chart> 

{{< /highlight >}}

В этом примере фрагмента кода имя пользовательского свойства - IsInList, и значение - 'True'.

{{% /alert %}}
