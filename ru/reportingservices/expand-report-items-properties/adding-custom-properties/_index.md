{{% alert color="primary" %}}

Вы можете добавлять пользовательские свойства к некоторым элементам отчёта, чтобы расширить их использование, например, Оглавление, линейные стрелки и т.д. Этот раздел описывает этот процесс.

Чтобы добавить пользовательские свойства, вам необходимо отредактировать файл кода документа RDL в следующих шагах:

1. Как показано на следующем рисунке, откройте ваш проект, перейдите к обозревателю решений и щелкните правой кнопкой мыши выбранный файл отчёта, затем выберите пункт меню 'View Code'.

![todo:image_alt_text](adding-custom-properties_1.png)

2. Отредактируйте XML-файл кода. Например, если вы хотите добавить пользовательское свойство для элемента отчёта диаграммы, вам нужно добавить код, аналогичный красному тексту в следующем примере.

**Пример**

{{< highlight csharp >}}

<chart Name="chart1">
    <Left>5.5cm</Left>
    <Top>0.5cm</Top>
      ......
    <Style>
      ......
    </style>     
    <CustomProperties>
      <CustomProperty>
        <Name>IsInList</Name>
        <Value>True</Value>
      </CustomProperty>
    </CustomProperties>
</chart> 

{{< /highlight >}}

В этом примере фрагмента кода пользовательское свойство называется IsInList, а значение — 'True'.

{{% /alert %}}
