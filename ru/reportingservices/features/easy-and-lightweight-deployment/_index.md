**Aspose.Pdf for Reporting Services** является пользовательским расширением рендеринга для Microsoft SQL Server 2016/2017/2019/2022 Reporting Services и Power BI Report Server. Aspose.Pdf for Reporting Services предоставляется в виде единственного MSI‑установщика, который может быть установлен на компьютерах, работающих с одним из следующих:

- Microsoft SQL Server 2016 Reporting Services  
- Microsoft SQL Server 2017 Reporting Services  
- Microsoft SQL Server 2019 Reporting Services  
- Microsoft SQL Server 2022 Reporting Services  
- Microsoft Power BI Report Server
 
Aspose.Pdf for Reporting Services легко развертывать и управлять, так как он состоит только из одной .NET сборки Aspose.Pdf.ReportingServices.dll, полностью написанной на C#, совместимой с CLS и содержащей только безопасный управляемый код. Требуется установить .NET Framework 4.8.1 на сервер.

Aspose.Pdf.ReportingServices.dll необходимо скопировать в каталог ReportServer\\bin, и файл конфигурации должен быть обновлен, чтобы Reporting Services знали о новом расширении рендеринга. Эти шаги выполняет установщик Aspose.Pdf for Reporting Services, но вы также можете выполнить их вручную, как описано дальше в этой документации.
