## Добро пожаловать в Aspose.PDF for Reporting Services!

Microsoft SQL Server Reporting Services удовлетворяет потребность, с которой сталкиваются многие организации — необходимость создания решений для бизнес‑аналитики и отчетности. До настоящего времени разработчикам приходилось внедрять отчёты в свои приложения, а организациям — приобретать дорогие и иногда проблемные сторонние решения для отчетности. Сейчас Microsoft SQL Server Reporting Services предлагает полноценное решение для распространения отчетов по всей компании, позволяя бизнесу принимать решения лучше и быстрее.

**Aspose.Pdf for Reporting Services** — это еще одно уникальное решение от Aspose, которое делает возможным генерацию PDF‑отчетов в Microsoft SQL Server 2016/2017/2019/2022 Reporting Services и Power BI Report Server. Все функции отчётов RDL, включая таблицы, матрицы, диаграммы и изображения, конвертируются с высочайшей точностью в PDF.

Microsoft SQL Server Reporting Services имеет встроенные возможности экспорта отчетов в виде PDF‑документов, но не предоставляет необходимую техническую поддержку конечному пользователю. Aspose.Pdf стремится обеспечить высочайшую и эффективную техническую поддержку.

Aspose.Pdf for Reporting Services создает документы на сервере без использования Adobe.Pdf SDK. Aspose.Pdf for Reporting Services внутри использует Aspose.Pdf for .NET – компонент мирового класса для серверной обработки документов и конвертации.
