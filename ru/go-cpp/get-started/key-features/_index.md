## Кроссплатформенное программное обеспечение

Поддерживает различные платформы и версии Go. См. [Системные требования](/pdf/ru/go-cpp/system-requirements/) для подробностей.

## Общие возможности

Поддерживает загрузку из и сохранение в широкий список [форматов файлов](/pdf/ru/go-cpp/supported-file-formats/).

## Поддерживаемые стандарты PDF

Aspose.PDF for Go поддерживает версии PDF 1.2, 1.3, 1.4, 1.5, 1.6 и 1.7.
