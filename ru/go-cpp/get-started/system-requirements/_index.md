## Поддерживаемые платформы Aspose.PDF for Go через C++

Пакет asposepdf — это мощный набор инструментов, который позволяет разработчикам напрямую манипулировать PDF‑файлами и помогает выполнять различные задачи с PDF.
Содержит уникальные функции для преобразования PDF в другие форматы.

Реализована поддержка платформ Linux x64, macOS x86_64, macOS arm64 и Windows x64. Использовано [cgo](https://go.dev/wiki/cgo).

Требуется версия динамической библиотеки, специфичная для платформы, из папки ‘lib’ в корневом каталоге пакета, для распространения полученного приложения:

- **libAsposePDFforGo_linux_amd64.so** для платформы Linux x64.
- **libAsposePDFforGo_darwin_arm64.dylib** для платформы macOS arm64.
- **libAsposePDFforGo_darwin_amd64.dylib** для платформы macOS x86_64.
- **AsposePDFforGo_windows_amd64.dll** для платформы Windows x64.

Для платформы Windows x64 требуется [MinGW-W64](https://www.mingw-w64.org/) установлен.
