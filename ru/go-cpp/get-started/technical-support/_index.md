Aspose предоставляет неограниченную бесплатную техническую поддержку для всех своих продуктов. Поддержка доступна всем пользователям, включая оценочную версию. Основным каналом поддержки являются Aspose.Forums.

{{% alert color="primary" %}}
Обратите внимание, что Aspose не предоставляет техническую поддержку по телефону. Телефонная поддержка доступна только для вопросов, связанных с продажами и покупкой.
{{% /alert %}}

## Бесплатный форум поддержки Aspose

Aspose.PDF предоставляет бесплатную техническую поддержку для всех своих продуктов. Если у вас есть вопросы по Aspose.PDF, ознакомьтесь со следующей статьей:

- Самый популярный источник поддержки — [Aspose.Forums](https://forum.aspose.com/). В [Aspose.PDF форум](https://forum.aspose.com/c/pdf/10) раздел, вы можете задать свой вопрос и получить ответ как можно скорее.
- Вы должны знать, что Aspose не предоставляет техническую поддержку по телефону. Звонки доступны только для вопросов по продажам и покупкам.
- Пожалуйста, помните, что у нас разница во временных зонах, это может повлиять на скорость ответов вам.

## Сообщить о проблеме или запросе функции

Если вам нужна помощь с проблемой в Aspose.PDF, следуйте следующим рекомендациям, чтобы убедиться, что она будет решена наиболее эффективным способом:

- Во-первых, проверьте, что вы используете последнюю версию Aspose.PDF перед сообщением о проблеме, см. [Aspose.PDF for Go Загрузки](https://github.com/aspose-pdf/aspose-pdf-go-cpp) чтобы узнать о последней версии.
- Возможно, решение вашей проблемы уже обсуждалось и имеется, для этого изучите форум и документацию, проверьте [Справочник API](https://reference.aspose.com/pdf/go-cpp/) перед тем как сообщить о проблеме.
- Когда формулируете свой вопрос, пожалуйста, включите оригинальный документ и, возможно, фрагмент вашего кода, вызывающего проблему. Вы можете упаковать эти файлы в один архив, если их несколько.
- Не волнуйтесь, доступ к прикреплённым файлам имеют только вы и разработчики Aspose.
