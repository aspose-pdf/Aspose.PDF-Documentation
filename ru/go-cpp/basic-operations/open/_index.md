## Открыть существующий PDF документ

Этот фрагмент кода демонстрирует базовые операции по работе с PDF с использованием Aspose.PDF for Go. Это открытие файла, сохранение изменений и правильное закрытие ресурсов. Это делает его фундаментальным примером для разработчиков, работающих с PDF‑документами.

Пример прост и понятен, что облегчает его понимание и модификацию. Он идеален для новичков или как шаблон для более сложных приложений.

Возможность открывать и сохранять PDF‑документы является основной потребностью во многих сценариях, таких как генерация отчётов, изменение документов или создание автоматизированных рабочих процессов.

Этот пример демонстрирует простоту использования API для базовой манипуляции PDF, которую можно расширить, включив продвинутые функции, такие как извлечение текста, аннотация или заполнение форм.

```go

    package main

    import "github.com/aspose-pdf/aspose-pdf-go-cpp"
    import "log"

    func main() {
        // Open(filename string) opens a PDF-document with filename
        pdf, err := asposepdf.Open("sample.pdf")
        if err != nil {
            log.Fatal(err)
        }
        // Save() saves previously opened PDF-document
        err = pdf.Save()
        if err != nil {
            log.Fatal(err)
        }
        // Close() releases allocated resources for PDF-document
        defer pdf.Close()
    }
```
