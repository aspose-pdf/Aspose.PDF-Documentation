- Aspose.PDF.SharePoint.License.wsp

Файл пакета решения SharePoint. Лицензия Aspose.PDF for SharePoint упакована как решение SharePoint для упрощения развертывания/отзыва в серверной ферме.

- readme.txt

Инструкции по установке лицензии. Установка лицензии выполняется из консоли сервера с помощью stsadm.exe. Шаги, необходимые для установки лицензии, приведены ниже.

**Note:** Путь опущен для наглядности. Возможно, вам потребуется добавить фактический путь к stsadm.exe и/или файлу решения при их выполнении.

1. Запустите stsadm, чтобы добавить решение в хранилище решений SharePoint:

stsadm.exe -o addsolution -filename Aspose.PDF.SharePoint.License.wsp

2. Разверните решение на всех серверах в ферме:

stsadm.exe -o deploysolution -name Aspose.PDF.SharePoint.License.wsp -immediate -force

3. Выполните административные таймерные задачи, чтобы завершить развертывание немедленно.

stsadm.exe -o execadmsvcjobs

**Примечание:** Вы получите предупреждение при выполнении шага развертывания, если служба администрирования Windows SharePoint Services не запущена. Stsadm.exe зависит от этой службы и службы таймера Windows SharePoint для репликации данных решения по ферме. Если эти службы не работают на вашем серверном ферме, вам может потребоваться развернуть лицензию на каждом сервере.
