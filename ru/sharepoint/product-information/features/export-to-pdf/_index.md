{{% alert color="primary" %}}

С помощью Aspose.PDF for SharePoint вы можете экспортировать списки, элементы списков и страницы Wiki SharePoint из библиотек SharePoint в формат PDF.

{{% /alert %}}

{{% alert color="primary" %}}

Для создания документов Aspose.PDF for SharePoint использует встроенную версию [Aspose.PDF for .NET](http://www.aspose.com/categories/.net-components/aspose.pdf-for-.net/default.aspx).


{{% /alert %}}
