## **Развертывание**

{{% alert color="primary" %}}

**Aspose.PDF for SharePoint выполняет следующие действия во время развертывания:**
- Установите Aspose.PDF.SharePoint.dll в Global Assembly Cache и добавьте запись SafeControl в файл web.config.
- Установите манифест функции и другие необходимые файлы в соответствующие каталоги.
- Зарегистрируйте функцию в базе данных SharePoint и сделайте её доступной для активации в области действия функции.

{{% /alert %}}


## **Активация**

{{% alert color="primary" %}}

**Aspose.PDF for SharePoint упакован как функция уровня сайта (коллекции сайтов) и может быть активирован и деактивирован в коллекциях сайтов.**

{{% /alert %}}

{{% alert color="primary" %}}

Во время активации функция вносит некоторые изменения в виртуальный каталог родительского веб-приложения коллекции сайтов: добавить страницу настроек конвертации в файл карты сайта. Скопировать необходимые файлы ресурсов в папку App_GlobalResources в виртуальном каталоге.

{{% /alert %}}
