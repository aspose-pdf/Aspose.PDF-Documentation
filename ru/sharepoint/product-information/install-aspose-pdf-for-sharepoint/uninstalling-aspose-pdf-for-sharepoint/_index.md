{{% alert color="primary" %}}

Чтобы удалить Aspose.PDF for SharePoint, просто запустите программу установки. Если Aspose.PDF for SharePoint уже установлен, программа установки предлагает его удалить.

![todo:image_alt_text](uninstalling-aspose-pdf-for-sharepoint_1.png)

Во время деинсталляции программа установки деактивирует функцию Aspose.PDF for SharePoint для всех коллекций сайтов и откатывает решение с фермы серверов.

{{% /alert %}}
