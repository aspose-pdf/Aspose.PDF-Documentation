{{% alert color="primary" %}}

Поддержка рабочих процессов является ключевой функцией Microsoft Office SharePoint Server. Рабочие процессы помогают автоматизировать перемещение документов в соответствии с бизнес-логикой и оптимизировать затраты и время организации документов. В этой статье демонстрируется, как использовать Aspose.PDF for SharePoint в рабочем процессе, который преобразует документ в PDF.

{{% /alert %}}
## **Настройка рабочего процесса**

В этом примере создаётся рабочий процесс, который преобразует любой новый элемент в библиотеке документов в формат PDF и сохраняет его в другой библиотеке документов. В примере в качестве исходной библиотеки используется библиотека **Personal Documents**, а в качестве целевой библиотеки — подпапка **Pdf** в библиотеке **Shared Documents**.

Aspose.PDF for SharePoint поддерживает преобразование файлов HTML, текста и изображений.

### **Создайте рабочий процесс с помощью SharePoint Designer**

1. Откройте **SharePoint Designer** и подключитесь к сайту, где будет реализован рабочий процесс.
1. Выберите **Workflows** в разделе **site objects** и затем откройте **List Workflow**.
1. Выберите библиотеку **Personal Documents**, чтобы создать и прикрепить новый список рабочего процесса к библиотеке документов.

   **Выбор Personal Documents из меню**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_1.png)


1. Создайте и прикрепите список рабочего процесса к библиотеке **Personal Documents**, введя название и описание рабочего процесса.
1. Нажмите **OK**, чтобы завершить этот шаг.

   **Создание списка рабочего процесса**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_2.png)



Появляется редактор шага рабочего процесса. Он используется для определения условий и действий для рабочих процессов. Теперь добавьте действие для конвертации нового документа в PDF без каких-либо условий, из **Aspose Actions**.

1. Выберите действие **Convert file to PDF via Aspose.PDF** из меню **Action**.

   **Выбор действия**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_3.png)


1. Настройте параметры действия:
   1. Установите параметр **this folder** в папку назначения.
   1. Либо оставьте другие параметры действия со значениями по умолчанию, либо установите их через окно свойств действия. Значение по умолчанию для параметра **Overwrite** — false.

      **Редактор рабочего процесса**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_4.png)



**Настройка целевой библиотеки**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_5.png)



**Настройка свойств**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_6.png)




1. В меню **Workflow** выберите **Workflow Settings**.
1. Выберите **запускать рабочий процесс автоматически при создании нового элемента** и очистите другие параметры в **Параметрах запуска**.

   **Настройка параметров запуска**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_7.png)



Дизайн рабочего процесса завершён.

1. Сохраните и опубликуйте рабочий процесс, чтобы внедрить его на сайте SharePoint.

### **Тестировать рабочий процесс**

Чтобы протестировать рабочий процесс:

1. Откройте сайт SharePoint и загрузите новый документ в библиотеку документов **Personal Documents**.
   Aspose.PDF for SharePoint поддерживает конвертацию файлов HTML, текстовых файлов и изображений (JPG, PNG, GIF, TIFF и BMP*) в PDF. Рабочий процесс настроен на автоматический запуск при создании нового элемента, поэтому файлы обрабатываются автоматически.
1. Обновите браузер.
   Статус рабочего процесса отображается в столбце рабочего процесса, **Aspose.PDF Workflow** в данном случае.

   **Добавление документа в исходную библиотеку**

![todo:image_alt_text](converting-a-file-to-pdf-via-workflow-activity_8.png)




1. Откройте целевую библиотеку документов, чтобы просмотреть конвертированный документ. **Shared Documents/Pdf** — путь в этом примере.
