{{% alert color="primary" %}}

[Aspose.PDF для .NET](/pdf/net/) представляет собой библиотеку классов, которая предоставляет функциональность для создания PDF-документов с нуля, а также возможности редактирования и управления существующими PDF-документами. Aspose.PDF доступен в двух различных продуктах [Aspose.PDF для .NET](/pdf/net/) и [Aspose.PDF для Java](/pdf/java/). Можно сказать, что Aspose.PDF "изначально" поддерживает:

- Платформу .NET и все языки программирования, нацеленные на платформу .NET, такие как C# или VB.NET.
- Платформу Java и язык программирования Java.

{{% /alert %}}

Обратите внимание, что также возможно использовать [Aspose.PDF для .NET](/pdf/net/) из множества языков программирования, включая: ASP, C++, ColdFusion, Delphi, JavaScript, Perl, PHP, PowerBuilder, Python, VBScript и др.

**Существует две версии библиотеки классов Aspose.PDF:**

- [Aspose.PDF для .NET](/pdf/net/), для использования с любым языком программирования, ориентированным на платформу .NET.
- [Aspose.PDF для Java](/pdf/java/), для использования с языком программирования Java.

Использование Aspose.PDF из любого другого языка программирования требует использования *технологии взаимодействия*. Технология взаимодействия должна позволять вашему языку программирования вызывать библиотеку классов, написанную для .NET или Java.
