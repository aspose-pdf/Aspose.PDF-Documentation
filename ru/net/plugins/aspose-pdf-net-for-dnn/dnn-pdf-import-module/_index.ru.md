## Введение

Модуль Aspose .NET PDF Import для DNN позволяет разработчикам получать/читать содержимое PDF-документов без необходимости использования другого программного обеспечения, такого как Adobe Acrobat или PDF-ридер.

### Особенности модуля

Этот модуль демонстрирует мощную функцию импорта, предоставляемую [Aspose.PDF](http://www.aspose.com/pdf-component-suite.aspx). Он добавляет простой контроль файлового браузера, выбор панели содержимого и кнопку **Импорт из PDF** на страницу, где добавлен модуль. При нажатии на кнопку содержимое документа отображается на экране немедленно.

## Использование

После установки модуля Aspose .NET PDF Import для DNN, его использование на вашем сайте становится действительно простым. Пожалуйста, выполните следующие простые шаги, чтобы начать

1. Убедитесь, что вы вошли в DNN как учетная запись уровня Host или Admin
1. Перейдите на страницу, где вы хотите добавить модуль Import
1. Из списка выберите **Aspose .NET PDF Import для DNN** и перетащите его в выбранное место на странице.

Вы успешно добавили модуль Aspose .NET PDF Import для DNN на вашу страницу. Теперь на странице появятся кнопка для выбора файла и кнопка с надписью **Импорт из PDF** вместе с выпадающим списком выбора панели содержимого. Любой пользователь может просто выбрать файл PDF и нажать кнопку **Импорт из PDF**, чтобы отобразить содержимое выбранного документа на странице.
