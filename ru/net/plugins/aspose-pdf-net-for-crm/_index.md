## Что такое Microsoft Dynamics CRM?

Microsoft Dynamics CRM — это пакет программного обеспечения для управления взаимоотношениями с клиентами, разработанный Microsoft. Изначально продукт ориентирован в основном на секторы продаж, маркетинга и обслуживания (служба поддержки), но Microsoft позиционирует Dynamics CRM как платформу XRM и призывает партнёров использовать её собственный (.NET-ориентированный) фреймворк для настройки. Это часть семейства бизнес-приложений Microsoft Dynamics.

Dynamics CRM представляет собой серверное клиентское приложение, которое, подобно Microsoft SharePoint, в первую очередь является веб-приложением на базе IIS, также поддерживающим обширные интерфейсы веб-сервисов. Клиенты получают доступ к Dynamics CRM либо через браузер, либо через плагин для Microsoft Outlook.
