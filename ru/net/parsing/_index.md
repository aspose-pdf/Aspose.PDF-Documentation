**Анализ PDF** документов — это термин, связанный с извлечением различных видов информации из PDF-файла. В этом разделе рассказывается, как:

- [Извлечь текст из PDF](/pdf/ru/net/extract-text-from-pdf/). Анализ текста или его извлечение — самая популярная операция с готовыми PDF-файлами. Вы узнаете об анализе текста из всего документа, отдельной страницы или определённой области на странице.
- [Извлечь изображения из PDF](/pdf/ru/net/extract-images-from-the-pdf-file/). Извлечение изображений аналогично операции, описанной выше для текста.
- [Извлечь шрифты из PDF](/pdf/ru/net/extract-fonts-from-pdf/). Извлечение шрифтов — это специфическая операция со шрифтами в PDF-файлах.
- [Извлечь данные из формы](/pdf/ru/net/extract-data-from-acroform/). Если у вас есть несколько PDF-документов с формами, возможно, вам потребуется извлечь данные из этих форм. Эта статья поможет понять, как извлекать данные AcroForms с помощью Aspose.PDF for .NET.
- [Извлечь текст из штампов](/pdf/ru/net/extract-text-from-stamps/) — получить текстовую информацию из вашего PDF-документа.
- [Извлечь данные из таблицы](/pdf/ru/net/extract-data-from-table-in-pdf/) — получить данные из таблицы в PDF-документе.
- [Извлечь векторные данные из PDF](/pdf/ru/net/extract-vector-data-from-pdf/) — вы можете получить векторные данные (путь, многоугольник, полилинию), такие как положение, цвет, ширина линии и т. д.
