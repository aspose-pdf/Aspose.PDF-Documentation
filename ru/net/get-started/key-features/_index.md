## Кроссплатформенное программное обеспечение

Поддерживает различные платформы и версии .NET. См. [Системные требования](/pdf/net/system-requirements/) для получения подробной информации.

## Общие функции

Aspose.PDF предоставляет мощную модель объектов документа (DOM) для изучения и изменения всех типов контента PDF. Поддерживает загрузку из и сохранение в широкий список [форматов файлов](/pdf/net/supported-file-formats/).

## Поддерживаемые стандарты PDF

- ISO 32000-1 (PDF 1.2, PDF 1.3, PDF 1.4, PDF 1.5, PDF 1.6, PDF 1.7).
- ISO 32000-2 (PDF 2.0).
- ISO 19005 (PDF/A), ISO 15930 (PDF/X), ISO 24517 (PDF/E), ISO 14289 (PDF/UA).

## Возможности ИИ

Поддерживает различные поставщики ИИ с возможностью загрузки данных из PDF-документов.
