## Обзор
API можно использовать для создания приложений в любой операционной системе, которая поддерживает .NET Standard 2.0 и более поздних версий или .NET Framework 4.0.

Для создания и управления PDF-документами на компьютере, на котором работает Aspose.PDF for .NET, не обязательно устанавливать Adobe Acrobat: Aspose.PDF for .NET — это механизм создания PDF-документов. Однако для просмотра PDF-документов, созданных с помощью Aspose.PDF for .NET, на компьютере должен быть установлен хотя бы Adobe Acrobat Reader. Его можно скачать бесплатно.

## Среды разработки
Вы можете использовать Aspose.PDF for .NET для разработки приложений в любой среде разработки, ориентированной на платформу .NET, но явно поддерживаются следующие среды:

- Microsoft Visual Studio 2010[^1].
- Microsoft Visual Studio 2011[^1].
- Microsoft Visual Studio 2012[^1].
- Microsoft Visual Studio 2013[^1].
- Microsoft Visual Studio 2015[^1].
- Microsoft Visual Studio 2017[^1].
- Microsoft Visual Studio 2019.
- Microsoft Visual Studio 2022.
- Microsoft Visual Studio Code.

## Поддерживаемые платформы
Поддерживаются следующие версии .NET Framework:

- .NET Framework 4.0.
- .NET Framework 4.8.1.
- .NET Standard 2.0 (.NET Core 3.1+).
- .NET 6.0.
- .NET 7.0.
- .NET 8.0.

[^1]: Поддержка Aspose.PDF for .NET осуществляется с использованием .NET Framework 4.0. Однако, поскольку .NET Framework 4.0 устарела, эта поддержка предоставляется исключительно в целях совместимости. Обратите внимание, что мы оставляем за собой право прекратить публикацию сборки для .NET Framework 4.0 в любое время.
