Вы можете удалить страницы из файла PDF с помощью Aspose.PDF для .NET. Чтобы удалить конкретную страницу из коллекции [PageCollection](https://reference.aspose.com/pdf/net/aspose.pdf/pagecollection).

## Удаление страницы из файла PDF

1. Вызовите метод [Delete](https://reference.aspose.com/pdf/net/aspose.pdf/pagecollection/methods/delete) и укажите индекс страницы
1. Вызовите метод [Save](https://reference.aspose.com/pdf/net/aspose.pdf.document/save/methods/4), чтобы сохранить обновленный файл PDF
Следующий фрагмент кода показывает, как удалить конкретную страницу из файла PDF на C#.

```csharp
// Для полных примеров и файлов данных, пожалуйста, перейдите на https://github.com/aspose-pdf/Aspose.PDF-for-.NET
// Путь к директории документов.
string dataDir = RunExamples.GetDataDir_AsposePdf_Pages();

// Открыть документ
Document pdfDocument = new Document(dataDir + "DeleteParticularPage.pdf");

// Удалить конкретную страницу
pdfDocument.Pages.Delete(2);

dataDir = dataDir + "DeleteParticularPage_out.pdf";
// Сохранить обновленный PDF
pdfDocument.Save(dataDir);
```
