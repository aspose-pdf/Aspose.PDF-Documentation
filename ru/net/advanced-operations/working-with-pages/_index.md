**Aspose.PDF for .NET** позволяет вставлять страницу в PDF-документ в любом месте файла, а также добавлять страницы в конец PDF-файла. В этом разделе показано, как добавлять страницы в PDF без Acrobat Reader.
Вы можете добавлять текст или изображения в заголовки и нижние колонтитулы вашего PDF-файла и выбирать разные заголовки в вашем документе с помощью библиотеки C# от Aspose.
Также попробуйте программно обрезать страницы в PDF-документе с помощью C#.

Этот раздел научит вас добавлять водяные знаки в ваш PDF-файл с помощью класса Artifact. Вы увидите пример программирования для этой задачи.
Добавьте номер страницы с помощью класса PageNumberStamp. Для добавления штампа в документ используйте классы ImageStamp и TextStamp. Используйте добавление водяного знака для создания фоновых изображений в вашем PDF-файле с **Aspose.PDF for .NET**.

Вы сможете выполнить следующие действия:

* [Добавить страницы](/pdf/ru/net/add-pages/) — добавляйте страницы в нужное место или в конец PDF-файла и удаляйте страницы из документа.
* [Перемещать страницы](/pdf/ru/net/move-pages/) — перемещайте страницы из одного документа в другой.
* [Удалить страницы](/pdf/ru/net/delete-pages/) — удалите страницу из вашего PDF-файла с помощью коллекции PageCollection.
* [Изменить размер страницы](/pdf/ru/net/change-page-size/) — вы можете изменить размер страницы PDF с помощью фрагмента кода, используя библиотеку Aspose.PDF.
* [Поворачивать страницы](/pdf/ru/net/rotate-pages/) — вы можете изменять ориентацию страниц в существующем PDF-файле.
* [Разделять страницы](/pdf/ru/net/split-document/) — вы можете разделить PDF-файлы на один или несколько PDF.
* [Добавлять верхние и/или нижние колонтитулы](/pdf/ru/net/add-headers-and-footers-of-pdf-file/) — добавляйте текст или изображения в верхние и нижние колонтитулы вашего PDF-файла.
* [Обрезать страницы](/pdf/ru/net/crop-pages/) — вы можете программно обрезать страницы в PDF-документе с различными свойствами страницы.
* [Добавление водяных знаков](/pdf/ru/net/add-watermarks/) — добавьте водяные знаки в свой PDF-файл с помощью класса Artifact.
* [Добавьте нумерацию страниц в PDF-файл](/pdf/ru/net/add-page-number/) — класс PageNumberStamp поможет вам добавить номер страницы в ваш PDF-файл.
* [Добавить фоны](/pdf/ru/net/add-backgrounds/) — фоновые изображения можно использовать для добавления водяного знака.
* [Штамповка](/pdf/ru/net/stamping/) — вы можете использовать класс ImageStamp для добавления изображения штампа в PDF-файл и класс TextStamp для добавления текста.
