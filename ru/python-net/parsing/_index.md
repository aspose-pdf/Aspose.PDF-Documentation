**Разбор PDF** документы — это термин, связанный с извлечением различных видов информации из PDF‑файла. В этом разделе рассматривается, как:

- [Извлечь текст из PDF](/pdf/python-net/extract-text-from-pdf/). Парсинг или извлечение текста — самая популярная операция с готовыми PDF. Вы узнаете о парсинге текста из всего документа, конкретной страницы или определённого региона на странице.
- [Извлечь изображения из PDF](/pdf/python-net/extract-images-from-the-pdf-file/). Извлечение изображений делает то же самое для изображений, что и операция выше для текста.
- [Извлечь шрифты из PDF](/pdf/python-net/extract-fonts-from-pdf/). Извлечение шрифтов — это специфическая операция с шрифтами в PDF.
- [Извлечь данные из формы](/pdf/python-net/extract-data-from-acroform/). Если у вас есть множество PDF‑документов с формами, вероятно, вам нужно получить данные из этих форм. Эта статья поможет понять, как извлекать данные AcroForms с помощью Aspose.PDF для Python через .NET.
- [Извлечь данные из таблицы](/pdf/python-net/extract-data-from-table-in-pdf/). Получите данные из таблицы в PDF‑документе.
- [Извлечь векторные данные из PDF](/pdf/python-net/extract-vector-data-from-pdf/). Вы можете получить векторные данные (контур, полигон, полилиния), такие как позиция, цвет, толщина линии и т.д.
