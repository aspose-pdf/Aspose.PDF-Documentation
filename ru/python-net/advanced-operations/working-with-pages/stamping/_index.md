Штамп в PDF‑документе аналогичен применению резинового штампа к бумажному документу.
Штамп в PDF‑файле предоставляет дополнительную информацию для PDF‑файла, например, защищает его от использования другими и подтверждает безопасность содержимого PDF‑файла. **Aspose.PDF for Python via .NET** позволяет добавлять в ваш PDF‑документ изображение‑штамп или текстовый штамп.

Ознакомьтесь с следующими разделами, чтобы узнать, как добавить штамп с помощью Python:

- [Добавление изображений‑штампов на страницу PDF](/pdf/python-net/image-stamps-in-pdf-page/) - добавить изображение‑штамп, контролировать качество изображения, использовать изображение‑штамп как фон вашего PDF‑файла.
- [Добавление текстовых штампов в PDF‑файл](/pdf/python-net/text-stamps-in-the-pdf-file/) - добавить текстовый штамп, определить выравнивание для объекта TextStamp, заполнить обводку текста как штамп в PDF‑файле.
- [Добавление штампов страниц в PDF‑файл](/pdf/python-net/page-stamps-in-the-pdf-file/) - добавить штампы страниц для наложения на целые страницы или разделы, полезно для водяных знаков, версии документов или добавления пользовательских фонов в PDF‑файлы.
- [Добавление штампов номеров страниц в PDF‑файл](/pdf/python-net/add-page-number/) - добавить номера страниц в ваш PDF‑документ, настроить их внешний вид и контролировать их размещение.
