## Заполнение поля формы в PDF-документе

Чтобы заполнить поле формы, получите поле из коллекции [Form](https://reference.aspose.com/pdf/python-net/aspose.pdf.forms/form/) объекта Document. затем установите значение поля, используя свойство [value](https://reference.aspose.com/pdf/python-net/aspose.pdf.forms/form/#properties) поля.

Этот пример выбирает [TextBoxField](https://reference.aspose.com/pdf/python-net/aspose.pdf.forms/textboxfield/) и устанавливает его значение, используя свойство Value.

```python

    import aspose.pdf as ap

    # Открыть документ
    pdfDocument = ap.Document(input_file)
    for formField in pdfDocument.form.fields:
        if formField.partial_name == "Field 1":
            # Изменить значение поля
            formField.value = "777"

    # Сохранить обновленный документ
    pdfDocument.save(output_pdf)
```
