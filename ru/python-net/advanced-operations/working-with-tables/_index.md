**Aspose.PDF for Python via .NET** позволяет работать с таблицами в PDF-файлах на продвинутом уровне. Этот идеальный инструмент помогает справляться с простотой PDF, извлекая таблицы. С помощью библиотеки Python вы можете легко создавать или добавлять таблицу в существующий PDF‑документ, определять, разорвется ли таблица на текущей странице, извлекать таблицу и удалять таблицы из существующих PDF.

Вы можете выполнить следующее:

- [Добавление таблиц](/pdf/python-net/adding-tables/) - создайте таблицу в PDF‑файле, объединяя столбцы или строки, учитывая границы, отступы и поля.
- [Извлечение таблиц](/pdf/python-net/extracting-table/) - вы можете извлечь таблицу из PDF‑файла.
- [Интеграция таблиц с источниками данных](/pdf/python-net/integrate-table/) - интегрируйте таблицу с базой данных, интегрируйте таблицу с источником Pandas DataFrame, используя библиотеку Python.
- [Удаление таблиц](/pdf/python-net/removing-tables/) - удалите таблицу или несколько таблиц из PDF‑документа.
- [Манипулирование таблицами в существующем PDF](/pdf/python-net/manipulating-tables/) - манипулируйте таблицами в вашем PDF, используя TableAbsorber.
