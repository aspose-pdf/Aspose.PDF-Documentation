**Aspose.PDF for Python via .NET** позволяет вам работать с таблицами в PDF‑файле на продвинутом уровне. Этот отличный инструмент помогает справляться с простотой PDF, извлекая таблицы. С ресурсом библиотеки Python вы можете легко создавать или добавлять таблицы в существующий PDF‑документ, определять, разорвётся ли таблица на текущей странице, извлекать таблицы и удалять их из существующих PDF.

Используйте этот раздел, когда нужно создавать отчёты на основе таблиц, извлекать структурированные данные из PDF‑файлов или обновлять табличный контент в существующих документах с помощью Python.

## Задачи, связанные с таблицами

Вы можете сделать следующее:

- [Добавление таблиц](/pdf/python-net/adding-tables/) - создайте таблицу в PDF-файле, объединив столбцы или строки, с учетом границ, полей и отступов.
- [Извлечение таблиц](/pdf/ru/python-net/extracting-table/) - вы можете извлечь таблицу из PDF‑файла.
- [Интеграция таблиц с источниками данных](/pdf/ru/python-net/integrate-table/) - интегрировать таблицу с базой данных, интегрировать таблицу с источником Pandas DataFrame с использованием библиотеки Python.
- [Манипулирование таблицами](/pdf/ru/python-net/manipulating-tables/) - проверять и обновлять существующие таблицы в PDF‑документах.
- [Удаление таблиц](/pdf/ru/python-net/removing-tables/) - удалять таблицу или несколько таблиц из PDF‑документа.
