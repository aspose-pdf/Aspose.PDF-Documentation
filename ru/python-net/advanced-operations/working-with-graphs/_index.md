## Что такое граф

Добавление графиков в PDF‑документы является очень распространённой задачей для разработчиков, работающих с Adobe Acrobat Writer или другими приложениями для обработки PDF. Существует множество типов графиков, которые можно использовать в PDF‑приложениях.
[Aspose.PDF для Python через .NET](/pdf/python-net/) также поддерживает добавление графиков в PDF‑документы. Для этой цели предоставлен класс Graph. Graph является элементом уровня абзаца, его можно добавить в коллекцию Paragraphs экземпляра Page. Экземпляр Graph содержит коллекцию Shapes.

Следующие типы фигур поддерживаются классом [Graph](https://reference.aspose.com/pdf/python-net/aspose.pdf.drawing/graph/):

- [Дуга](/pdf/python-net/add-arc/) - иногда также называемая флагом, представляет собой упорядоченную пару смежных вершин, но иногда также называется направленной линией.
- [Круг](/pdf/python-net/add-circle/) - отображает данные с помощью круга, разделённого на сектора. Мы используем круговой график (также называемый круговой диаграммой), чтобы показать, как данные представляют части целого или группы.
- [Кривая](/pdf/python-net/add-curve/) - является соединённым объединением проектных линий, каждая из которых пересекает три другие в обычных двойных точках.
- [Линия](/pdf/python-net/add-line) - линейные графики используются для отображения непрерывных данных и могут быть полезны при прогнозировании будущих событий, когда они показывают тенденции во времени.
- [Прямоугольник](/pdf/python-net/add-rectangle/) - является одной из многих базовых фигур, используемых в графиках; она может быть очень полезна при решении задач.
- [Эллипс](/pdf/python-net/add-ellipse/) - представляет собой набор точек на плоскости, образующий овальную, изогнутую форму.
