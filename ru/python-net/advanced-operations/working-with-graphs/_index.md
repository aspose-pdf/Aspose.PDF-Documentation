## Что такое Graph

Добавление графиков в PDF‑документы — очень распространённая задача для разработчиков, работающих с Adobe Acrobat Writer или другими приложениями для обработки PDF. Существует множество типов графиков, которые можно использовать в PDF‑приложениях.
[Aspose.PDF for Python via .NET](/pdf/ru/python-net/) Также поддерживает добавление графиков в PDF‑документы. Для этой цели предоставляется класс Graph. Graph — элемент уровня абзаца, и его можно добавить в коллекцию Paragraphs экземпляра Page. Экземпляр Graph содержит коллекцию Shapes.

Используйте этот раздел, когда необходимо рисовать векторную графику непосредственно на страницах PDF в Python, будь то диаграммы, схемы, иллюстрации или пользовательские аннотации к страницам.

## Поддерживаемые формы Graph

Следующие типы форм поддерживаются [Graph](https://reference.aspose.com/pdf/python-net/aspose.pdf.drawing/graph/) класс:

- [arc](/pdf/ru/python-net/add-arc/) - иногда также называемый флагом, представляет собой упорядоченную пару соседних вершин, но иногда его называют направленной линией.
- [circle](/pdf/ru/python-net/add-circle/) - отображает данные с помощью круга, разделённого на сектора. Мы используем круговую диаграмму (также известную как круговой график), чтобы показать, как данные представляют части единого целого или группы.
- [curve](/pdf/ru/python-net/add-curve/) - представляет собой связное объединение проективных линий, каждая из которых пересекается с тремя другими в обычных двойных точках.
- [line](/pdf/ru/python-net/add-line) - линейные графики используются для отображения непрерывных данных и могут быть полезны при прогнозировании будущих событий, когда они показывают тенденции во времени.
- [rectangle](/pdf/ru/python-net/add-rectangle/) - является одной из многих базовых форм, которые вы увидите на графиках; она может быть очень полезна при решении задачи.
- [ellipse](/pdf/ru/python-net/add-ellipse/) - представляет собой набор точек на плоскости, создающих овальную, изогнутую форму.
