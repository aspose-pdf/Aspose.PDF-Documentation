`PdfContentEditor` Класс в Aspose.PDF Facades предназначен для задач редактирования на уровне содержимого в существующих PDF‑документах. Он предоставляет упрощённый API для обновления интерактивных элементов, замены текста, работы с вложениями и ссылками, а также настройки поведения документа в приложениях Python via .NET.
