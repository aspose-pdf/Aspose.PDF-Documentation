## Ограничения версии оценки

Мы хотим, чтобы наши клиенты тщательно протестировали наши компоненты перед покупкой, поэтому версия оценки позволяет использовать её так, как обычно.

- **PDF, созданный с оценочным водяным знаком.** Версия оценки Aspose.PDF for Python предоставляет полный набор функций продукта, однако все страницы создаваемых PDF‑документов помечаются водяным знаком "Evaluation Only. Created with Aspose.PDF. Copyright 2002-2020 Aspose Pty Ltd" вверху.

>Если вы хотите протестировать Aspose.PDF без ограничений версии оценки, вы также можете запросить 30‑дневную временную лицензию. Пожалуйста, обратитесь к [How to get a Temporary License?](https://purchase.aspose.com/temporary-license)

## Классическая лицензия

Лицензию можно загрузить из файла или объекта потока. Самый простой способ установить лицензию — поместить файл лицензии в ту же папку, что и файл Aspose.PDF.dll, и указать имя файла без пути, как показано в примере ниже.

Если вы используете любой другой компонент Aspose для Python вместе с Aspose.PDF for Python, пожалуйста, укажите пространство имён для лицензии, как в примере [Aspose.Pdf.License class]().

```python

    license_file = LICENSE_FILE
    license = ap.License()
    license.set_license(license_file)
```
