PDF расшифровывается как Portable Document Format и используется для отображения документов в электронном виде, независимом от программного обеспечения, аппаратного обеспечения или операционной системы, на которой они просматриваются.

PDF — открытый стандарт, поддерживаемый сегодня Международной организацией по стандартизации (ISO).

Исходная цель заключалась в сохранении и защите содержимого и макета документа — независимо от платформы или программы, на которой он просматривается. Поэтому PDF трудно редактировать, и иногда даже извлечение информации из него представляет собой проблему.

Но **Aspose.PDF for Node.js via C++** может помочь вам справиться с большинством задач, которые возникают при работе с PDF‑документом.

Вы можете сделать следующее:

- [Оптимизировать PDF](/pdf/ru/nodejs-cpp/optimize-pdf/) - оптимизировать содержание страниц, оптимизировать размер файла, удалить неиспользуемые объекты, сжать все изображения для успешной оптимизации документа.
- [Оптимизировать ресурсы PDF](/pdf/ru/nodejs-cpp/optimize-pdf-resources/) - оптимизировать содержание страниц, оптимизировать размер файла, удалить неиспользуемые объекты, сжать все изображения для успешной оптимизации документа.
- [Объединить PDF](/pdf/ru/nodejs-cpp/merge-pdf/) - объединить несколько PDF‑файлов в один PDF‑документ с помощью Node.js.
- [Разделить PDF](/pdf/ru/nodejs-cpp/split-pdf/) - разбить страницы PDF на отдельные PDF‑файлы в Node.js via C++.
