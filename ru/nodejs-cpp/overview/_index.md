_Эта страница представляет обзор функций Aspose.PDF for Node.js via C++._

**Aspose.PDF for Node.js via C++** позволяет разработчикам манипулировать PDF‑файлами напрямую в среде Node.js. Это решение Node.js построено на использовании технологии WebAssembly и основано на Aspose.PDF for .NET. Вы можете легко использовать aspose-pdf for Node.js via C++ и затем следовать инструкциям по установке.

**О расширениях файлов CJS и MJS**

Node.js поддерживает две модульные системы для организации и обмена кодом. Существуют:

- Модули CommonJS
- модули ECMAScript

Модули CommonJS были классическим методом обработки зависимостей в Node.js, тогда как модули ECMAScript являются более новым дополнением и следуют стандартному формату, используемому в современных браузерах. 'cjs', 'mjs' — это расширения, используемые для указания форматов модулей (CommonJS и ECMAScript Modules), используемых в файлах JavaScript. 

В фрагментах кода вы заметите разницу в работе с модулем AsposePDFforNode.js. Основное различие между 'require' и 'import' заключается в их случаях использования. **Require** ограничен импортом целых модулей, тогда как **Import** обладает гибкостью импортировать как целые модули, так и отдельные экспорты из этих модулей.

**Два разных подхода**

JavaScript предлагает два подхода — 'then' и 'await' — для обработки асинхронных операций.

Метод 'then' работает с обещаниями (promises), управляя как успешными результатами, так и ошибками в последовательном порядке. В свою очередь, 'await' заставляет JavaScript ждать, буквально приостанавливая выполнение функции, пока это обещание не будет разрешено и не вернёт свой результат.

Aspose.PDF for Node.js поддерживает широкий набор функций, таких как:

- Установить, получить и удалить информацию о PDF‑файле
- Удалить закладки
- Удалить аннотации
- Извлечь текст из PDF
- Оптимизировать PDF документ
- Оптимизировать ресурсы 
- Разделить PDF на два файла.
- Объединить два файла в PDF
- Восстановить PDF
- Функции конвертации

Кроме того, Aspose.PDF for Node.js via C++ можно использовать для простого преобразования PDF‑документов в различные форматы с высокой производительностью и отличным качеством. См. раздел [Aspose.PDF Поддерживаемые форматы файлов](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). Также, этот набор инструментов поддерживает шифрование и расшифровку PDF‑документов.
