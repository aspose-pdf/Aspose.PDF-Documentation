# Установка

**Aspose.PDF for Node.js via C++** — это простой и безопасный набор инструментов, используемый для работы с PDF непосредственно в среде Node.js.

Чтобы установить и использовать Aspose.PDF for Node.js via C++ из *ZIP‑архива, выполните следующие инструкции:

- скачайте архив с сайта Aspose.PDF
- извлеките файлы из ZIP-архива
- переименуйте ваш файл *.lic в 'aspose.lic', если он существует

Перейдите по ссылке для [Прямая загрузка](https://releases.aspose.com/pdf/nodejscpp/)

Для установки и использования Aspose.PDF for Node.js via C++ из NPMJS следуйте приведённым ниже инструкциям:

- npm i asposepdfnodejs
- переименуйте ваш файл *.lic в ‘aspose.lic’, если он существует
