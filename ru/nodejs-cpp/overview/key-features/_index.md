## Поддерживаемые версии PDF

Aspose.PDF for Node.js поддерживает версии PDF 1.2, 1.3, 1.4, 1.5, 1.6 и 1.7.

## Функции гиперссылок & вложений

Набор средств позволяет добавлять:

- Удалить вложения из PDF

## Функции безопасности PDF

API предлагает 40‑битное или 128‑битное шифрование. Также можно установить пользовательские и мастер‑пароли для повышения безопасности. Манипуляцию содержимым в файлах, такой как копирование содержимого, печать, ограниченная печать, изменение аннотаций, заполнение FormField, а также сборку PDF‑файла можно ограничить с помощью этой библиотеки.

- Зашифровать PDF файл
- Расшифровать PDF файл
- Сменить пароль
- Подписать PDF
