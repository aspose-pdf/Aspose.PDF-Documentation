Aspose.PDF для Node.js поддерживает версии PDF 1.2, 1.3, 1.4, 1.5, 1.6 и 1.7.

## PDF Security Features

API предлагает шифрование с длиной ключа 40 или 128 бит. Можно также установить пользовательский и главный пароли для большей безопасности. Манипуляции с содержимым файлов, такие как копирование контента, печать, ухудшенная печать, модификация аннотаций, заполнение полей формы, а также сборка PDF файла могут быть ограничены с использованием этой библиотеки.

- Шифрование PDF файла
- Дешифрование PDF файла
- Изменение пароля
- Подпись PDF
