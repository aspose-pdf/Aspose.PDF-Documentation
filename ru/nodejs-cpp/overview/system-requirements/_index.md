## Обзор

Для обработки PDF‑документов с использованием [Aspose.PDF for Node.js via C++](https://products.aspose.com/pdf/nodejs-cpp/), не требует сторонних библиотек и программ.
Требуется установленный [Node.js](https://nodejs.org/). Вы должны иметь Node.js версии 16 или выше. Мы настоятельно рекомендуем использовать LTS-версии.
Node.js может быть [установлен](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) по‑разному.

## Поддерживаемые операционные системы

Все поддерживаемые версии [Linux](https://en.wikipedia.org/wiki/Linux) и [macOS](https://www.apple.com/macos/).
