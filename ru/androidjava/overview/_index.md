_Эта страница — обзор функций Aspose.PDF for Android via Java._

Aspose.PDF for Android Java — это API обработки PDF, позволяющий Android‑приложениям читать, писать, изменять \u0026 отображать PDF‑документы. Aspose.PDF for Android via Java предоставляет очень простой интерфейс программирования приложений (API) для разработчиков, который легко изучить и использовать.

Aspose.PDF for Android via Java поддерживает широкий спектр функций, таких как:

- сжатие документов;
- создание и изменение таблиц;
- поддержка графических объектов;
- расширенная функциональность гиперссылок;
- расширенные элементы управления безопасностью;
- обработка пользовательских шрифтов;
- интеграция с источниками данных;
- добавление или удаление закладок;
- создать оглавление;
- добавлять, обновлять, удалять вложения и аннотации;
- импортировать или экспортировать данные PDF-формы;
- добавлять, заменять или удалять текст и изображения;
- разделять, соединять, извлекать или вставлять страницы;
- преобразовывать страницы в изображение;
- печать PDF‑документов и многое другое.

Кроме того, Aspose.PDF for Android via Java можно использовать для простого преобразования текста, изображений, конвертации PDF в различные форматы документов с отличной производительностью и хорошим качеством.
