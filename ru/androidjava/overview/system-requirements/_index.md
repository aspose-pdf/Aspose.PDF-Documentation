## Обзор

Для создания PDF‑документов машина, на которой будет запускаться [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) не обязана иметь [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) установленный как Aspose.PDF for Java сам по себе является движком создания PDF‑документов. Однако для того, чтобы просматривать PDF‑документы, созданные Aspose.PDF for Android через Java, машине требуется как минимум [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). Это может быть [скачано](https://www.adobe.com/acrobat/pdf-reader.html) бесплатно.

## Поддерживаемые операционные системы

Aspose.PDF for Android via Java поддерживает любые мобильные устройства, работающие на Java runtime, включая, но не ограничиваясь:

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## Среды разработки

Вы можете использовать Aspose.PDF for Android via Java для разработки приложений в любой среде разработки, нацеленной на платформу Java под Android, но поддерживаются следующие среды:

- Android Studio и инструменты SDK
