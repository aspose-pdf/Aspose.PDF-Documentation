## Общие особенности

- Aspose.PDF for Android via Java поддерживает любое мобильное устройство, на котором работает Java runtime
- API предлагает 40-битное или 128-битное шифрование

## Поддерживаемые версии PDF

Aspose.PDF for Android via Java поддерживает версии PDF 1.2, 1.3, 1.4, 1.5, 1.6 и 1.7.

## Графика

- Поддерживаются различные формы графиков, такие как линия, круг, дуга и кривая.
- Графики могут быть нарисованы в любом месте страницы, указанном координатами.

- Графики могут быть повернуты, масштабированы или заполнены, и можно определить стили линий с тире и политирасой.
- Графики рисуются на основе координат и имеют абсолютное, а не относительное положение.
- Поддерживаются цветовые пространства CMYK и Gray.
