## Компонент Aspose.PDF for Android via Java

{{% alert color="primary" %}}

**Aspose.PDF for Android via Java** — это компонент, который позволяет разработчикам программно создавать и манипулировать PDF‑документами, будь то простые или сложные. Aspose.PDF for Android via Java позволяет разработчикам вставлять таблицы, графики, изображения, гиперссылки, пользовательские шрифты — и многое другое — в PDF‑документы. Его также можно использовать для сжатия PDF‑документов. Aspose.PDF for Android via Java предоставляет отличные функции безопасности для разработки защищённых PDF. Самой отличительной особенностью Aspose.PDF for Android via Java является поддержка создания PDF как через API, так и из XML‑шаблонов.

{{% /alert %}} 

## Описание продукта

**Aspose.PDF for Android Java** — это API обработки PDF, которое позволяет Android‑приложениям читать, записывать, манипулировать и отображать PDF‑документы. API поддерживает широкий спектр стандартных файловых форматов, включая HTML, TXT, EPUB, XPS и изображения.

**Aspose.PDF for Android via Java** — это доступное решение, которое предлагает варианты сжатия, поддержку графики, создание документов с нуля, манипуляцию стилями текста\u0026 форматированием, добавление, изменение, извлечение и удаление вложений, аннотаций и другие популярные расширенные функции.67

{{% alert color="primary" %}} 

Вы можете загрузить [Aspose.PDF for Android via Java](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf-android-via-java) для оценки. Загрузка оценочной версии идентична загрузке покупной версии. Оценочная версия просто становится лицензированной, когда вы добавляете несколько строк кода. ## Установка Aspose.PDF for Android через Java из репозитория Aspose

Aspose размещает все Java API на [Репозиторий Aspose](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf-android-via-java). Вы можете легко использовать Aspose.PDF for Android через Java API напрямую в своих проектах Maven с простыми настройками.
