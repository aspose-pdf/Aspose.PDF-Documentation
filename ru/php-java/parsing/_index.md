Парсинг PDF документов — это термин, связанный с извлечением различных видов информации из PDF файла. Этот раздел охватывает как:

- [Извлечь текст из PDF](/pdf/ru/php-java/extract-text-from-pdf/). Парсинг или извлечение текста — это самая популярная операция с готовыми PDF. Вы узнаете о парсинге текста из всего документа, отдельной страницы или определенного региона на странице.
- [Извлечь изображения из PDF](/pdf/ru/php-java/extract-images-from-the-pdf-file/). Извлечение изображений выполняет ту же функцию для изображений, что и операция выше для текста.
- [Извлечь шрифты из PDF](/pdf/ru/php-java/extract-fonts-from-pdf/). Извлечение шрифтов — это специфическая операция со шрифтами в PDF.
- [Извлечь данные из таблицы в PDF](/pdf/ru/php-java/extract-data-from-table-in-pdf/).
 Learn how to extract tabular from PDF using Aspose.PDF for PHP.  
- [Извлечение данных из формы](/pdf/ru/php-java/extract-data-from-acroform/). Если у вас есть множество PDF-документов с формами, вероятно, вам нужно получить данные из этих форм. Эта статья поможет понять, как извлечь данные AcroForms с помощью Aspose.PDF для PHP.
