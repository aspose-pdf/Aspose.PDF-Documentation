Aspose.PDF for Python via Java Facades 是一组类，允许您在不深入了解 PDF 文档结构的情况下执行各种操作。

在本节中，您将学习如何：

- [使用 PdfFileEditor] - 工具，允许对 PDF 文件进行操作（拆分、合并等）
- [使用 PdfFileSignature] - 工具，用证书签署 PDF 文件
- [使用 Form] - 用于操作 Acro Forms 的工具
