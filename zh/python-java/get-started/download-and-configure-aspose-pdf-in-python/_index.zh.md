## 下载所需库

下载下面提到的所需库。这些是通过 Java 执行 Aspose.PDF for Python 示例所需的。

- Aspose: [Aspose.PDF for Python via Java 组件](https://releases.aspose.com/pdf/pythonjava/)
- 或 PyPI: [Python 包索引](https://pypi.org/project/aspose-pdf-for-python-via-java/)
- [JPype](https://pypi.python.org/pypi/JPype1)
- JDK 1.8+

## 从社交编码网站下载示例

以下版本的运行示例可以在下面提到的社交编码网站上下载：

### GitHub

#### Aspose.PDF for Python via Java 示例

[Aspose.PDF for Python via Java](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Plugins/Aspose_Pdf_Java_for_Python)

### 如何配置源代码

请按照以下简单步骤来打开并扩展源代码：

1. 从[官方网站](https://www.python.org/downloads)仅安装 Python 3.7+
2. 从[Sourceforge](http://sourceforge.net/projects/jpype/files/JPype/)安装 JPype 1.5.0+
3. 从[Java 下载](https://www.oracle.com/java/technologies/downloads/)安装 Java JDK 1.8+
4. 下载 Aspose.PDF for Python via Java API: [Aspose.PDF for Python via Java 组件](https://releases.aspose.com/pdf/pythonjava/)
5. 或者使用 pip 安装 Aspose.PDF for Python via Java API: [PyPI](https://pypi.org/project/aspose-pdf-for-python-via-java/)
