{{% alert color="primary" %}}

Aspose.PDF for JasperReports 需要以下软件才能运行：

- Java 2 Platform Standard Edition 5.0 或更高版本。

- JasperReports 6.7.0 或更高版本。

- JasperServer 6.7.0 或更高版本（如果您想将 Aspose.PDF for JasperReports 与 JasperServer 一起使用）。

{{% /alert %}}
