## 通用特性

- Aspose.PDF for Android via Java 支持任何运行 Java 运行时的移动设备，包括
- API 提供 40 位或 128 位加密

## 支持的 PDF 版本

Aspose.PDF for Android via Java 支持 PDF 版本 1.2、1.3、1.4、1.5、1.6 和 1.7。

## 图形

- 支持不同形状的图形，例如直线、圆形、弧线和曲线。
- 图形可以在由坐标指定的任意页面位置绘制。
- 图形可以旋转、缩放或填充，并且可以定义虚线和多段虚线样式。
- 图形基于坐标绘制，使用的是过时的绝对位置，而非相对位置。
- 支持 CMYK 和灰度颜色空间。
