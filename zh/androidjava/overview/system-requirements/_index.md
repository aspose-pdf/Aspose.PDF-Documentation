## 概述

为了生成 PDF 文档，运行 [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) 的机器不需要安装 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html)，因为 Aspose.PDF for Java 本身就是一个 PDF 文档创建引擎。但是，为了查看由 Aspose.PDF for Android via Java 生成的 PDF 文档，机器至少需要安装一个 [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html)。可以免费[下载](https://www.adobe.com/acrobat/pdf-reader.html)。

## 支持的操作系统

Aspose.PDF for Android via Java 支持运行 Java 运行时的任何移动设备，包括但不限于：

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## 开发环境

您可以使用 Aspose.PDF for Android via Java 在任何针对 Android 平台的 Java 开发环境中开发应用程序，但以下环境明确支持：

- Android Studio 和 SDK 工具
