## 概述

要生成 PDF 文档，需要运行的机器 [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) 不需要具备 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) 已安装的 Aspose.PDF for Java 本身是一个 PDF 文档创建引擎。然而，为了查看由 Aspose.PDF for Android 通过 Java 生成的 PDF 文档，机器至少需要一个 [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). 它可以 [下载](https://www.adobe.com/acrobat/pdf-reader.html) 免费。

## 支持的操作系统

Aspose.PDF for Android via Java 支持任何运行 Java 运行时的移动设备，包括但不限于：

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## 开发环境

您可以使用 Aspose.PDF for Android via Java 在任何针对 Android 上 Java 平台的开发环境中开发应用程序，但以下环境得到明确支持：

- Android Studio 和 SDK 工具
