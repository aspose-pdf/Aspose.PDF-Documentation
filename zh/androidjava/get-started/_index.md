## 什么是 PDF 文件？

PDF 代表可移植文档格式，用于以电子形式显示文档，且不依赖于查看它们的软件、硬件或操作系统。最初由 Adobe® 系统公司开发，作为基于 PostScript 格式的通用兼容文件格式，它已成为交换文档和信息的国际事实标准。

PDF 文件在任何操作系统、设备或软件应用程序上查看时，显示的内容和布局完全相同。

PDF 格式允许您集成各种类型的内容——文本、图像和矢量图形、视频、动画、音频文件、3D 模型、交互式字段、超链接、按钮等。

PDF 可以轻松创建、读取和使用，人人皆可。

提供设置不同访问级别的选项，以保护内容和整个文档，例如水印、密码或数字签名。

然而，仅仅打开文档是不够的。在处理 PDF 时，您将需要新建文档、编辑文档或将其转换为所需的格式。Aspose.PDF for Android via Java 将帮助您解决这些任务。

## 为什么使用 Aspose.PDF for Android via Java？

在您的项目中使用 Aspose.PDF for Android via Java 可为您带来以下收益：

- 功能种类繁多
- 必定可在 Java Oracle JDK 上运行
- 便利性和易用性
- JavaDoc 格式的 API 参考

Aspose.PDF for Android via Android via Java 可以集成到任何应用程序中，例如 JSP/JSF Web 应用或 Windows 应用程序。

Aspose.PDF for Android via Android via Java 速度快且轻量。它高效地创建 PDF 文档，并帮助您的应用程序提升性能。
