本节描述了一种快速简便的方法，通过使用 Aspose.PDF for Python via .NET 库以编程方式完成 PDF 文档。该节还讨论了如何使用 Aspose.PDF for Python 来发现并映射现有包含 AcroForms 的 PDF 中可用的字段。

AcroForms 允许用户电子化填写、提交和存储表单，使其成为收集信息的便捷高效方式。

- [AcroForms](/pdf/python-net/acroforms/) - 创建表单，填写表单字段，提取表单数据，在您的 PDF 中使用 Python 库添加和删除字段。
- [XFA Forms](/pdf/python-net/xfa-forms/) - 转换 XFA，使用 IgnoreNeedsRendering。
