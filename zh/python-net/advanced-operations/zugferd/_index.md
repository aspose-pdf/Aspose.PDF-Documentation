## 什么是 ZUGFeRD

电子发票是创建、发送、接收和处理电子发票文档的过程。它从头到尾完全数字化，有助于企业在 B2B 环境中节省成本并提高效率。买卖双方都能从电子发票中受益。

随着技术的进步，数百万公司正在采用电子发票以享受其优势。然而，他们需要标准来确保电子发票能够被任何软件轻松读取和理解，无论其生成方式如何。

在众多电子发票数据格式中，ZUGFeRD 标准在讲德语的国家和欧盟地区越来越得到认可。

ZUGFeRD 是德国的电子发票标准。它代表 “Zentraler User Guide des Forums elektronische Rechnung Deutschland”，即德国电子发票论坛的中央用户指南。

该格式是一种混合体，结合了人类可阅读的 PDF 文档和机器可读取的 XML 数据文件。它实现了企业与公共部门之间发票信息的统一且可互操作的交换。

它还符合长期归档和税务合规的法律要求。ZUGFeRD 拥有不同的版本和配置文件，以适应不同的需求和场景。最新版本是 ZUGFeRD 2.0，兼容欧洲电子发票标准 EN 16931。
ZUGFeRD 为用户提供诸多好处和成本节约，例如更快的处理速度、降低错误率、改善现金流以及降低环境影响。
