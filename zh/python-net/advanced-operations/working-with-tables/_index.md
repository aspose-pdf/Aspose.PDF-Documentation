**Aspose.PDF for Python via .NET** 使您能够高级地处理 PDF 文件中的表格。这个完美的工具通过提取表格来克服 PDF 的简单性。使用 Python 库，您可以轻松地在现有 PDF 文档中创建或添加表格，确定表格是否会在当前页面换行，提取表格，并从现有 PDF 中删除表格。

您可以执行以下操作：

- [添加表格](/pdf/python-net/adding-tables/) - 在 PDF 文件中创建表格，合并列或行，并考虑边框、边距和填充。
- [提取表格](/pdf/python-net/extracting-table/) - 您可以从 PDF 文件中提取表格。
- [将表格与数据源集成](/pdf/python-net/integrate-table/) - 将表格与数据库集成，使用 Python 库将表格与 Pandas DataFrame 源集成。
- [删除表格](/pdf/python-net/removing-tables/) - 从 PDF 文档中删除单个或多个表格。
- [在现有 PDF 中操作表格](/pdf/python-net/manipulating-tables/) - 使用 TableAbsorber 操作 PDF 中的表格。
