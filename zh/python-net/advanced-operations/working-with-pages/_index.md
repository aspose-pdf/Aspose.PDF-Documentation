**Aspose.PDF for Python via .NET** 允许您在文件中的任意位置向 PDF 文档插入页面，以及向 PDF 文件的末尾添加页面。本节展示了如何在没有 Acrobat Reader 的情况下向 PDF 添加页面。
您可以在 PDF 文件的页眉和页脚中添加文本或图像，并使用 Aspose 的 Python 库在文档中选择不同的页眉。
此外，尝试使用 Python 以编程方式裁剪 PDF 文档中的页面。

本节教您如何使用 Artifact 类在 PDF 文件中添加水印。您可以查看此任务的编程示例。
使用 [PageNumberStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/pagenumberstamp/) 类添加页码。要在文档中添加盖章，请使用 [ImageStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/imagestamp/) 和 [TextStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/textstamp/) 类。使用 **Aspose.PDF for Python via .NET** 添加水印以在 PDF 文件中创建背景图像。

您可以执行以下操作：

- [添加页面](/pdf/python-net/add-pages/) - 在所需位置或 PDF 文件末尾添加页面，并从文档中删除页面。
- [移动页面](/pdf/python-net/move-pages/) - 将页面从一个文档移动到另一个文档。
- [删除页面](/pdf/python-net/delete-pages/) - 使用 PageCollection 集合从 PDF 文件中删除页面。
- [提取页面](/pdf/python-net/extract-pages/) - 使用 Aspose.PDF for Python 从 PDF 文件中提取页面。
- [旋转页面](/pdf/python-net/rotate-pages/) - 您可以更改现有 PDF 文件中页面的方向。
- [更改页面大小](/pdf/python-net/change-page-size/) - 更改 PDF 文档的页面大小。
- [添加页眉和页脚](/pdf/python-net/add-headers-and-footers-of-pdf-file/) - 在 PDF 文件的页眉和页脚中添加文本或图像。
- [裁剪 PDF 页面](/pdf/python-net/crop-pages/) - 您可以使用不同的页面属性以编程方式裁剪 PDF 文档中的页面。
- [获取和设置页面属性](/pdf/python-net/get-and-set-page-properties/) - 访问并修改 PDF 页面各项属性，如尺寸、旋转和颜色属性。
- [盖章](/pdf/python-net/stamping/) - 通过 Python 向 PDF 页面添加图像盖章、页面盖章、页码盖章和文本盖章。
