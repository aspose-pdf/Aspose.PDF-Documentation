PDF 文档中的盖章类似于在纸质文件上使用橡胶章。
PDF 文件中的盖章提供额外信息，例如保护 PDF 文件不被他人使用并确认 PDF 内容的安全性。**Aspose.PDF for Python via .NET** 允许在您的 PDF 文档中添加图像或文字盖章。

请查看以下章节，了解如何使用 Python 添加盖章：

- [在 PDF 页面添加图像盖章](/pdf/python-net/image-stamps-in-pdf-page/) - 添加图像盖章，控制图像质量，将图像盖章设为 PDF 文件的背景。
- [在 PDF 文件添加文字盖章](/pdf/python-net/text-stamps-in-the-pdf-file/) - 添加文字盖章，定义 TextStamp 对象的对齐方式，在 PDF 文件中填充描边文字作为盖章。
- [在 PDF 文件添加页面盖章](/pdf/python-net/page-stamps-in-the-pdf-file/) - 添加页面盖章以覆盖整页或部分区域，可用于水印、文档版本控制或为 PDF 文件添加自定义背景。
- [在 PDF 文件添加页码盖章](/pdf/python-net/add-page-number/) - 向您的 PDF 文档添加页码，定制其外观，并控制其位置。
