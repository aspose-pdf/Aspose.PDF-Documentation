PDF 中的注释是交互式元素，允许您添加备注、突出显示文本、绘制形状、附加文件以及执行其他操作，从而提升浏览、交互性和互动体验。

Aspose.PDF for Python 库支持多种类型的注释，包括文本注释（例如文本和弹出注释）、高亮注释（例如文本标记）、图形注释（例如圆形、折线、多边形、直线和墨迹）、多媒体注释（包括屏幕、声音、控件和 3D）等。

注释可用于评论、审阅、标记文档、提供反馈或添加附加信息。注释有助于建立协作、促进沟通，并提升对文档内容的理解。它们增强协作性，改进交流，使 PDF 文档对用户而言更具活力和吸引力。
