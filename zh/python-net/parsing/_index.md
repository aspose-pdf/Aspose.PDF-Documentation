**Parse PDF** 文档是与从 PDF 文件提取各种信息相关的术语。本节介绍如何：

- [Extract Text from PDF](/pdf/python-net/extract-text-from-pdf/)。文本解析或提取是最常见的操作，适用于现成的 PDF。您将学习如何从整个文档、特定页面或页面的特定区域进行文本解析。
- [Extract Images from PDF](/pdf/python-net/extract-images-from-the-pdf-file/)。图像提取在功能上与上述文本提取相同。
- [Extract Fonts from PDF](/pdf/python-net/extract-fonts-from-pdf/)。字体提取是针对 PDF 中字体的特定操作。
- [Extract Data from the Form](/pdf/python-net/extract-data-from-acroform/)。如果您有大量包含表单的 PDF 文档，可能需要获取这些表单中的数据。本文将帮助您了解如何使用 Aspose.PDF for Python via .NET 提取 AcroForms 数据。
- [Extract Data from Table](/pdf/python-net/extract-data-from-table-in-pdf/)。获取 PDF 文档中表格的数据。
- [Extract Vector Data from PDF](/pdf/python-net/extract-vector-data-from-pdf/)。您可以获取矢量数据（路径、多边形、折线），例如位置、颜色、线宽等。
