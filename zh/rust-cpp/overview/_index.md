## Aspose.PDF for Rust via C++

_此页面是 Aspose.PDF for Rust 功能的概览._

Rust 是一种现代的系统级编程语言，旨在提供性能、可靠性和内存安全。它由 Mozilla Research 开发，现在由 Rust 基金会维护，Rust 旨在提供类似 C 和 C++ 等语言的强大控制力，同时避免常见的陷阱，如空指针解引用、缓冲区溢出和数据竞争。

Rust 以强大、安全且快速的编程语言脱颖而出。它非常适合需要对内存和性能进行细粒度控制的开发者，同时也重视现代开发实践和安全保证。

无论您是在构建高性能软件、安全系统，还是仅仅想探索下一代语言，Rust 都是值得学习和采用的强有力候选者。

使用 Aspose.PDF for Rust via C++ 库，您可以实现从头创建 PDF 文件的丰富功能，或在无需安装 Adobe Acrobat 的情况下完全处理现有的 PDF 文档。Aspose.PDF for Rust 支持 **Wide Variety of Functions**。

### 添加、删除或旋转页面

该 [处理 PDF 页面](/pdf/zh/rust-cpp/working-with-pages/) 本节说明了如何在 PDF 中添加、删除、旋转页面。

您可以添加、删除和旋转 PDF 页面，获取和设置页面属性，并设置背景颜色，从而提升文档的组织性和可访问性。

### 将 PDF 页面转换为其他格式

将 PDF 页面转换为图像可以让内容对那些可能难以直接查看或交互 PDF 文档的用户更加友好。Aspose.PDF 允许使用 Rust 将 PDF 转换为不同的格式。包括：

- TIFF
- BMP
- JPG
- PNG
- DICOM
- SVG
- DOC
- DOCX
- EPUB
- PPTX
- TEX
- TXT
- XLSX
- XPS

您可以访问 [转换 PDF 格式](/pdf/zh/rust-cpp/conversion/) 页面了解详情。
