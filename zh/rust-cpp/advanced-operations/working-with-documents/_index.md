PDF 代表可移植文档格式，用于以电子形式显示文档，而不受查看它们的软件、硬件或操作系统的限制。

PDF 是一种开放标准，现由国际标准化组织（ISO）维护。

最初的目标是保留和保护文档的内容和布局——不论在何种平台或计算机程序上查看。这就是 PDF 难以编辑，有时甚至提取信息都颇具挑战性的原因。

但是 **Aspose.PDF for Rust via C++** 可以帮助您应对在处理 PDF 文档时出现的大多数任务。

您可以执行以下操作：

- [优化 PDF](/pdf/zh/rust-cpp/optimize-pdf/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现成功的文档优化。
- [优化 PDF 资源](/pdf/zh/rust-cpp/optimize-pdf-resources/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现成功的文档优化。
- [PDF 排版](/pdf/zh/rust-cpp/pdf-imposition/) - 如何使用 Aspose.PDF 将 PDF 文件制作成小册子或 N-Up。
