## 跨平台软件

支持各种平台和 Rust 版本。请参阅 [系统要求](/pdf/zh/rust-cpp/system-requirements/) 详情请参阅。

## 通用功能

支持从广泛的列表中加载和保存 [文件格式](/pdf/zh/rust-cpp/supported-file-formats/).

## 支持的 PDF 标准

Aspose.PDF for Rust 支持 PDF 版本 1.2、1.3、1.4、1.5、1.6 和 1.7。

- new PDF
- open PDF
- save PDF
- save_as
- page_insert
- page_add
- page_count
- page_delete
- extract_text
- set_license

- optimize

- grayscale
- rotate

- repair
- page_add_text

- PDF, DOC, DOCX, XLSX, PPTX, JPEG, PNG, BMP, TIFF, EPUB, DICOM, SVG, XPS, TEX, TXT
