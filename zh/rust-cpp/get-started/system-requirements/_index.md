## Aspose.PDF for Rust via C++ 的支持平台

asposepdf 包是一个强大的工具包，允许开发人员直接操作 PDF 文件并帮助完成各种 PDF 任务。
包含将 PDF 转换为其他格式的独特功能。

已实现对 Linux x64、macOS x86_64、macOS arm64 和 Windows x64 平台的支持。

在包根目录的 'lib' 文件夹中，需要平台特定版本的动态库来分发生成的应用程序： 

- 适用于 Linux x64 平台的 libAsposePDFforRust_linux_amd64.so 
- 适用于 macOS arm64 平台的 libAsposePDFforRust_darwin_arm64.dylib 
- 适用于 macOS x86_64 平台的 libAsposePDFforRust_darwin_amd64.dylib 
- AsposePDFforRust_windows_amd64.dll 用于 Windows x64 平台。
