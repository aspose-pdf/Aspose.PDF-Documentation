XFA 代表 XML 表单架构。它是一组专有的 XML 规范，最初于 1999 年用于网页表单，现在已可用于 PDF。

要更详细地研究使用表单的可能性，请研究本节中的以下文章：

-[转换 XFA 表单](/pdf/zh/php-java/convert-form/) - 使用 PHP 将 XFA 转换为 AcroForm。

-[填写 AcroForm](/pdf/zh/php-java/fill-form/) - 要填写表单字段，从 Document 对象的 Form 集合中获取字段。

-[提取数据 AcroForm](/pdf/zh/php-java/extract-form/) - 获取所有和单个字段的值等。
