## 什么是 PDF 文件？

PDF 由 Adobe 于 1990 年代创建，旨在实现两件事。其一是您应该能够在任何硬件或操作系统上打开文档，而无需拥有创建它们的应用程序——只需一个 PDF 阅读器，而如今大多数网页浏览器都能满足这一需求。其二是无论在何处打开 PDF，文档的布局都应保持一致。

然而，仅仅打开文档并不足够。在处理 PDF 时，您会需要创建全新的文档、编辑文档或将其转换为所需的格式。

## 为什么使用 Aspose.PDF for Node.js via C++？

在您的项目中使用 Aspose.PDF for Node.js 可为您带来以下收益：

- 功能种类繁多
- 用于在 Node.js 环境中处理 PDF 的工具包
- 基于 Aspose.PDF for .NET
