PDF 代表 Portable Document Format，用于在电子形式中显示文档，且不依赖于观看时所使用的软件、硬件或操作系统。

PDF 是一种开放标准，现由国际标准化组织（ISO）维护。

最初的目标是保留和保护文档的内容和版式——无论在何种平台或计算机程序上查看。这也是 PDF 难以编辑，有时甚至从中提取信息也颇具挑战性的原因。

但是 **Aspose.PDF for Node.js via C++** 可以帮助您应对在处理 PDF 文档时出现的大多数任务。

您可以执行以下操作：

- [优化 PDF](/pdf/zh/nodejs-cpp/optimize-pdf/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现成功的文档优化。
- [优化 PDF 资源](/pdf/zh/nodejs-cpp/optimize-pdf-resources/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现成功的文档优化。
- [合并 PDF](/pdf/zh/nodejs-cpp/merge-pdf/) - 使用 Node.js 将多个 PDF 文件合并为一个 PDF 文档。
- [拆分 PDF](/pdf/zh/nodejs-cpp/split-pdf/) - 在 Node.js via C++ 中将 PDF 页面拆分为单独的 PDF 文件。
