_此页面概述了 Aspose.PDF for Node.js via C++ 的功能。_

**Aspose.PDF for Node.js via C++** 允许开发人员直接在 Node.js 环境中操作 PDF 文件。此 Node.js 解决方案基于 WebAssembly 技术并基于 Aspose.PDF for .NET。您可以轻松使用 aspose-pdf for Node.js via C++，然后遵循安装说明。

**关于 CJS 和 MJS 文件扩展名**

Node.js 支持两种模块系统用于组织和共享代码。它们是：

- CommonJS 模块
- ECMAScript 模块

CommonJS 模块是 Node.js 中经典的依赖处理方式，而 ECMAScript 模块是较新的补充，并遵循现代浏览器使用的标准格式。'cjs'、'mjs' — 是用于在 JavaScript 文件中指定模块格式（CommonJS 和 ECMAScript 模块）的扩展名。 

在代码示例中，你会注意到使用 AsposePDFforNode.js 模块的差异。'require' 与 'import' 的主要区别在于它们的使用场景。**Require** 只能导入整个模块，而 **Import** 则灵活地可以导入整个模块以及该模块中的特定导出。

**两种不同的方法**

JavaScript 提供两种方法，'then' 和 'await'，用于处理异步操作。

‘then’ 方法配合 Promise 使用，以顺序方式管理成功结果和错误。相反，‘await’ 让 JavaScript 等待，实际暂停函数执行，直至该 Promise 解决并返回其结果。

Aspose.PDF for Node.js 支持多种功能，例如：

- 设置、获取和删除 PDF 文件信息
- 删除书签
- 删除批注
- 从 PDF 中提取文本
- 优化 PDF 文档
- 优化资源 
- 将 PDF 拆分为两个文件。
- 将两个文件合并为 PDF
- 修复 PDF
- 转换功能

此外，Aspose.PDF for Node.js via C++ 可用于轻松将 PDF 文档转换为各种格式，具备高性能和完美质量。参见本节 [Aspose.PDF 支持的文件格式](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). 同时，此工具包支持对 PDF 文档进行加密和解密。
