## 通用特性

- **Aspose.PDF for Node.js via C++** 基于 WebAssembly 技术构建，并且基于 Aspose.PDF for .NET
- 设置、获取和删除 PDF 文件信息
- 删除书签
- 删除批注
- 从 PDF 中提取文本
- 优化 PDF 文档
- 优化资源 
- 将 PDF 拆分为两个文件
- 将两个文件合并为 PDF
- 修复 PDF

## 支持的 PDF 版本

Aspose.PDF for Node.js 支持 PDF 版本 1.2、1.3、1.4、1.5、1.6 和 1.7。

## PDF 安全功能

API 提供 40 位或 128 位加密。还可以设置用户密码和主密码以增强安全性。通过使用此库，可以限制对文件内容的操作，例如复制内容、打印、受限打印、注释修改、表单字段填写以及 PDF 文件的合并。

- 加密 PDF 文件
- 解密 PDF 文件
- 更改密码
- 签署 PDF
