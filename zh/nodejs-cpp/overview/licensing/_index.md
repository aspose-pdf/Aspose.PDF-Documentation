## 评估版的限制

我们希望客户在购买前彻底测试我们的组件，因此评估版允许您像平常一样使用它。

>如果您想在不受评估版限制的情况下测试 Aspose.PDF for Node.js via C++，也可以请求 30 天的临时许可证。请参阅 [如何获取临时许可证？](https://purchase.aspose.com/temporary-license/)

- **PDF 已创建并附有评估水印。**
Aspose.PDF for Node.js via C++ 的评估版本提供完整的产品功能，但生成的 PDF 文档的所有页面顶部都会被水印标记为 "Evaluation Only. Created with Aspose.PDF. Copyright 2002-2023 Aspose Pty Ltd."

- **可以处理的集合项数量限制。**
在任何集合的评估版本中，您只能处理四个元素（例如，仅 4 页、4 个表单字段等）。


## 经典许可证

许可证文件名必须为 "aspose.lic". 您需要将收到的许可证文件重命名，并将其放置在 Product 文件夹中。
