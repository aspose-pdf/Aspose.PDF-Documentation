## 评估版本的限制

我们希望客户在购买前彻底测试我们的组件，因此评估版本允许您正常使用。

>如果您希望在没有评估版本限制的情况下测试 Aspose.PDF for Node.js via C++，您还可以申请 30 天的临时许可证。请参阅[如何获取临时许可证？](https://purchase.aspose.com/temporary-license/)

- **带有评估水印的 PDF 创建。**
Aspose.PDF for Node.js via C++ 的评估版本提供完整的产品功能，但生成的 PDF 文档中的所有页面都会被标记为“仅供评估”。
 创建者：Aspose.PDF。版权所有 2002-2023 Aspose Pty Ltd。

- **可以处理的集合项目数量限制。**
在评估版本中，从任何集合中，您只能处理四个元素（例如，只有4页，4个表单字段等）。

## 经典许可证

许可证文件名必须为 "aspose.lic"。您需要重命名收到的许可证文件，并将其放置在产品文件夹中。
