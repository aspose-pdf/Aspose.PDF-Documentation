## 概述

使用该 [Aspose.PDF for Node.js via C++](https://products.aspose.com/pdf/nodejs-cpp/), 不需要第三方库和程序。
需要已安装 [Node.js](https://nodejs.org/). 您必须拥有 Node.js 版本 16 或更高。我们强烈建议使用 LTS 版本。
Node.js 可以 [已安装](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) 以不同方式。

## 受支持的操作系统

所有受支持的版本 [Linux](https://en.wikipedia.org/wiki/Linux) 和 [macOS](https://www.apple.com/macos/).
