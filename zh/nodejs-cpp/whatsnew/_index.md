我们很高兴宣布 Aspose.PDF for Node.js via C++ 的首次公开发布。

**Aspose.PDF for Node.js via C++** 允许开发者直接在 Web 上操作 PDF 文件。此 Node.js 解决方案基于 WebAssembly 技术构建，并依赖于 Aspose.PDF for .NET。您可以轻松使用 aspose-pdf for Node.js via C++，然后遵循安装说明。

请访问 [产品页面](https://products.aspose.com/pdf/nodejs-cpp/) 了解更多详情，并从 [Aspose.PDF for Node.js via C++ 下载页](https://releases.aspose.com/pdf/nodejscpp/) 获取首个发布版。

## **主要支持的功能**

- 设置、获取和删除 PDF 文件信息
- 删除书签
- 删除批注
- 从 PDF 中提取文本
- 优化 PDF 文档
- 优化资源 
- 将 PDF 拆分为两个文件
- 将两个文件合并为 PDF
- 修复 PDF
