**Aspose.PDF 类库有三个版本：**

[Aspose.PDF](/pdf/) 是一个产品系列，提供从头生成 PDF 文档的功能，以及编辑/操作现有文档的能力。Aspose.PDF 有三个不同的产品：[Aspose.PDF for .NET](/pdf/net/)、[Aspose.PDF for Java](/pdf/java/) 和 [Aspose.PDF for C++](/pdf/cpp/)。我们可以说 Aspose.PDF “原生”支持：

- .NET Framework 及所有针对 .NET 平台的编程语言，如 C# 或 VB.NET。
- Java 平台和 Java 编程语言。
- 32 位和 64 位操作系统以及 C++ 编程语言。

{{% /alert %}}

请注意，还可以从许多编程语言中使用 [Aspose.PDF for .NET](/pdf/net/)，包括：ASP、C++、ColdFusion、Delphi、JavaScript、Perl、PHP、PowerBuilder、Python、VBScript 等。

**基于 Aspose.PDF for .NET 类库的产品：**

除了上述三个产品外，Aspose 还提供了一套适用于不同平台和语言的解决方案：

- [Aspose.PDF for Python via .NET](/pdf/python-net/) 用于开发适用于不同操作系统的 32 位和 64 位 Python 应用程序。
- [Aspose.PDF for Reporting Services](/pdf/reportingservices/) 使在 Microsoft SQL Server Reporting Services 和 Power BI Report Server 中生成 PDF 报告成为可能。
- [Aspose.PDF for SharePoint](/pdf/sharepoint/) 允许将列表、列表项和 SharePoint Wiki 页面导出为 PDF 文件格式。

Aspose.PDF 产品的完整列表可以在 [此页面](/pdf/) 找到。它包括基于 [Aspose.PDF for Java](/pdf/java/) 和 PDF [Aspose.PDF for C++](/pdf/cpp/) 的产品。

**从其他语言访问 Aspose.PDF 对象**

从其他编程语言使用 Aspose.PDF 需要一种 *互操作性技术*。该技术必须使您的语言能够调用为 .NET、Java 或 C++ 编写的类库。我们建议使用 [COM Interop](/pdf/net/use-aspose-pdf-for-net-via-com-interop/) 技术与 [Aspose.PDF for .NET](/pdf/net/) 类库结合使用，以适用于多种编程语言：
- ASP。
- Delphi。
- Perl。
- PHP。
- PowerBuilder。
- VBScript。
- Visual Basic。

有关推荐与 [Aspose.PDF for Java](/pdf/java/) 和 PDF [Aspose.PDF for C++](/pdf/cpp/) 一起使用的互操作性技术的信息，如 `Inline::Java` 和 `JPype`，可以在相应的 [页面](/pdf/java/plugins/) 中找到。
