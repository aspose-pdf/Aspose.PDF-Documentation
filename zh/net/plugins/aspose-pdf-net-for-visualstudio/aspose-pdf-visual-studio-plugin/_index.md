## 介绍

Microsoft Visual Studio 是 Microsoft 的集成开发环境 (IDE)。它用于为 Microsoft Windows 开发计算机程序，以及网站、Web 应用程序和 Web 服务。Visual Studio 使用 Microsoft 软件开发平台，如 Windows API、Windows 窗体、Windows Presentation Foundation、Windows Store 和 Microsoft Silverlight。它可以生成本机代码和托管代码。

### Aspose.PDF for .NET

Aspose.PDF for .NET 是一个 PDF 文档创建和操作组件，使您的 .NET 应用程序能够读取、写入和操作现有 PDF 文档，而无需使用 Adobe Acrobat。

有关 [Aspose.PDF for .NET API](https://products.aspose.com/pdf/net/) 的更多详细信息。

### 使用 Visual Studio 探索 Aspose.PDF for .NET 示例

Aspose Visual Studio 插件 - **探索 Aspose.PDF for .NET 示例** 是一个很好的工具，可以快速下载和探索 Aspose.PDF for .NET API 示例。它通过提供一个非常简单的选项，无缝选择、下载和打开最新的示例项目，节省了大量时间和精力，而无需离开您的 Visual Studio。

此版本提供以下功能：

- 支持 Visual Studio 2010、Visual Studio 2012 和 Visual Studio 2013。
- 可以从 Visual Studio 的 **文件 > Aspose** 菜单轻松启动。
- 节省时间并减少学习曲线。
- 使您能够选择并打开托管在 Github.com 上的最新 Aspose.PDF for .NET API 示例。
- 允许您打开 **C#** 或 **VB.NET** 示例项目。
- 选定的项目会自动在 Visual Studio 中打开，并设置所有所需的引用，以便为您提供一个可以运行和探索的环境。
- 自动从 **NuGet** 下载最新的 API 库和来自 **GitHub** 的示例。

## 安装

安装 Aspose Visual Studio 插件非常简单，请按照以下简单步骤操作

- 双击下载的 msi 文件。
- 按照步骤操作，您可以在每个屏幕上简单地按“下一步”以使用标准设置进行安装。

安装完成后，您将收到成功对话框。

![Aspose.PDF for .NET 示例安装对话框](aspose-pdf-visual-studio-plugin_2.png)

**注意：** 请确保重新启动 Visual Studio 以使更改生效。

## 使用

安装后，您可以轻松启动和使用交互式插件，如下所述：
**1.** 通过在 Visual Studio 中选择 **文件 > Aspose**，然后选择 **新建 Aspose.PDF 示例项目** 来打开插件。

![新建 Aspose.PDF 示例项目对话框](aspose-pdf-visual-studio-plugin_3.png)

**2.** 第一次打开插件时，您会看到如下所示的 API 选择对话框。它将花费几分钟时间从 **NuGet** 下载最新的 API 及其库，以及从 **GitHub** 下载最新的示例源代码。

![Aspose.PDF 示例下载对话框](aspose-pdf-visual-studio-plugin_4.png)

**3.** 在选择要查看示例的 API 对话框中，选择或更改示例项目将创建的位置，并选择或更改示例项目类型为 **C#** 或 **VB.NET**，然后单击 **继续** 在 Visual Studio 中打开示例项目。

![Aspose.PDF 示例项目选择文件夹对话框](aspose-pdf-visual-studio-plugin_1.png)

**4.** 根据在 Visual Studio 中选择的项目类型 **C#** 或 **VB.NET** 创建新的示例项目。

#### 如何配置源代码

您需要安装以下内容才能打开和扩展源代码

- Visual Studio 2010。

请按照以下简单步骤开始

1. 下载/克隆源代码。
1. 打开 Visual Studio 2010 并选择 **文件** > **打开项目**。
1. 浏览到您下载的最新源代码并打开 **AsposeVisualStudioPlugin.sln**。

#### 源代码概述

解决方案中有两个项目

- **AsposeVisualStudioPlugin** - 包含与 Visual Studio 的集成、选择和自动下载 Aspose API 和示例。
- **Installer** - Wix 安装程序项目，用于生成插件的 msi 安装文件。
