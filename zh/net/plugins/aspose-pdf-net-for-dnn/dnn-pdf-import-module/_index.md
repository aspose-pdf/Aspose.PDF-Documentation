## 介绍

Aspose .NET PDF 导入模块允许开发人员获取/读取 PDF 文档的内容，而无需任何其他软件，如 Adobe Acrobat 或 PDF 阅读器。

### 模块特性

该模块演示了 [Aspose.PDF](https://products.aspose.com/pdf/net/) 提供的强大导入功能。它在添加模块的页面上添加了一个简单的文件浏览器控件、内容面板选择和 **从 PDF 导入** 按钮。点击该按钮后，文档内容会立即显示在屏幕上。

## 系统要求和支持的平台

### 系统要求

为了设置 Aspose .NET PDF 导入模块，您需要满足以下要求：

- DNN 7.0+。

如果您希望在其他版本的 DNN 上安装此模块，请随时与我们联系。

Aspose .NET PDF 导入模块目前支持

- DNN 7.0+。

如果您希望在其他版本的 DNN 上安装此模块，请随时与我们联系。

## 安装

下载后，请按照以下步骤将模块安装到您的 DNN 网站中：

1. 以 **Host** 或其他超级用户级别帐户登录到您的网站。
1. 导航到主机菜单并选择 **扩展**。
1. 点击 **安装扩展向导**。
1. 按照指示浏览到下载的 zip 文件的位置，选择它，然后点击 **打开**。
1. 点击 **下一步**，接受许可，继续安装。完成后点击 **返回** 按钮。

有关更多详细信息，请查看 DNN 的 [此模块安装视频](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)。

**注意：** 如果在上传模块时出现错误，这是由于您 DNN 安装的 web.config 中的 maxRequestLength 限制。打开 web.config 并通过设置 **maxRequestLength=”20480″** 将 maxRequestLength 更新为 20MB，然后再次尝试上传模块。

## 使用

安装 Aspose .NET PDF 导入模块后，在您的网站上开始使用它非常简单。请按照以下简单步骤开始

1. 确保您以 Host 或 Admin 级别帐户登录到 DNN。
1. 导航到您想要添加导入模块的页面。
1. 从顶部功能区选择 **模块** -> **添加新模块**。

1. 从列表中选择 **Aspose .NET PDF 导入模块** 并将其拖动到页面上您选择的位置。

您已成功将 Aspose .NET PDF 导入模块添加到您的页面。页面上将出现一个文件浏览和一个名为 **从 PDF 导入** 的按钮，以及内容面板选择下拉菜单。任何人都可以简单地选择一个 PDF 文件并点击 **从 PDF 导入** 按钮，以在页面上列出所选文档的内容。
