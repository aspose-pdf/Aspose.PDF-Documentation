## 什么是 Microsoft Dynamics CRM？

Microsoft Dynamics CRM 是由 Microsoft 开发的客户关系管理软件包。开箱即用，该产品主要关注销售、市场营销和服务（帮助台）领域，但 Microsoft 一直将 Dynamics CRM 市场化为 XRM 平台，并鼓励合作伙伴使用其专有的（基于 .NET 的）框架进行自定义。它是 Microsoft Dynamics 商业应用程序系列的一部分。

Dynamics CRM 是一个服务器-客户端应用程序，类似于 Microsoft SharePoint，主要是一个基于 IIS 的 Web 应用程序，同时也支持广泛的 Web 服务接口。客户端可以通过浏览器或 Microsoft Outlook 的厚客户端插件访问 Dynamics CRM。

## Aspose.PDF for .NET

Aspose.PDF for .NET 是一个 PDF 文档创建和处理组件，使您的 .NET 应用程序能够读取、写入和处理现有的 PDF 文档，而无需使用 Adobe Acrobat。
