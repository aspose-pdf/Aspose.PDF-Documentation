## 概览
为了生成和操作 PDF 文档，运行 Aspose.PDF for .NET 的机器不需要安装 Adobe Acrobat：Aspose.PDF for .NET 是一个 PDF 文档创建引擎。然而，为了查看由 Aspose.PDF for .NET 生成的 PDF 文档，机器至少需要安装 Adobe Acrobat Reader。它可以免费下载。

## 开发环境
您可以在任何针对.NET平台的开发环境中使用Aspose.PDF for .NET开发应用程序，但以下环境得到了明确支持：

- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2011
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022
