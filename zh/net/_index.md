## 欢迎来到 Aspose.PDF for .NET

{{% alert color="primary" %}}

Aspose.PDF 是一个 .NET 类库，允许开发人员以编程方式动态创建 PDF 文档，无论是简单的还是复杂的。Aspose.PDF for .NET 允许开发人员将表格、图形、图像、超链接、自定义字体等插入 PDF 文档中。此外，还可以压缩 PDF 文档。该产品遵循所有 PDF 标准，包括 ISO 32000-1 (PDF 1.7)、ISO 32000-2 (PDF 2.0) 以及许多扩展，如 PDF/A、PDF/UA、PDF/X、PDF/E。

Aspose.PDF for .NET 提供以下关键功能：

- 从 HTML、Doc/Docx、Pptx、Markdown、Json、Jpeg 等格式转换文档。
- 支持与各种 AI 提供商的集成，能够从 PDF 文档加载数据。
- 通过 API 和 XML 文件创建 PDF 文档。
- 出色的基于证书的安全性和密码保护功能，以开发安全的 PDF 文档。
- 兼容强大的数字签名（包括 PAdES）。
- PDF 表单创建（交互式表单、AcroForms、动态 XFA 表单）。

还有更多功能。
