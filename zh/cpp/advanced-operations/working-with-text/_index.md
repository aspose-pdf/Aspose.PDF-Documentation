本部分文档向您展示如何在 PDF 中添加和格式化文本，如何旋转或替换文本，如何使用 Aspose.PDF for C++ 在文件中添加 PDF 工具提示。

Aspose.PDF 拥有处理 PDF 文档中文本内容的最佳 C++ 库。

您可以执行以下操作：

- [向 PDF 文件添加文本](/pdf/zh/cpp/add-text-to-pdf-file/) - 向 PDF 添加文本，使用流和文件中的字体，添加 HTML 字符串，添加超链接等。
- [PDF 工具提示](/pdf/zh/cpp/pdf-tooltip/) - 您可以通过使用 C++ 添加不可见按钮来为搜索到的文本添加工具提示。
- [PDF 内部的文本格式化](/pdf/zh/cpp/text-formatting-inside-pdf/) - 在格式化文档内部的文本时，您可以为文档添加许多功能。 添加行缩进，添加文本边框，添加下划线文本，使用 Aspose.PDF 库添加换行符。
- [在 PDF 中替换文本](/pdf/zh/cpp/replace-text-in-pdf/) - 在 PDF 文档的所有页面中替换文本。您首先需要使用 TextFragmentAbsorber。
- [旋转 PDF 内的文本](/pdf/zh/cpp/rotate-text-inside-pdf/) - 使用 TextFragment 类的旋转属性旋转 PDF 内的文本。
- [搜索并获取 PDF 文档页面中的文本](/pdf/zh/cpp/search-and-get-text-from-pdf/) - 您可以使用 TextFragmentAbsorber 类来搜索和获取页面中的文本。
- [确定换行符](/pdf/zh/cpp/determine-line-break/) - 本主题解释如何跟踪多行文本片段的换行。
