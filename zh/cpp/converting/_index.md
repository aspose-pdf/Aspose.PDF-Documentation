**Aspose.PDF for C++** 库使您能够将 PDF 文档转换为各种格式，并从其他格式转换为 PDF。

我们的 **Aspose.PDF for C++** 库允许您成功、快速和轻松地将您的 PDF 文档转换为最流行的格式，反之亦然。

## 如何使用 Aspose.PDF 进行转换

Aspose.PDF 支持最多数量的流行文档格式，既可以加载也可以保存。

请注意当前部分仅描述流行的转换。
有关支持的格式的完整列表，请参见 [支持的 Aspose.PDF 文件格式](https://docs.aspose.com/pdf/cpp/supported-file-formats/)。

探索带有代码片段的文档转换章节。

Word 文档是尽可能多功能且可编辑的。手动将 PDF 转换为 Word 是一项非常耗时的任务。本文将向您展示如何在 C++ 中以编程方式将 PDF 转换为 Word。

- [将 PDF 转换为 Microsoft Word](/pdf/cpp/convert-pdf-to-word/) - 您可以使用 C++ 将 PDF 文档转换为 Word 格式

数字格式不仅用于使表格中的数据更易于阅读，还用于使表格更易于使用。当然，如果您需要将此类数据从 PDF 文档转换为 Excel 格式，请使用我们的 Aspose.PDF 库。

- [将 PDF 转换为 Microsoft Excel](/pdf/cpp/convert-pdf-to-excel/) - 本节介绍如何将 PDF 文档转换为 XLSX、ODS、CSV 和 SpreadSheetML

PowerPoint 格式用于创建各种演示文稿。PPT 文件包含大量幻灯片或页面，包含各种信息。

- [将 PDF 转换为 Microsoft PowerPoint](/pdf/cpp/convert-pdf-to-powerpoint/) - 在这里我们讨论通过跟踪转换过程将 PDF 转换为 PowerPoint

超文本标记语言是一种超文本文档描述语言，是创建网页的标准语言。使用 Aspose.PDF for C++，您可以轻松转换 HTML 文档。

- [将 PDF 文件转换为 HTML 格式](/pdf/cpp/convert-pdf-to-html/) - 将您的 PDF 文档转换为 HTML 文件，作为单独的页面或单个页面

有许多图像格式需要出于不同目的转换为 PDF。Aspose.PDF 支持最流行的图像格式，反之亦然。

- [将图像格式转换为 PDF 文件](/pdf/cpp/convert-images-format-to-pdf/) - 将不同格式的图像转换为 PDF 文件
- [将 PDF 转换为各种图像格式](/pdf/cpp/convert-pdf-to-images-format/) - 将 PDF 页面转换为 JPEG、PNG 和其他格式的图像

本节包括以下格式：EPUB、Markdown、PCL、XPS、LATex/TeX、文本和 PostScript。

- [将其他文件格式转换为 PDF](/pdf/cpp/convert-other-files-to-pdf/) - 本主题描述了与 EPUB、XPS、Postscript、文本和其他格式的转换。
- [将 PDF 文件转换为其他格式](/pdf/cpp/convert-pdf-to-other-files/) - 本主题描述了将 PDF 文档转换为各种格式的方法

PDF/A 是 PDF 的一个版本，专为电子文档的长期归档而设计。坦率地说，从外部来看，确定它是 PDF 还是 PDF/A 是非常困难的。为了检查此文件，会使用验证器。查看以下文章以实现高质量的 PDF 到 PDF/A 的转换，反之亦然。

- [将 PDF 转换为 PDF/A 格式](/pdf/cpp/convert-pdf-to-pdfa/) - Aspose.PDF 的 C++ 库提供了一种简单的方法将 PDF 转换为 PDF/A
