## 一般特性

- 完全用 C++ 编写
- 需要 C++ 环境。
- 支持 Windows Forms 和 ASP.C++ 应用程序。
- HTML 和 Microsoft Help 格式的 API 参考。
- 32 位操作系统
- 64 位操作系统

## 支持的 PDF 版本

Aspose.PDF for C++ 支持 PDF 版本 1.2, 1.3, 1.4, 1.5, 1.6 和 1.7。
