{{% alert color="primary" %}}

Aspose.PDF for Java 库无法支持具有特定语言-国家组合的区域设置（例如 "en-KR"）。但是，API 中存在为 Aspose.PDF 设置经典区域设置的功能，可以通过调用 com.aspose.pdf.LocaleOptions.setLocale() 方法来使用。
