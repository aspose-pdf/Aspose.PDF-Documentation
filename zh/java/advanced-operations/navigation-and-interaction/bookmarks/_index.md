PDF 书签是阅读 PDF 文件时一个方便且必不可少的工具。它们允许用户快速跳转到 PDF 文档中的其他位置、在页面之间导航以及快速查看 PDF 的内容，就像目录一样。
在本节中，您将学习如何：

- [添加和删除书签](/pdf/zh/java/add-and-delete-bookmark/)
- [获取、更新和展开书签](/pdf/zh/java/get-update-and-expand-bookmark/)
