## Aspose.PDF - 将SVG转换为PDF

要使用**Aspose.PDF Java for PHP**将SVG文件转换为PDF格式，只需调用**SvgToPdf**模块。

PHP代码

```php
# 使用SVG加载选项实例化LoadOption对象
$options = new SvgLoadOptions();

# 创建文档对象
$pdf = new Document($dataDir . 'Example.svg', $options);

# 将输出保存为XLS格式
$pdf->save($dataDir . "SVG.pdf");

print "文档已成功转换";

```
