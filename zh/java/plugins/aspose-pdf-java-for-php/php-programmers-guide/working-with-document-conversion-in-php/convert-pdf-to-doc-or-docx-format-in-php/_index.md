## Aspose.PDF - 将PDF转换为DOC或DOCX

要使用 **Aspose.PDF Java for PHP** 将PDF文档转换为DOC或DOCX格式，只需调用 **PdfToDoc** 模块。

PHP代码

```php

# 打开目标文档
$pdf = new Document($dataDir . 'input1.pdf');

# 保存合并后的输出文件（目标文档）
$pdf->save($dataDir . "output.doc");

print "文档已成功转换";

```
