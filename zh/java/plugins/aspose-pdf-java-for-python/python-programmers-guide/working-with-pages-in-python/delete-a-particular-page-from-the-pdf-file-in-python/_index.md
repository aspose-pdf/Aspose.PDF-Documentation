要使用 **Aspose.PDF Java for Python** 从 PDF 文档中删除特定页面，只需调用 **DeletePage** 类。

```python

doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'

# 删除特定页面
pdf.getPages().delete(2)

# 保存新生成的 PDF 文件
doc.save(self.dataDir + "output.pdf")

print "页面删除成功!"

```
