要在 Python 中使用 **Aspose.PDF Java for Python** 优化 PDF 文档以适应网络，只需调用 **Optimize** 类的 **optimize_web** 方法。

```python

doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'

# 优化以适应网络
doc.optimize();

# 保存输出文档
doc.save(self.dataDir + "Optimized_Web.pdf")

print "优化后的 PDF 适用于网络，请检查输出文件。"
```
