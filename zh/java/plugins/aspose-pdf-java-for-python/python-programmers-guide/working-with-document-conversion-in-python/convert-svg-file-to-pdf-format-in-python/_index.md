## 如何在Python中将SVG文件转换为PDF格式

要使用**Aspose.PDF Java for Python**将SVG文件转换为PDF格式，只需调用**SvgToPdf**模块。

Python代码：

```python
options = self.SvgLoadOptions();
doc=self.Document()
pdf = self.Document()
pdf=self.dataDir +'input1.pdf'
将输出保存为XLS格式
doc.save(self.dataDir + "SVG1.pdf");
print "文档已成功转换"
```
