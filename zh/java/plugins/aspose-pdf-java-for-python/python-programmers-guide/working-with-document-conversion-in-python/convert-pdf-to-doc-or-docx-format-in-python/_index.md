要使用**Aspose.PDF Java for Python**将PDF文档转换为DOC或DOCX格式，只需调用**PdfToDoc**模块。

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "Document has been converted successfully"
```
