importBookmarksWithXml 方法允许您从 XML 文件中导入书签到 PDF 文件。

要导入书签：

1. 创建一个 PdfBookmarkEditor 对象，并使用 bindPdf 方法绑定 PDF 文件。
1. 调用 importBookmarksWithXml 方法。
1. 使用 save 方法保存更新后的 PDF 文件。
