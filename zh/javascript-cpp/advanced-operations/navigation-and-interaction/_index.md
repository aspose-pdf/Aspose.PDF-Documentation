- [书签](/pdf/zh/javascript-cpp/bookmark/) - 大型出版物通常包含一个书签框架，可以在书签窗格中轻松查看和选择，允许您点击书签跳转到它所代表的页面或章节。书签窗格是一个内容感知元素，仅当打开的 PDF 文档包含书签结构时才在侧边栏中可见。
