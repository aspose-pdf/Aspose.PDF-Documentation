## 通用功能

- **Aspose.PDF for JavaScript via C++** 构建于 WebAssembly 技术之上，基于 Aspose.PDF for .NET
- 设置、获取和删除 PDF 文件信息
- 删除书签
- 删除注释
- 删除 JavaScripts
- 从 PDF 中提取文本
- 优化 PDF 文档
- 优化资源
- 将 PDF 拆分为两个文件
- 合并两个文件为 PDF
- 修复 PDF

## 支持的 PDF 版本

Aspose.PDF for JavaScript 支持 PDF 版本 1.2, 1.3, 1.4, 1.5, 1.6 和 1.7。

## 超链接和附件功能

工具包允许添加：

- 从 PDF 中删除附件
- PDF 文档内其他页面的超链接快捷方式

## PDF 安全功能

API 提供 40 位或 128 位加密。为了提高安全性，还可以设置用户和主密码。可以使用此库限制文件中的内容操作，比如复制内容、打印、降级打印、注释修改、表单字段填写以及 PDF 文件的组装。

- 加密 PDF 文件
- 解密 PDF 文件
- 更改密码
- 签署 PDF
