## 通过 C++ 的 Aspose.PDF for Go

_此页面是  Aspose.PDF for Go 功能的概述._

Go，通常称为 Golang，是一种由 Google 开发的现代编程语言，旨在应对可扩展且高效的软件开发所面临的挑战。该语言于 2009 年由 Robert Griesemer、Rob Pike 和 Ken Thompson 创建，融合了简洁性、速度和可靠性，因而深受开发者喜爱。

Go 语言具备诸如垃圾回收、强类型以及 goroutine 和 channel 等并发机制等有益特性。这些特性使开发者能够编写健壮的应用程序，高效地同时处理多个任务，这对基于云的服务和 Web 服务器至关重要。

Go 的流行源于其易用性、极简设计和无缝的可伸缩性。它的编译速度极快，生成高度优化的二进制文件，降低了运行时开销。随着包括工具和库在内的生态系统不断壮大，Go 在微服务、DevOps 和大数据等领域被广泛采用。

该语言同样强调社区驱动的开发，促进开发者之间的协作。Go 对编码的务实态度以及对性能的专注，使其成为现代高性能应用的理想选择。

Go 自 2018 年以来一直很受欢迎，它是开发者最想添加到其技术栈的最流行语言。

使用此库，您可以实现从头创建 PDF 文件的丰富功能，或在无需安装 Adobe Acrobat 的情况下完整处理现有 PDF 文档。Aspose.PDF for Go 支持 **广泛的功能**。

### 将 PDF 页面转换为其他格式

将 PDF 页面转换为图像可以使内容对可能难以直接查看或交互 PDF 文档的用户更易访问。Aspose.PDF 允许使用 Go 将 PDF 转换为不同的格式。有以下格式：

- TIFF
- BMP
- JPG
- PNG
- DICOM
- SVG
- DOC
- DOCX
- EPUB
- PPTX
- TEX
- TXT
- XLSX
- XPS

您可以访问 [转换 PDF 格式](/pdf/zh/go-cpp/conversion/) 页面了解详情。
