PDF 是 Portable Document Format（可移植文档格式）的缩写，用于以电子形式显示文档，且不受查看时所使用的软件、硬件或操作系统的影响。

PDF 是一种开放标准，现由国际标准化组织 (ISO) 维护。

最初的目标是保留和保护文档的内容和布局——无论在何种平台或计算机程序上查看。这就是为什么 PDF 难以编辑，有时甚至提取其中的信息也是一项挑战。

但 **Aspose.PDF for Go via C++** 可以帮助您应对在处理 PDF 文档时出现的大多数任务。

您可以执行以下操作：

- [优化 PDF](/pdf/zh/go-cpp/optimize-pdf/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现文档的成功优化。
- [优化 PDF 资源](/pdf/zh/go-cpp/optimize-pdf-resources/) - 优化页面内容，优化文件大小，删除未使用的对象，压缩所有图像，以实现文档的成功优化。
