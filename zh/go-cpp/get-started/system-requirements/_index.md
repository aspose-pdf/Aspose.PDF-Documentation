## 通过 C++ 的 Aspose.PDF for Go 支持平台

asposepdf 包是一个强大的工具包，允许开发人员直接操作 PDF 文件，并帮助完成各种 PDF 任务。
包含将 PDF 转换为其他格式的独特功能。

已实现对 Linux x64、macOS x86_64、macOS arm64 和 Windows x64 平台的支持。已使用 [cgo](https://go.dev/wiki/cgo).

在包的根目录下的 ‘lib’‑folder 中的平台特定动态库版本是分发生成的应用程序所必需的：

- **libAsposePDFforGo_linux_amd64.so** 用于 Linux x64 平台。
- **libAsposePDFforGo_darwin_arm64.dylib** 用于 macOS arm64 平台。
- **libAsposePDFforGo_darwin_amd64.dylib** 用于 macOS x86_64 平台。
- **AsposePDFforGo_windows_amd64.dll** 用于 Windows x64 平台。

Windows x64 平台需要 [MinGW-W64](https://www.mingw-w64.org/) 已安装。
