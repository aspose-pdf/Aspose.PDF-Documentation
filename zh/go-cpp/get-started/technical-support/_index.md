Aspose 为其所有产品提供无限的免费技术支持。支持面向所有用户开放，包括试用用户。主要的支持渠道是 Aspose.Forums。

{{% alert color="primary" %}}
请注意，Aspose不提供电话技术支持。电话支持仅适用于销售和采购问题。
{{% /alert %}}

## Aspose 免费支持论坛

Aspose.PDF 让您可以为其所有产品使用免费技术支持。如果您对 Aspose.PDF 有疑问，请查看下一篇文章：

- 最常用的支持渠道是 [Aspose.Forums](https://forum.aspose.com/). 在 [Aspose.PDF 论坛](https://forum.aspose.com/c/pdf/10) 部分，您可以提出问题并尽快得到答案。
- 您应当了解，Aspose 不提供电话技术支持。您只能在销售和采购问题上致电。
- 请记住，我们存在时区差异，这可能会影响我们对您的响应速度。

## 报告问题或功能请求

如果您需要在 Aspose.PDF 上的问题帮助，请遵循以下建议，以确保以最有效的方式解决问题：

- 首先，在报告问题之前，请检查您使用的是最新的 Aspose.PDF 版本，参见 [Aspose.PDF for Go 下载](https://github.com/aspose-pdf/aspose-pdf-go-cpp) 以了解最新版本。
- 也许您问题的解决方案已经被讨论并已有答案，为此，请研究论坛和文档，检查 [API 参考](https://reference.aspose.com/pdf/go-cpp/) 在报告问题之前。
- 在提出问题时，请附上原始文档以及可能导致问题的代码片段。如果有多个文件，您可以将这些文件压缩成一个文档。
- 别担心，只有您和 Aspose 的开发人员可以访问这些粘性文件。
