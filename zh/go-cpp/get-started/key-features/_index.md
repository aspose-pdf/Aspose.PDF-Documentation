## 跨平台软件

支持多种平台和 Go 版本。请参阅 [系统要求](/pdf/zh/go-cpp/system-requirements/) 了解详情。

## 通用功能

支持从广泛的列表中加载并保存到 [文件格式](/pdf/zh/go-cpp/supported-file-formats/).

## 受支持的 PDF 标准

Aspose.PDF for Go 支持 PDF 版本 1.2、1.3、1.4、1.5、1.6 和 1.7。
