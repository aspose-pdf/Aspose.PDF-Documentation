{{% alert color="primary" %}}

将多个 PDF 文件合并/串联为单个 PDF 文件是 PDF 文件处理应用程序中非常流行且需求量大的功能。我们已在其中引入了此重要功能 [Aspose.PDF for SharePoint 2.2.0](https://releases.aspose.com/pdf/sharepoint/new-releases/aspose.pdf-for-sharepoint-2.2.0/) 版本。合并两个文件是指通过将第二个文件添加到第一个文件的末尾来创建单个文件。

{{% /alert %}}

## **合并 PDF 文件**

将来自 SharePoint 文档库的多个 PDF 文件合并为一个 PDF，操作如下：

1.  从 SharePoint 文档库中选择要合并的 PDF 文件。

![todo:image_alt_text](merge-pdf-files_1.png)

2.  在库工具中点击 Aspose Tools 选项卡。

![todo:image_alt_text](merge-pdf-files_2.png)

3.  在库工具中点击 “合并为 PDF” 选项，将所有选定的 PDF 文件合并为结果 PDF。

![todo:image_alt_text](merge-pdf-files_3.png)

4.  系统将弹出提示，下载/保存具有适当名称的结果 PDF 文件。

![todo:image_alt_text](merge-pdf-files_4.png)
