{{% alert color="primary" %}}

Aspose.PDF for SharePoint 支持将 SharePoint 文档库中的 PDF 文档转换为 HTML 格式的功能。在本文中，我们将演示 PDF 转 HTML 的转换。

{{% /alert %}}

## **将 PDF 文档转换为 HTML**

将 PDF 文档从 SharePoint 文档库转换为 HTML，步骤如下：

1. 在 PDF 文档的 ECB 菜单中点击 **Convert to HTML**。

![todo:image_alt_text](convert-pdf-to-html_1.png)

2. 下载并将生成的 HTML 文件保存到磁盘。

![todo:image_alt_text](convert-pdf-to-html_2.png)
