## **向 PDF 文件添加条形码**

{{% alert color="primary" %}}

Aspose.PDF for SharePoint 允许您向 PDF 文档添加条形码。用户可以在添加到库中的 PDF 文档的每一页左下角添加条形码。下面的图片展示了添加条形码后的 PDF 文档可能的外观。

**左下角的条形码**

![todo:image_alt_text](add-barcode-to-a-pdf-file_1.png)

{{% /alert %}}

{{% alert color="primary" %}}

要为特定库启用条形码功能，请如下面所示，在 **Library Tools** 中的 **Aspose PDF Watermark Tools** 选项卡下点击 **Watermark Settings** 按钮。

**PDF 水印设置**

![todo:image_alt_text](add-barcode-to-a-pdf-file_2.png)

在为特定库启用条形码后，Aspose.PDF for SharePoint 会在添加到该库的任何 PDF 文档中添加条形码。

{{% /alert %}}
