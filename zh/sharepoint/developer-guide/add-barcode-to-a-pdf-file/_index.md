## **向 PDF 文件添加条形码**



{{% alert color="primary" %}}



Aspose.PDF for SharePoint 允许您向 PDF 文档添加条形码。用户可以在添加到库中的 PDF 文档的每一页的左下角添加条形码。下图展示了添加条形码后的 PDF 文档的样子。



**左下角的条形码**



![todo:image_alt_text](add-barcode-to-a-pdf-file_1.png)



{{% /alert %}}



{{% alert color="primary" %}}



要为特定库启用条形码功能，请使用**库工具**中的**Aspose PDF 水印工具**选项卡中的**水印设置**按钮，如下所示。



**PDF 水印设置**




![todo:image_alt_text](add-barcode-to-a-pdf-file_2.png)

After you enable barcodes for the specific library, Aspose.PDF for SharePoint adds a barcode to any PDF document added to that library.

在为特定库启用条形码后，Aspose.PDF for SharePoint 会向添加到该库的任何 PDF 文档添加条形码。

{{% /alert %}}
