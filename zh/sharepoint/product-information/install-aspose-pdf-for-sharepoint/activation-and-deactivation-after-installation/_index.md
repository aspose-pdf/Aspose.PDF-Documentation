{{% alert color="primary" %}}

在安装期间，Aspose.PDF for SharePoint 会为所有选定的站点集合激活。安装后，您可以在站点集合的根网站上使用“站点操作”菜单来激活和停用 Aspose.PDF for SharePoint。

{{% /alert %}}
