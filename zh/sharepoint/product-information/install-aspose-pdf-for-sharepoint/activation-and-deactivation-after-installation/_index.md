{{% alert color="primary" %}}

在安装期间，Aspose.PDF for SharePoint 会为所有选定的网站集合激活。安装后，您可以使用网站集合根网站的“网站操作”菜单来激活和停用 Aspose.PDF for SharePoint。

{{% /alert %}}
