## **服务器要求**

要安装 Aspose.PDF for SharePoint，您需要一台运行 Microsoft SharePoint Server 2010 的计算机。

## **客户端要求**

您需要使用浏览器访问 SharePoint 网站。Aspose.PDF for SharePoint 已在以下浏览器上进行测试：

- 微软 Internet Explorer 8
- Mozilla 火狐 3.5
- Google Chrome 3
- Safari 4
