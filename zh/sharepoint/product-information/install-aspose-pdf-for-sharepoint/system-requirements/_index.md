## **服务器要求**

要安装 Aspose.PDF for SharePoint，您需要一台运行 Microsoft SharePoint Server 2010 的计算机。

## **客户端要求**

您需要一个浏览器来访问 SharePoint 网站。Aspose.PDF for SharePoint 已在以下浏览器中进行了测试：

- Microsoft Internet Explorer 8
- Mozilla FireFox 3.5
- Google Chrome 3
- Safari 4
