{{% alert color="primary" %}}

Aspose.PDF for SharePoint 可以作为 Aspose.PDF.SharePoint.zip 压缩档下载。

{{% /alert %}}

**此压缩档包含：**

- Aspose.PDF.SharePoint.wsp
  SharePoint 解决方案文件。Aspose.PDF for SharePoint 打包为 SharePoint 解决方案，以便在服务器场中进行部署/撤销及功能激活/停用。
- Aspose_LicenseAgreement.rtf

**最终用户许可协议：**

- Aspose.PDF for SharePoint.pdf

**用户文档：**

- Aspose.PDF for SharePoint Documentation.chm

**包含公共 API 参考的用户文档：**

- setup.exe

**安装程序：**

- setup.exe.config

**安装配置文件：**

在继续之前，安装程序会检查以下条件：

- SharePoint 2010 已安装。
- 用户有权限安装 SharePoint 解决方案。
- SharePoint 数据库在线。
- SharePoint 管理服务已启动。
- SharePoint 计时器服务已启动。需要 SharePoint 管理服务和计时器服务，因为某些设置操作依赖于计时器作业来传播到服务器场中的所有服务器。

**安装 Aspose.PDF for SharePoint 的步骤：**

- 将 Aspose.PDF.SharePoint 压缩包解压到本地驱动器。
- 运行 setup.exe 并按照屏幕上的说明进行操作。

**安装程序执行以下操作：**

- 检查安装先决条件。如果任何检查失败，设置将不会继续。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_1.png)

- 显示最终用户许可协议。用户必须接受协议才能继续。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_2.png)

- 显示部署目标选择对话框。
 用户选择要激活该功能的 Web 应用程序和站点集合。请参见下图。

- 将功能部署到服务器场。

- 激活选定站点集合的功能并配置其父 Web 应用程序。
- 显示已部署和激活该功能的 Web 应用程序和站点集合列表。
