{{% alert color="primary" %}}

Aspose.PDF for SharePoint 可下载为 Aspose.PDF.SharePoint.zip 存档。

{{% /alert %}}

**此存档包含：**

- Aspose.PDF.SharePoint.wsp
  SharePoint 解决方案文件。Aspose.PDF for SharePoint 已打包为 SharePoint 解决方案，以便在服务器农场中实现部署/撤回以及功能的激活/停用。
- Aspose_LicenseAgreement.rtf

**最终用户许可协议：**

- Aspose.PDF for SharePoint.pdf

**用户文档：**

- Aspose.PDF for SharePoint Documentation.chm

**带公共 API 参考的用户文档：**

- setup.exe

**设置程序:**

- setup.exe.config

**设置配置文件:**

设置程序在继续之前会检查以下条件：

- 已安装 SharePoint 2010。
- 用户有权限安装 SharePoint 解决方案。
- SharePoint 数据库已上线。
- SharePoint 管理服务已启动。
- SharePoint 计时器服务已启动。需要 SharePoint 管理服务和计时器服务，因为某些设置操作依赖计时任务将更改传播到服务器场中的所有服务器。

**安装 Aspose.PDF for SharePoint：**

- 将 Aspose.PDF.SharePoint zip 解压到本地磁盘。
- 运行 setup.exe 并按照屏幕上的指示操作。

**设置程序执行以下操作：**

- 检查安装前提条件。如果任何检查失败，设置将不会继续。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_1.png)



- 显示最终用户许可协议。用户必须接受该协议才能继续。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_2.png)



- 显示部署目标选择对话框。用户选择要激活功能的 Web 应用程序和站点集合。见下图。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_3.png)



- 将功能部署到服务器场。

![todo:image_alt_text](installing-aspose-pdf-for-sharepoint_4.png)



- 为选定的站点集合激活该功能并配置其父 Web 应用程序。
- 显示已部署并激活该功能的 Web 应用程序和站点集合列表。
