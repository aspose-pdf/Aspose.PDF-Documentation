{{% alert color="primary" %}}

要卸载 Aspose.PDF for SharePoint，只需运行安装程序。如果 Aspose.PDF for SharePoint 已经安装，安装程序会建议将其移除。

![todo:image_alt_text](uninstalling-aspose-pdf-for-sharepoint_1.png)

在卸载期间，安装程序会为所有站点集合停用 Aspose.PDF for SharePoint 功能，并从服务器场撤回该解决方案。

{{% /alert %}}
