{{% alert color="primary" %}}

要卸载 Aspose.PDF for SharePoint，只需运行设置程序。如果 Aspose.PDF for SharePoint 已安装，设置程序会建议移除它。

![todo:image_alt_text](uninstalling-aspose-pdf-for-sharepoint_1.png)

在卸载期间，设置程序会停用所有网站集合的 Aspose.PDF for SharePoint 功能，并从服务器场中撤回该解决方案。

{{% /alert %}}
