### **欢迎使用 Aspose.PDF for SharePoint 文档！**
Aspose.PDF for SharePoint 是一种解决方案，允许用户将列表、列表项和 SharePoint Wiki 页面导出为 PDF 文件格式。

{{% /alert %}}


{{% alert color="primary" %}}
Aspose.PDF for SharePoint 旨在与 Microsoft SharePoint Server 2010 配合使用。除了 Microsoft SharePoint Server 2010 外，没有其他额外的系统要求。

本文档描述了 Aspose.PDF for SharePoint 的 [功能](/pdf/zh/sharepoint/features/), [安装](/pdf/zh/sharepoint/install-aspose-pdf-for-sharepoint/), [评估限制](/pdf/zh/sharepoint/evaluate-aspose-pdf/), [许可](/pdf/zh/sharepoint/license-aspose-pdf-for-sharepoint/), 常见用例和设置。
