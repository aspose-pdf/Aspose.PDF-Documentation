### **欢迎访问Aspose.PDF for SharePoint文档！**

Aspose.PDF for SharePoint是一个解决方案，允许用户将列表、列表项和SharePoint Wiki页面导出为PDF文件格式。



{{% /alert %}}



{{% alert color="primary" %}}

Aspose.PDF for SharePoint旨在与Microsoft SharePoint Server 2010一起使用。除Microsoft SharePoint Server 2010外，没有额外的系统要求。



本文档描述了Aspose.PDF for SharePoint的[功能](/pdf/sharepoint/features/)、[安装](/pdf/sharepoint/install-aspose-pdf-for-sharepoint/)、[评估限制](/pdf/sharepoint/evaluate-aspose-pdf/)、[许可](/pdf/sharepoint/license-aspose-pdf-for-sharepoint/)、常见用例和设置。
