{{% alert color="primary" %}}

Cette page répertorie les modifications de l'API publique qui ont été introduites dans Aspose.PDF pour Java 4.5.0. Elle inclut non seulement les nouvelles méthodes publiques et celles obsolètes, mais aussi une description de tous les changements dans le comportement en coulisses dans Aspose.PDF pour Java qui peuvent affecter le code existant. Tout comportement introduit pouvant être perçu comme une régression et modifiant le comportement existant est particulièrement important et est documenté ici.

{{% /alert %}}

Les classes suivantes ont été déplacées du package **com.aspose.pdf** au package **com.aspose.pdf.devices**.

- BmpDevice
- ColorDepth
- CompressionType
- Device
- DocumentDevice
- EmfDevice
- FormPresentationMode
- GifDevice
- GraphicsDevice
- ImageDevice
- ImageGenerator
- JpegDevice
- Margins
- PageDevice
- PngDevice
- Resolution
- ShapeType
- TextDevice
- TiffDevice
- TiffSettings
