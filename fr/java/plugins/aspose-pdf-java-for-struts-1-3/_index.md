## Cadre Web Apache Struts

Le cadre web Apache Struts est une solution open-source gratuite pour créer des applications web Java.

Les applications web diffèrent des sites web conventionnels en ce sens qu'elles peuvent créer une réponse dynamique. De nombreux sites web ne diffusent que des pages statiques. Une application web peut interagir avec des bases de données et des moteurs de logique métier pour personnaliser une réponse.

Une façon de séparer les préoccupations dans une application logicielle est d'utiliser une architecture Modèle-Vue-Contrôleur (MVC). Le Modèle représente le code métier ou de la base de données, la Vue représente le code de conception de la page, et le Contrôleur représente le code de navigation. Le cadre Struts est conçu pour aider les développeurs à créer des applications web qui utilisent une architecture MVC.

[En savoir plus](http://struts.apache.org/birdseye.html)

### Aspose.PDF pour Java

Aspose.PDF pour Java est un composant de création de documents PDF qui permet à vos applications Java de lire, écrire et manipuler des documents PDF sans utiliser Adobe Acrobat.

Aspose.PDF pour Java est un composant proposé à un prix abordable qui offre une incroyable richesse de fonctionnalités, notamment : options de compression PDF, création et manipulation de tableaux, prise en charge des graphiques, fonctions d'image, fonctionnalité de lien hypertexte étendue, contrôles de sécurité avancés et gestion de polices personnalisées.

Aspose.PDF pour Java vous permet de créer des fichiers PDF directement via l'API fournie et des modèles XML. Utiliser Aspose.PDF pour Java vous permettra également d'ajouter des capacités PDF à vos applications en un rien de temps.

## Aspose.Words Java pour Struts 1.3

Aspose.Words Java pour Struts 1.3 est une application de librairie en ligne simple écrite en utilisant Struts 1.3 et Maven.

- L'application web a pour but de montrer l'intégration et l'utilisation de [Aspose.Words pour Java](https://products.aspose.com/words/java/) au sein du framework Struts 1.3 et Maven.
- Elle fonctionne dans un conteneur web comme Apache Tomcat et nécessite maven pour construire l'application web .war pour le déploiement.

- Le cas d'utilisation actuel dans l'application exporte des enregistrements de la grille de livres au format de document MS-WORD en créant un document MS-WORD, en y créant une table et en y écrivant des textes / données.

## Exigences du système

Voici les exigences du système pour Aspose.PDF Java pour l'application web Struts 1.3 :

- Java 1.6 ou supérieur installé.
- Serveur Apache Tomcat 7 ou supérieur
- Maven 3
- Composant Aspose.PDF.

## Plates-formes supportées

Voici les plates-formes supportées :

- Aspose.PDF 10.4.0 ou supérieur.
- IDE Java (Eclipse, NetBeans, IntelliJ, ...).
