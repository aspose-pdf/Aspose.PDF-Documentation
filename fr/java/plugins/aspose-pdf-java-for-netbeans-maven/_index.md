## Introduction

### NetBeans IDE

NetBeans est un IDE Java complet écrit entièrement en Java. Il est principalement destiné au développement en Java mais prend également en charge d'autres langages, en particulier, PHP, C/C++, et HTML5.

NetBeans IDE vous permet de développer rapidement et facilement des applications Java de bureau, mobiles et web, ainsi que des applications HTML5 avec HTML, JavaScript et CSS. Il est gratuit et open-source et dispose d'une large communauté d'utilisateurs et de développeurs à travers le monde.

L'équipe NetBeans soutient activement le produit et recherche des suggestions de fonctionnalités auprès de la communauté élargie. Chaque version est précédée d'une période de test et de retour d'information de la communauté.

### Aspose.PDF pour Java

[Aspose.PDF pour Java](https://products.aspose.com/pdf/java/) est une API de création de documents PDF robuste qui permet à vos applications Java de lire, écrire et manipuler des documents PDF sans utiliser Adobe Acrobat.

Aspose.PDF pour Java offre une incroyable richesse de fonctionnalités, incluant des options de compression PDF, la création et la manipulation de tableaux, le support des graphiques, des fonctions d'image, une fonctionnalité étendue de liens hypertexte, des contrôles de sécurité avancés et la gestion des polices personnalisées.

### Aspose.PDF Java pour NetBeans (Maven)

- Aspose.PDF Java pour NetBeans (Maven) est un Plugin pour **NetBeans IDE** par **Aspose.** Le Plugin est destiné aux développeurs utilisant la plateforme Maven pour les développements Java et souhaitant utiliser Aspose.PDF pour Java dans leurs projets. Le Plugin vous permet de créer des projets maven pour utiliser l'API Aspose.PDF pour Java et également de télécharger [Exemples de Code](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) de l'API.

## ASSISTANTS

Le plugin contient deux assistants

### Projet Maven Aspose.PDF (assistant)

- Cet assistant de nouveau projet permet aux développeurs de créer un projet **Maven** pour utiliser Aspose.PDF pour Java à partir de Nouveau Projet -> Maven-> Projet Maven Aspose.PDF.
- La référence de la dépendance Maven de l'API Aspose.PDF pour Java est automatiquement récupérée depuis le [dépôt Aspose Maven](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf) et ajoutée dans le pom.xml.
- Le projet créé contiendra toujours la version la plus récente disponible de la dépendance **Maven** pour l'API Aspose.PDF pour Java.
- Les étapes de l'assistant sont également présentes avec l'option de téléchargement des exemples de code pour utiliser l'API Aspose.PDF pour Java.

### Exemple de code Aspose.PDF (assistant)

- Cet assistant de nouveau fichier vous permet de copier les [exemples de code](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) téléchargés dans votre projet pour utiliser Aspose.PDF pour Java à partir de **Nouveau Fichier -> Java -> Exemple de code Aspose.PDF.**
- Les exemples disponibles sont affichés sous forme d'arborescence à partir de laquelle l'utilisateur peut les sélectionner par catégorie.

- Tous les exemples de la catégorie sélectionnée seront copiés dans le dossier de paquets **com.aspose.pdf.examples** du projet ainsi que les ressources nécessaires dans le dossier **src/main/resources** nécessaires pour exécuter les exemples.
- Les exemples de code d'Aspose.PDF pour l'API Java sont destinés à démontrer la variété de l'API.
- L'assistant recherchera également des mises à jour nouvellement disponibles à partir du [répertoire des exemples d'Aspose.PDF pour Java.](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)

## Téléchargement

### Télécharger NetBeans

Vous devez avoir NetBeans IDE préinstallé avant de télécharger le plugin **Aspose.PDF Java pour NetBeans (Maven)**.

Le plugin **Aspose.PDF Java pour NetBeans (Maven)** prend en charge **NetBeans IDE 8.0 et 8.1**. Nous vous recommandons d'utiliser uniquement la dernière version de NetBeans IDE.

Pour télécharger NetBeans IDE,

1. Accédez à : <https://netbeans.apache.org/download/index.html>

1. Cliquez sur Télécharger pour télécharger le bundle approprié de NetBeans IDE.

Téléchargez uniquement le bundle de téléchargement NetBeans pris en charge par Java EE / SE de la dernière version actuelle, c'est-à-dire **NetBeans IDE 8.1**.

### Télécharger Aspose.PDF Java pour NetBeans (Maven)

- Le plugin peut être directement téléchargé depuis l'URL d'hébergement des plugins NetBeans : <http://plugins.netbeans.org/plugin/63218>
  et peut ensuite être installé en utilisant l'option de l'IDE :
  - Allez à **Outils** > **Plugins** > **Téléchargé**
  - Cliquez sur **Ajouter des plugins...** et recherchez le fichier plugin avec extension .nbm téléchargé pour **Aspose.PDF Java pour NetBeans (Maven)**.
  - Cliquez sur **"Installer"** pour installer le plugin
- Ou il peut être directement installé (sans téléchargement manuel depuis l'URL d'hébergement des plugins NetBeans) en utilisant les options de l'IDE :
  - Allez à **Outils** > **Plugins** > **Plugins disponibles**
  - Dans la boîte de recherche, tapez : "Aspose"
  - Sélectionnez le plugin **Aspose.PDF Java NetBeans (Maven)**
  - Cliquez sur **"Installer"**

### Comment appliquer la licence Aspose ?

Ce plugin utilise une version d'évaluation de Aspose.PDF. Une fois que vous êtes satisfait de votre évaluation, vous pouvez acheter une licence sur le [site Aspose](https://purchase.aspose.com/buy).  
Pour supprimer le message d'évaluation et les limitations des fonctionnalités, une licence produit doit être appliquée. Vous recevrez un fichier de licence après avoir acheté le produit. Veuillez suivre les étapes ci-dessous pour appliquer la licence.

- Assurez-vous que le fichier de licence est nommé **Aspose.PDF.Java.lic**
- Placez le fichier **Aspose.PDF.Java.lic** dans le dossier qui contient Aspose.PDF.jar
- Utilisez le code suivant pour activer la licence :

```java
License license = new License();
license.setLicense("Aspose.PDF.Java.lic");
```
