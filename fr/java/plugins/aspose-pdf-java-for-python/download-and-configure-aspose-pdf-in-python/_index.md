## Comment configurer le code source

Veuillez suivre ces étapes simples afin d'ouvrir et d'étendre le code source tout en utilisant :

1. Installer uniquement Python 2.7.x.x depuis : <https://www.python.org/downloads/>
2. Installer JPype depuis : <http://sourceforge.net/projects/jpype/files/JPype/>
3. Installer Java JDK depuis : <http://www.oracle.com/technetwork/java/javase/downloads>
4. Télécharger Aspose.PDF pour les API Java que vous souhaitez utiliser dans vos projets python depuis : <http://www.aspose.com/community/files/72/java-components/default.aspx> et copier les .jars des API téléchargées dans le dossier %Python%/lib.
5. Cloner [Aspose.PDF Java for Python Examples](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Plugins/Aspose_Pdf_Java_for_Python) depuis github.
6. Cloner [Aspose.PDF Java for Python Examples](http://asposepdfjavapython.codeplex.com/) depuis CodePlex.
