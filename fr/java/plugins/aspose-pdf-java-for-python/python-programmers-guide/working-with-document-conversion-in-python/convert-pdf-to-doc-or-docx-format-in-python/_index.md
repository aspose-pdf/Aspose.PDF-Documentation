Pour convertir un document PDF en format DOC ou DOCX en utilisant **Aspose.PDF Java pour Python**, il suffit d'invoquer le module **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "Le document a été converti avec succès"
```
