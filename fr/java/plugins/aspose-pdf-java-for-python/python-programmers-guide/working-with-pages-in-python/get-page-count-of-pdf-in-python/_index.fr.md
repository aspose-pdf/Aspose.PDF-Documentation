Pour obtenir le nombre de pages d'un document Pdf en utilisant **Aspose.PDF Java pour Python**, il suffit d'invoquer la classe **GetNumberOfPages**.

```Python
doc = self.Document()
pdf = self.Document()
pdf = self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "Nombre de Pages:" . page_count

```
