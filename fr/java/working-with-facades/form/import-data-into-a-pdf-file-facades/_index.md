## Importer des données depuis XML dans un fichier PDF (façades)

La classe [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form) vous permet d'importer des données à partir d'un fichier XML vers le fichier PDF en utilisant la méthode importXml. Pour importer des données depuis XML, vous devez créer un objet de la classe Form, puis appeler la méthode importXml en utilisant l'objet FileInputStream. Enfin, vous pouvez enregistrer le fichier PDF en utilisant la méthode save de la classe Form.
