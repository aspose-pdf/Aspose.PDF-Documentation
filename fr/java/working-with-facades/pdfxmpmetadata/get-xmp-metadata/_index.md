Pour obtenir les métadonnées XMP d'un fichier PDF, vous devez créer un objet [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) et lier le fichier PDF en utilisant la méthode [bindPdf(..)](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Facade#bindPdf-com.aspose.pdf.IDocument-). Vous pouvez passer des propriétés de métadonnées XMP spécifiques à l'objet [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) pour obtenir leurs valeurs.
