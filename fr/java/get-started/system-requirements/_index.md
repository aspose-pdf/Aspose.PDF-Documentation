## Aperçu

Pour produire des documents PDF, la machine pour exécuter [Aspose.PDF pour Java](https://products.aspose.com/pdf/java/) n'a pas besoin d'avoir [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) installé car Aspose.PDF pour Java est lui-même un moteur de création de documents PDF. Cependant, pour visualiser les documents PDF produits par Aspose.PDF pour Java, la machine doit avoir au moins un [Adobe Acrobat Reader](https://www.adobe.com/acrobat/pdf-reader.html). Il peut être téléchargé gratuitement.

## Systèmes d'exploitation pris en charge

Aspose.PDF pour Java prend en charge tout système d'exploitation 32 bits ou 64 bits qui exécute le runtime Java, y compris, mais sans s'y limiter :

### Windows


- Microsoft Windows 2000 Professionnel (x64, x86)
- Microsoft Windows 2000 Serveur (x64, x86)
- Microsoft Windows 2003 Serveur (x64, x86)
- Microsoft Windows 2008 Serveur (x64, x86)
- Microsoft Windows 2012 Serveur (x64, x86)
- Microsoft Windows 2012 R2 Serveur (x64, x86)
- Microsoft Windows 2016 Serveur (x64, x86)
- Microsoft Windows 2019 Serveur (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Azure

### Linux

- Linux (Ubuntu, CentOS, et autres)

### Mac

- Mac OS X

## Versions Java prises en charge

Aspose.PDF pour Java prend en charge les versions Java suivantes :

- J2SE 8.0 (1.8) ou supérieur (*par exemple Java 10*)
- Prise en charge d'Aspose.Pdf pour Java sur IBM i (Iseries ou As/400)

Vous pouvez utiliser Aspose.PDF pour Java pour développer des applications dans n'importe quel environnement de développement qui cible la plateforme Java, mais les environnements suivants sont explicitement pris en charge :

- Microsoft Visual Studio 2003
- Microsoft Visual Studio 2005
