## Plateformes prises en charge par Aspose.PDF for Rust via C++

Le package asposepdf est un ensemble d'outils puissant qui permet aux développeurs de manipuler directement les fichiers PDF et aide à réaliser diverses tâches liées aux PDF.
Contient des fonctionnalités uniques pour convertir le PDF en d'autres formats.

Prise en charge implémentée pour les plateformes Linux x64, macOS x86_64, macOS arm64 et Windows x64.

La version spécifique à la plateforme de la bibliothèque dynamique du dossier 'lib' dans le répertoire racine du paquet est requise pour distribuer l'application résultante : 

- libAsposePDFforRust_linux_amd64.so pour la plateforme Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib pour la plateforme macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib pour la plateforme macOS x86_64 
- AsposePDFforRust_windows_amd64.dll pour la plateforme Windows x64.
