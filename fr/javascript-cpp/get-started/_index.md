## Qu'est-ce qu'un fichier PDF ?

Le PDF a été créé par Adobe dans les années 1990 pour atteindre deux objectifs. Le premier est que vous devriez pouvoir ouvrir les documents sur n'importe quel matériel ou système d'exploitation, sans avoir besoin de l'application utilisée pour les créer—tout ce dont vous avez besoin est un lecteur PDF, et de nos jours, la plupart des navigateurs web remplissent ce rôle. Le second est que, où que vous ouvriez un PDF, la mise en page du document devrait être la même.

Cependant, il ne suffit pas simplement d'ouvrir votre document. En travaillant avec des PDF, vous serez confronté à la nécessité de créer un tel document à nouveau, de l'éditer ou de le convertir dans le format dont vous avez besoin.

## Pourquoi utiliser Aspose.PDF pour JavaScript via C++ ?

Utiliser Aspose.PDF pour JavaScript dans votre projet vous offre les avantages suivants :

- large variété de fonctions
- boîte à outils utilisée pour travailler avec des PDF directement dans le navigateur web
- basé sur Aspose.PDF pour .NET
