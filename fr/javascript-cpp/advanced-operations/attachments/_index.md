Dans cette section, nous expliquerons comment travailler avec les pièces jointes dans les PDF en utilisant Aspose.PDF pour JavaScript via C++.

Une pièce jointe est un fichier supplémentaire qui est attaché à un document parent, il peut s'agir de divers types de fichiers, tels que pdf, word, image ou d'autres fichiers.

Vous apprendrez comment ajouter des pièces jointes à un pdf, obtenir les informations d'une pièce jointe, la sauvegarder dans un fichier, supprimer la pièce jointe d'un PDF par programmation avec JavaScript.
