### **Bienvenue dans la documentation Aspose.PDF for SharePoint !**
Aspose.PDF for SharePoint est une solution qui permet aux utilisateurs d'exporter des listes, des éléments de liste et des pages Wiki SharePoint au format PDF.

{{% /alert %}}


{{% alert color="primary" %}}
Aspose.PDF for SharePoint est conçu pour être utilisé avec Microsoft SharePoint Server 2010. Aucun autre prérequis système n'est nécessaire en dehors de Microsoft SharePoint Server 2010.

Cette documentation décrit Aspose.PDF for SharePoint [fonctionnalités](/pdf/fr/sharepoint/features/), [installation](/pdf/fr/sharepoint/install-aspose-pdf-for-sharepoint/), [limitations d'évaluation](/pdf/fr/sharepoint/evaluate-aspose-pdf/), [licence](/pdf/fr/sharepoint/license-aspose-pdf-for-sharepoint/), cas d'utilisation courants et paramètres.

{{% /alert %}}
