## Qu'est-ce qu'un fichier PDF ?

Le PDF a été créé par Adobe dans les années 1990 pour atteindre deux objectifs. Le premier est que vous puissiez ouvrir les documents sur n'importe quel matériel ou système d'exploitation, sans avoir besoin de l'application utilisée pour les créer — tout ce dont vous avez besoin est un lecteur PDF, et de nos jours la plupart des navigateurs web remplissent cette fonction. Le second est que, où que vous ouvriez un PDF, la mise en page du document doit rester identique.

Cependant, il ne suffit pas de simplement ouvrir votre document. En travaillant avec les PDF, vous serez amené à créer un tel document à nouveau, le modifier ou le convertir dans le format dont vous avez besoin.
