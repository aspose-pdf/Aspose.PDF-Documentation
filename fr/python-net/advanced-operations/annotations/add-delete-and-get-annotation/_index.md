**Quelles sont les annotations dans les documents PDF ?**

Les annotations dans les documents PDF sont des éléments qui permettent aux utilisateurs de placer du contenu d'une certaine manière dans le document. Les annotations peuvent être utilisées pour ajouter différents types de notes, mettre en évidence du texte, dessiner des formes, joindre des fichiers et effectuer d'autres actions qui améliorent la visualisation, l'interactivité et l'interaction au sein d'un document.

Nous avons regroupé les différents types d'annotations disponibles pour la bibliothèque Aspose.PDF pour Python en groupes :

- [Annotation de texte PDF](/pdf/python-net/text-annotation/)
- [Annotation de surlignage PDF](/pdf/python-net/highlights-annotation/)
- [Annotation de figures PDF](/pdf/python-net/figures-annotation/)
- [Annotations collantes PDF](/pdf/python-net/sticky-annotations/)
- [Annotations de lien](/pdf/python-net/link-annotations/)
- [Annotations supplémentaires](/pdf/python-net/extra-annotations/)
