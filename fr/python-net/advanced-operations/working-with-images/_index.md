Il existe de nombreuses méthodes et outils pour éditer et manipuler des images dans les documents PDF.
En plus des fonctions habituelles d'ajout et de suppression d'images du fichier PDF, la bibliothèque Aspose.PDF pour Python permet également d'extraire des images.
**Aspose.PDF for Python** est un outil intelligent et efficace pour travailler avec des images dans un PDF existant.

Vous pouvez faire ce qui suit :

- [Ajouter une image à un fichier PDF existant](/pdf/python-net/add-image-to-existing-pdf-file/) - ajouter des images et des références d'une seule image dans le document PDF, puis contrôler la qualité.
- [Supprimer des images du fichier PDF](/pdf/python-net/delete-images-from-pdf-file/) - voir l'extrait de code pour supprimer des images du fichier PDF.
- [Extraire des images d'un fichier PDF](/pdf/python-net/extract-images-from-pdf-file/) - l'article suivant montre comment extraire des images d'un fichier PDF à l'aide de la bibliothèque Python.
- [Rechercher et obtenir des images d'un document PDF](/pdf/python-net/search-and-get-images-from-pdf-document/) - vous pouvez obtenir une image d'une page individuelle et rechercher parmi les images de toutes les pages avec Python.
- [Remplacer une image dans un fichier PDF existant](/pdf/python-net/replace-image-in-existing-pdf-file/) - voir notre extrait de code, il montre comment remplacer une image dans un fichier PDF.
