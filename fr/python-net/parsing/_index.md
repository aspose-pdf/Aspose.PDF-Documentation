**Parse PDF** documents est un terme lié à l'extraction de divers types d'informations à partir d'un fichier PDF. Cette section couvre comment:

- [Extraire le texte du PDF](/pdf/python-net/extract-text-from-pdf/). L'analyse ou l'extraction de texte est l'opération la plus populaire avec les PDF prêts à l'emploi. Vous apprendrez l'analyse de texte d'un document complet, d'une page particulière ou d'une région particulière d'une page.
- [Extraire les images du PDF](/pdf/python-net/extract-images-from-the-pdf-file/). L'extraction d'images fait la même chose pour les images que l'opération ci-dessus pour le texte.
- [Extraire les polices du PDF](/pdf/python-net/extract-fonts-from-pdf/). L'extraction de polices est une opération spécifique avec les polices dans les PDF.
- [Extraire les données du formulaire](/pdf/python-net/extract-data-from-acroform/). Si vous avez un grand nombre de documents PDF avec des formulaires, vous avez probablement besoin d'obtenir les données de ces formulaires. Cet article vous aidera à comprendre comment extraire les données des AcroForms avec Aspose.PDF pour Python via .NET.
- [Extraire les données d'un tableau](/pdf/python-net/extract-data-from-table-in-pdf/). Obtenez les données d'un tableau dans un document PDF.
- [Extraire les données vectorielles du PDF](/pdf/python-net/extract-vector-data-from-pdf/). Vous pouvez obtenir les données vectorielles (chemin, polygone, polyligne), telles que la position, la couleur, l'épaisseur du trait, etc.
