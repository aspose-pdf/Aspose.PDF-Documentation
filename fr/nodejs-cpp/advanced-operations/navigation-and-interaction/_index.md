- [Signets](/pdf/fr/nodejs-cpp/bookmark/)- les grandes publications incluent généralement un cadre de signets qui peut être facilement consulté et sélectionné dans le volet Signets, vous permettant de cliquer sur un signet pour accéder à la page ou au chapitre qu’il représente. Le volet Signets est un élément conscient du contenu et n’est visible dans la barre latérale que si le document PDF ouvert contient une structure de signets.
