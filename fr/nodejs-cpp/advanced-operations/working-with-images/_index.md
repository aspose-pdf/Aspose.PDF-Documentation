Utilisez **Aspose.PDF for Node.js via C++** pour travailler avec des images dans les documents PDF. Il est possible d’ajouter et de supprimer des images dans votre fichier PDF dans l’environnement Node.js.

Vous pouvez faire ce qui suit :

- [Ajouter une image au fichier PDF](/pdf/fr/nodejs-cpp/add-image-to-pdf/) - ajouter des images à PDF dans l'environnement Node.js.
- [Supprimer les images du fichier PDF](/pdf/fr/nodejs-cpp/delete-images-from-pdf-file/) - vérifier l'extrait de code pour supprimer des images du fichier PDF.
