Dans cette section, nous expliquerons comment travailler avec les pièces jointes dans PDF en utilisant Aspose.PDF for Node.js via C++.

Une pièce jointe est un fichier supplémentaire attaché à un document parent, il peut s'agir de divers types de fichiers, tels que pdf, word, image ou d'autres fichiers.

Vous apprendrez comment supprimer les pièces jointes du PDF de manière programmatique avec Node.js.

- [Suppression de la pièce jointe d'un PDF existant](/pdf/fr/nodejs-cpp/removing-attachment-from-an-existing-pdf/)
