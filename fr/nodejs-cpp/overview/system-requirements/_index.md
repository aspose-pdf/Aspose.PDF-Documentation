## Vue d'ensemble

Pour traiter des documents PDF en utilisant le [Aspose.PDF for Node.js via C++](https://products.aspose.com/pdf/nodejs-cpp/), n’a pas besoin de bibliothèques et de programmes tiers.
Nécessite installé [Node.js](https://nodejs.org/). Vous devez disposer de la version 16 ou supérieure de Node.js. Nous recommandons fortement d’utiliser les versions LTS.
Node.js peut être [installé](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) de différentes manières.

## Systèmes d'exploitation pris en charge

Toutes les versions prises en charge de [Linux](https://en.wikipedia.org/wiki/Linux) et [macOS](https://www.apple.com/macos/).
