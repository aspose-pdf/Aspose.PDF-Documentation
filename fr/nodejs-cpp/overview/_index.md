_Cette page est une vue d'ensemble des fonctionnalités d'Aspose.PDF for Node.js via C++._

**Aspose.PDF for Node.js via C++** permet aux développeurs de manipuler les fichiers PDF directement dans l'environnement Node.js. Cette solution Node.js repose sur l'utilisation de la technologie WebAssembly et est basée sur Aspose.PDF for .NET. Vous pouvez facilement utiliser aspose-pdf pour Node.js via C++ à partir de et ensuite suivre les instructions d'installation.

**À propos des extensions de fichiers CJS et MJS**

Node.js prend en charge deux systèmes modulaires pour organiser et partager le code. Il y a :

- Modules CommonJS
- Modules ECMAScript

Les modules CommonJS étaient la méthode classique de traitement des dépendances dans Node.js, tandis que les modules ECMAScript sont le complément plus récent et suivent le format standard utilisé dans les navigateurs modernes. 'cjs', 'mjs' - sont des extensions utilisées pour spécifier les formats de modules (CommonJS et Modules ECMAScript) utilisés dans les fichiers JavaScript. 

Dans les extraits de code, vous remarquerez une différence de travail avec le module AsposePDFforNode.js. La distinction principale entre 'require' et 'import' réside dans leurs cas d'utilisation. **Require** se limite à l'importation de modules entiers, alors que **Import** offre la flexibilité d'importer à la fois des modules entiers et des exportations spécifiques de ces modules.

**Deux approches différentes**

JavaScript offre deux approches, 'then' et 'await', pour gérer les opérations asynchrones.

La méthode 'then' fonctionne avec les promesses pour gérer à la fois les résultats réussis et les erreurs dans un ordre séquentiel. En revanche, 'await' fait attendre JavaScript, suspendant littéralement l'exécution de la fonction jusqu'à ce que la promesse se résolve et renvoie son résultat.

Aspose.PDF pour Node.js prend en charge une grande variété de fonctions telles que :

- Définir, obtenir et supprimer les informations du fichier PDF
- Supprimer les signets
- Supprimer les annotations
- Extraire du texte d'un PDF
- Optimiser le document PDF
- Optimiser les ressources 
- Diviser le PDF en deux fichiers.
- Fusionner deux fichiers en PDF
- Réparer le PDF
- Fonctionnalités de conversion

De plus, Aspose.PDF for Node.js via C++ peut être utilisé pour convertir facilement des documents PDF en divers formats avec des performances élevées et une qualité parfaite. Voir la section [Formats de fichiers pris en charge par Aspose.PDF](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). De plus, cet ensemble d'outils prend en charge le chiffrement et le déchiffrement des documents PDF.
