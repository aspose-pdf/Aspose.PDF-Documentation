**Analyser PDF** les documents sont un terme lié à l'extraction de différents types d'informations à partir d'un fichier PDF. Cette section décrit comment :

- [Extraire des images du PDF](/pdf/fr/nodejs-cpp/extract-images-from-the-pdf-file/). L'extraction d'images permet d'extraire les images et de les enregistrer comme fichiers séparés.
- [Extraire du texte du PDF](/pdf/fr/nodejs-cpp/extract-text-from-pdf/). L'analyse du texte ou son extraction est l'opération la plus populaire avec les PDF prêts à l'emploi. Vous apprendrez l'analyse du texte à partir d'un document complet, d'une page particulière ou d'une région particulière dans une page.
- [Extraire des tableaux du PDF](/pdf/fr/nodejs-cpp/extract-tables-from-the-pdf-file/). Convertir le PDF au format CSV en utilisant Aspose.PDF for Node.js via C++.
