## Télécharger les bibliothèques nécessaires

Téléchargez les bibliothèques nécessaires mentionnées ci-dessous. Celles-ci sont requises pour exécuter des exemples d'Aspose.PDF pour Python via Java.

- Aspose: [Aspose.PDF pour Python via Java Component](https://releases.aspose.com/pdf/pythonjava/)
- ou PyPI: [L'index des paquets Python](https://pypi.org/project/aspose-pdf-for-python-via-java/)
- [JPype](https://pypi.python.org/pypi/JPype1)
- JDK 1.8+

### Comment configurer le code source

Veuillez suivre ces étapes simples pour ouvrir et étendre le code source tout en utilisant :

1. Installez uniquement Python 3.7+ depuis [Site Officiel](https://www.python.org/downloads)
2. Installez JPype 1.5.0+ depuis [Sourceforge](http://sourceforge.net/projects/jpype/files/JPype/)
3. Installez Java JDK 1.8+ depuis [Téléchargements Java](https://www.oracle.com/java/technologies/downloads/)
4. Téléchargez Aspose.PDF pour Python via l'API Java : [Aspose.PDF pour Python via Composant Java](https://releases.aspose.com/pdf/pythonjava/)
5. Ou installez Aspose.PDF pour Python via l'API Java en utilisant pip : [PyPI](https://pypi.org/project/aspose-pdf-for-python-via-java/)
