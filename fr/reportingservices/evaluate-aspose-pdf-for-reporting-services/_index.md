{{% alert color="primary" %}}

Cette page indique la différence entre la version sous licence et la version d'évaluation d'Aspose.PDF for Reporting Services

{{% /alert %}}

Vous pouvez facilement télécharger [Aspose.PDF pour Reporting Services](https://downloads.aspose.com/pdf/reportingservices) pour l'évaluation. Le téléchargement d'évaluation est identique au téléchargement acheté. La version d'évaluation devient simplement sous licence lorsque vous placez le fichier de licence dans le dossier contenant Aspose.PDF.ReportingServices.dll ou quelques lignes de code pour initialiser la licence lors de l'utilisation du composant avec Report Viewer en mode local. La version d'évaluation d'Aspose.PDF pour Reporting Services (sans licence spécifiée) fournit toutes les fonctionnalités du produit, mais elle insère un filigrane d'évaluation en haut du document lors du rendu du fichier .RDL au format PDF. Vous pouvez consulter la page suivante pour plus d'instructions sur [Comment licencier Aspose.PDF pour Reporting Services](/pdf/fr/reportingservices/license-aspose-pdf-for-reporting-services/)

![todo:image_alt_text](evaluate-aspose-pdf-for-reporting-services_1.png)

{{% alert color="primary" %}}

Si vous souhaitez tester Aspose.PDF pour Reporting Services sans les limitations de la version d'évaluation, vous pouvez également demander une licence temporaire de 30 jours. Veuillez vous référer à [Comment obtenir une licence temporaire ?](<https://about.aspose.com/>)

{{% /alert %}}
