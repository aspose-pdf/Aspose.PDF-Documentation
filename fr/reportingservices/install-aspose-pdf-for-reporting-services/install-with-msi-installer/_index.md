Vous pouvez installer Aspose.Pdf pour Reporting Services en utilisant un programme d'installation MSI. Exécutez Aspose.Pdf.ReportingServices.msi et suivez les étapes proposées par l'installateur. L'installateur copiera l'assembly et les autres fichiers dans le répertoire spécifié et installera le produit sur l'instance par défaut de Reporting Services. Vous n'avez pas besoin de copier ou de modifier des fichiers manuellement, sauf si vous souhaitez ajouter des paramètres de configuration spéciaux comme décrit dans la section 'Configure Aspose.Pdf for Reporting Services'.

L'installation automatique est la meilleure option qui fonctionne dans la plupart des cas. Cependant, il se peut que vous deviez installer le produit manuellement dans certaines situations, comme :
 
- L'installation automatique échoue en raison de problèmes de sécurité d'E/S.
- Vous devez installer le produit sur une instance nommée (pas par défaut) de Reporting Services 2016 ou sur plusieurs instances.
- Vous mettez à jour vers la dernière version et souhaitez simplement remplacer l'assembly au lieu de désinstaller l'ancienne version et d'installer la nouvelle à l'aide de l'installateur MSI.
 
{{% alert color="primary" %}}

Remarque : Prenez note que, dans ce dernier cas, vous pourriez vous retrouver avec d’autres composants (comme la documentation hors ligne) qui ne sont pas mis à jour vers la version correspondante.

{{% /alert %}}
