## Bienvenue à Aspose.PDF for Reporting Services !

Microsoft SQL Server Reporting Services répond à un besoin auquel de nombreuses organisations sont confrontées - le besoin de créer des solutions d'intelligence décisionnelle et de reporting. Jusqu'à présent, les développeurs devaient intégrer les rapports dans leurs applications, ou les organisations devaient acheter des solutions de reporting tierces coûteuses et parfois problématiques. Aujourd'hui, Microsoft SQL Server Reporting Services propose une solution complète pour distribuer les rapports à l'échelle de l'entreprise ; permettant aux entreprises de prendre des décisions plus rapidement et mieux.

**Aspose.Pdf for Reporting Services** est une autre solution unique d'Aspose, qui rend possible la génération de rapports PDF dans Microsoft SQL Server 2016/2017/2019/2022 Reporting Services et Power BI Report Server. Toutes les fonctionnalités des rapports RDL, y compris les tableaux, matrices, graphiques et images, sont converties avec le plus haut degré de précision en PDF.

Microsoft SQL Server Reporting Services possède des capacités intégrées pour exporter les rapports sous forme de documents PDF, mais il ne fournit pas le support technique nécessaire aux utilisateurs finaux. Aspose.Pdf s'efforce de fournir un support technique suprême et efficace.

Aspose.Pdf for Reporting Services crée des documents sur le serveur sans utiliser le SDK Adobe.Pdf. Aspose.Pdf for Reporting Services utilise en interne Aspose.Pdf for .NET – le composant de classe mondiale pour le traitement et les conversions de documents côté serveur.
