## Détails

La classe [PdfPageEditor](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor) dans l'espace de noms [Aspose.Pdf.Facades](https://docs-qa.aspose.com/display/pdftemp/Aspose.Pdf.Facades+namespace) contient une propriété nommée [PageSize](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor/properties/pagesize) qui peut être utilisée pour changer la taille de page d'une page individuelle ou de plusieurs pages à la fois. La propriété Pages peut être utilisée pour assigner les numéros des pages sur lesquelles la nouvelle taille de page doit être appliquée. La classe PageSize contient une liste de différentes tailles de page en tant que membres. N'importe lequel des membres de cette classe peut être assigné à la propriété PageSize de la classe [PdfPageEditor](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor). Vous pouvez également obtenir la taille de page de n'importe quelle page en utilisant la méthode GetPageSize et en passant le numéro de page.
