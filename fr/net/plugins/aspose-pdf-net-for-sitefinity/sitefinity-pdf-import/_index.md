## Introduction

### Introduction

#### Qu'est-ce que Sitefinity ?

Sitefinity est un système de gestion de contenu (CMS) web moderne basé sur ASP.NET, offrant une gestion intuitive du contenu web et un environnement de développement robuste pour les professionnels des affaires. Voici les projets que nous avons créés pour ce CMS populaire.

#### Aspose.PDF for .NET

Aspose.PDF for .NET est un composant de création et de manipulation de documents PDF qui permet à vos applications .NET de lire, écrire et manipuler des documents PDF existants sans utiliser Adobe Acrobat. Il vous permet également de créer des formulaires et de gérer les champs de formulaire intégrés dans un document PDF.

Aspose.PDF for .NET est abordable et offre une incroyable richesse de fonctionnalités, y compris des options de compression PDF ; création et manipulation de tableaux ; support pour les objets graphiques ; fonctionnalité de lien hypertexte étendue ; contrôles de sécurité avancés ; gestion de polices personnalisées ; intégration avec des sources de données ; ajout ou suppression de signets ; création de tables des matières ; ajout, mise à jour, suppression de pièces jointes et d'annotations ; importation ou exportation de données de formulaire PDF ; ajout, remplacement ou suppression de texte et d'images ; division, concaténation, extraction ou insertion de pages ; transformation de pages en image ; impression de documents PDF et bien plus encore.

#### Aspose .NET PDF Import pour Sitefinity

L'importation PDF Sitefinity est un add-on open source de [Aspose](http://www.aspose.com/) qui permet aux développeurs de lire le contenu de n'importe quel document PDF sans nécessiter d'autre logiciel. Cet add-on démontre la puissante fonctionnalité d'importation fournie par [Aspose.PDF](https://products.aspose.com/pdf/net/). Il ajoute un simple contrôle de navigateur de fichiers et un bouton **Importer depuis PDF** sur la page où l'add-on est ajouté. En cliquant sur le bouton, le contenu du document est récupéré à partir du fichier et affiché immédiatement à l'écran.

## Installation et utilisation

### Téléchargement et installation

#### Téléchargement

Vous pouvez télécharger Aspose .NET Pdf Import pour Sitefinity à partir de l'un des emplacements suivants :

- [Github](https://github.com/asposemarketplace/Aspose_for_Sitefinity/releases).
- [Sourceforge](https://sourceforge.net/projects/asposesitefinity/files/).
- [Bitbucket](https://bitbucket.org/asposemarketplace/aspose-for-sitefinity/downloads).

#### Installation

Une fois téléchargé, veuillez suivre ces étapes pour installer l'add-on dans votre site Web Sitefinity :

##### Étape 1 : Copier les fichiers dans votre installation Sitefinity

Veuillez extraire le fichier ZIP téléchargé. Vous aurez besoin d'un accès FTP ou direct au dossier d'installation de Sitefinity sur le serveur pour effectuer les opérations suivantes :

1. Copiez Aspose.PDF.dll et Aspose.SiteFinity.PDFImport.dll dans le dossier **bin** de l'installation de Sitefinity.
1. Copiez le dossier **Addons** à la racine de l'installation de Sitefinity où se trouve le dossier **bin**.

##### Étape 2 : Enregistrer l'add-on Aspose .NET PDF Import pour Sitefinity dans Sitefinity

1. Connectez-vous à votre CMS Sitefinity avec un compte **Administrateur**. La page de connexion peut être atteinte à <http://www.mywebsite.com/sitefinity>
1. Cliquez sur **Administration** puis sur **Paramètres**.
   La page des paramètres de base apparaît.
1. **Cliquez sur le lien Avancé.**.
   La page des paramètres apparaît.
1. Dans le panneau de gauche, cliquez sur **Toolboxes** suivi de **Toolboxes**, puis **PageControls**, **Sections** et **ContentToolboxSection**, puis **Tools.**.
1. Cliquez sur **Créer nouveau**.
   Le formulaire d'enregistrement du widget apparaît.
1. Remplissez les champs du formulaire comme suit :
   1. Assurez-vous que **Activé** est sélectionné.
   1. Ajoutez ~/Addons/AsposePDFImport/AsposePDFImport.ascx dans le champ **Type de contrôle CLR ou chemin virtuel**.
   1. Ajoutez **Nom**, **Titre** et **Description** comme suit :
      AsposePDFImport
      Importation PDF Aspose
      Importer des documents PDF dans Sitefinity en utilisant Aspose .NET PDF Import pour Sitefinity
   1. Vous pouvez laisser tous les autres champs tels quels.
1. Une fois terminé, cliquez sur **Enregistrer les modifications**.
   Le widget est enregistré dans la boîte à outils et peut être utilisé dans Sitefinity. Les paramètres sont également montrés dans l'image ci-dessous.

### Utilisation et démonstration vidéo

#### Utilisation

Après avoir installé et configuré l'add-on Aspose .NET PDF Import pour Sitefinity, il est vraiment simple de commencer à l'utiliser sur votre site Web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous d'être connecté à Sitefinity avec un compte de niveau Administrateur.
1. Accédez à la page où vous souhaitez ajouter l'add-on Sitefinity PDF Import. Assurez-vous que la page est ouverte en mode édition.
1. Dans le menu **Faire glisser des widgets** à droite, sélectionnez **Aspose PDF Import** et faites-le glisser à la position souhaitée.

Vous avez réussi à ajouter l'add-on Aspose .NET PDF Import pour Sitefinity à votre page. Un navigateur de fichiers et un bouton intitulé **Importer depuis PDF** apparaîtront maintenant sur la page. Quiconque peut simplement sélectionner n'importe quel document PDF et cliquer sur le bouton Importer depuis PDF pour voir le contenu du document sélectionné sur la page.
