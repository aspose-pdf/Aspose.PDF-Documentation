## Qu'est-ce que Microsoft Dynamics CRM ?

Microsoft Dynamics CRM est un logiciel de gestion de relations clients développé par Microsoft. De base, le produit se concentre principalement sur les secteurs des ventes, du marketing et du service (support technique), mais Microsoft a promu Dynamics CRM comme une plateforme XRM et a encouragé ses partenaires à utiliser son cadre propriétaire (basé sur .NET) pour le personnaliser. Il fait partie de la famille d'applications commerciales Microsoft Dynamics.

Dynamics CRM est une application serveur-client, qui, comme Microsoft SharePoint, est principalement une application web basée sur IIS qui prend également en charge de vastes interfaces de services web. Les clients accèdent à Dynamics CRM soit en utilisant un navigateur, soit par un plugin client lourd pour Microsoft Outlook.

## Aspose.PDF pour .NET

Aspose.PDF pour .NET est un composant de création et de manipulation de documents PDF qui permet à vos applications .NET de lire, écrire et manipuler des documents PDF existants sans utiliser Adobe Acrobat.
