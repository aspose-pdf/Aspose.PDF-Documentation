## Introduction

Le générateur de documents PDF OneClick d'Aspose .NET est un add-on open source utilisé pour générer et exporter des PDF depuis le CRM. Vous pouvez concevoir un modèle dans Word et l'attacher dans le CRM et générer le document comme vous le souhaitez. Vous pouvez utiliser le générateur de documents PDF OneClick avec les entités Compte, Contact, Lead, Opportunité et Devis. Il offre la possibilité d'attacher le document généré avec l'enregistrement lui-même et vous pouvez également télécharger le document. Cet add-on peut également résoudre les ensembles d'options et les champs de recherche. De plus, il vous permet de générer le PDF à partir du bouton de la barre supérieure ainsi que via un bouton en un clic sur le formulaire.

Le générateur de documents PDF OneClick d'Aspose .NET peut être utilisé avec le CRM sur site et le CRM en ligne.

Les principales fonctionnalités de cet add-on sont :

- Créer un modèle dans Word et l'attacher dans le CRM. (utilisez la fonctionnalité de champ de fusion de Word où vous souhaitez inclure des données CRM)
- Vous permet d'ajouter des ensembles d'options ainsi que des recherches.
- Fonctionne avec Compte, Contact, Lead, Opportunité et Devis.
- Un bouton en un clic qui peut générer un document.
- Les documents générés peuvent être téléchargés ainsi qu'attachés au même enregistrement d'entité.
- Un bouton de barre est également fourni pour choisir le modèle et générer le document.

Cet add-on fonctionnera avec toutes les versions de **Microsoft Dynamics CRM 2015 et CRM Online**.

## Exigences système et plateformes prises en charge

### Exigences système

Pour installer et utiliser le générateur de documents PDF OneClick d'Aspose .NET pour Microsoft Dynamics CRM, vous devez avoir l'une des versions de CRM suivantes :

- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
- .NET Framework 4.5 ou supérieur.

## Installation ou désinstallation

### Comment installer

#### Importation de la solution

- Téléchargez le fichier de solution. (Géré si vous avez l'intention de le désinstaller).
- Ouvrez le CRM et allez dans Importer la solution.
- Cliquez sur Parcourir et sélectionnez le fichier de solution téléchargé, puis cliquez sur suivant.
- Cliquez sur Suivant et attendez que la solution soit importée.
- Cliquez sur fermer lorsque la solution est importée avec succès.

#### Configuration de la page Web

- Téléchargez le code source pour le site Web.
- Copiez sur le serveur. (Recommandé sous inetpub).
- Ouvrez IIS. (Sur le serveur où vous avez l'intention de déployer le site Web, accessible depuis le serveur CRM).
- Créez un nouveau site Web.
- Choisissez le chemin physique et configurez le site Web.
- Parcourez pour vérifier s'il est accessible.
- Ouvrez le CRM et ajoutez un nouvel enregistrement sous Aspose >> Entité de configuration Aspose et spécifiez le nom et l'URL de la page Web déployée comme ci-dessous.

![Générateur de documents PDF OneClick](oneclick-pdf-document-generator_1.png)

### Comment désinstaller

- Supprimez le site Web de IIS.
- Allez dans la solution et sélectionnez "Générateur de documents PDF OneClick".
- Appuyez sur le bouton Supprimer. Appuyez sur OK pour confirmer.
