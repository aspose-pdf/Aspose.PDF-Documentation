## Introduction

Le module Aspose .NET PDF Import pour DNN permet aux développeurs de lire le contenu des documents PDF sans nécessiter d'autres logiciels tels qu'Adobe Acrobat ou un lecteur PDF.

### Fonctionnalités du module

Ce module démontre la puissante fonctionnalité d'importation fournie par [Aspose.PDF](http://www.aspose.com/pdf-component-suite.aspx). Il ajoute un simple contrôle de navigateur de fichiers, une sélection de panneau de contenu et un bouton **Importer depuis PDF** sur la page où le module est ajouté. En cliquant sur le bouton, le contenu du document est immédiatement affiché à l'écran.

## Installation

Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site DNN :

1. Connectez-vous à votre site en tant que compte **Host** ou un autre compte de niveau Supersuser.
1. Naviguez vers le menu Host et sélectionnez **Extensions**.
1. Comme indiqué, naviguez jusqu'à l'emplacement du fichier zip téléchargé, sélectionnez-le puis cliquez sur **Ouvrir**.
1. Cliquez sur **Suivant**, acceptez la licence, continuez à travers l'installation. Une fois terminé, cliquez sur le bouton **Retour**.

Veuillez consulter [cette vidéo d'installation de module](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails

**Note :** Si vous obtenez une erreur lors du téléchargement du module, cela est dû à la limite de maxRequestLength dans le web.config de votre installation DNN. Ouvrez web.config et mettez à jour maxRequestLength à 20MB en réglant **maxRequestLength="20480"** et essayez de télécharger le module à nouveau.

## Utilisation

Après avoir installé Aspose .NET PDF Import pour le module DNN, il est vraiment simple de commencer à l'utiliser sur votre site web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous que vous êtes connecté à DNN en tant que compte de niveau Host ou Admin
1. Naviguez vers la page où vous souhaitez ajouter le module d'Import
1. Depuis la liste, sélectionnez **Aspose .NET PDF Import for DNN** et faites-le glisser à l'endroit de votre choix sur la page.

Vous avez réussi à ajouter le module Aspose .NET PDF Import for DNN à votre page. Un navigateur de fichiers et un bouton intitulé **Importer depuis PDF** apparaîtront désormais sur la page avec un menu déroulant de sélection de panneau de contenu. N'importe qui peut simplement sélectionner un fichier PDF et cliquer sur le bouton **Importer depuis PDF** pour lister le contenu du document sélectionné sur la page.
