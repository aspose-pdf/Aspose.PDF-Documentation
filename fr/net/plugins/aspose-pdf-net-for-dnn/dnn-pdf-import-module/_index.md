## Introduction

Aspose .NET PDF Import pour le module DNN permet aux développeurs d'obtenir/lire le contenu des documents PDF sans nécessiter d'autre logiciel tel qu'Adobe Acrobat ou un lecteur PDF.

### Fonctionnalités du module

Ce module démontre la puissante fonctionnalité d'importation fournie par [Aspose.PDF](https://products.aspose.com/pdf/net/). Il ajoute un simple contrôle de navigateur de fichiers, une sélection de panneau de contenu, et un bouton **Importer depuis PDF** sur la page où le module est ajouté. En cliquant sur le bouton, le contenu du document est affiché à l'écran immédiatement.

## Exigences système et plateformes prises en charge

### Exigences système

Pour configurer Aspose .NET Pdf Import pour le module DNN, vous devez respecter les exigences suivantes :

- DNN 7.0+.

N'hésitez pas à nous contacter si vous souhaitez installer ce module sur d'autres versions de DNN.

### Plateformes prises en charge

Aspose .NET Pdf Import pour le module DNN prend actuellement en charge

- DNN 7.0+.

N'hésitez pas à nous contacter si vous souhaitez installer ce module sur d'autres versions de DNN.

## Installation

Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site DNN :

1. Connectez-vous à votre site en tant que **Hôte** ou un autre compte de niveau Supersuser.
1. Accédez au menu Hôte et sélectionnez **Extensions**.
1. Cliquez sur **Assistant d'installation d'extension**.
1. Comme indiqué, parcourez l'emplacement du fichier zip téléchargé, sélectionnez-le puis cliquez sur **Ouvrir**.
1. Cliquez sur **Suivant**, acceptez la licence, continuez à travers l'installation. Une fois terminé, cliquez sur le bouton **Retour**.

Veuillez consulter [cette vidéo d'installation du module](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails.

**Remarque :** Si vous obtenez une erreur lors du téléchargement du module, cela est dû à la limite maxRequestLength dans le web.config de votre installation DNN. Ouvrez web.config et mettez à jour maxRequestLength à 20 Mo en définissant **maxRequestLength="20480"** et essayez de télécharger le module à nouveau.

## Utilisation

Après avoir installé Aspose .NET PDF Import pour le module DNN, il est vraiment simple de commencer à l'utiliser sur votre site web. Veuillez suivre ces étapes simples pour commencer

1. Assurez-vous d'être connecté à DNN en tant que compte Hôte ou Admin.
1. Accédez à la page où vous souhaitez ajouter le module d'importation.
1. Sélectionnez **Modules** -> **Ajouter un nouveau module** dans le ruban supérieur.

1. Dans la liste, sélectionnez **Aspose .NET PDF Import pour DNN** et faites-le glisser à l'endroit de votre choix sur la page.

Vous avez réussi à ajouter le module Aspose .NET PDF Import pour DNN à votre page. Un navigateur de fichiers et un bouton intitulé **Importer depuis PDF** apparaîtront maintenant sur la page avec un menu déroulant de sélection de panneau de contenu. Quiconque peut simplement sélectionner un fichier PDF et cliquer sur le bouton **Importer depuis PDF** pour lister le contenu du document sélectionné sur la page.
