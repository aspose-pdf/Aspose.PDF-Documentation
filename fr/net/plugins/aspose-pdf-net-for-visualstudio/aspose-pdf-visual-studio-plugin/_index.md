## Introduction

Microsoft Visual Studio est un environnement de développement intégré (IDE) de Microsoft. Il est utilisé pour développer des programmes informatiques pour Microsoft Windows, ainsi que des sites web, des applications web et des services web. Visual Studio utilise des plateformes de développement de logiciels Microsoft telles que Windows API, Windows Forms, Windows Presentation Foundation, Windows Store et Microsoft Silverlight. Il peut produire à la fois du code natif et du code géré.

### Aspose.PDF pour .NET

Aspose.PDF pour .NET est un composant de création et de manipulation de documents PDF qui permet à vos applications .NET de lire, écrire et manipuler des documents PDF existants sans utiliser Adobe Acrobat.

Pour plus de détails sur [Aspose.PDF pour l'API .NET](http://www.aspose.com/.net/Pdf-component.aspx)

### Explorez les exemples Aspose.PDF pour .NET en utilisant Visual Studio

Plugin Visual Studio Aspose - **Explorez les exemples Aspose.PDF pour .NET** est un excellent outil pour télécharger rapidement et explorer les exemples de l'API Aspose.PDF pour .NET. Il permet de gagner beaucoup de temps et d'efforts en offrant une option très simple pour sélectionner, télécharger et ouvrir les projets d'exemples les plus récents sans avoir à quitter votre Visual Studio.

Cette version offre les fonctionnalités suivantes :

- Prend en charge Visual Studio 2010, Visual Studio 2012 et Visual Studio 2013.
- Facile à lancer depuis le menu **Fichier > Aspose** dans Visual Studio
- Économise du temps et réduit la courbe d'apprentissage.
- Vous permet de sélectionner et d'ouvrir les derniers exemples de l'API Aspose.PDF pour .NET hébergés sur Github.com.
- Vous permet d'ouvrir des projets d'exemples en **C#** ou **VB.NET**
- Un projet sélectionné est automatiquement ouvert dans Visual Studio avec toutes les références nécessaires configurées pour vous offrir un environnement prêt à l'exécution et à l'exploration.
- Télécharge automatiquement les dernières bibliothèques API de **NuGet** et les exemples de **GitHub**.

## Utilisation

Une fois installé, vous pouvez facilement lancer et utiliser le plugin interactif comme décrit ci-dessous :
**1.** Ouvrez le plugin en sélectionnant **Fichier > Aspose** suivi de **Nouveau projet d'exemple Aspose.PDF** dans Visual Studio.

![New Aspose.PDF Example Project dialog](aspose-pdf-visual-studio-plugin_3.png)

**2.** La première fois que vous ouvrez le plugin, vous verrez le dialogue de sélection de l'API comme indiqué ci-dessous. Cela prendra quelques minutes pour télécharger la dernière API et leurs bibliothèques depuis **NuGet** et les derniers exemples de code source depuis **GitHub**.

![Aspose.PDF Example Downloading dialog](aspose-pdf-visual-studio-plugin_4.png)
![Dialogue de téléchargement d'exemple Aspose.PDF](aspose-pdf-visual-studio-plugin_4.png)

**3.** Dans le dialogue Sélectionner une API pour voir les exemples, sélectionnez ou modifiez l'emplacement où les projets d'exemple seront créés et sélectionnez ou modifiez également le type de projet d'exemples comme **C#** ou **VB.NET** et cliquez sur **Continuer** pour ouvrir le projet d'exemple dans Visual Studio.

![Dialogue de sélection de dossier de projet d'exemple Aspose.PDF](aspose-pdf-visual-studio-plugin_1.png)

**4.** Nouveaux projets d'exemples selon la sélection du type de projet **C#** ou **VB.NET** dans Visual Studio.
