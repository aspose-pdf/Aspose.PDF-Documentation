## Vue d'ensemble
Pour produire et manipuler des documents PDF, la machine sur laquelle Aspose.PDF pour .NET est exécuté n'a pas besoin d'avoir Adobe Acrobat installé : Aspose.PDF pour .NET est un moteur de création de documents PDF. Cependant, pour visualiser les documents PDF produits par Aspose.PDF pour .NET, la machine doit au moins disposer d'Adobe Acrobat Reader. Il peut être téléchargé gratuitement.
