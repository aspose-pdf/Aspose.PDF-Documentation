## Aperçu
L'API peut être utilisée pour construire des applications sur tout système d'exploitation qui prend en charge .NET Standard 2.0 et ultérieur ou .NET Framework 4.0.

Pour produire et manipuler des documents PDF, la machine sur laquelle Aspose.PDF for .NET fonctionne n'a pas besoin d'avoir Adobe Acrobat installé : Aspose.PDF for .NET est un moteur de création de documents PDF. Cependant, pour visualiser les documents PDF produits par Aspose.PDF for .NET, la machine doit avoir au moins un Adobe Acrobat Reader. Il peut être téléchargé gratuitement.

## Environnements de développement
Vous pouvez utiliser Aspose.PDF for .NET pour développer des applications dans n'importe quel environnement de développement qui cible la plateforme .NET, mais les environnements suivants sont explicitement pris en charge :

- Microsoft Visual Studio 2010[^1].
- Microsoft Visual Studio 2011[^1].
- Microsoft Visual Studio 2012[^1].
- Microsoft Visual Studio 2013[^1].
- Microsoft Visual Studio 2015[^1].
- Microsoft Visual Studio 2017[^1].
- Microsoft Visual Studio 2019.
- Microsoft Visual Studio 2022.
- Microsoft Visual Studio Code.

## Frameworks pris en charge
Les versions suivantes de .NET Framework sont prises en charge :

- .NET Framework 4.0.
- .NET Framework 4.8.1.
- .NET Standard 2.0 (.NET Core 3.1+).
- .NET 6.0.
- .NET 7.0.
- .NET 8.0.

[^1]: Aspose.PDF for .NET est pris en charge avec .NET Framework 4.0. Cependant, étant donné que .NET Framework 4.0 est obsolète, ce support est fourni uniquement à des fins de compatibilité. Veuillez noter que nous nous réservons le droit de cesser de publier l'assembly pour .NET Framework 4.0 à tout moment.
