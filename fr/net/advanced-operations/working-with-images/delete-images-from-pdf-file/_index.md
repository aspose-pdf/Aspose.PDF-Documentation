Le code suivant fonctionne également avec la bibliothèque [Aspose.PDF.Drawing](/pdf/net/drawing/).

Pour supprimer une image d'un fichier PDF :

1. Créez un objet Document et ouvrez le fichier PDF d'entrée.
1. Obtenez la Page qui contient l'image de la collection de Pages de l'objet Document.
1. Les images sont contenues dans la collection d'Images, trouvée dans la collection de Ressources de la page.
1. Supprimez une image avec la méthode Delete de la collection d'Images.
1. Enregistrez le résultat en utilisant la méthode Save de l'objet Document.

Le code suivant montre comment supprimer une image d'un fichier PDF.

```csharp
// Pour des exemples complets et des fichiers de données, veuillez aller à https://github.com/aspose-pdf/Aspose.PDF-for-.NET
// Le chemin vers le répertoire des documents.
string dataDir = RunExamples.GetDataDir_AsposePdf_Images();

// Ouvrir le document
Document pdfDocument = new Document(dataDir+ "DeleteImages.pdf");

// Supprimer une image particulière
pdfDocument.Pages[1].Resources.Images.Delete(1);

dataDir = dataDir + "DeleteImages_out.pdf";
// Sauvegarder le fichier PDF mis à jour
pdfDocument.Save(dataDir);
```
