Les documents PDF peuvent être visualisés et imprimés, mais pas toujours facilement modifiés. Cependant, la **bibliothèque Aspose.PDF pour Python via C++** vous aidera à faire face même à la tâche la plus difficile dans le travail avec les PDF.

Vous êtes capable de faire ce qui suit :

- [Optimiser PDF](/pdf/fr/python-cpp/optimize-pdf/) - optimiser le contenu des pages, optimiser la taille du fichier, supprimer les objets inutilisés, compresser toutes les images pour une optimisation réussie du document.
- [Fusionner PDF](/pdf/fr/python-cpp/merge-pdf-documents/) - fusionner plusieurs fichiers PDF en un seul document PDF à l'aide de Python.
- [Diviser PDF](/pdf/fr/python-cpp/split-document/) - diviser les pages PDF en fichiers PDF individuels dans vos applications Python.
