## Aperçu

Pour produire des documents PDF, la machine pour exécuter [Aspose.PDF pour PHP via Java](https://products.aspose.com/pdf/php-java/) n'a pas besoin d'avoir [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) installé car Aspose.PDF pour PHP via Java est lui-même un moteur de création de documents PDF. Cependant, pour visualiser les documents PDF produits par Aspose.PDF pour PHP via Java, la machine a besoin d'au moins un [Adobe Acrobat Reader](https://www.adobe.com/acrobat/pdf-reader.html). Il peut être téléchargé gratuitement.

## Prérequis

- JDK
- PHP/Java Bridge
- Serveur Web comme Tomcat - version 9.0
- PHP versions 7 et 8
