## Composant Aspose.PDF for Android via Java

{{% alert color="primary" %}}

**Aspose.PDF for Android via Java** est un composant qui permet aux développeurs de créer et de manipuler des documents PDF de manière programmatique, qu'ils soient simples ou complexes. Aspose.PDF for Android via Java permet aux développeurs d'insérer des tableaux, des graphiques, des images, des hyperliens, des polices personnalisées - et plus encore - dans les documents PDF. Il peut également être utilisé pour compresser les documents PDF. Aspose.PDF for Android via Java offre d'excellentes fonctionnalités de sécurité pour le développement de PDF sécurisés. La caractéristique la plus distinctive d'Aspose.PDF for Android via Java est qu'il prend en charge la création de PDF à la fois via une API et à partir de modèles XML.

{{% /alert %}} 

## Description du produit

**Aspose.PDF for Android Java** est une API de traitement PDF qui permet aux applications Android de lire, écrire, manipuler et afficher des documents PDF. L'API prend en charge un large éventail de formats de fichiers standard, y compris HTML, TXT, EPUB, XPS et images.

**Aspose.PDF for Android via Java** est une solution abordable qui offre des options de compression, une prise en charge graphique, la création de documents à partir de zéro, la manipulation des styles de texte \u0026 du formatage, l'ajout, la modification, l'extraction et la suppression de pièces jointes, d'annotations, et d'autres fonctionnalités avancées populaires.67

## Installation

### Évaluer Aspose.PDF for Android via Java

{{% alert color="primary" %}} 

Vous pouvez télécharger [Aspose.PDF pour Android via Java](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf-android-via-java) pour l'évaluation. Le téléchargement d'évaluation est le même que le téléchargement acheté. La version d'évaluation devient simplement sous licence lorsque vous ajoutez quelques lignes de code à [appliquer la licence](/pdf/fr/androidjava/licensing/).

Si vous souhaitez tester Aspose.PDF pour Android via Java sans les limitations de la version d'évaluation, vous pouvez également demander une Licence Temporaire de 30 jours. Veuillez vous référer à [Comment obtenir une Licence Temporaire ?](https://purchase.aspose.com/temporary-license)

{{% /alert %}} 

## Installation d'Aspose.PDF pour Android via Java depuis le référentiel Aspose

Aspose héberge toutes les API Java sur [Dépôt Aspose](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf-android-via-java). Vous pouvez facilement utiliser Aspose.PDF for Android via Java API directement dans vos projets Maven avec des configurations simples.
