## Aperçu

Pour produire des documents PDF, la machine pour exécuter [Aspose.PDF pour Android via Java](https://products.aspose.com/pdf/android-java/) n'a pas besoin d'avoir [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) installé car Aspose.PDF pour Java est lui-même un moteur de création de documents PDF. Cependant, pour visualiser les documents PDF produits par Aspose.PDF pour Android via Java, la machine doit disposer au moins d'un [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). Il peut être [téléchargé](https://www.adobe.com/acrobat/pdf-reader.html) gratuitement.

## Systèmes d'exploitation pris en charge

Aspose.PDF pour Android via Java prend en charge tout appareil mobile qui exécute l'environnement d'exécution Java, y compris, mais sans s'y limiter :

### Android

- API Android 32
- API Android 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## Environnements de Développement

Vous pouvez utiliser Aspose.PDF pour Android via Java pour développer des applications dans n'importe quel environnement de développement qui cible la plateforme Java sous Android, mais les environnements suivants sont explicitement pris en charge :

- Android Studio et outils SDK
