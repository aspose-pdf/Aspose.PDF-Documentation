## 概要
Aspose.PDF for .NETが動作するマシンは、Adobe Acrobatをインストールする必要はありません。Aspose.PDF for .NETはPDFドキュメント生成エンジンです。ただし、Aspose.PDF for .NETで生成されたPDFドキュメントを表示するためには、少なくともAdobe Acrobat Readerが必要です。無料でダウンロードできます。

- Microsoft Visual Studio 2010

- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013

- Microsoft Visual Studio 2019
