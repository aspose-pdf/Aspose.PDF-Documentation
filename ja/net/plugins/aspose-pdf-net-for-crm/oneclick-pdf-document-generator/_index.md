## はじめに

Aspose .NET OneClick PDF Document Generatorは、CRMからPDFを生成およびエクスポートするためのオープンソースアドオンです。Wordでテンプレートを設計し、CRMに添付して、希望の形式でドキュメントを生成できます。OneClick PDF Document Generatorは、アカウント、連絡先、リード、機会、見積もりエンティティと共に使用できます。生成されたドキュメントをレコード自体に添付するオプションがあり、ドキュメントをダウンロードすることもできます。このアドオンは、オプションセットやルックアップフィールドも解決できます。さらに、トップリボンボタンからPDFを生成することも、フォーム上のワンクリックボタンを介して行うことができます。

Aspose .NET OneClick PDF Document Generatorは、オンプレミスのCRMおよびCRM Onlineで使用できます。

このアドオンの主な機能は次のとおりです。

- Word内でテンプレートを作成し、CRMに添付します。（CRMデータを含めたい場所でWordのマージフィールド機能を使用）
- オプションセットやルックアップを追加できます。
- アカウント、連絡先、リード、機会、見積もりで動作します。
- ドキュメントを生成できるワンクリックボタン。
- 生成されたドキュメントはダウンロードでき、同じエンティティレコードに添付できます。
- テンプレートを選択してドキュメントを生成するためのリボンボタンも提供されています。

このアドオンは、**Microsoft Dynamics CRM 2015およびCRM Online**のすべてのバージョンで動作します。

## システム要件とサポートされているプラットフォーム

### システム要件

Microsoft Dynamics CRM用のAspose .NET OneClick PDF Document Generatorをインストールして使用するには、次のいずれかのCRMバージョンが必要です。

- Microsoft Dynamics CRM 2015。
- Microsoft Dynamics CRM Online。
- .NET Framework 4.5以上。

このアドオンのインストールまたは使用に関して問題が発生した場合は、お気軽にお問い合わせください。

## インストールまたはアンインストール

### インストール方法

#### ソリューションのインポート

- ソリューションファイルをダウンロードします。（アンインストールする場合は管理されたものを選択）。
- CRMを開き、ソリューションのインポートに移動します。
- ブラウズをクリックし、ダウンロードしたソリューションファイルを選択して次へ進みます。
- 次へをクリックし、ソリューションのインポートを待ちます。
- ソリューションが正常にインポートされたら、閉じるをクリックします。

#### ウェブページの構成

- ウェブサイトのソースコードをダウンロードします。
- サーバーにコピーします。（inetpubの下に配置することを推奨）。
- IISを開きます。（ウェブサイトをデプロイする予定のサーバーで、CRMサーバーからアクセス可能）。
- 新しいウェブサイトを作成します。
- 物理パスを選択し、ウェブサイトを構成します。
- アクセス可能かどうかを確認するためにブラウズします。
- CRMを開き、Aspose>> Aspose Configuration Entityの下に新しいレコードを追加し、デプロイされたウェブページの名前とURLを以下のように指定します。

![OneClick PDF Document Generator](oneclick-pdf-document-generator_1.png)

### アンインストール方法

- IISからウェブサイトを削除します。
- ソリューションに移動し、「OneClick PDF Document Generator」を選択します。
- 削除ボタンを押します。確認のためにOKを押します。
