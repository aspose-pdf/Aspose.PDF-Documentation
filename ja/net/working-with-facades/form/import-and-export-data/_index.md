## フィールドの値をJSONファイルにエクスポート

Aspose.Pdf.Facadesは、フォームフィールドを操作するための代替APIを提供します。このスニペットは、このAPIを使用してフィールド値をエクスポートおよびインポートする方法を示しています。

```cs
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form();
// Open Document
form.BindPdf(_dataDir + "Sample-Form-01.pdf");

// Create XFDF file.
System.IO.FileStream jsonStream = new FileStream("Sample-Form-01.json", FileMode.Create);

// Export data
form.ExportJson(jsonStream);

// Close file stream
jsonStream.Close();

// Close the document
form.Dispose();
```
## JSONファイルからフィールドに値をインポートする

このコードスニペットは、Aspose.Pdf.Facades APIを使用して、JSONファイルからPDFドキュメントのフォームフィールドに値をインポートする方法を示しています。FileStreamはJSONファイルを処理するために使用されます。

```cs

Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form();
// ドキュメントを開く
form.BindPdf("Sample-Form-01.pdf");

// XFDFファイルを作成
System.IO.FileStream jsonStream = new FileStream("Sample-Form-01.json", FileMode.Open);

// データをエクスポート
form.ImportJson(jsonStream);

// ファイルストリームを閉じる
jsonStream.Close();

// ドキュメントを閉じる
form.Dispose();
```
