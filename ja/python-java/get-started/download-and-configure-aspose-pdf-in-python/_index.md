## 必要なライブラリのダウンロード

以下に記載されている必要なライブラリをダウンロードしてください。これらは、Java経由でPython用のAspose.PDFを実行するために必要です。

- Aspose: [Aspose.PDF for Python via Java Component](https://releases.aspose.com/pdf/pythonjava/)
- または PyPI: [The Python Package Index](https://pypi.org/project/aspose-pdf-for-python-via-java/)
- [JPype](https://pypi.python.org/pypi/JPype1)
- JDK 1.8+

## ソーシャルコーディングサイトからの例のダウンロード

以下に記載されているソーシャルコーディングサイトで、実行可能な例のリリースをダウンロードできます。

### GitHub

#### Python via Java 用の Aspose.PDF の例

[Aspose.PDF for Python via Java](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Plugins/Aspose_Pdf_Java_for_Python)

### ソースコードの設定方法

以下の簡単な手順に従って、ソースコードを開き、拡張してください。

1. Python 3.7以上を[公式サイト](https://www.python.org/downloads)からインストールします
2. JPype 1.5.0以上を[Sourceforge](http://sourceforge.net/projects/jpype/files/JPype/)からインストールします
3. Java JDK 1.8以上を[Java Downloads](https://www.oracle.com/java/technologies/downloads/)からインストールします
4. Aspose.PDF for Python via Java APIをダウンロードします: [Aspose.PDF for Python via Java Component](https://releases.aspose.com/pdf/pythonjava/)
5. または、pipを使用してAspose.PDF for Python via Java APIをインストールします: [PyPI](https://pypi.org/project/aspose-pdf-for-python-via-java/)
