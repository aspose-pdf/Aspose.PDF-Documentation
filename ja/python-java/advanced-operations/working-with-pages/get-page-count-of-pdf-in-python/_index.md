<ins>**Aspose.PDF Java for Python**を使用してPDFドキュメントのページ数を取得するには、**GetNumberOfPages**クラスを呼び出します。

**Pythonコード**
```
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "Page Count:" . page_count

```
