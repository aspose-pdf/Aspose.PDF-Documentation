**高度な操作**は、Aspose.PDFで作成されたドキュメント（[基本操作](/pdf/ja/python-java/basic-operations/)で説明されています）や、Adobe Acrobat、Google Docs、Microsoft Office、Open Office、その他のPDFプロデューサーで作成されたPDFであっても、既存のPDFファイルをプログラムで処理する方法についてのセクションです。

次の異なる方法を学びます：

- [ドキュメントの操作](/pdf/ja/python-java/working-with-documents/) - ドキュメントを圧縮、分割、結合し、ドキュメント全体に対して他の操作を行います。
- [ページの操作](/pdf/ja/python-java/working-with-pages/) - ページを追加、移動または削除、トリミングし、透かしやスタンプなどを追加します。
- [フォームの操作](/pdf/ja/python-java/working-with-forms/) - インタラクティブなPDFドキュメントを扱い、フォームフィールドを追加し、データを抽出します。
- [テキストの操作](/pdf/ja/python-java/working-with-text/) - PDFにテキストを追加、フォーマット、検索および置換します。
