## **PDF ファイルにバーコードを追加する**

{{% alert color="primary" %}}

Aspose.PDF for SharePoint は PDF ドキュメントにバーコードを追加できる機能を提供します。ユーザーはライブラリに追加された PDF ドキュメントの各ページの左下隅にバーコードを追加できます。以下の画像は、バーコードが追加された PDF ドキュメントがどのように見えるかのイメージです。

**左下隅のバーコード**

![todo:image_alt_text](add-barcode-to-a-pdf-file_1.png)

{{% /alert %}}

{{% alert color="primary" %}}

特定のライブラリでバーコード機能を有効にするには、以下のように **Library Tools** の **Aspose PDF Watermark Tools** タブにある **Watermark Settings** ボタンを使用します。

**PDF 透かし設定**

![todo:image_alt_text](add-barcode-to-a-pdf-file_2.png)

特定のライブラリでバーコードを有効にすると、Aspose.PDF for SharePoint はそのライブラリに追加されたすべての PDF ドキュメントにバーコードを付加します。

{{% /alert %}}
