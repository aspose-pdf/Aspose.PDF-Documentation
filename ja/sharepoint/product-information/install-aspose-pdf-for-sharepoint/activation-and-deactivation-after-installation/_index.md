{{% alert color="primary" %}}

インストール中に、Aspose.PDF for SharePoint は選択されたすべてのサイトコレクションに対して有効化されます。インストール後、サイトコレクションのルート Web サイトの Site Actions メニューを使用して、Aspose.PDF for SharePoint を有効化および無効化できます。

{{% /alert %}}
