## **サーバー要件**

Aspose.PDF for SharePoint をインストールするには、Microsoft SharePoint Server 2010 が実行されているコンピュータが必要です。

## **クライアント要件**

SharePoint サイトにアクセスするにはブラウザが必要です。Aspose.PDF for SharePoint は以下のブラウザでテストされています。

- Microsoft Internet Explorer 8
- Mozilla FireFox 3.5
- Google Chrome 3
- Safari 4
