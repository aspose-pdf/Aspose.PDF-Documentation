## クロスプラットフォーム ソフトウェア

さまざまなプラットフォームと Rust バージョンをサポートします。参照 [システム要件](/pdf/ja/rust-cpp/system-requirements/) 詳細について。

## 一般的な機能

幅広いリストからの読み込みと保存をサポートします [ファイル形式](/pdf/ja/rust-cpp/supported-file-formats/).

## サポートされている PDF 標準

Aspose.PDF for Rust は PDF バージョン 1.2、1.3、1.4、1.5、1.6 および 1.7 をサポートしています。
