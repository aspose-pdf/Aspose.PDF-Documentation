Aspose は、すべての製品に対して無制限の無料テクニカルサポートを提供しています。サポートは評価版を含むすべてのユーザーが利用可能です。主なサポート手段は Aspose.Forums です。

{{% alert color="primary" %}}
Aspose は電話によるテクニカルサポートを提供していないことにご留意ください。電話サポートは販売および購入に関する質問にのみ利用可能です。
{{% /alert %}}

## Aspose 無料サポートフォーラム

Aspose.PDF はすべての製品に対して無料のテクニカルサポートを利用できるようにします。Aspose.PDF に関する質問がある場合は、次の記事をご確認ください：

- 最も一般的なサポートの情報源は [Aspose.Forums](https://forum.aspose.com/). 以下の [Aspose.PDF フォーラム](https://forum.aspose.com/c/pdf/10) セクションでは、質問を投稿すれば、できるだけ早く回答が得られます。
- Aspose は電話での技術サポートを提供していないことをご承知おきください。電話での問い合わせは販売および購入に関する質問にのみ利用可能です。
- 時差があることをご理解ください。これにより、回答までにかかる時間が影響を受ける可能性があります。

## 問題の報告または機能リクエスト

Aspose.PDF に関する問題で支援が必要な場合は、次の推奨事項に従って、最も効率的な方法で解決できるようにしてください:

- まず、問題を報告する前に最新の Aspose.PDF バージョンを使用しているか確認してください、参照 [Aspose.PDF for Rust をダウンロード](https://releases.aspose.com/pdf/rustcpp/) 最新バージョンについて確認するには。
- おそらく、問題の解決策はすでに議論されており、解決策が存在しています。そのため、フォーラムとドキュメントを調べて、確認してください [API リファレンス](https://reference.aspose.com/pdf/rust-cpp/) 問題を報告する前に。
- 質問を作成する際には、元のドキュメントと、問題を引き起こす可能性のあるコードのフラグメントを含めてください。複数ある場合は、これらのファイルを 1 つの文書に zip することができます。
- 心配しないでください。Aspose の開発者とあなただけがスティッキーファイルにアクセスできます。
