## C++ を介した Aspose.PDF for Rust のサポートプラットフォーム

asposepdf パッケージは、開発者が PDF ファイルを直接操作できる強力なツールキットであり、PDF に関するさまざまなタスクを支援します。
PDF を他の形式に変換するための固有の機能が含まれています。

Linux x64、macOS x86_64、macOS arm64、および Windows x64 プラットフォームのサポートが実装されました。

パッケージのルートディレクトリにある ‘lib’ フォルダーからのプラットフォーム固有の動的ライブラリのバージョンが、生成されたアプリケーションの配布に必要です： 

- libAsposePDFforRust_linux_amd64.so は Linux x64 プラットフォーム用です 
- libAsposePDFforRust_darwin_arm64.dylib は macOS arm64 プラットフォーム用です 
- libAsposePDFforRust_darwin_amd64.dylib は macOS x86_64 プラットフォーム用です 
- AsposePDFforRust_windows_amd64.dll（Windows x64 プラットフォーム用）。
