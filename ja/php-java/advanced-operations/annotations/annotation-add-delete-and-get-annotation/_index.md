**PDFドキュメントにおけるアノテーションとは何ですか？**

これらは、ファイルの内容を拡張し、他のユーザーのために編集やコメントを追加するためにファイルに追加する追加のオブジェクトです。また、ドキュメント内のテキストをより読みやすくしたり、強調表示したり、下線を引いたり、まったく新しいテキストを追加したりすることも可能です。
