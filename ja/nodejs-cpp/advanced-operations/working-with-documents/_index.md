PDF は Portable Document Format の略で、ソフトウェア、ハードウェア、またはオペレーティングシステムに依存せずに電子的な形で文書を表示するために使用されます。

PDF は現在、国際標準化機構（ISO）によって維持されているオープン標準です。

元々の目的は、プラットフォームやコンピュータプログラムに関係なく、文書の内容とレイアウトを保護・保存することでした。このため PDF は編集が難しく、情報の抽出さえも時に課題となります。

しかし **Aspose.PDF for Node.js via C++** は、PDF ドキュメントを扱う際に発生するタスクのほとんどに対処するのに役立ちます。

以下のことが可能です:

- [PDF の最適化](/pdf/ja/nodejs-cpp/optimize-pdf/) - ページコンテンツの最適化、ファイルサイズの最適化、未使用オブジェクトの削除、すべての画像を圧縮して文書の最適化に成功させます。
- [PDF リソースの最適化](/pdf/ja/nodejs-cpp/optimize-pdf-resources/) - ページコンテンツの最適化、ファイルサイズの最適化、未使用オブジェクトの削除、すべての画像を圧縮して文書の最適化に成功させます。
- [PDF の結合](/pdf/ja/nodejs-cpp/merge-pdf/) - Node.js を使用して複数の PDF ファイルを単一の PDF ドキュメントに結合します。
- [PDF を分割](/pdf/ja/nodejs-cpp/split-pdf/) - Node.js と C++ を使用して PDF ページを個別の PDF ファイルに分割します。
