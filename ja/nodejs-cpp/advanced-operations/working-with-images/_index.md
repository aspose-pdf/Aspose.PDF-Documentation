PDF ドキュメントの画像操作には **Aspose.PDF for Node.js via C++** を使用してください。Node.js 環境の PDF ファイルに画像を追加および削除することが可能です。

以下のことができます：

- [PDF ファイルに画像を追加](/pdf/ja/nodejs-cpp/add-image-to-pdf/) - Node.js 環境で PDF に画像を追加します。
- [PDF ファイルから画像を削除する](/pdf/ja/nodejs-cpp/delete-images-from-pdf-file/) - PDF ファイルから画像を削除するコードスニペットを確認してください。
