PDF は、アプリケーションソフトウェア、ハードウェア、オペレーティングシステムに依存しない形式で文書やその他の参照資料を提示するための標準を提供する目的で開発されました。PDF ファイルの内容はテキストに限定されず、ハイパーリンク、画像、クリック可能なボタンやフォーム、電子署名、透かしなどが含まれます。そのため、内容を編集または変更するために PDF ファイルを他の形式に変換する必要があることがしばしばあります。 
**Our Aspose.PDF for Node.js via C++** ツールは、PDF 文書を最も一般的な形式に迅速かつ簡単に変換できるようにします。サポートされている形式の完全な一覧については、セクションをご覧ください。 [Aspose.PDF がサポートするファイル形式](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/).

Aspose.PDF for Node.js via C++ は、Node.js 環境で PDF ドキュメントをさまざまな形式に変換することを可能にします。また、Aspose.PDF の変換品質を確認し、Aspose.PDF コンバータ アプリでオンライン結果を表示できます。コードスニペットを使用したドキュメント変換のセクションをご覧ください。

## Node.js via C++ を使用した変換方法

Microsoft Word ドキュメントは、最も柔軟で編集可能です。本記事では、Aspose.PDF を使用して Node.js 環境で PDF を Word に変換する方法を学びます。

- [PDF を DOC に変換](/pdf/ja/nodejs-cpp/convert-pdf-to-doc/) - Node.js を使用して PDF ドキュメントを Word 形式に変換できます。

数値フォーマットは、テーブル内のデータを読みやすくするだけでなく、テーブルを使いやすくするためにも必要です。もちろん、PDF ドキュメントから Excel 形式にそのようなデータを変換する必要がある場合は、当社の Aspose.PDF for Node.js をご利用ください。

- [PDF を Microsoft XLSX に変換](/pdf/ja/nodejs-cpp/convert-pdf-to-xlsx/) - このセクションでは、PDF ドキュメントを XLSX に変換する方法を説明します。

PowerPoint 形式はさまざまなプレゼンテーションを作成するために使用されます。PPT ファイルには、さまざまな情報を含む多数のスライドまたはページが含まれています。

- [PDF を Microsoft PowerPoint に変換](/pdf/ja/nodejs-cpp/convert-pdf-to-powerpoint/) - ここでは、変換プロセスを追跡しながら PDF を PowerPoint に変換することについて説明します。

HyperText Markup Language はハイパーテキスト文書記述言語であり、ウェブページを作成するための標準言語です。Aspose.PDF for Node.js を使用すると、HTML ドキュメントを簡単に変換したり、その逆も行うことができます。

さまざまな目的で PDF に変換する必要がある画像フォーマットは多数あります。Aspose.PDF for Node.js は最も一般的な画像フォーマットをサポートします。

- [PDF をさまざまな画像フォーマットに変換](/pdf/ja/nodejs-cpp/convert-pdf-to-images-format/) - PDF ページを JPEG、PNG、SVG、その他のフォーマットの画像として変換

このセクションには次のようなフォーマットが含まれます: EPUB、XPS、TeX、Text、そしてグレースケール PDF。

- [PDF ファイルを他のフォーマットに変換](/pdf/ja/nodejs-cpp/convert-pdf-to-other-files/) - このトピックでは、PDF ドキュメントをさまざまなフォーマットに変換する方法について説明します

PDF/A は、電子文書の長期保存のために設計された PDF のバージョンです。
正直に言えば、外部からはそれが PDF か PDF/A かを判断するのは非常に難しいです。このファイルを確認するには、バリデータが使用されます。PDF から PDF/A、またはその逆への高品質変換については、以下の記事をご覧ください。

- [PDF を PDF/A 形式に変換する](/pdf/ja/nodejs-cpp/convert-pdf-to-pdfa/) - Aspose.PDF の Node.js ツールキットは、PDF を PDF/A に変換する簡単な方法を提供します。

- [PDF/A を PDF 形式に変換する](/pdf/ja/nodejs-cpp/convert-pdfa-to-pdf/) - Node.js を使用して PDF/A を PDF 形式に変換すると、簡単、迅速、高品質です。
