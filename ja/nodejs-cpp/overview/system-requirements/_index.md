## 概要

PDF 文書を処理するには、 [Aspose.PDF for Node.js via C++](https://products.aspose.com/pdf/nodejs-cpp/), サードパーティのライブラリやプログラムは必要ありません。
インストールが必要です [Node.js](https://nodejs.org/). Node.js バージョン 16 以上が必要です。LTS バージョンの使用を強く推奨します。
Node.js は [インストール](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) さまざまな方法で。

## サポートされているオペレーティングシステム

すべてのサポート対象バージョンの [Linux](https://en.wikipedia.org/wiki/Linux) および [macOS](https://www.apple.com/macos/).
