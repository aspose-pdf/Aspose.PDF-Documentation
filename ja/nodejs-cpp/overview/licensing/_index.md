## 評価版の制限

購入前に顧客が当社のコンポーネントを十分にテストできるように、評価版では通常通り使用できるようにしています。

>評価版の制限なしで Aspose.PDF for Node.js via C++ をテストしたい場合は、30日間の Temporary License をリクエストすることもできます。詳細はご参照ください。 [一時ライセンスの取得方法は？](https://purchase.aspose.com/temporary-license/)

- **評価用ウォーターマークが付いた PDF。**
Aspose.PDF for Node.js via C++ の評価版は完全な製品機能を提供しますが、生成された PDF 文書のすべてのページの上部には "Evaluation Only. Created with Aspose.PDF. Copyright 2002-2023 Aspose Pty Ltd." というウォーターマークが付けられます。

- **処理できるコレクション アイテムの数の制限。**
評価版では、任意のコレクションからは最大 4 つの要素しか処理できません（例: ページ 4 枚、フォーム フィールド 4 個など）。


## クラシック ライセンス

ライセンスファイル名は "aspose.lic" にする必要があります。受け取ったライセンスファイルの名前を変更し、Product フォルダーに配置してください。
