Aspose.PDF for Node.js via C++ の最初のパブリックリリースを発表できることを嬉しく思います。

**Aspose.PDF for Node.js via C++** は、開発者が Web 上で PDF ファイルを直接操作できるようにします。この Node.js ソリューションは WebAssembly テクノロジーの使用を基盤としており、Aspose.PDF for .NET をベースにしています。aspose-pdf for Node.js via C++ を簡単に使用でき、インストール手順に従ってください。

ご訪問ください [製品ページ](https://products.aspose.com/pdf/nodejs-cpp/) さらに詳しい情報を確認し、最初のリリースをダウンロードする機会を得る [Aspose.PDF for Node.js via C++](https://releases.aspose.com/pdf/nodejs-cpp/).

## **主要なサポート機能**

- PDF ファイル情報の設定、取得、削除
- ブックマークの削除
- 注釈の削除
- PDFからテキストを抽出する
- PDFドキュメントを最適化する
- リソースを最適化する 
- PDFを2つのファイルに分割する
- 2つのファイルをPDFに結合する
- PDFを修復する
