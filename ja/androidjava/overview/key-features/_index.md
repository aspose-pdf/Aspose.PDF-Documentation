## 一般的な機能

- Aspose.PDF for Android via Java は、Java ランタイムを実行するすべてのモバイルデバイス（含む）をサポートします。
- API は 40 ビットまたは 128 ビットの暗号化を提供します

## サポートされている PDF バージョン

Aspose.PDF for Android via Java は PDF バージョン 1.2、1.3、1.4、1.5、1.6 および 1.7 をサポートしています。

## フォント

- 14 のコアフォント。
- PostScript Type1 フォント。
- TrueType フォント。
- Type 3 fonts。
- PDF への Font 情報の埋め込み
- CJK fonts。
- Unicode のサポート。

## テキスト

- テキストを置換する。
- テキストを抽出する（PDF全体、特定のページ、またはページ範囲から）。
- テキストを追加する。

## グラフ

- 線、円、弧、曲線など、さまざまな形状のグラフがサポートされています。
- グラフは、座標で指定された任意のページ位置に描画できます。
- グラフは回転、拡大縮小、塗りつぶしが可能で、破線やポリ破線の線スタイルを定義できます。
- グラフは座標に基づいて描画され、相対的ではなく旧式の位置を持ちます。
- CMYK および Gray カラースペースがサポートされています。

## 画像

- PDF ファイルに画像を追加します。
- 画像を削除します。
- 画像を置き換えます。
- 画像を抽出する（PDF全体、特定のページ、またはページ範囲から）。
- PDF から JPEG への変換（特定のページまたはすべてのページ）。
- PDF から TIFF への変換（単一ページまたはすべてのページ、またはピクセル化された TIFF 画像へ）。
- PDF から PNG への変換（特定のページまたはすべてのページ）。
- PDF から BMP への変換（特定のページまたはすべてのページ）。
- PDF から SVG への変換。
- PDFからEMFへの変換。
- PDFからGIFへの変換。
