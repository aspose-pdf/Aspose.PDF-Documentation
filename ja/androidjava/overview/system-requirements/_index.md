## 概要

PDF ドキュメントを生成するために、実行するマシンは [Aspose.PDF for Android via Java](https://products.aspose.com/pdf/android-java/) を持つ必要はありません。 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) Aspose.PDF for Java 自体は PDF 文書作成エンジンとしてインストールされます。しかし、Java を介して Aspose.PDF for Android が生成した PDF 文書を表示するには、マシンに最低でも [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). それは [ダウンロードできます](https://www.adobe.com/acrobat/pdf-reader.html) 無料で。

## 対応オペレーティングシステム

Aspose.PDF for Android via Java は、Java ランタイムを実行するすべてのモバイルデバイスをサポートします（含むが、これに限定されません）:

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## 開発環境

Java 経由で Aspose.PDF for Android を使用して、Android 上の Java プラットフォームを対象とする任意の開発環境でアプリケーションを開発できますが、以下の環境は明示的にサポートされています:

- Android Studio と SDK ツール
