**Aspose.PDF Java for Python**を使用してPDFドキュメントをDOCまたはDOCX形式に変換するには、**PdfToDoc**モジュールを呼び出します。

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "ドキュメントは正常に変換されました"
```
