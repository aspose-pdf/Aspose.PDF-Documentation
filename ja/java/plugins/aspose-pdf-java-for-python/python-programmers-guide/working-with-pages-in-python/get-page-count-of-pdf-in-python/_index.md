**Aspose.PDF Java for Python**を使用してPdfドキュメントのページ数を取得するには、**GetNumberOfPages**クラスを呼び出すだけです。

```Python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "ページ数:" . page_count

```
