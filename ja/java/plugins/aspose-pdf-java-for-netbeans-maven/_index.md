## はじめに

### NetBeans IDE

NetBeansは、完全にJavaで記述されたフル機能のJava IDEです。主にJavaでの開発を目的としていますが、特にPHP、C/C++、HTML5などの他の言語もサポートしています。

NetBeans IDEを使用すると、Javaデスクトップ、モバイル、およびWebアプリケーション、ならびにHTML、JavaScript、およびCSSを使用したHTML5アプリケーションを迅速かつ簡単に開発できます。これは無料でオープンソースであり、世界中に多くのユーザーと開発者のコミュニティがあります。

NetBeansチームは製品を積極的にサポートしており、広範なコミュニティからの機能提案を求めています。すべてのリリースの前には、コミュニティによるテストとフィードバックの時間が設けられています。

### Aspose.PDF for Java

[Aspose.PDF for Java](https://products.aspose.com/pdf/java/)は、JavaアプリケーションがAdobe Acrobatを使用せずにPDFドキュメントを読み取り、書き込み、操作できるようにする堅牢なPDFドキュメント作成APIです。

Aspose.PDF for Javaは驚くほど豊富な機能を提供します。これには、PDF圧縮オプション、テーブルの作成と操作、グラフサポート、画像機能、広範なハイパーリンク機能、拡張されたセキュリティコントロール、およびカスタムフォントの処理が含まれます。

### Aspose.PDF Java for NetBeans (Maven)

- Aspose.PDF Java for NetBeans (Maven)は、**Aspose.**による**NetBeans IDE**用のプラグインです。このプラグインは、Java開発のためにMavenプラットフォームを使用し、プロジェクトでAspose.PDF for Javaを使用したい開発者向けに設計されています。このプラグインを使用すると、Aspose.PDF for Java APIを使用するためのMavenプロジェクトを作成し、APIの[コード例](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)をダウンロードすることができます。
- プラグインは、**NetBeans IDE**内でAspose.PDF for Java APIを快適に使用するための以下の機能を提供します。

## ウィザード

プラグインには2つのウィザードが含まれています。

### Aspose.PDF Maven Project (ウィザード)

- この新しいプロジェクトウィザードを使用すると、開発者は新しいプロジェクト -> Maven-> Aspose.PDF Maven ProjectからAspose.PDF for Javaを使用するための**Maven**プロジェクトを作成できます。
- Aspose.PDF for Java APIのMaven依存関係の参照は、自動的に[Aspose Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf)から取得され、pom.xmlに追加されます。
- 作成されたプロジェクトには、常に最新のAspose.PDF for Java API用の**Maven**依存関係が含まれます。
- ウィザードのステップには、Aspose.PDF for Java APIを使用するためのコード例をダウンロードするオプションもあります。

### Aspose.PDF コード例 (ウィザード)

- この新しいファイルウィザードを使用すると、ダウンロードした[コード例](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)をプロジェクトにコピーして、**新しいファイル -> Java -> Aspose.PDF コード例**からAspose.PDF for Javaを使用できます。
- 使用可能なサンプルはツリーフォーマットで表示され、ユーザーはカテゴリ別に選択できます。

- 選択されたカテゴリ内のすべての例は、必要なリソースと共にプロジェクトの**com.aspose.pdf.examples**パッケージフォルダーにコピーされ、例を実行するために必要な**src/main/resources**フォルダーに含まれます。
- Aspose.PDF for Java APIのコード例は、APIの多様性を示すことを目的としています。
- ウィザードは、[Aspose.PDF for Javaの例のリポジトリ](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)から新しく利用可能な更新を探して更新します。

## ダウンロード

### NetBeansのダウンロード

**Aspose.PDF Java for NetBeans (Maven)** プラグインをダウンロードする前に、NetBeans IDEが事前にインストールされている必要があります。

**Aspose.PDF Java for NetBeans (Maven)** プラグインは、**NetBeans IDE 8.0および8.1**をサポートしています。NetBeans IDEの最新バージョンのみを使用することをお勧めします。

NetBeans IDEをダウンロードするには、

1. 次のURLに移動します:<https://netbeans.apache.org/download/index.html>

1. ダウンロードをクリックして、NetBeans IDEの適切なバンドルをダウンロードします。

Java EE / SEがサポートされているNetBeansの最新リリースバンドル、つまり**NetBeans IDE 8.1**のみをダウンロードしてください。

### NetBeans (Maven) 用の Aspose.PDF Java をダウンロード

- プラグインはNetBeansプラグインホスティングURLから直接ダウンロードできます: <http://plugins.netbeans.org/plugin/63218>
  その後、IDEのオプションを使用してインストールできます:
  - **ツール** > **プラグイン** > **ダウンロード済み**に移動
  - **プラグインを追加...**をクリックし、**Aspose.PDF Java for NetBeans (Maven)**のダウンロードした.nbm拡張子のプラグインファイルを参照します。
  - **"インストール"**をクリックしてプラグインをインストールします
- または、IDEオプションを使用して（NetBeansプラグインホスティングURLから手動でダウンロードせずに）直接インストールすることもできます:
  - **ツール** > **プラグイン** > **利用可能なプラグイン**に移動
  - 検索ボックスに「Aspose」と入力
  - **Aspose.PDF Java NetBeans (Maven)**プラグインを選択
  - **"インストール"**をクリック

### Asposeライセンスを適用する方法?

このプラグインは、Aspose.PDFの評価版を使用しています。評価に満足したら、[Asposeのウェブサイト](https://purchase.aspose.com/buy)でライセンスを購入できます。評価メッセージと機能制限を削除するには、製品ライセンスを適用する必要があります。製品を購入すると、ライセンスファイルが送られてきます。ライセンスを適用する手順は以下の通りです。

- ライセンスファイルが**Aspose.PDF.Java.lic**という名前になっていることを確認してください。
- **Aspose.PDF.Java.lic**ファイルをAspose.PDF.jarが含まれているフォルダに配置してください。
- ライセンスを有効化するには、以下のコードを使用してください。

```java
License license = new License();
license.setLicense("Aspose.PDF.Java.lic");
```

### #Aspose.PDF Java for NetBeans (Maven) のソースコードを設定する方法

以下の簡単なステップに従うことで、NetBeans IDE における **"Aspose.PDF Java for NetBeans (Maven)"** プラグインのソースコードの設定がスムーズに成功します。

1. ソースコードをダウンロードまたはクローンします。
1. **File** > Open Project を選択します。
1. ダウンロードした最新のプロジェクトソースを参照し、プロジェクトを開きます。
1. Aspose.PDF Java for NetBeans (Maven) プラグインコードの拡張準備が整いました。
