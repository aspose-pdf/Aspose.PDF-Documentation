## Aspose.PDF - ページの削除

**Aspose.PDF Java for PHP**を使用してPDFドキュメントから特定のページを削除するには、**DeletePage**クラスを呼び出します。

PHPコード

```php

# 対象のドキュメントを開く
$pdf = new Document($dataDir . 'input1.pdf');

# 特定のページを削除する
$pdf->getPages()->delete(2);

# 新しく生成されたPDFファイルを保存する
$pdf->save($dataDir . "output.pdf");

print "ページが正常に削除されました！";

```
