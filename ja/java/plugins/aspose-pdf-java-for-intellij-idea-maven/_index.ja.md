## はじめに

### IntelliJ IDEA

IntelliJ IDEAは、ソフトウェア開発者向けのJava統合開発環境（IDE）です。JetBrainsによって開発されています。元々Java用に作成されましたが、現在ではJavaScript、Groovy、HTML、CSS、RSS、R、Haskell、PHP、Ruby、Python、Scala、Clojure、Kotlinを含む多様な言語をサポートしています。

2010年のInfoworldの報告において、IntelliJはEclipse、IntelliJ IDEA、NetBeans、Oracle JDeveloperの4つの主要なJavaプログラミングツールの中で最高のテストセンタースコアを獲得しました。

### Aspose.PDF for Java

Aspose.PDF for Javaは、JavaアプリケーションがAdobe Acrobatを使用せずにPDFドキュメントを読み取り、書き込み、操作できるようにするPDFドキュメント作成コンポーネントです。

### Aspose.PDF Java for IntelliJ IDEA - Maven

- Aspose.PDF Java for IntelliJ IDEA (Maven) は、mavenプロジェクトを作成できるIntelliJ IDEAプラグインです（例：
 作成されたプロジェクトには、Aspose.PDF for Java API maven 依存関係が含まれています）と、Aspose.PDF for Java API を使用するための API のソースコード例をダウンロードします。
- このプラグインは、お気に入りの IntelliJ IDEA 内で Aspose.PDF for Java API を使用するのに役立ちます。

![todo:image_alt_text](https://i.imgur.com/KWKGljg.png)

ウィザード:
このプラグインには 2 つのウィザードが含まれています

ウィザード 1: Aspose.PDF Maven プロジェクト

- プラグインの Aspose Maven プロジェクトウィザードは、開発者が Aspose.PDF for Java API Maven プロジェクトを作成できるようにします。
- Aspose.PDF for Java API maven 依存関係の参照は、自動的に [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) から取得され、pom.xml に追加されます。
- 作成されたプロジェクトには、常に Aspose.PDF for Java API の最新の利用可能なバージョンの Maven 依存関係が含まれます。
- ウィザードのステップには、[Aspose.PDF for Java API を使用するためのソースコード例](https://github.com/aspose-pdf/Aspose.PDF-for-Java) をダウンロードするオプションも含まれています

ウィザード 2: Aspose.PDF 例
- Aspose.PDF Examples Wizardは、Aspose.PDF for Java APIのダウンロード済みのソースコード例をプロジェクトに作成/コピーすることができます。選択したカテゴリのすべての例は、必要なリソースと共にプロジェクトの "com.aspose.pdf.examples" パッケージフォルダ内にコピーされ、例を実行するのに必要な "src/main/resources" フォルダ内に配置されます。
- Aspose.PDF for Java APIのソースコード例は、APIのさまざまな使用法を示すことを目的としています。

## システム要件およびサポートされているプラットフォーム

### システム要件

- Microsoft Windows 8/7/Vista/2003/XP (64ビットを含む) / Linux GNOMEまたはKDEデスクトップ / Mac OS X 10.5以上
- 最小1 GB RAM、推奨2 GB RAM
- 300 MBのハードディスク空き容量 + 少なくとも1 Gのキャッシュ用
- 最小1024x768の画面解像度
- JDK 1.8以上を推奨
- インターネット接続 (推奨2 MB以上の速度)

### サポートされているプラットフォーム

プラグインは以下のIntelliJ IDEAバージョンをサポートしています:

- 13.0.1 (コミュニティ / アルティメットエディション)
- 13.1.2 (CE / UE)
- 14.0.2 (CE/ UE)

- 14.1.5 (CE/ UE)

## ダウンロード

### IntelliJ IDEAのダウンロード

Aspose.PDF Java for IntelliJ IDEA (Maven) プラグインをダウンロードする前に、まずIntelliJ IDEA CE / UEをインストールする必要があります。

IntelliJ IDEAをダウンロードするには

1. <https://www.jetbrains.com/idea/download/> にアクセスします。
1. 使用中のOSを選択します。
1. UEまたはCEのダウンロードをクリックします。

### IntelliJ IDEAプラグイン (Maven) のためのAspose.PDF Javaのダウンロード

- プラグインは [JetBrainsプラグインホスティングURL](https://goo.gl/z06gC0) - IntelliJ IDEAプラグインリポジトリから直接ダウンロード可能です。その後、IDEオプションを使用してインストールできます。
  - ファイル > 設定...>プラグインに移動します。
  - 「ディスクからプラグインをインストール」を選択します。
- また、JetBrainsプラグインホスティングURLから手動でダウンロードせずに、IDEオプションを使用して直接インストールすることもできます。
  - ファイル > 設定...>プラグインに移動します。
  - 「リポジトリを参照」を選択します。
  - 検索ボックスに「Aspose」と入力します。
  - 「Aspose.PDF Java for IntelliJ IDEA Maven」プラグインを選択してインストールします。
