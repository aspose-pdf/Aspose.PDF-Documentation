{{% alert color="primary" %}}

Aspose.PDF for Javaライブラリは、特定の言語-国の組み合わせ（例えば「en-KR」）を持つロケールをサポートできません。しかし、Aspose.PDFに古典的なロケールを設定する機能がAPIに存在し、com.aspose.pdf.LocaleOptions.setLocale()メソッドを呼び出すことで利用できます。
