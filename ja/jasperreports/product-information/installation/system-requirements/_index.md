{{% alert color="primary" %}}

Aspose.PDF for JasperReportsを実行するには、次のソフトウェアが必要です：

- Java 2 Platform Standard Edition 5.0以上。

- JasperReports 6.7.0以上。

- JasperServer 6.7.0以上（Aspose.PDF for JasperReportsをJasperServerで使用する場合）。

{{% /alert %}}
