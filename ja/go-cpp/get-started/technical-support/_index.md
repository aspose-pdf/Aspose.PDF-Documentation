Aspose はすべての製品に対して無制限の無料テクニカルサポートを提供しています。サポートは評価版を含むすべてのユーザーが利用可能です。主なサポート窓口は Aspose.Forums です。

{{% alert color="primary" %}}
Aspose は電話での技術サポートを提供していないことにご注意ください。電話サポートは販売および購入に関する質問にのみ利用可能です。
{{% /alert %}}

## Aspose 無料サポートフォーラム

Aspose.PDF はすべての製品に対して無料の技術サポートを利用できるようにします。Aspose.PDF に関する質問がある場合は、次の記事をご確認ください：

- 最も一般的なサポートの情報源は [Aspose.Forums](https://forum.aspose.com/)。In the [Aspose.PDF フォーラム](https://forum.aspose.com/c/pdf/10) セクションでは、質問を投稿すればできるだけ早く回答が得られます。
- Aspose が電話でのテクニカルサポートを提供していないことをご存知おきください。電話でのお問い合わせは販売および購入に関する質問のみ利用可能です。
- 時差があることをご留意ください。これにより、回答までの速度に影響する可能性があります。

## 問題または機能リクエストを報告する

Aspose.PDF に関する問題で支援が必要な場合は、次の推奨事項に従って、最も効率的な方法で解決できるようにしてください：

- まず、問題を報告する前に最新の Aspose.PDF バージョンを使用しているか確認してください。参照 [Aspose.PDF for Go ダウンロード](https://github.com/aspose-pdf/aspose-pdf-go-cpp) 最新バージョンについて調べるには。
- おそらく、問題の解決策はすでに議論されており、解決策が存在します。そのため、フォーラムとドキュメントを調べて確認してください [APIリファレンス](https://reference.aspose.com/pdf/go-cpp/) 問題を報告する前に。
- 質問を作成する際には、元のドキュメントと、可能であれば問題を引き起こすコードの断片を含めてください。複数ある場合は、これらのファイルをひとつのドキュメントに zip することができます。
- 心配しないでください。アクセスできるのは、あなたと Aspose の開発者だけです。スティッキーファイルはそれらだけが利用できます。
