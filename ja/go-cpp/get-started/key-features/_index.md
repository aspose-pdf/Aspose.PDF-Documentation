さまざまなプラットフォームと Go バージョンをサポートしています。参照 [システム要件](/pdf/ja/go-cpp/system-requirements/) 詳細については。

## 一般機能

幅広いリストからの読み込みおよび保存をサポートしています [ファイル形式](/pdf/ja/go-cpp/supported-file-formats/).

## サポートされている PDF 標準

Aspose.PDF for Go は PDF バージョン 1.2、1.3、1.4、1.5、1.6 および 1.7 をサポートしています。
