## 一般的な機能

- 主要な PDF 標準および仕様のほとんどをサポートします。
- BMP、GIF、JPEG、PNG などの複数の画像形式で PDF を読込およびエクスポートする機能。
- PDF ドキュメントの基本情報（例: 作成者、クリエイター）を設定します。
- PDF ページのプロパティ（幅、高さ、CropBox、BleedBox など）を設定します。
- ページ番号付け、ブックマークレベル、ページサイズなどを設定します。
- テキスト、段落、見出し、ハイパーリンク、グラフ、添付ファイルなどを操作できます。

## サポートされている PDF 標準

- ISO 32000-1 (PDF 1.2, PDF 1.3, PDF 1.4, PDF 1.5, PDF 1.6, PDF 1.7).
- ISO 32000-2 (PDF 2.0).
- ISO 19005 (PDF/A), ISO 15930 (PDF/X), ISO 24517 (PDF/E), ISO 14289 (PDF/UA).

## 変換機能

Aspose.PDF for Python via .NET ライブラリを使用すると、PDF ドキュメントを最も一般的な形式に、またはその逆に、迅速かつ簡単に変換できます。

- PDF を Word、Excel、PowerPoint に変換します。
- PDF を画像形式に変換します。
- PDF ファイルを HTML 形式に、またはその逆に変換します。
- PDF を EPUB、テキスト、XPS などに変換します。
- EPUB、Markdown、テキスト、XPS、PostScript、XML、LaTeX を PDF に変換します。

## サポートされている PDF バージョン

Aspose.PDF for Python は PDF バージョン 1.2、1.3、1.4、1.5、1.6、1.7、2.0 をサポートしています。
