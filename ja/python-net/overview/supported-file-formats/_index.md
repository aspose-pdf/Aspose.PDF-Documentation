Abstract: 本稿では、Aspose.PDF for Python が対応できるファイル形式を詳細に示した包括的な表を提供します。ライブラリがさまざまな文書タイプをロードおよび保存できる機能を概説し、異なるファイル形式の管理における汎用性について洞察を提供します。対象となる形式には、PDF、EPUB、HTML、XML などの広く知られたタイプが含まれ、これらはすべてロードと保存の両方が可能です。また、JPEG、PNG、TIFF といった画像形式についても、ロードと保存の両方がサポートされています。ただし、CGM、MHT、PCL、PS、XSLFO、MD などの一部の形式はロードのみ可能であり、XLS、XLSX、PPTX、DOC、DOCX、MobiXML などは保存のみが可能です。この表は、Aspose.PDF for Python の形式互換性を理解しようとするユーザーにとって、文書および画像形式のサポートにおける強みと制限を示す実用的なガイドとなります。
