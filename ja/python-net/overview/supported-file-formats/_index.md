Abstract: この記事では、Aspose.PDF for Python が処理できるファイル形式を詳しく説明した包括的な表を提供します。さまざまな文書タイプを読み込んだり保存したりするためのライブラリの機能の概要を説明し、さまざまなファイル形式を管理するための汎用性についての洞察を提供しています。対象となる形式には、PDF、EPUB、HTML、XML などのよく知られたタイプが含まれ、これらはすべてライブラリでロードおよび保存できます。また、JPEG、PNG、TIFF などの画像形式もどちらの操作でもサポートしています。ただし、CGM、MHT、PCL、PS、XSLFO、MD などの一部のフォーマットはロードのみが可能ですが、XLS、XLSX、PPTX、DOC、DOCX、MobiXML などの他のフォーマットは保存のみ可能です。この表は、Aspose.PDF for Python のフォーマット互換性を理解したいユーザーのための実践的なガイドとなり、ドキュメントと画像フォーマットのサポートにおける長所と制限の両方に焦点を当てています。
