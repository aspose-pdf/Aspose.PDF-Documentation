_このページは Aspose.PDF for Python の機能の概要です。_

**Aspose.PDF for Python via .NET**、Microsoft Office® や Adobe Acrobat Automation を必要とせずに開発者が PDF ドキュメントを操作できる PDF 処理 APIです。ライブラリの機能と可能性の詳細については、[Aspose.PDF for Python](https://products.aspose.com/pdf/python-net/) のランディングページをご覧ください。

Aspose.PDF for Python は、次のような多種多様な機能をサポートします。

- ほとんどの確立された PDF 標準と PDF 仕様をサポートします。
- BMP、GIF、JPEG、PNG を含む複数の画像形式で PDF を読み取り・エクスポートする機能。
- PDF ドキュメントの基本情報（例：作者、作成者）の設定。
- PDF ページプロパティ（例：幅、高さ、CropBox、BleedBox など）の構成。
- ページ番号付け、ブックマークレベル、ページサイズなどの設定。
- テキスト、段落、見出し、ハイパーリンク、グラフ、添付ファイルなどを操作する機能。

さらに、Aspose.PDF for Python via .NET を使用すると、EPUB、Markdown、テキスト、XPS、PostScript、XML、LaTex を PDF に簡単に変換したり、PDF をさまざまな文書形式に変換したりすることができ、優れたパフォーマンスと高品質を実現します。

最も人気のある Aspose.PDF 機能のいくつかを示す [Free Online Apps](https://products.aspose.app/pdf/applications) をお試しください。
