{{% alert color="primary" %}}

**Aspose.PDF for Python via .NET** は、開発者がアプリケーションに PDF 処理機能を追加できるライブラリです。API を使用して 32 ビットおよび 64 ビットのアプリケーションを構築し、Adobe Acrobat を使用せずに PDF ファイルの生成、読み取り、変換、操作が可能です。

{{% /alert %}}

## 製品説明

**Aspose.PDF for Python** は、フォーム処理、メタデータ情報の取得・設定、テキストやページの操作、注釈の管理、ブックマークや透かしの追加・削除、添付ファイル、カスタムフォントの取り扱いなど、さまざまな文書処理タスクを実行できます。

**Aspose.PDF for Python** は、Python 3.5 以降がインストールされている Windows や Linux などのさまざまなオペレーティングシステム向けに、32 ビットおよび 64 ビットの Python アプリケーションを開発するために使用できます。

## インストール

## .NET 経由の Aspose.PDF for Python の評価

簡単に Aspose.PDF for Python を評価用にダウンロードできます。評価用ダウンロードは購入時のダウンロードと同一です。評価版は、ライセンスを適用するコードを数行追加するだけでライセンス版に切り替わります。

評価用に [Aspose.PDF for Python](https://releases.aspose.com/pdf/pythonnet/) をダウンロードできます。評価ダウンロードは購入ダウンロードと同じです。評価版は、数行のコードを追加して [ライセンスを適用](/pdf/python-net/licensing/) するとライセンスが有効になります。

ライセンスが指定されていない Aspose.PDF の評価版は、製品の全機能を提供しますが、2 つの制限があります。評価用の透かしが挿入され、任意のコレクションの要素は最大 4 つまでしか表示/編集できません。

{{% alert color="primary" %}}

評価版の制限なしで Aspose.PDF for Python をテストしたい場合は、30 日間の一時ライセンスをリクエストすることもできます。詳細は [一時ライセンスの取得方法](https://purchase.aspose.com/temporary-license) をご参照ください。

{{% /alert %}}

## .NET 経由の Aspose.PDF for Python のインストール

直接ダウンロード用のリンクから [pip](https://pypi.org/project/aspose-pdf/) を使用して、aspose-pdf for Python を簡単に利用できます。
パッケージを取得するには 'pip install aspose-pdf' を実行してください。すでに Aspose.PDF for Python をお持ちで最新バージョンを取得したい場合は、'pip install --upgrade aspose-pdf' を実行してください。
