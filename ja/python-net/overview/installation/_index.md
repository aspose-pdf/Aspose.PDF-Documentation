**.NET 経由の Python 用 Aspose.pdf ** は、開発者がアプリケーションに PDF 処理機能を追加できるようにするライブラリです。API を使用すると、Adobe Acrobat を使用せずに PDF ファイルを生成、読み取り、変換、操作できる 32 ビットおよび 64 ビットのアプリケーションを構築できます。

{{% /alert %}}

## [商品説明]

**Aspose.pdf for Python**を使用すると、フォーム処理、メタデータ情報の取得と設定、テキストとページの操作、注釈の管理、ブックマークとウォーターマークの追加または削除、添付ファイル、カスタムフォントの処理など、さまざまなドキュメント処理タスクを実行できます。

**Aspose.pdf for Python**は、Python 3.5以降がインストールされているさまざまなオペレーティングシステム（WindowsやLinuxなど）用の32ビットおよび64ビットPythonアプリケーションを開発するために使用できます。

## インストール

## .NET 経由で Python 用 Aspose.PDF を評価してください

ダウンロードできます [Aspose.PDF for Python](https://releases.aspose.com/pdf/pythonnet/) 評価用。評価版のダウンロードは、購入済みのダウンロードと同じです。評価版は、に数行のコードを追加するとライセンスが付与されます。 [ライセンスを申請する](/pdf/ja/python-net/licensing/).

Aspose.PDF の評価版 (ライセンス指定なし) では、製品の全機能が提供されますが、2 つの制限があります。評価ウォーターマークが挿入されること、コレクションの 4 つの要素しか表示/編集できないことです。

{{% alert color="primary" %}}

評価版の制限なしに Aspose.PDF for Python をテストしたい場合は、30 日間の一時ライセンスをリクエストすることもできます。「」を参照してください。 [仮免許の取得方法は？](https://purchase.aspose.com/temporary-license)

{{% /alert %}}

## .NET 経由で Python 用 Aspose.PDF をインストールする

Python用のaspose-pdfは、直接ダウンロードするためのリンクから簡単に使用できます [ピップ](https://pypi.org/project/aspose-pdf/).
'pip install aspose-pdf' を実行してパッケージを取得します。Python 用の Aspose.PDF が既にあり、最新バージョンを入手したい場合は、「pip install--upgrade aspose-pdf」を実行してください。
