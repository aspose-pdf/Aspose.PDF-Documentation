このセクションでは、.NET ライブラリ経由の Aspose.PDF for Python を使用して PDF ドキュメントをすばやく簡単に作成する方法について説明します。また、Aspose.PDF for Python を使用して、既存の PDF 内で使用可能なフィールドを AcroForms で検出してマッピングする方法についても説明します。

AcroFormsを使用すると、ユーザーはフォームに電子的に入力、送信、保存できるため、情報を収集するための便利で効率的な方法になります。

- [アクロフォーム](/pdf/ja/python-net/acroforms/) -Pythonライブラリを使用して、フォームの作成、フォームフィールドへの入力、フォームからのデータの抽出、PDF内のフィールドの追加と削除を行います。
- [XFA フォーム](/pdf/ja/python-net/xfa-forms/) -XFA を変換します。無視を使用するにはレンダリングが必要です。
