私たちは皆、PDF ファイルにテキストを追加する必要があることがあります。例えば、本文の下に翻訳を追加したり、画像の横にキャプションを配置したり、単に申込書に記入したりする場合です。すべてのテキスト要素を自分の希望するスタイルで書式設定できると便利です。PDF ファイルで最も一般的なテキスト操作は、PDF へのテキスト追加、PDF 内のテキスト書式設定、ドキュメント内のテキスト置換と回転です。**Aspose.PDF for Python via .NET** は、PDF コンテンツとやり取りするために必要なすべてを備えた最適なソリューションです。

以下のことが可能です。

- [PDF ファイルにテキストを追加](/pdf/python-net/add-text-to-pdf-file/) - PDF にテキストを追加し、ストリームやファイルからフォントを使用し、HTML 文字列やハイパーリンクなどを追加できます。
- [PDF ツールチップ](/pdf/python-net/pdf-tooltip/) - Python を使用して不可視ボタンを追加することで、検索されたテキストにツールチップを追加できます。
- [PDF 内のテキスト書式設定](/pdf/python-net/text-formatting-inside-pdf/) - テキストを書式設定する際にドキュメントに追加できる機能が多数あります。行インデント、テキスト枠線、下線テキスト、改行の追加などを Aspose.PDF ライブラリで実現できます。
- [PDF のテキスト置換](/pdf/python-net/replace-text-in-pdf/) - PDF ドキュメントのすべてのページでテキストを置換します。まず TextFragmentAbsorber を使用する必要があります。
- [PDF 内のテキスト回転](/pdf/python-net/rotate-text-inside-pdf/) - TextFragment クラスの回転プロパティを使用して、PDF 内のテキストを回転させます。
- [PDF ドキュメントのページからテキストを検索・取得](/pdf/python-net/search-and-get-text-from-pdf/) - テキストの検索と取得には TextFragmentAbsorber クラスを使用できます。
- [FloatingBox の使用](/pdf/python-net/floating-box/) - このページでは、フローティングボックス内のテキストを書式設定する方法を説明しています。
