Python でページレベルの PDF 操作を実行する必要がある場合は、このセクションを使用してください。

**Aspose.pdf for Python via .NET**を使用すると、ページの挿入、削除、移動、抽出、回転、サイズ変更、切り抜き、ヘッダー、フッター、ページ番号、透かし、スタンプを適用してレイアウトやレビューのワークフローを行うことができます。

## 対象ページタスク

次のタスクが対象です。

- [ページを追加する](/pdf/ja/python-net/add-pages/) -特定の位置にページを挿入したり、文書の末尾にページを追加したりできます。
- [ページの移動](/pdf/ja/python-net/move-pages/) -文書内または文書間でページを移動します。
- [ページを削除する](/pdf/ja/python-net/delete-pages/) -PDF ドキュメントからページを削除します。
- [ページの抽出](/pdf/ja/python-net/extract-pages/) -選択したページを新しいPDFファイルに抽出します。
- [ページの回転](/pdf/ja/python-net/rotate-pages/) -既存の PDF ファイルのページの向きを変更します。
- [ページサイズの変更](/pdf/ja/python-net/change-page-size/) -PDF ドキュメントのページサイズを変更します。
- [ヘッダーとフッターの追加](/pdf/ja/python-net/add-headers-and-footers-of-pdf-file/) -繰り返しのテキスト、画像、または構造化されたヘッダー/フッターコンテンツを追加します。
- [PDF ページのトリミング](/pdf/ja/python-net/crop-pages/) -ページジオメトリ設定を使用してページコンテンツをトリミングします。
- [ページプロパティの取得と設定](/pdf/ja/python-net/get-and-set-page-properties/) -サイズ、回転、色属性など、PDFページのさまざまなプロパティにアクセスして変更します。
- [スタンピング](/pdf/ja/python-net/stamping/) -テキスト、画像、ページ、ページ番号のスタンプを PDF ページに適用します。
