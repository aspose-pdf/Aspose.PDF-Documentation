このセクションでは、Aspose.PDF for Python via .NET を使用して PDF の添付ファイルを操作する方法について説明します。
添付ファイルとは、親ドキュメントに付加される追加のファイルで、PDF、Word、画像、その他のファイルなど、さまざまな種類があります。
PDF に添付ファイルを追加し、添付ファイルの情報を取得してファイルに保存し、Python を使用してプログラム的に PDF から添付ファイルを削除する方法を学びます。
