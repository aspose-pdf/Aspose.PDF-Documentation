- [アクション](/pdf/python-net/actions/) - 本記事では、さまざまなタイプのアクションの扱い方を学びます。
- [ブックマーク](/pdf/python-net/bookmarks/)- 大規模な出版物には、ブックマークペインで簡単に表示・選択できるブックマークのフレームワークが含まれることが一般的です。ブックマークをクリックすると、対応するページや章へジャンプできます。ブックマークペインはコンテンツに応じた要素で、開いている PDF 文書にブックマーク構造がある場合にのみサイドバーに表示されます。
- [リンク](/pdf/python-net/links/)- Python ライブラリを使用して PDF 文書内のリンクを操作します。
