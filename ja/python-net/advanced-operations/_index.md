**高度な操作** は、既存の PDF ファイルをプログラムで扱う方法に関するセクションです。Aspose.PDF で作成されたドキュメント（[基本操作](/pdf/python-net/basic-operations/) で説明）や、Adobe Acrobat、Google Docs、Microsoft Office、Open Office、またはその他の PDF 作成ツールで作成された PDF も対象です。

さまざまな方法を学びます：

- [ドキュメントの操作](/pdf/python-net/working-with-documents/) - 圧縮、分割、結合を行い、文書全体に対するその他の操作を実行します。
- [ページの操作](/pdf/python-net/working-with-pages/) - ページの追加、移動、削除、トリミング、透かしやスタンプの追加などを行います。
- [画像の操作](/pdf/python-net/working-with-images/) - PDF 文書に画像を追加、抽出、削除します。
- [添付ファイル](/pdf/python-net/attachments/) - Python で PDF に添付ファイルをプログラム的に追加および削除する方法を学びます。
- [ナビゲーションとインタラクション](/pdf/python-net/navigation-and-interaction/) - アクション、ブックマークの処理やページのナビゲーションを行います。
- [注釈](/pdf/python-net/annotations/) - 注釈によりユーザーは PDF ページにカスタムコンテンツを追加できます。PDF 文書から注釈を追加、削除、変更できます。
- [テーブルの操作](/pdf/python-net/working-with-tables/) - PDF にテーブルを挿入、装飾し、表データを抽出します。
- [フォームの操作](/pdf/python-net/working-with-forms/) - インタラクティブな PDF 文書を扱い、フォームフィールドを追加し、データを抽出します。
- [テキストの操作](/pdf/python-net/working-with-text/) - PDF のテキストを追加、書式設定、検索、置換します。
- [PDF 文書の比較](/pdf/python-net/compare-pdf-documents/) - PDF 文書の内容を比較できます。
- [PDF のメタデータ](/pdf/python-net/pdf-file-metadata/) - 文書のメタデータを取得または設定し、XMP データを扱います。
- [グラフの操作](/pdf/python-net/working-with-graphs/) - ページ上の図形を操作します。
- [オペレータの操作](/pdf/python-net/working-with-operators/) - PDF の低レベル操作を行います。
- [アーティファクト](/pdf/python-net/artifacts/) - PDF の透かしやその他の特殊オブジェクトを扱います。
- [PDF レイヤーの操作](/python-net/work-with-pdf-layers/) - レイヤーをロックし、要素を抽出、フラット化、PDF レイヤーをマージします。
