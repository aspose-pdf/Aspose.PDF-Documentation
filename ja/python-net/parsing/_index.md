**Parse PDF** ドキュメントは、PDF ファイルからさまざまな情報を抽出することに関連する用語です。このセクションでは、以下の内容をカバーします:

- [PDFからテキストを抽出](/pdf/python-net/extract-text-from-pdf/). テキストのパースまたは抽出は、既成の PDF で最も一般的な操作です。全文書、特定のページ、またはページ内の特定領域からのテキスト抽出について学べます。
- [PDFから画像を抽出](/pdf/python-net/extract-images-from-the-pdf-file/). 画像抽出は、上記のテキスト抽出と同様に画像に対して行われます。
- [PDFからフォントを抽出](/pdf/python-net/extract-fonts-from-pdf/). フォント抽出は、PDF 内のフォントに特化した操作です。
- [フォームからデータを抽出](/pdf/python-net/extract-data-from-acroform/). フォーム付きの PDF ドキュメントが多数ある場合、これらのフォームからデータを取得する必要があるでしょう。本記事では、Aspose.PDF for Python via .NET を使用して AcroForms データを抽出する方法を解説します。
- [テーブルからデータを抽出](/pdf/python-net/extract-data-from-table-in-pdf/). PDF ドキュメント内のテーブルからデータを取得します。
- [PDFからベクトルデータを抽出](/pdf/python-net/extract-vector-data-from-pdf/). パス、ポリゴン、ポリラインなどのベクトルデータ（位置、色、線幅など）を取得できます。
