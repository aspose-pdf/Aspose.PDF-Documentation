**Aspose.Pdf for Reporting Services** は、Microsoft SQL Server 2016/2017/2019/2022 Reporting Services および Power BI Report Server 用のカスタムレンダリング拡張機能です。Aspose.Pdf for Reporting Services は、次のいずれかが実行されているコンピュータにインストールできる単一の MSI インストーラとして提供されます：

- Microsoft SQL Server 2016 Reporting Services  
- Microsoft SQL Server 2017 Reporting Services  
- Microsoft SQL Server 2019 Reporting Services  
- Microsoft SQL Server 2022 Reporting Services  
- Microsoft Power BI Report Server
 
Aspose.Pdf for Reporting Services は、.NET アセンブリ Aspose.Pdf.ReportingServices.dll が 1 つだけで構成されており、完全に C# で記述され、CLS に準拠し、安全なマネージドコードのみを含むため、展開および管理が簡単です。サーバーには .NET Framework 4.8.1 がインストールされている必要があります。

Aspose.Pdf.ReportingServices.dll は ReportServer\\bin ディレクトリにコピーし、設定ファイルを更新して Reporting Services が新しいレンダリング拡張機能を認識できるようにする必要があります。これらの手順は Aspose.Pdf for Reporting Services インストーラーによって実行されますが、本文書でさらに説明するように手動で実行することも可能です。
