## What is a PDF File?

Adobe created PDF in the 1990s to make documents viewable on any hardware or operating system, without requiring the application used to create them. Additionally, PDF was designed to ensure that the layout of the document remains the same regardless of where it is opened. However, simply opening a PDF may not suffice, as one may need to create, edit, or convert a PDF document to a different format, which requires additional tools.

## How to start working

To begin working with Java, it is important to have the **JDK (Java Development Kit)** installed on your machine. Ensure that the JDK is installed and properly set up on your system by setting the JAVA_HOME environment variable to its installation directory. This step is crucial for the successful execution of Java programs.

Installing the JDK typically involves downloading the JDK distribution package from the official Oracle website and following the installation instructions provided. Once installed, you can verify the installation by opening a command prompt and running the java -version command, which should display the installed Java version.

After installing and setting up the JDK, you can proceed with developing Java applications. This includes writing Java code using a text editor or an integrated development environment (IDE) of your choice. You can then compile your Java source code using the *javac* command, followed by executing the compiled bytecode using the java command.

By ensuring that you have the JDK installed and properly configured on your machine, you can confidently start working with Java and harness its powerful capabilities for developing robust and scalable applications.
