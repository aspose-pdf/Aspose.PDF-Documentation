**Aspose.PDF for PHP via Java** lets you insert a page to a PDF document at any location in the file, add pages to the end of a PDF file, and delete pages from your document. This section shows how to add pages to a PDF without Acrobat Reader.
You can add text or images in the headers and footers of your PDF file, and choose different headers in your document with PHP library by Aspose. 
You can change the page orientation of pages in an existing PDF file with PHP library. Aspose.PDF allows you to change and get  PDF page size with code snippet.

Also, try to crop pages in PDF document programmatically using PHP.

This section learn you how to add watermarks in your PDF file using Artifact class. You will check the programming sample for this task. 
Add Page number using PageNumberStamp class. For adding a Stamp in your document use ImageStamp and TextStamp classes. Use adding a watermark for creating background images in your PDF file with **Aspose.PDF for PHP via Java**.

You are able to do the following:

- [Add Pages](/pdf/php-java/add-pages/) - add pages at desired location or to the end of a PDF file and delete a page from you document.
- [Move Pages](/pdf/php-java/move-pages/) - move pages from one document to another.
- [Get and Set Page Properties](/pdf/php-java/get-and-set-page-properties/) - you can learn how to get numbers in PDF file, get page properties and determine page color.
- [Delete Pages](/pdf/php-java/delete-pages/) - delete page from your PDF file using PageCollection collection.
- [Change Page size](/pdf/php-java/change-page-size) - you can change PDF page size with code snippet using Aspose.PDF library.
- [Rotate Pages](/pdf/php-java/rotate-pages/) - you can change the page orientation of pages in an existing PDF file.
- [Split Pages](/pdf/php-java/split-document/) - you can split PDF files into one or multiple PDF.
- [Add Headers and/or Footers](/pdf/php-java/add-headers-and-footers-of-pdf-file/) - add text or images in the headers and footers of your PDF file .
- [Crop Pages](/pdf/php-java/crop-pages/) - you can crop pages in PDF document programmatically with different Page Properties.
- [Add Watermarks](/pdf/php-java/add-watermarks/) - add watermarks in your PDF file with Artifact Class.
- [Add Page Numbering in PDF File](/pdf/php-java/add-page-number/) - PageNumberStamp class will help you to add a Page Number in your PDF file.
- [Add Backgrounds](/pdf/php-java/add-backgrounds/) - background images can be used to add a watermark.
- [Stamping](/pdf/php-java/stamping/) - you can use the ImageStamp class to add an image stamp to a PDF file and TextStamp class for adding a text.
