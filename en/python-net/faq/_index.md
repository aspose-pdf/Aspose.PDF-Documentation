As a result, it may be impossible to convert a document using the ConvertErrorAction.None mode. Another scenario for using this option is when your document is already mostly or fully PDF/A compliant, and you want to avoid unnecessary changes. More permissive PDF/A versions (e.g., PDF/A-2 or PDF/A-3) are more likely to be successfully converted with the ConvertErrorAction.None option.
