The current Java `FormEditorExamples` class provides direct appearance-related coverage for `decorateField`, `setFieldAlignment`, `setFieldAlignmentVertical`, `setFieldAppearance`, `setFieldLimit`, and `setFieldCombNumber`.

In this subsection:

- `decorateField` shows how to apply background, text, and border colors through `FormFieldFacade`
- `setFieldAlignment` and `setFieldAlignmentVertical` show how to change alignment directly with `FormEditor`
- `setFieldAppearance` shows how to apply annotation flags to a field
- `setFieldLimit` and `setFieldCombNumber` show how to constrain text-entry behavior
