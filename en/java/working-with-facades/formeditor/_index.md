The Java `FormEditorExamples` class focuses on structure and appearance changes for existing PDF forms.

The example set covers:

- Creating text box, check box, combo box, list box, radio button, and submit button fields
- Adding or deleting list items in existing form controls
- Moving, removing, renaming, and copying fields
- Converting a single-line field to a multiline field
- Decorating a field and setting horizontal or vertical alignment
- Setting field appearance flags, text limits, and comb numbers
- Assigning JavaScript and submit actions to form buttons
