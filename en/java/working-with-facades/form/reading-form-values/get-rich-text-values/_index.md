The Java facade examples currently document general field value inspection only:

- list field names with `getFieldNames()`
- read values with `getField(fieldName)`

There is no separate rich-text extraction example in `FormExamples.java`. If a rich-text-specific Java sample is added later, this page should be rewritten from that source.
