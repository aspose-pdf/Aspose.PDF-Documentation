The current Java sample set contains one direct inspection workflow: `FormExamples.inspectFormFields(...)`.

That example shows how to:

- list all field names with `getFieldNames()`
- read each field value with `getField(fieldName)`

Several child pages in this section remain as scope notes because the repository does not currently provide separate Java examples for those specialized reading tasks.
