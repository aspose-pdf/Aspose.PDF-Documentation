The current `FormExamples.java` source file focuses on field value inspection through `getFieldNames()` and `getField(...)`.

`FormFieldFacade` does appear elsewhere in the Java facade examples, specifically in `FormEditorExamples.decorateField(...)`, but that example is about field styling rather than reading form values. Until a dedicated Java sample for field facade inspection is added, this page should remain a scope note.
