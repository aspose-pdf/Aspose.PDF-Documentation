The current `FormExamples.java` source file includes `flattenAllFields(...)`, but it does not include a separate example for flattening only a selected subset of fields.

If you need selective flattening in Java, do not assume it is covered by the current repository samples. Use the supported `flattenAllFields(...)` example as the documented baseline, or add a new dedicated Java sample before documenting a narrower workflow here.
