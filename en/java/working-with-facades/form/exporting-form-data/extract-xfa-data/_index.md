The current `FormExamples.java` source file does not provide an XFA extraction method.

This section is limited to the Java workflows that are actually present in the repository:

- export form data to XML, FDF, and XFDF
- import form data from XML, FDF, and XFDF
- fill, flatten, rename, and inspect AcroForm fields

If a dedicated Java XFA extraction example is added to the sample set later, this page should be updated to follow that source directly.
