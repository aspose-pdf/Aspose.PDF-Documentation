The Java `PdfFileSecurityExamples` class demonstrates the main security workflows available through the Facades API.

Use this section to learn how to:

- encrypt a PDF with user and owner passwords
- apply or update document privileges
- decrypt a PDF with an owner password
- change passwords and optionally reset security settings
