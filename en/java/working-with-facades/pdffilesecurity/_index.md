The Java `PdfFileSecurityExamples` class demonstrates the main file-security operations available through the Facades API.

The current examples show how to:

- Encrypt a PDF with user and owner passwords
- Apply or restrict permissions such as printing and copying
- Use AES encryption with a 256-bit key size
- Decrypt a PDF with the owner password
- Change passwords and optionally reset security settings
- Set document privileges with or without passwords
