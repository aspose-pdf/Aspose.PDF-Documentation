The Java `PdfFileInfoExamples` class is focused on inspection and metadata maintenance tasks.

It includes examples for:

- Reading the PDF version
- Inspecting document privileges and allowed actions
- Retrieving page width, height, rotation, and offsets
- Reading standard and custom metadata values
- Updating or clearing document metadata
- Saving new metadata with XMP
