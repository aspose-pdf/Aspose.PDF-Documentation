The Java `PdfFileStampExamples` class demonstrates repeated-content and page-marking workflows built on `PdfFileStamp`.

The examples in this section include:

- Adding an image stamp to a PDF
- Adding text or image headers
- Adding text or image footers
- Applying custom margins to headers and footers
- Inserting page numbers with default or custom placement
- Using Roman numeral page numbering with a custom starting value
