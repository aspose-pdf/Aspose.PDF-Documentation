The current Java example for this article is `createNupPdfDocument`.

It demonstrates how to call `makeNUp(...)` with a 2-by-2 layout and save the generated output PDF.

The same source file also includes `tryCreateNupPdfDocument`, which uses the boolean return value to handle failure without throwing a custom exception.
