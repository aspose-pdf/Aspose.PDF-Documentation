The advanced operations section groups the Java workflows used for editing, inspecting, securing, and enriching existing PDF documents beyond the basic create, open, save, merge, and split operations.

Key topics in this section include:

- annotations, links, actions, and bookmarks
- pages, tables, text, images, graphs, and vector graphics
- forms, layers, tagged PDF, and document metadata
- attachments, artifacts, security, signatures, and ZUGFeRD
