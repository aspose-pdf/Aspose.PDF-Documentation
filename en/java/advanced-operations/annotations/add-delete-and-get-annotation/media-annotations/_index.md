Aspose.PDF supports a broad annotation model, but the current Java examples in this repository focus on text, markup, link, shape, watermark, redaction, and annotation-transfer workflows.

At the moment, there is no dedicated Java example class here for:

- sound annotations
- screen annotations
- rich media annotations
- 3D annotations

This page is kept to preserve the section structure. For source-backed Java examples in this repo, use the sibling annotation topics that map directly to `workingwithannotations` example classes.
