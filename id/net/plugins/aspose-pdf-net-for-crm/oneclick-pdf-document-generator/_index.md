## Pendahuluan

Aspose .NET OneClick PDF Document Generator adalah add-on open source yang digunakan untuk menghasilkan dan mengekspor PDF dari CRM. Anda dapat merancang template di Word dan melampirkannya di CRM serta menghasilkan dokumen sesuai keinginan Anda. Anda dapat menggunakan OneClick PDF Document Generator dengan Entitas Akun, Kontak, Lead, Kesempatan, dan Penawaran. Ini menyediakan opsi untuk melampirkan dokumen yang dihasilkan dengan catatan itu sendiri dan Anda juga dapat mengunduh dokumen tersebut. Add-on ini juga dapat menyelesaikan opsi set dan bidang pencarian. Selain itu, ini memungkinkan Anda untuk menghasilkan PDF dari tombol pita atas serta melalui tombol satu klik di formulir.

Aspose .NET OneClick PDF Document Generator dapat digunakan dengan CRM on-premises dan CRM Online.

Fitur utama dari Addon ini adalah:

- Buat Template di dalam Word dan lampirkan di CRM. (gunakan fitur merge field di Word di mana Anda ingin menyertakan data CRM)
- Memungkinkan Anda untuk menambahkan Optionset serta pencarian.
- Bekerja dengan Akun, Kontak, Lead, Kesempatan, dan Penawaran.
- Tombol satu klik yang dapat Menghasilkan dokumen.
- Dokumen yang dihasilkan dapat diunduh serta dilampirkan dengan catatan entitas yang sama.
- Tombol pita juga disediakan untuk memilih template dan menghasilkan dokumen.

Add-on ini akan bekerja dengan semua versi **Microsoft Dynamics CRM 2015 dan CRM Online**.

## Menginstal atau Menghapus Instalasi

### Cara Menginstal

#### Mengimpor Solusi

- Unduh File Solusi. (Dikelola jika Anda berniat untuk menghapusnya).
- Buka CRM dan pergi ke Impor Solusi.
- Klik Telusuri dan pilih file solusi yang diunduh dan klik berikutnya.
- Klik Berikutnya dan tunggu solusi untuk diimpor.
- Klik tutup ketika solusi berhasil diimpor.

#### Mengonfigurasi Halaman Web

- Unduh kode sumber untuk situs web.
- Salin ke server. (Disarankan di bawah inetpub).
- Buka IIS. (Di server tempat Anda berniat untuk menerapkan situs web, dapat diakses dari server CRM).
- Buat Situs Web baru.
- Pilih jalur fisik dan konfigurasikan situs web.
- Telusuri untuk memeriksa apakah dapat diakses.
- Buka CRM dan tambahkan catatan baru di bawah Aspose>> Entitas Konfigurasi Aspose dan tentukan Nama dan URL Halaman Web yang diterapkan seperti di bawah ini.

![OneClick PDF Document Generator](oneclick-pdf-document-generator_1.png)

### Cara Menghapus Instalasi

- Hapus Situs Web dari IIS.
- Pergi ke solusi dan pilih "OneClick PDF Document Generator".
- Tekan tombol Hapus. Tekan OK untuk konfirmasi.
