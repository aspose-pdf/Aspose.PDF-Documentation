## Pendahuluan

Aspose .NET OneClick PDF Document Generator adalah add-on sumber terbuka yang digunakan untuk menghasilkan dan mengekspor PDF dari CRM. Anda dapat mendesain template dalam word dan melampirkannya dalam CRM serta menghasilkan dokumen sesuai keinginan Anda. Anda dapat menggunakan OneClick PDF Document Generator dengan Entitas Account, Contact, Lead, Opportunity, dan Quote. Ini menyediakan opsi untuk melampirkan dokumen yang dihasilkan dengan catatan itu sendiri dan Anda juga dapat mengunduh dokumennya. Add-on ini juga dapat menyelesaikan set opsi dan bidang pencarian. Selain itu, memungkinkan Anda untuk menghasilkan PDF dari tombol pita Atas serta melalui tombol klik tunggal di formulir.

Aspose .NET OneClick PDF Document Generator dapat digunakan dengan CRM on-premises dan CRM Online.

Fitur utama dari Addon ini adalah:

- Membuat Template di dalam word dan melampirkannya di CRM.
- Buat Template di dalam word dan lampirkan di CRM.
- Memungkinkan Anda untuk menambahkan Optionset serta lookups.
- Bekerja dengan Account, Contact, Lead, Opportunity dan Quote.
- Tombol klik satu kali yang dapat Menghasilkan dokumen.
- Dokumen yang dihasilkan dapat diunduh serta dilampirkan dengan catatan entitas yang sama.
- Tombol pita juga disediakan untuk memilih template dan menghasilkan dokumen.

Add-on ini akan bekerja dengan semua versi **Microsoft Dynamics CRM 2015 dan CRM Online**.

## Memasang atau Mencopot Pemasangan

### Cara Memasang

#### Mengimpor Solusi

- Unduh Berkas Solusi. (Dikelola jika Anda berniat mencopotnya)
- Buka CRM dan pergi ke Impor Solusi.
- Klik Telusuri dan pilih berkas solusi yang diunduh lalu klik lanjut.
- Klik Lanjut dan tunggu sampai solusi diimpor.
- Klik tutup ketika solusi berhasil diimpor.

#### Mengkonfigurasi Halaman Web

- Unduh kode sumber untuk situs web.
- Salin di server. (Direkomendasikan di bawah inetpub)
- Buka IIS. (Di server tempat Anda berniat menerapkan situs web, dapat diakses dari server CRM)
- Buat Situs Web baru.
- Pilih jalur fisik dan konfigurasikan situs web.
- Telusuri untuk memeriksa apakah dapat diakses.
- Buka CRM dan tambahkan catatan baru di bawah Aspose>> Entitas Konfigurasi Aspose dan tentukan Nama dan URL dari Halaman Web yang diterapkan seperti di bawah ini.

![OneClick PDF Document Generator](oneclick-pdf-document-generator_1.png)
![OneClick PDF Document Generator](oneclick-pdf-document-generator_1.png)

### Cara Mencopot

- Hapus Situs Web dari IIS.
- Pergi ke solusi dan pilih "OneClick PDF Document Generator".
- Tekan tombol Hapus. Tekan OK untuk konfirmasi.
