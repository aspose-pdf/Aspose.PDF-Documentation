## Apa itu Microsoft Dynamics CRM?

Microsoft Dynamics CRM adalah paket perangkat lunak manajemen hubungan pelanggan yang dikembangkan oleh Microsoft. Secara default, produk ini fokus terutama pada sektor Penjualan, Pemasaran, dan Layanan (meja bantuan), tetapi Microsoft telah memasarkan Dynamics CRM sebagai platform XRM dan telah mendorong mitra untuk menggunakan kerangka kerja miliknya (.NET berbasis) untuk menyesuaikannya. Ini adalah bagian dari keluarga aplikasi bisnis Microsoft Dynamics.

Dynamics CRM adalah aplikasi server-klien, yang, seperti Microsoft SharePoint, pada dasarnya adalah aplikasi web berbasis IIS yang juga mendukung antarmuka layanan web yang luas. Klien mengakses Dynamics CRM baik dengan menggunakan Browser atau dengan plug-in klien tebal ke Microsoft Outlook.

## Aspose.PDF for .NET

Aspose.PDF for .NET adalah komponen pembuatan dan manipulasi dokumen PDF yang memungkinkan aplikasi .NET Anda untuk membaca, menulis, dan memanipulasi dokumen PDF yang ada tanpa menggunakan Adobe Acrobat.
Untuk detail lebih lanjut tentang API Aspose.PDF for .NET

## Aspose.PDF untuk CRM

Aspose .NET dapat digunakan dengan Dynamics CRM dalam berbagai cara. Ketika Anda bekerja dengan dokumen, baik itu pembuatan dokumen atau pemrosesan dokumen, Anda perlu melihat API Aspose yang dapat memberikan hasil cepat dengan jumlah baris kode yang minimal. Periksa contoh add-on kami di bawah ini, add-on ini akan membantu Anda menggunakan Aspose dalam aplikasi Anda.
