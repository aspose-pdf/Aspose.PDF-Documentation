## Apa itu Microsoft Dynamics CRM?

Microsoft Dynamics CRM adalah paket perangkat lunak manajemen hubungan pelanggan yang dikembangkan oleh Microsoft. Secara default, produk ini terutama fokus pada sektor Penjualan, Pemasaran, dan Layanan (help desk), namun Microsoft telah memasarkan Dynamics CRM sebagai platform XRM dan telah mendorong mitra untuk menggunakan kerangka kerja berbasis (.NET) miliknya untuk mengkustomisasinya. Ini adalah bagian dari keluarga aplikasi bisnis Microsoft Dynamics.

Dynamics CRM adalah aplikasi server-klien, yang, seperti Microsoft SharePoint, terutama adalah aplikasi web berbasis IIS yang juga mendukung antarmuka layanan web yang luas. Klien mengakses Dynamics CRM baik dengan menggunakan Browser atau dengan plug-in klien tebal ke Microsoft Outlook.

## Aspose.PDF untuk .NET

Aspose.PDF untuk .NET adalah komponen pembuatan dan manipulasi dokumen PDF yang memungkinkan aplikasi .NET Anda untuk membaca, menulis, dan memanipulasi dokumen PDF yang ada tanpa menggunakan Adobe Acrobat.
