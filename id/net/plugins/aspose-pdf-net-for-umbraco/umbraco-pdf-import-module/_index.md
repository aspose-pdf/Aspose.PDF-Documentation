## Pendahuluan

**Aspose.PDF for .NET** adalah komponen pembuatan dan manipulasi dokumen PDF yang memungkinkan aplikasi .NET Anda untuk membaca, menulis, dan memanipulasi dokumen PDF yang ada tanpa menggunakan Adobe Acrobat. Ini juga memungkinkan Anda untuk membuat formulir dan mengelola bidang formulir yang tertanam dalam dokumen PDF.

**Aspose.PDF for .NET** terjangkau dan menawarkan kekayaan fitur yang luar biasa termasuk opsi kompresi PDF; pembuatan dan manipulasi tabel; dukungan untuk objek grafik; fungsionalitas hyperlink yang luas; kontrol keamanan yang diperluas; penanganan font kustom; integrasi dengan sumber data; menambah atau menghapus bookmark; membuat daftar isi; menambah, memperbarui, menghapus lampiran dan anotasi; mengimpor atau mengekspor data formulir PDF; menambah, mengganti atau menghapus teks dan gambar; membagi, menggabungkan, mengekstrak atau menyisipkan halaman; mengubah halaman menjadi gambar; mencetak dokumen PDF dan banyak lagi.

### **Fitur Modul**

Umbraco PDF Import adalah tambahan open source dari [Aspose](http://www.aspose.com/) yang memungkinkan pengembang untuk mendapatkan/membaca konten dari dokumen PDF mana pun tanpa memerlukan perangkat lunak lain. Tambahan ini menunjukkan fitur impor yang kuat yang disediakan oleh [Aspose.PDF](https://products.aspose.com/pdf/net/). Ini menambahkan kontrol penjelajah file sederhana dan tombol **Impor dari PDF** di halaman tempat tambahan ditambahkan. Ketika mengklik tombol tersebut, konten dokumen diambil dari file dan ditampilkan di layar segera.

## Menginstal

Setelah diunduh, silakan ikuti langkah-langkah berikut untuk menginstal paket ini ke dalam situs web Umbraco Anda:

1. Masuk ke bagian **Developer** Umbraco, misalnya <http://www.myblog.com/umbraco/>.
1. Dari pohon, perluas folder **Packages**.
   Dari sini ada dua cara untuk menginstal paket: pilih **Instal paket lokal** atau telusuri **Repositori Paket Umbraco.**.
1. Jika Anda menginstal **paket lokal**, jangan mengekstrak paket tetapi muat zip ke dalam Umbraco.
1. Ikuti instruksi di layar.

**Catatan:** Anda mungkin mendapatkan kesalahan ‘Panjang permintaan maksimum terlampaui’ saat menginstal. Anda dapat dengan mudah memperbaiki masalah ini dengan memperbarui nilai ‘maxRequestLength’ di file web.config Umbraco Anda.

```xml
  <httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
```

## Menggunakan

Setelah Anda menginstal makro, sangat mudah untuk mulai menggunakannya di situs web Anda:

1. Pastikan Anda masuk ke bagian **Developer** Umbraco, misalnya <http://www.myblog.com/umbraco/>.
1. Klik **Settings** di daftar bagian di sudut kiri bawah layar.
1. Perluas node **Templates** dan pilih template yang ingin Anda tambahkan makro, misalnya Blog post.
1. Pilih posisi di template yang dipilih di mana Anda ingin tombol tersebut.
1. Klik **Insert Macro** di pita atas.
1. Dari **Choose a macro**, pilih makro yang diinstal dan klik **OK**.

Anda telah berhasil menambahkan template. Tombol bertuliskan **Impor dari Pdf** sekarang muncul di semua halaman yang dibuat menggunakan template ini. Siapa pun dapat dengan mudah mengklik tombol tersebut dan mengimpor konten dari dokumen PDF.
