## Pendahuluan

Kontrol Ekspor GridView ke PDF adalah kontrol server ASP.NET yang memungkinkan mengekspor konten GridView ke dalam dokumen PDF menggunakan [Aspose.PDF](https://products.aspose.com/pdf/net/). Ini menambahkan tombol **Ekspor ke PDF** di atas kontrol GridView. Mengklik tombol tersebut secara dinamis mengekspor konten kontrol GridView ke dokumen PDF dan kemudian secara otomatis mengunduh file yang diekspor ke lokasi disk yang dipilih oleh pengguna dalam beberapa detik saja.

### Fitur Modul

Versi awal kontrol ini menyediakan fitur-fitur berikut:

- Dapatkan salinan offline dari konten GridView online favorit Anda untuk pengeditan, berbagi, dan pencetakan dalam dokumen PDF yang sangat populer.
- Turunan dari kontrol GridView ASP.NET default dan oleh karena itu memiliki semua fitur dan propertinya.
- Bekerja dengan semua versi .NET mulai dari .NET 2.0.
- Kemampuan untuk menyesuaikan/melokalisasi teks tombol Ekspor.
- Opsi untuk mengekspor dalam mode Lanskap jika konten GridView lebih lebar dan tidak muat dalam mode Potret default.
- Terapkan tampilan dan nuansa tema Anda sendiri pada tombol Ekspor menggunakan CSS.
- Opsi untuk menambahkan judul kustom di atas dokumen yang diekspor.
- Opsi untuk menyimpan setiap dokumen yang diekspor di server pada jalur disk yang dapat dikonfigurasi.
- Opsi untuk mengekspor halaman saat ini atau semua halaman ketika paging diaktifkan.

## Menginstal

Sangat sederhana dan mudah untuk menginstal Kontrol Ekspor GridView ke PDF, silakan ikuti langkah-langkah sederhana ini

### **Untuk Visual Studio 2010, 2012 dan 2013**

1. Ekstrak file zip yang diunduh yaitu Aspose.PDF.GridViewExport_1.0.zip.
1. Klik dua kali file VSIX Aspose.PDF.GridViewExport.vsix.
1. Sebuah dialog akan muncul yang menunjukkan versi Visual Studio yang tersedia dan didukung yang terinstal di mesin Anda.
1. Pilih yang ingin Anda tambahkan ke Kontrol Ekspor GridView ke PDF.
1. Klik Instal.

Anda akan mendapatkan dialog sukses setelah instalasi selesai.

**Catatan:** Harap pastikan untuk memulai ulang Visual Studio agar perubahan dapat diterapkan.

### **Untuk Visual Studio 2005, 2008 dan edisi Express**

Silakan ikuti langkah-langkah ini untuk mengintegrasikan Kontrol Ekspor GridView ke PDF di Visual Studio untuk drag and drop yang mudah seperti kontrol ASP.NET lainnya

1. Ekstrak file zip yang diunduh yaitu Aspose.PDF.GridViewExport.NET2.0_1.0.zip.
1. Pastikan untuk menjalankan Visual Studio sebagai Administrator.

Di menu Alat, klik Pilih Item Toolbox.

1. Klik Telusuri.
   Kotak dialog Buka muncul.
1. Telusuri ke folder yang diekstrak dan pilih Aspose.PDF.GridViewExport.dll.
1. Klik OK.

Ketika Anda membuka kontrol aspx atau ascx di Toolbox sisi kiri, Anda akan melihat ExportGridViewToPdf di bawah Tab Umum

### Menambahkan kontrol ExportGridViewToPdf secara manual

Jika Anda mengalami masalah menggunakan metode di atas yang menggunakan Toolbox Visual Studio, Anda dapat menambahkan kontrol ini secara manual ke aplikasi ASP.NET Anda yang berjalan di .NET framework lebih dari 2.0

1. Jika Anda menggunakan Visual Studio, pastikan untuk menjalankannya sebagai Administrator.
1. Tambahkan referensi ke **Aspose.PDF.GridViewExport.dll** yang tersedia dalam paket unduhan yang diekstrak di proyek ASP.NET atau aplikasi web Anda. Pastikan aplikasi web/Visual Studio Anda memiliki akses penuh ke folder ini, jika tidak, Anda mungkin mendapatkan pengecualian Akses ditolak.
1. Tambahkan baris ini ke bagian atas halaman, kontrol, atau MasterPage.

```csharp
 <%@ Register assembly="Aspose.PDF.GridViewExport" namespace="Aspose.PDF.GridViewExport" tagprefix="aspose" %>
```

1. Tambahkan yang berikut ke tempat di halaman ASP.NET Anda, kontrol, atau masterpage di mana Anda ingin kontrol ditambahkan

```csharp
 <aspose:ExportGridViewToPdf ID="ExportGridViewToPdf1" runat="server"></aspose:ExportGridViewToPdf>
```

### **Properti Kontrol Ekspor GridView ke PDF Aspose .NET**

Properti berikut diekspos untuk mengonfigurasi dan menggunakan fitur-fitur keren yang disediakan oleh kontrol ini

|**Nama Properti**|**Tipe**|**Contoh/Nilai yang Mungkin**|**Deskripsi**|
| :- | :- | :- | :- |
|ExportButtonText|string|Ekspor ke PDF|Anda dapat menggunakan properti ini untuk menimpa teks default yang ada.|
|ExportButtonCssClass|string|btn btn-primary|Kelas CSS yang diterapkan pada div luar tombol ekspor. Untuk menerapkan CSS pada tombol, Anda dapat menggunakan .yourClass input.|
|ExportInLandscape|bool|true atau false|Jika true, ini mengubah orientasi dokumen keluaran menjadi lanskap. Defaultnya adalah Potret.|
| | | | |
|ExportFileHeading|string|Contoh Laporan Ekspor GridView|Anda dapat menggunakan tag HTML untuk menambahkan gaya pada judul Anda.|
|ExportOutputPathOnServer|string|c:/temp|Jalur Disk keluaran lokal di server di mana salinan ekspor secara otomatis disimpan. Aplikasi harus memiliki akses tulis ke jalur ini.|
|ExportDataSource|object|allRowsDataTable|Mengatur objek dari mana kontrol data-bind ini mengambil daftar item data. Objek tersebut harus memiliki semua data yang perlu diekspor. Properti ini digunakan selain properti DataSource normal dan berguna ketika paging kustom diaktifkan dan halaman saat ini hanya mengambil baris yang akan ditampilkan di layar.|
|LicenseFilePath|string| |Jalur lokal di server ke file lisensi. Misalnya c:/inetpub/Aspose.PDF.lic.|

Contoh kontrol Ekspor GridView ke PDF dengan semua properti yang digunakan ditunjukkan di bawah ini

```csharp
<aspose:ExportGridViewToPdf Width="800px" ID="ExportGridViewToPdf1" ExportButtonText="Export to Pdf"
    CssClass="table table-hover table-bordered" ExportButtonCssClass="myClass" ExportOutputFormat="Doc"
    ExportInLandscape="true" ExportOutputPathOnServer="c:\\temp" ExportFileHeading="<h4>Example Report</h4>"
    OnPageIndexChanging="ExportGridViewToPdf1_PageIndexChanging" PageSize="5" AllowPaging="True"
    LicenseFilePath="c:\\inetpub\\Aspose.PDF.lic"
    runat="server" CellPadding="4" ForeColor="#333333" GridLines="Both">
</aspose:ExportGridViewToPdf>
```

- Aspose.PDF.GridViewExport.Website - Proyek web untuk menguji GridView PDF yang dapat diekspor ke Word.
