## Pendahuluan

Aspose .NET PDF Import untuk Modul DNN memungkinkan pengembang untuk mendapatkan/membaca konten dokumen PDF tanpa memerlukan perangkat lunak lain seperti Adobe Acrobat atau pembaca PDF.

### Fitur Modul

Modul ini menunjukkan fitur impor yang kuat yang disediakan oleh [Aspose.PDF](https://products.aspose.com/pdf/net/). Ini menambahkan kontrol penjelajah file sederhana, pemilihan Panel Konten, dan tombol **Impor dari PDF** di halaman tempat modul ditambahkan. Ketika mengklik tombol tersebut, konten dokumen ditampilkan di layar segera.

## Persyaratan Sistem dan Platform yang Didukung

### Persyaratan Sistem

Untuk mengatur Aspose .NET Pdf Import untuk Modul DNN, Anda perlu memenuhi persyaratan berikut:

- DNN 7.0+.

Silakan hubungi kami jika Anda ingin menginstal modul ini di versi DNN lainnya.

### Platform yang Didukung

Aspose .NET Pdf Import untuk Modul DNN saat ini mendukung

## Menginstal

Setelah diunduh, silakan ikuti langkah-langkah berikut untuk menginstal modul ke situs DNN Anda:

1. Masuk ke situs Anda sebagai **Host** atau akun level Supersuser lainnya.
1. Navigasikan ke menu Host dan pilih **Extensions**.
1. Klik pada **Install Extension Wizard**.
1. Sesuai petunjuk, telusuri lokasi file zip yang diunduh, pilih, lalu klik **Open**.
1. Klik **Next**, terima lisensi, lanjutkan melalui proses instalasi. Setelah selesai, klik tombol **Return**.

Silakan periksa [video instalasi modul ini](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) dari DNN untuk detail lebih lanjut.

**Catatan:** Jika Anda mendapatkan kesalahan saat mengunggah modul, ini disebabkan oleh batas maxRequestLength dalam web.config instalasi DNN Anda. Buka web.config dan perbarui maxRequestLength menjadi 20MB dengan mengatur **maxRequestLength=”20480″** dan coba unggah modul lagi.

## Menggunakan

Setelah Anda menginstal Aspose .NET PDF Import untuk modul DNN, sangat mudah untuk mulai menggunakannya di situs web Anda. Silakan ikuti langkah-langkah sederhana ini untuk memulai

1. Pastikan Anda masuk ke DNN sebagai akun level Host atau Admin.
1. Navigasikan ke halaman tempat Anda ingin menambahkan modul Impor.
1. Pilih **Modules** -> **Add New Module** dari Ribbon atas.

1. Dari daftar, pilih **Aspose .NET PDF Import untuk DNN** dan seret ke tempat yang Anda pilih di halaman.

Anda telah berhasil menambahkan modul Aspose .NET PDF Import untuk DNN ke halaman Anda. Sebuah penjelajah file dan tombol bertuliskan **Impor dari PDF** sekarang akan muncul di halaman bersama dengan dropdown pemilihan Panel Konten. Siapa pun dapat dengan mudah memilih file PDF dan mengklik tombol **Impor dari PDF** untuk menampilkan konten dokumen yang dipilih di halaman.
