## Ekspor nilai dari bidang ke file JSON

Aspose.Pdf.Facades menyediakan API alternatif untuk bekerja dengan bidang formulir. Cuplikan ini menunjukkan cara mengekspor dan mengimpor nilai bidang menggunakan API ini.

```cs
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form();
// Buka Dokumen
form.BindPdf(_dataDir + "Sample-Form-01.pdf");

// Buat file XFDF.
System.IO.FileStream jsonStream = new FileStream("Sample-Form-01.json", FileMode.Create);

// Ekspor data
form.ExportJson(jsonStream);

// Tutup aliran file
jsonStream.Close();

// Tutup dokumen
form.Dispose();
```
## Impor nilai ke bidang dari file JSON

Cuplikan kode ini menunjukkan cara mengimpor nilai ke bidang formulir dari dokumen PDF dari file JSON menggunakan Aspose.Pdf.Facades API. FileStream digunakan untuk menangani file JSON.

```cs

Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form();
// Buka Dokumen
form.BindPdf("Sample-Form-01.pdf");

// Buat file XFDF.
System.IO.FileStream jsonStream = new FileStream("Sample-Form-01.json", FileMode.Open);

// Ekspor data
form.ImportJson(jsonStream);

// Tutup file stream
jsonStream.Close();

// Tutup dokumen
form.Dispose();
```
