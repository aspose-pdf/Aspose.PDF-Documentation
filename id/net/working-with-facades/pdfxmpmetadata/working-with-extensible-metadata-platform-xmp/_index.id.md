Extensible Metadata Platform (XMP) adalah standar yang dibuat oleh Adobe Systems Inc. Standar ini dikembangkan untuk memproses dan menyimpan metadata yang terstandarisasi dan milik pribadi. Metadata ini dapat disematkan ke dalam berbagai format file, tetapi dalam artikel ini kita akan fokus hanya pada format file PDF. Kita akan melihat bagaimana kita dapat menyematkan Metadata dalam file PDF menggunakan namespace Aspose.Pdf.Facades dalam Aspose.PDF untuk .NET. Kita akan menggunakan kelas [PdfXmpMetadata](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfxmpmetadata) untuk memanipulasi XMP dalam dokumen PDF.

{{% /alert %}}

## Latar Belakang

Sebuah file PDF melewati banyak tahap selama masa hidupnya. Kami membuat dokumen PDF dan kemudian meneruskannya ke orang atau departemen lain untuk diproses lebih lanjut. Namun, selama proses ini, kita perlu melacak berbagai aspek dari perubahan yang dilakukan. XMP berfungsi untuk melacak perubahan atau informasi lain tentang data dalam file.

## Penjelasan

Untuk memanipulasi XMP menggunakan Aspose.Pdf.Facades, kita akan menggunakan kelas [PdfXmpMetadata](http://www.aspose.com/docs/display/pdfnet/PdfXmpMetadata+Class). Kita akan menggunakan kelas ini untuk memanipulasi properti Meta data yang telah ditentukan sebelumnya. Kelas [PdfXmpMetadata](http://www.aspose.com/docs/display/pdfnet/PdfXmpMetadata+Class) menambahkan properti ini ke file PDF. Ini juga membantu untuk membuka dan menyimpan file PDF yang mana kita menambahkan Meta data. Jadi, dengan menggunakan kelas [PdfXmpMetadata](http://www.aspose.com/docs/display/pdfnet/PdfXmpMetadata+Class), kita dapat dengan mudah memanipulasi XMP dalam file PDF.
