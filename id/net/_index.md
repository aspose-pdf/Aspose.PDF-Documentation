## Selamat datang di Aspose.PDF for .NET

{{% alert color="primary" %}}

Aspose.PDF adalah pustaka kelas .NET yang memungkinkan pengembang untuk membuat dokumen PDF, baik sederhana maupun kompleks, secara programatis. Aspose.PDF for .NET memungkinkan pengembang untuk menyisipkan tabel, grafik, gambar, hyperlink, font kustom - dan banyak lagi - ke dalam dokumen PDF. Selain itu, juga dimungkinkan untuk mengompresi dokumen PDF. Produk ini mematuhi semua standar PDF termasuk ISO 32000-1 (PDF 1.7), ISO 32000-2 (PDF 2.0)
dan banyak ekstensi seperti PDF/A, PDF/UA, PDF/X, PDF/E.

Aspose.PDF for .NET menyediakan fitur kunci berikut:

- Mengonversi dokumen dari HTML, Doc/Docx, Pptx, Markdown, Json, Jpeg dan lainnya.
- Mendukung berbagai penyedia AI dengan kemampuan untuk memuat data dari dokumen PDF.
- Mendukung pembuatan dokumen PDF melalui API dan dari file XML.
- Fitur keamanan berbasis sertifikat yang sangat baik dan perlindungan kata sandi untuk mengembangkan dokumen PDF yang aman.
- Kompatibel dengan tanda tangan digital yang kuat (termasuk PAdES).
- Pembuatan Formulir PDF (formulir interaktif, AcroForms, formulir XFA dinamis).

dan banyak lagi.

{{% /alert %}}
