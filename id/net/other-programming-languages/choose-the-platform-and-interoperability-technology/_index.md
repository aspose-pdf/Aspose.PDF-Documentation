**Ada tiga versi dari perpustakaan kelas Aspose.PDF:**

[Aspose.PDF](/pdf/) adalah keluarga produk yang menyediakan fungsionalitas untuk menghasilkan dokumen PDF dari awal, serta kemampuan untuk mengedit/memanipulasi yang sudah ada. Aspose.PDF tersedia dalam tiga produk yang berbeda: [Aspose.PDF for .NET](/pdf/net/), [Aspose.PDF untuk Java](/pdf/java/), dan [Aspose.PDF untuk C++](/pdf/cpp/). Kita dapat mengatakan bahwa Aspose.PDF “secara native” mendukung:

- .NET Framework dan semua bahasa pemrograman yang menargetkan platform .NET, seperti C# atau VB.NET.
- Platform Java dan bahasa pemrograman Java.
- Sistem operasi 32-bit dan 64-bit serta bahasa pemrograman C++.

{{% /alert %}}

Harap dicatat bahwa juga dimungkinkan untuk menggunakan [Aspose.PDF for .NET](/pdf/net/) dari banyak bahasa pemrograman, termasuk: ASP, C++, ColdFusion, Delphi, JavaScript, Perl, PHP, PowerBuilder, Python, VBScript, dll.

## Dukungan untuk bahasa lain

{{% alert color="primary" %}}

Bagian ini berisi saran dan tautan dokumentasi mengenai cara menggunakan Aspose.PDF dalam berbagai bahasa pemrograman dan mencakup topik berikut:

- Pembungkus siap pakai yang tersedia untuk [Aspose.PDF for .NET](/pdf/net/).
- Cara membuat objek, memanggil metode, dan mengakses properti objek Aspose.PDF for .NET.

Jika Anda mengikuti prinsip-prinsip yang diuraikan dalam bagian ini, Anda akan dapat memanfaatkan banyak fitur Aspose.PDF dalam proyek Anda.

{{% /alert %}}

**Produk berbasis perpustakaan kelas Aspose.PDF for .NET:**

Selain tiga produk yang disebutkan di atas, Aspose mengusulkan serangkaian solusi untuk berbagai platform dan bahasa:

- [Aspose.PDF untuk Python melalui .NET](/pdf/python-net/) digunakan untuk mengembangkan aplikasi Python 32-bit dan 64-bit untuk berbagai sistem operasi.
- [Aspose.PDF untuk Layanan Pelaporan](/pdf/reportingservices/) memungkinkan menghasilkan laporan PDF di Microsoft SQL Server Reporting Services dan Power BI Report Server.
- [Aspose.PDF untuk SharePoint](/pdf/sharepoint/) memungkinkan mengekspor daftar, item daftar, dan halaman Wiki SharePoint ke format file PDF.

Daftar lengkap produk Aspose.PDF dapat ditemukan di [halaman](/pdf/). Ini termasuk produk berbasis [Aspose.PDF untuk Java](/pdf/java/) dan PDF [Aspose.PDF untuk C++](/pdf/cpp/).

**Mengakses objek Aspose.PDF dari bahasa lain**

Menggunakan Aspose.PDF dari bahasa pemrograman lain memerlukan *teknologi interoperabilitas*. Teknologi ini harus memungkinkan bahasa Anda untuk memanggil perpustakaan kelas yang ditulis untuk .NET, Java, atau C++. Kami menyarankan teknologi [COM Interop](/pdf/net/use-aspose-pdf-for-net-via-com-interop/) dengan perpustakaan kelas [Aspose.PDF for .NET](/pdf/net/) untuk digunakan dengan berbagai bahasa pemrograman:
- ASP.
- Delphi.
- Perl.
- PHP.
- PowerBuilder.
- VBScript.
- Visual Basic.

Informasi tentang teknologi interoperabilitas yang disarankan untuk digunakan dengan [Aspose.PDF untuk Java](/pdf/java/) dan PDF [Aspose.PDF untuk C++](/pdf/cpp/) seperti `Inline::Java` dan `JPype` dapat ditemukan di [halaman](/pdf/java/plugins/) produk yang bersangkutan.
