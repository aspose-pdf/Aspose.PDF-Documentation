{{% alert color="primary" %}}

Dengan Aspose.PDF for SharePoint Anda dapat mengekspor daftar, item daftar, dan halaman Wiki SharePoint dari perpustakaan SharePoint ke format PDF.

{{% /alert %}}

{{% alert color="primary" %}}

Untuk menghasilkan dokumen, Aspose.PDF for SharePoint bergantung pada versi bawaan dari [Aspose.PDF for .NET](http://www.aspose.com/categories/.net-components/aspose.pdf-for-.net/default.aspx).


{{% /alert %}}
