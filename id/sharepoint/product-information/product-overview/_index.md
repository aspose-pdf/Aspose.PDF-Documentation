### **Selamat datang di Dokumentasi Aspose.PDF untuk SharePoint!**
Aspose.PDF untuk SharePoint adalah solusi yang memungkinkan pengguna mengekspor daftar, item daftar, dan halaman Wiki SharePoint ke format file PDF.

{{% /alert %}}


{{% alert color="primary" %}}
Aspose.PDF untuk SharePoint dirancang untuk digunakan dengan Microsoft SharePoint Server 2010. Tidak ada persyaratan sistem tambahan selain Microsoft SharePoint Server 2010.

Dokumentasi ini menjelaskan Aspose.PDF untuk SharePoint [fitur](/pdf/id/sharepoint/features/), [instalasi](/pdf/id/sharepoint/install-aspose-pdf-for-sharepoint/), [batasan evaluasi](/pdf/id/sharepoint/evaluate-aspose-pdf/), [lisensi](/pdf/id/sharepoint/license-aspose-pdf-for-sharepoint/), kasus penggunaan umum dan pengaturan.
