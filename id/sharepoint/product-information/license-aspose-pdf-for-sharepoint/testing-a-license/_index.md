{{% alert color="primary" %}}

Untuk menguji bahwa lisensi telah dipasang dengan benar, konversi dokumen apa pun ke format yang diinginkan. Jika dokumen tidak mengandung Peringatan Evaluasi, lisensi telah diaktifkan dengan sukses.

{{% /alert %}}
