{{% alert color="primary" %}}

Untuk menguji bahwa lisensi telah dipasang dengan benar, konversikan dokumen apa pun ke format yang diinginkan. Jika dokumen tidak mengandung Evaluation Warning, lisensi telah diaktifkan dengan sukses.

{{% /alert %}}
