{{% alert color="primary" %}}

Selama instalasi, Aspose.PDF for SharePoint diaktifkan untuk semua koleksi situs yang dipilih. Setelah instalasi, Anda dapat menggunakan menu Site Actions pada situs web root dari sebuah koleksi situs untuk mengaktifkan dan menonaktifkan Aspose.PDF for SharePoint.

{{% /alert %}}
