PDF adalah singkatan dari Portable Document Format, yang digunakan untuk menampilkan dokumen dalam bentuk elektronik yang independen dari perangkat lunak, perangkat keras, atau sistem operasi tempat dokumen tersebut dilihat.

PDF merupakan standar terbuka, yang dikelola oleh International Organisation for Standardisation (ISO) saat ini.

Tujuan awalnya adalah untuk mempertahankan dan melindungi konten serta tata letak sebuah dokumen – tidak peduli platform atau program komputer apa yang digunakan untuk melihatnya. Inilah mengapa PDF sulit untuk diedit dan terkadang bahkan mengekstrak informasi darinya menjadi tantangan.

Namun **Aspose.PDF for Node.js via C++** dapat membantu Anda menangani sebagian besar tugas yang muncul saat bekerja dengan dokumen PDF.

Anda dapat melakukan hal berikut:

- [Optimalkan PDF](/pdf/id/nodejs-cpp/optimize-pdf/) - mengoptimalkan konten halaman, mengoptimalkan ukuran file, menghapus objek yang tidak digunakan, mengompresi semua gambar untuk optimasi dokumen yang berhasil.
- [Optimalkan Sumber Daya PDF](/pdf/id/nodejs-cpp/optimize-pdf-resources/) - mengoptimalkan konten halaman, mengoptimalkan ukuran file, menghapus objek yang tidak digunakan, mengompresi semua gambar untuk optimasi dokumen yang berhasil.
- [Gabungkan PDF](/pdf/id/nodejs-cpp/merge-pdf/) - gabungkan beberapa file PDF menjadi satu dokumen PDF menggunakan Node.js.
- [Pisah PDF](/pdf/id/nodejs-cpp/split-pdf/) - pisahkan halaman PDF menjadi file PDF masing-masing dalam Node.js via C++.
