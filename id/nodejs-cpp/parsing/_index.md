**Parse PDF** dokumen adalah istilah yang terkait dengan ekstraksi berbagai jenis informasi dari file PDF. Bagian ini mencakup cara:

- [Ekstrak Gambar dari PDF](/pdf/id/nodejs-cpp/extract-images-from-the-pdf-file/). Ekstraksi Gambar memungkinkan mengekstrak gambar dan menyimpannya sebagai file terpisah.
- [Ekstrak Teks dari PDF](/pdf/id/nodejs-cpp/extract-text-from-pdf/). Parsing Teks atau Ekstraksi adalah operasi paling populer dengan PDF yang sudah jadi. Anda akan mempelajari tentang parsing teks dari seluruh dokumen, halaman tertentu, atau wilayah tertentu di sebuah halaman.
- [Ekstrak Tabel dari PDF](/pdf/id/nodejs-cpp/extract-tables-from-the-pdf-file/). Konversi PDF ke format CSV menggunakan Aspose.PDF for Node.js via C++.
