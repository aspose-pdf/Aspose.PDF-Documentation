# Instalasi

**Aspose.PDF for Node.js via C++** adalah toolkit yang mudah dan aman digunakan untuk bekerja dengan PDF secara langsung di lingkungan Node.js. 

Untuk menginstal dan menggunakan Aspose.PDF for Node.js via C++ dari arsip -*ZIP, ikuti instruksi berikut:

- unduh arsip dari situs Aspose.PDF
- ekstrak file dari arsip ZIP
- ganti nama file *.lic Anda menjadi 'aspose.lic' jika ada

Ikuti tautan untuk [Unduhan Langsung](https://releases.aspose.com/pdf/nodejscpp/)

Untuk menginstal dan menggunakan Aspose.PDF for Node.js via C++ dari NPMJS, ikuti instruksi berikut:

- npm i asposepdfnodejs
- ganti nama file *.lic Anda menjadi ‘aspose.lic’ jika ada
