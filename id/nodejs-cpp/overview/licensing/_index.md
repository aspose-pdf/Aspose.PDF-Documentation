## Batasan versi evaluasi

Kami ingin pelanggan kami menguji komponen kami secara menyeluruh sebelum membeli sehingga versi evaluasi memungkinkan Anda menggunakannya seperti biasanya.

>Jika Anda ingin menguji Aspose.PDF for Node.js via C++ tanpa batasan versi evaluasi, Anda juga dapat meminta Temporary License 30‑day. Silakan merujuk ke [Bagaimana cara mendapatkan Lisensi Sementara?](https://purchase.aspose.com/temporary-license/)

- **PDF dibuat dengan watermark evaluasi.**
Versi evaluasi dari Aspose.PDF for Node.js via C++ menyediakan semua fungsi produk, tetapi semua halaman dalam dokumen PDF yang dihasilkan diberi watermark “Evaluation Only. Created with Aspose.PDF. Copyright 2002-2023 Aspose Pty Ltd.” di bagian atas.

- **Batas jumlah item koleksi yang dapat diproses.**
Dalam versi evaluasi dari semua koleksi, Anda hanya dapat memproses empat elemen (misalnya, hanya 4 halaman, 4 bidang formulir, dll.).


## Lisensi Klasik

Nama file lisensi harus "aspose.lic". Anda perlu mengganti nama file lisensi yang diterima dan menempatkannya di folder Product.
