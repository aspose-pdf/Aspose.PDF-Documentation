## Gambaran Umum

Untuk memproses dokumen PDF menggunakan [Aspose.PDF for Node.js via C++](https://products.aspose.com/pdf/nodejs-cpp/), tidak memerlukan perpustakaan dan program pihak ketiga.
Memerlukan terinstal [Node.js](https://nodejs.org/). Anda harus memiliki versi Node.js 16 atau lebih tinggi. Kami sangat menyarankan menggunakan versi LTS.
Node.js dapat [diinstal](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) dengan cara berbeda.

## Sistem Operasi yang Didukung

Semua versi yang didukung dari [Linux](https://en.wikipedia.org/wiki/Linux) dan [macOS](https://www.apple.com/macos/).
