Aspose menyediakan dukungan teknis gratis tanpa batas untuk semua produknya. Dukungan tersedia untuk semua pengguna, termasuk evaluasi. Saluran utama dukungan adalah Aspose.Forums.

{{% alert color="primary" %}}
Harap dicatat bahwa Aspose tidak menyediakan dukungan teknis melalui telepon. Dukungan telepon hanya tersedia untuk pertanyaan penjualan dan pembelian.
{{% /alert %}}

## Forum Dukungan Gratis Aspose

Aspose.PDF memungkinkan Anda menggunakan dukungan teknis gratis untuk semua produknya. Jika Anda memiliki pertanyaan tentang Aspose.PDF, lihat artikel berikut:

- Sumber dukungan yang paling populer adalah [Aspose.Forums](https://forum.aspose.com/). Di [Aspose.PDF forum](https://forum.aspose.com/c/pdf/10) bagian, Anda dapat mengajukan pertanyaan Anda dan mendapatkan jawaban sesegera mungkin.
- Anda harus tahu bahwa Aspose tidak menyediakan dukungan teknis melalui telepon. Panggilan hanya tersedia untuk pertanyaan penjualan dan pembelian.
- Harap ingat, bahwa kami memiliki perbedaan zona waktu, ini dapat memengaruhi kecepatan respons kepada Anda.

## Laporkan Masalah atau Permintaan Fitur

Jika Anda memerlukan bantuan dengan masalah pada Aspose.PDF, ikuti  rekomendasi berikut untuk memastikan masalah tersebut diselesaikan dengan cara yang paling efisien:

- Pertama, pastikan Anda menggunakan versi terbaru Aspose.PDF sebelum melaporkan masalah, lihat [Unduh Aspose.PDF untuk Rust](https://releases.aspose.com/pdf/rustcpp/) untuk mengetahui versi terbaru.
- Mungkin solusi untuk masalah Anda sudah dibahas dan memiliki solusi, untuk itu, pelajari forum dan dokumentasi, periksa [Referensi API](https://reference.aspose.com/pdf/rust-cpp/) sebelum melaporkan masalah.
- Saat Anda menyusun pertanyaan, harap sertakan dokumen asli dan mungkin potongan kode Anda yang menyebabkan masalah. Anda dapat mengompres file-file ini menjadi satu dokumen jika ada beberapa.
- Jangan khawatir, hanya Anda dan pengembang Aspose yang memiliki akses ke file sticky.
