## Platform yang Didukung oleh Aspose.PDF untuk Rust via C++

Paket asposepdf adalah toolkit yang kuat yang memungkinkan pengembang memanipulasi file PDF secara langsung dan membantu melakukan berbagai tugas untuk PDF.
Berisi fitur unik untuk mengonversi PDF ke format lain.

Mendukung Linux x64, macOS x86_64, macOS arm64, dan platform Windows x64.

Versi khusus platform dari pustaka dinamis yang berada di folder ‘lib’ dalam direktori root paket diperlukan untuk mendistribusikan aplikasi yang dihasilkan: 

- libAsposePDFforRust_linux_amd64.so untuk platform Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib untuk platform macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib untuk platform macOS x86_64 
- AsposePDFforRust_windows_amd64.dll untuk platform Windows x64.
