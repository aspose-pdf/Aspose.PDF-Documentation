## Perangkat lunak lintas platform

Mendukung berbagai platform dan versi Rust. Lihat [Persyaratan Sistem](/pdf/id/rust-cpp/system-requirements/) untuk detail.

## Fitur Umum

Mendukung pemuatan dari dan penyimpanan ke daftar luas [format file](/pdf/id/rust-cpp/supported-file-formats/).

## Standar PDF yang Didukung

Aspose.PDF for Rust mendukung versi PDF 1.2, 1.3, 1.4, 1.5, 1.6, dan 1.7.

## Fitur berdasarkan jenis fungsi

### Operasi inti

- new PDF
- open PDF
- save PDF
- save_as
- page_insert
- page_add
- page_count
- page_delete
- extract_text
- set_license

### Mengatur

- optimize
- optimize_resource
- grayscale
- rotate
- set_background
- repair
- page_add_text
- page_set_size
- page_rotate
- page_grayscale

### Mengonversi dan Menyimpan

- PDF, DOC, DOCX, XLSX, PPTX, JPEG, PNG, BMP, TIFF, EPUB, DICOM, SVG, XPS, TEX, TXT
