## Platform yang Didukung untuk Aspose.PDF for Go via C++

Paket asposepdf adalah toolkit yang kuat yang memungkinkan pengembang untuk memanipulasi file PDF secara langsung dan membantu melakukan berbagai tugas untuk PDF.
Berisi fitur unik untuk mengonversi PDF ke format lain.

Menerapkan dukungan untuk platform Linux x64, macOS x86_64, macOS arm64, dan Windows x64. Digunakan [cgo](https://go.dev/wiki/cgo).

Versi khusus platform dari pustaka dinamis dari folder 'lib' dalam direktori root paket diperlukan untuk mendistribusikan aplikasi yang dihasilkan:

- **libAsposePDFforGo_linux_amd64.so** untuk platform Linux x64.
- **libAsposePDFforGo_darwin_arm64.dylib** untuk platform macOS arm64.
- **libAsposePDFforGo_darwin_amd64.dylib** untuk platform macOS x86_64.
- **AsposePDFforGo_windows_amd64.dll** untuk platform Windows x64.

Platform Windows x64 memerlukan [MinGW-W64](https://www.mingw-w64.org/) diinstal.
