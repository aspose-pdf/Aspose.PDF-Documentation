## Perangkat lunak lintas platform

Mendukung berbagai platform dan versi Go. Lihat [Persyaratan Sistem](/pdf/id/go-cpp/system-requirements/) untuk detail.

## Fitur Umum

Mendukung pemuatan dari dan penyimpanan ke daftar luas [format file](/pdf/id/go-cpp/supported-file-formats/).

## Standar PDF yang Didukung

Aspose.PDF for Go mendukung versi PDF 1.2, 1.3, 1.4, 1.5, 1.6, dan 1.7.
