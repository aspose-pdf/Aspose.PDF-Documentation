flattenAllFields(..) metode dari kelas [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form) memungkinkan Anda untuk memipihkan semua bidang dari formulir PDF. Anda perlu membuat objek [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form), mengikat file PDF sumber dan kemudian memanggil metode flattenAllFields(...) dan akhirnya menyimpan PDF yang diperbarui menggunakan metode save(..).

Cuplikan kode berikut menunjukkan kepada Anda cara memipihkan semua bidang dari file PDF.
