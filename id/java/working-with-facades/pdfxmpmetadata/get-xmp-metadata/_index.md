Untuk mendapatkan metadata XMP dari file PDF, Anda perlu membuat objek [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) dan mengikat file PDF menggunakan metode [bindPdf(..)](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Facade#bindPdf-com.aspose.pdf.IDocument-). Anda dapat melewatkan properti metadata XMP tertentu ke objek [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) untuk mendapatkan nilainya.

Cuplikan kode berikut menunjukkan cara mendapatkan metadata XMP dari file PDF.
