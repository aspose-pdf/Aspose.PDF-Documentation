# Pengenalan Aspose.PDF Java untuk Python

## JPype

JPype adalah upaya untuk memungkinkan program python mengakses penuh ke perpustakaan kelas java. Ini dicapai bukan melalui mengimplementasikan ulang Python, seperti yang telah dilakukan oleh Jython/JPython, tetapi melalui penghubungan pada tingkat asli di kedua Mesin Virtual.

Baca lebih lanjut di [readthedocs.org](http://jpype.readthedocs.org/en/latest/userguide.html)

## Aspose.PDF untuk Java

Aspose.PDF untuk Java adalah perpustakaan kelas lanjutan untuk Java yang memungkinkan Anda melakukan berbagai tugas pemrosesan dokumen secara langsung dalam aplikasi Java Anda.

Aspose.PDF untuk Java mendukung pemrosesan kata (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS, SWF dan semua format gambar. Dengan Aspose.PDF Anda dapat menghasilkan, memodifikasi, dan mengonversi dokumen tanpa menggunakan Microsoft Word.

Baca lebih lanjut di [aspose.com](https://products.aspose.com/words/java)

## Aspose.PDF Java untuk Python

Proyek Aspose.PDF untuk Python menunjukkan bagaimana berbagai tugas dapat dilakukan menggunakan API Aspose.PDF Java di Python. Proyek ini bertujuan untuk menyediakan contoh berguna bagi Pengembang Python yang ingin memanfaatkan Aspose.PDF untuk Java dalam Proyek Python mereka menggunakan [JPype](http://jpype.readthedocs.org/en/latest/userguide.html).
