Untuk mengonversi dokumen PDF ke format DOC atau DOCX menggunakan **Aspose.PDF Java for Python**, cukup panggil modul **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "Dokumen telah berhasil dikonversi"
```
