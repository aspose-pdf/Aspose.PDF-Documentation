## Fitur Umum

- Aspose.PDF for Android via Java mendukung semua perangkat seluler yang menjalankan runtime Java termasuk
- API menawarkan enkripsi 40-bit atau 128-bit

## Versi PDF yang didukung

Aspose.PDF for Android via Java mendukung versi PDF 1.2, 1.3, 1.4, 1.5, 1.6, dan 1.7.

## Grafik

- Berbagai bentuk grafik, seperti garis, lingkaran, busur, dan kurva didukung.
- Grafik dapat digambar pada lokasi halaman mana pun yang ditentukan oleh koordinat.
- Grafik dapat diputar, diskalakan, atau diisi, dan gaya garis dash serta poly-dash dapat didefinisikan.
- Grafik digambar berdasarkan koordinat dan memiliki posisi usang alih-alih relatif.
- Ruang warna CMYK dan Gray didukung.
