Gunakan bagian ini ketika Anda perlu melakukan operasi PDF tingkat halaman di Python.

Dengan **Aspose.PDF for Python via .NET**, Anda dapat menyisipkan, menghapus, memindahkan, mengekstrak, memutar, mengubah ukuran, dan memotong halaman, kemudian menerapkan header, footer, nomor halaman, watermark, dan stempel untuk alur kerja tata letak dan peninjauan.

## Tugas Halaman yang Dicakup

Tugas berikut dicakup:

- [Menambahkan Halaman](/pdf/id/python-net/add-pages/) - menyisipkan halaman pada posisi tertentu atau menambahkan halaman ke akhir dokumen.
- [Memindahkan Halaman](/pdf/id/python-net/move-pages/) - memindahkan halaman dalam satu dokumen atau antar dokumen.
- [Menghapus Halaman](/pdf/id/python-net/delete-pages/) - menghapus halaman dari dokumen PDF.
- [Mengekstrak Halaman](/pdf/id/python-net/extract-pages/) - mengekstrak halaman terpilih ke dalam file PDF baru.
- [Memutar Halaman](/pdf/id/python-net/rotate-pages/) - ubah orientasi halaman dalam file PDF yang ada.
- [Mengubah Ukuran Halaman](/pdf/id/python-net/change-page-size/) - ubah ukuran dimensi halaman dalam dokumen PDF.
- [Menambahkan Header dan Footer](/pdf/id/python-net/add-headers-and-footers-of-pdf-file/) - tambahkan teks yang berulang, gambar, atau konten header/footer terstruktur.
- [Memotong Halaman PDF](/pdf/id/python-net/crop-pages/) - memotong konten halaman menggunakan pengaturan geometri halaman.
- [Mendapatkan dan Mengatur Properti Halaman](/pdf/id/python-net/get-and-set-page-properties/) - mengakses dan memodifikasi berbagai properti halaman PDF, seperti ukuran, rotasi, dan atribut warna.
- [Penstempelan](/pdf/id/python-net/stamping/) - menerapkan stempel teks, gambar, halaman, dan nomor halaman ke halaman PDF.
