**Aspose.PDF for Python via .NET** memungkinkan Anda menyisipkan halaman ke dokumen PDF di lokasi mana saja dalam file serta menambahkan halaman ke akhir file PDF. Bagian ini menunjukkan cara menambahkan halaman ke PDF tanpa Acrobat Reader.
Anda dapat menambahkan teks atau gambar di header dan footer file PDF Anda, serta memilih header yang berbeda dalam dokumen Anda dengan library Python oleh Aspose.
Juga, coba potong halaman dalam dokumen PDF secara programatis menggunakan Python.

Bagian ini mengajarkan Anda cara menambahkan watermark dalam file PDF Anda menggunakan kelas Artifact. Anda akan memeriksa contoh pemrograman untuk tugas ini.
Tambahkan nomor halaman menggunakan kelas [PageNumberStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/pagenumberstamp/). Untuk menambahkan Stempel dalam dokumen Anda gunakan kelas [ImageStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/imagestamp/) dan [TextStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/textstamp/). Gunakan penambahan watermark untuk membuat gambar latar belakang dalam file PDF Anda dengan **Aspose.PDF for Python via .NET**.

- [Memindahkan Halaman](/pdf/python-net/move-pages/) - memindahkan halaman dari satu dokumen ke dokumen lain.
- [Menghapus Halaman](/pdf/python-net/delete-pages/) - menghapus halaman dari file PDF Anda menggunakan koleksi PageCollection.
- [Mengekstrak Halaman](/pdf/python-net/extract-pages/) - mengekstrak halaman dari file PDF Anda menggunakan Aspose.PDF untuk Python.
- [Memutar Halaman](/pdf/python-net/rotate-pages/) - Anda dapat mengubah orientasi halaman dalam file PDF yang ada.
- [Mengubah Ukuran Halaman](/pdf/python-net/change-page-size/) - ubah ukuran halaman dari dokumen PDF Anda.
- [Menambahkan Header dan Footer](/pdf/python-net/add-headers-and-footers-of-pdf-file/) - tambahkan teks atau gambar di header dan footer file PDF Anda.
- [Memotong Halaman PDF](/pdf/python-net/crop-pages/) - Anda dapat memotong halaman dalam dokumen PDF secara programatis dengan berbagai Properti Halaman.
- [Mendapatkan dan Mengatur Properti Halaman](/pdf/python-net/get-and-set-page-properties/) - akses dan ubah berbagai properti halaman PDF, seperti ukuran, rotasi, dan atribut warna.
- [Menambahkan Stempel](/pdf/python-net/stamping/) - tambahkan stempel gambar, stempel halaman, stempel nomor halaman, dan stempel teks ke halaman PDF melalui Python.
