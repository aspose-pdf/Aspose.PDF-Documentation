**Aspose.PDF for Python via .NET** menyediakan alat lengkap untuk bekerja dengan tabel dalam file PDF. Anda dapat membuat dan menambahkan tabel, mengontrol tata letak tabel dan aliran halaman, mengekstrak data tabel, memperbarui tabel yang ada, dan menghapus satu atau beberapa tabel dari PDF yang sudah ada.

Gunakan bagian ini ketika Anda perlu membuat laporan berbasis tabel, mengekstrak data terstruktur dari PDF, atau memperbarui konten tabel dalam dokumen yang ada dengan Python.

## Tugas Tabel yang Dicakup

Anda dapat melakukan hal berikut:

- [Menambahkan Tabel](/pdf/id/python-net/adding-tables/) - membuat tabel dalam file PDF, termasuk baris/kolom yang digabung, batas, margin, dan padding.
- [Mengekstrak Tabel](/pdf/id/python-net/extracting-table/) - mengekstrak data tabel dari file PDF yang ada.
- [Mengintegrasikan Tabel dengan Sumber Data](/pdf/id/python-net/integrate-table/) - membangun tabel dari basis data dan data DataFrame pandas di Python.
- [Memanipulasi Tabel](/pdf/id/python-net/manipulating-tables/) - memeriksa dan memperbarui tabel yang ada di dalam dokumen PDF.
- [Menghapus Tabel](/pdf/id/python-net/removing-tables/) - menghapus satu atau beberapa tabel dari dokumen PDF.
