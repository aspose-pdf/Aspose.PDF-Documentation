## Dasar-dasar AcroForms

AcroForms adalah formulir PDF interaktif yang berorientasi halaman yang diperkenalkan oleh Adobe dan termasuk dalam spesifikasi PDF sejak versi 1.2. Mereka didukung secara luas dan dapat digunakan untuk penangkapan data, validasi, dan pertukaran dalam alur kerja lintas platform.

Dalam alur kerja berbasis AcroForm, data formulir dapat dipertukarkan dalam format seperti FDF dan XFDF. Hal ini menjadikan AcroForms cocok untuk skenario di mana Anda perlu mengisi pra‑kolom dari sistem eksternal, mengekstrak nilai yang dikirim, atau menyinkronkan data formulir antar dokumen.

Aspose.PDF for Python via .NET menyediakan API untuk membuat dan mengelola AcroForms secara programatik, termasuk mengisi kolom, mengekspor data, memodifikasi struktur formulir, dan menambahkan aksi pengiriman.
