Di bagian ini, kami akan menjelaskan cara bekerja dengan lampiran dalam PDF menggunakan Aspose.PDF for Python via .NET.
Lampiran adalah file tambahan yang ditempelkan pada dokumen induk, dapat berupa berbagai jenis file, seperti pdf, word, gambar, atau file lainnya.
Anda akan mempelajari cara menambahkan lampiran ke pdf, mendapatkan informasi tentang lampiran, menyimpannya ke file, serta menghapus lampiran dari PDF secara programatis dengan Python.

Gunakan artikel-artikel ini ketika Anda perlu menyematkan file pendukung di dalam PDF, menghapus file yang disematkan dari dokumen yang ada, atau mengemas materi terkait bersama‑sama sebagai portofolio PDF.
