Artikel ini menunjukkan cara bekerja dengan anotasi berbasis teks dalam dokumen PDF menggunakan Aspose.PDF for Python via .NET.

Skrip contoh menunjukkan beberapa alur kerja anotasi teks:

- anotasi teks bebas
- anotasi sorotan
- anotasi garis bawah
- anotasi bergelombang
- anotasi coret

## Anotasi FreeText

### Tambahkan Anotasi FreeText 

Anotasi teks bebas memungkinkan Anda menempatkan komentar teks yang terlihat langsung pada halaman PDF. Contoh ini menambahkan anotasi teks bebas sederhana ke halaman pertama.

#### Buka PDF sumber

```python
document = ap.Document(infile)
```

#### Buat dan konfigurasikan anotasi teks bebas

```python
free_text_annotation = ap.annotations.FreeTextAnnotation(
    document.pages[1],
    ap.Rectangle(299, 713, 308, 720, True),
    ap.annotations.DefaultAppearance(),
)
free_text_annotation.title = "Aspose User"
free_text_annotation.color = ap.Color.light_green
```

#### Tambahkan anotasi dan simpan PDF

```python
document.pages[1].annotations.append(free_text_annotation)
document.save(outfile)
```

#### Contoh lengkap

```python
def free_text_annotation_add(infile, outfile):
    document = ap.Document(infile)

    free_text_annotation = ap.annotations.FreeTextAnnotation(
        document.pages[1],
        ap.Rectangle(299, 713, 308, 720, True),
        ap.annotations.DefaultAppearance(),
    )
    free_text_annotation.title = "Aspose User"
    free_text_annotation.color = ap.Color.light_green

    document.pages[1].annotations.append(free_text_annotation)
    document.save(outfile)
```

### Dapatkan Anotasi FreeText 

Untuk memeriksa anotasi teks bebas, filter anotasi halaman pertama berdasarkan `FREE_TEXT` ketik dan cetak setiap persegi panjang anotasi.

#### Muat dokumen dan kumpulkan anotasi teks bebas

```python
document = ap.Document(infile)
free_text_annotations = [
    annotation
    for annotation in document.pages[1].annotations
    if annotation.annotation_type == ap.annotations.AnnotationType.FREE_TEXT
]
```

#### Cetak persegi panjang anotasi

```python
for annotation in free_text_annotations:
    print(annotation.rect)
```

#### Contoh lengkap

```python
def free_text_annotation_get(infile, outfile):
    document = ap.Document(infile)
    free_text_annotations = [
        annotation
        for annotation in document.pages[1].annotations
        if annotation.annotation_type == ap.annotations.AnnotationType.FREE_TEXT
    ]

    for annotation in free_text_annotations:
        print(annotation.rect)
```

### Hapus Anotasi FreeText 

Alur kerja ini menghapus semua anotasi teks bebas dari halaman pertama dan menyimpan PDF yang diperbarui.

#### Temukan dan hapus anotasi teks bebas

```python
document = ap.Document(infile)
free_text_annotations = [
    annotation
    for annotation in document.pages[1].annotations
    if annotation.annotation_type == ap.annotations.AnnotationType.FREE_TEXT
]

for annotation in free_text_annotations:
    document.pages[1].annotations.delete(annotation)

document.save(outfile)
```

#### Contoh lengkap

```python
def free_text_annotation_delete(infile, outfile):
    document = ap.Document(infile)
    free_text_annotations = [
        annotation
        for annotation in document.pages[1].annotations
        if annotation.annotation_type == ap.annotations.AnnotationType.FREE_TEXT
    ]

    for annotation in free_text_annotations:
        document.pages[1].annotations.delete(annotation)

    document.save(outfile)
```


## Anotasi Penandaan Teks

### Sorotan Anotasi

#### Tambah Sorotan Teks

Anotasi sorotan menekankan bagian-bagian dokumen tanpa mengubah konten dasar. Contoh ini menambahkan anotasi sorotan ke halaman pertama.

```python
document = ap.Document(infile)

highlight_annotation = ap.annotations.HighlightAnnotation(
    document.pages[1],
    ap.Rectangle(300, 750, 320, 770, True),
)

document.pages[1].annotations.append(highlight_annotation)
document.save(outfile)
```

```python
def text_highlight_annotation_add(infile, outfile):
    document = ap.Document(infile)

    highlight_annotation = ap.annotations.HighlightAnnotation(
        document.pages[1],
        ap.Rectangle(300, 750, 320, 770, True),

    document.save(outfile)
```

#### Dapatkan Sorotan Teks

Untuk memeriksa anotasi sorotan, filter anotasi halaman berdasarkan `HIGHLIGHT` ketik dan cetak persegi panjang mereka.

```python
document = ap.Document(infile)

    annotation
    for annotation in document.pages[1].annotations
    if annotation.annotation_type == ap.annotations.AnnotationType.HIGHLIGHT

for annotation in highlight_annotations:

```

### Anotasi Garis Bawah

#### Tambahkan Anotasi Garis Bawah Teks

Anotasi garis bawah menandai teks dengan garis bawah yang terlihat. Contoh ini menambahkan anotasi garis bawah dasar dan mengatur metadata serta warnanya.

```python
document = ap.Document(infile)

underline_annotation = ap.annotations.UnderlineAnnotation(
    document.pages[1],
    ap.Rectangle(299.988, 713.664, 308.708, 720.769, True),
)
underline_annotation.title = "Aspose User"
underline_annotation.subject = "Inserted Underline 1"
underline_annotation.flags = ap.annotations.AnnotationFlags.PRINT
underline_annotation.color = ap.Color.blue

document.pages[1].annotations.append(underline_annotation)
document.save(outfile)
```

```python
def text_underline_annotation_add(infile, outfile):
    document = ap.Document(infile)

    underline_annotation = ap.annotations.UnderlineAnnotation(
        document.pages[1],
        ap.Rectangle(299.988, 713.664, 308.708, 720.769, True),

    underline_annotation.title = "Aspose User"
    underline_annotation.subject = "Inserted Underline 1"
    underline_annotation.flags = ap.annotations.AnnotationFlags.PRINT
    underline_annotation.color = ap.Color.blue

    document.pages[1].annotations.append(underline_annotation)

```

#### Tambah Teks Garis Bawah Anotasi Ratakan 

Jika Anda ingin garis bawah menjadi bagian dari konten halaman alih-alih tetap menjadi anotasi interaktif, Anda dapat meratakannya setelah menambahkannya.

```python

underline_annotation.color = ap.Color.blue

document.pages[1].annotations.append(underline_annotation)
underline_annotation.flatten()

document.save(outfile)
```

#### Tambahkan Anotasi Garis Bawah Teks dengan Quad Points

Quad points memungkinkan Anda mendefinisikan area yang tepat ditandai untuk anotasi garis bawah. Ini berguna ketika Anda membutuhkan kontrol lebih daripada sekadar persegi panjang.

```python
document = ap.Document(infile)
rect = ap.Rectangle(299.988, 713.664, 308.708, 720.769, True)

underline_annotation = ap.annotations.UnderlineAnnotation(document.pages[1], rect)
underline_annotation.title = "Aspose User"
underline_annotation.subject = "Inserted Underline with Quad Points"
underline_annotation.flags = ap.annotations.AnnotationFlags.PRINT
underline_annotation.color = ap.Color.blue
underline_annotation.quad_points = [
    ap.Point(rect.llx, rect.lly),
    ap.Point(rect.urx, rect.lly),
    ap.Point(rect.urx, rect.ury),
    ap.Point(rect.llx, rect.ury),
]

document.pages[1].annotations.append(underline_annotation)
document.save(outfile)
```

```python
def text_underline_with_quad_points_add(infile, outfile):
    document = ap.Document(infile)
    rect = ap.Rectangle(299.988, 713.664, 308.708, 720.769, True)

    underline_annotation = ap.annotations.UnderlineAnnotation(document.pages[1], rect)
    underline_annotation.title = "Aspose User"
    underline_annotation.subject = "Inserted Underline with Quad Points"
    underline_annotation.flags = ap.annotations.AnnotationFlags.PRINT
    underline_annotation.color = ap.Color.blue
    underline_annotation.quad_points = [
        ap.Point(rect.llx, rect.lly),
        ap.Point(rect.urx, rect.lly),
        ap.Point(rect.urx, rect.ury),
        ap.Point(rect.llx, rect.ury),
    ]

    document.pages[1].annotations.append(underline_annotation)
    document.save(outfile)
```

#### Hapus Anotasi Garis Bawah Teks berdasarkan Judul

```python
document = ap.Document(infile)

underline_annotations = [
```

#### Dapatkan Teks Anotasi Garis Bawah yang Ditandai

Alur kerja ini mengonversi setiap anotasi garis bawah menjadi sebuah `UnderlineAnnotation` objek dan mengekstrak teks yang ditandai.

```python
document = ap.Document(infile)

underline_annotations = [
    annotation
    for annotation in document.pages[1].annotations
    if annotation.annotation_type == ap.annotations.AnnotationType.UNDERLINE
]

for annotation in underline_annotations:
    ua = cast(ap.annotations.UnderlineAnnotation, annotation)
    print(f"Marked text: {ua.get_marked_text()}")
```

```python
def text_underline_marked_text_get(infile, outfile):
    document = ap.Document(infile)

    underline_annotations = [
        annotation
        for annotation in document.pages[1].annotations
        if annotation.annotation_type == ap.annotations.AnnotationType.UNDERLINE
    ]

    for annotation in underline_annotations:
        ua = cast(ap.annotations.UnderlineAnnotation, annotation)
        print(f"Marked text: {ua.get_marked_text()}")
```

#### Dapatkan Teks Anotasi Garis Bawahi Fragmen yang Ditandai

Jika Anda membutuhkan setiap fragmen yang ditandai secara terpisah, Anda dapat mengiterasi melalui koleksi yang dikembalikan oleh `get_marked_text_fragments()`.

```python
document = ap.Document(infile)

underline_annotations = [
    annotation
    for annotation in document.pages[1].annotations
    if annotation.annotation_type == ap.annotations.AnnotationType.UNDERLINE
]

for annotation in underline_annotations:
    ua = cast(ap.annotations.UnderlineAnnotation, annotation)
    for fragment in ua.get_marked_text_fragments():
        print(f"Fragment text: {fragment.text}")
```

```python
def text_underline_marked_fragments_get(infile, outfile):
    document = ap.Document(infile)

    underline_annotations = [
        annotation
        for annotation in document.pages[1].annotations
        if annotation.annotation_type == ap.annotations.AnnotationType.UNDERLINE
    ]

    for annotation in underline_annotations:
        ua = cast(ap.annotations.UnderlineAnnotation, annotation)
        for fragment in ua.get_marked_text_fragments():
            print(f"Fragment text: {fragment.text}")
```


### Anotasi Bergelombang

#### Tambahkan Anotasi Squiggly

Anotasi bergelombang sering digunakan untuk menandai ejaan, tata bahasa, atau area perhatian dalam teks. Contoh ini menambahkan anotasi bergelombang ke halaman pertama.

```python
document = ap.Document(infile)
page = document.pages[1]

squiggly_annotation = ap.annotations.SquigglyAnnotation(
    page,
    ap.Rectangle(67, 317, 261, 459, True),
)
squiggly_annotation.title = "John Smith"
squiggly_annotation.color = ap.Color.blue

page.annotations.append(squiggly_annotation)
document.save(outfile)
```

```python
def text_squiggly_annotation_add(infile, outfile):
```
