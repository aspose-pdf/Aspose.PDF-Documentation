The `PdfFileSecurity` Kelas dalam Aspose.PDF Facades digunakan untuk melindungi dokumen PDF dan mengontrol bagaimana pengguna dapat berinteraksi dengan mereka. Ini mendukung alur kerja keamanan umum seperti enkripsi berbasis kata sandi, dekripsi, rotasi kata sandi, dan manajemen izin untuk mencetak, menyalin, dan mengedit.
