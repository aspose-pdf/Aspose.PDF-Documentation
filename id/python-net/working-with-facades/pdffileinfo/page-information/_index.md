AlternativeHeadline: Mengambil Informasi Halaman PDF dan Offset
Abstract: Halaman PDF dapat berbeda dalam ukuran, rotasi, dan posisi konten internal. Fungsi Get Page Information dan Get Page Offset memberikan pengembang tampilan lengkap tentang tata letak setiap halaman. Get Page Information mengembalikan lebar, tinggi, dan rotasi halaman, sementara Get Page Offset mengambil offset X dan Y dalam inci. Bersama-sama, metode ini memungkinkan penyelarasan tepat teks, gambar, anotasi, dan konten lainnya, mendukung alur kerja otomatis baik satu halaman maupun multi‑halaman.
