## Apa itu File PDF?

PDF dibuat oleh Adobe pada tahun 1990-an untuk mencapai dua hal. Pertama, Anda harus dapat membuka dokumen di perangkat keras atau sistem operasi apa pun, tanpa perlu memiliki aplikasi yang digunakan untuk membuatnya—yang Anda perlukan hanyalah pembaca PDF, dan saat ini kebanyakan browser web sudah memenuhi kebutuhan tersebut. Kedua, di mana pun Anda membuka PDF, tata letak dokumen harus terlihat sama.

Namun, tidak cukup hanya membuka dokumen Anda. Dalam bekerja dengan PDF, Anda akan menghadapi kebutuhan untuk membuat dokumen tersebut kembali, mengeditnya, atau mengkonversinya ke format yang Anda perlukan.
