Abstract: Artikel ini menyediakan tabel komprehensif yang merinci format file yang dapat ditangani oleh Aspose.PDF for Python. Ini menjelaskan kemampuan perpustakaan untuk memuat dan menyimpan berbagai jenis dokumen, memberikan wawasan tentang fleksibilitasnya dalam mengelola format file yang berbeda. Format yang dicakup termasuk tipe yang dikenal luas seperti PDF, EPUB, HTML, dan XML, yang semuanya dapat dimuat dan disimpan oleh perpustakaan. Itu juga mendukung format gambar seperti JPEG, PNG, dan TIFF untuk kedua operasi. Namun, beberapa format seperti CGM, MHT, PCL, PS, XSLFO, dan MD hanya tersedia untuk pemuatan, sementara yang lain seperti XLS, XLSX, PPTX, DOC, DOCX, dan MobiXML hanya tersedia untuk penyimpanan. Tabel ini berfungsi sebagai panduan praktis bagi pengguna yang ingin memahami kompatibilitas format Aspose.PDF for Python, menyoroti baik keunggulan maupun keterbatasannya dalam dukungan format dokumen dan gambar.
