## Overview

Para procesar documentos PDF utilizando [Aspose.PDF para Node.js a través de C++](https://products.aspose.com/pdf/nodejs-cpp/), no se necesitan bibliotecas y programas de terceros.
Requiere tener instalado [Node.js](https://nodejs.org/). Debe tener la versión 16 de Node.js o superior. Recomendamos encarecidamente usar versiones LTS.
Node.js se puede [instalar](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) de diferentes maneras.

## Supported Operating Systems

Todas las versiones compatibles de [Linux](https://en.wikipedia.org/wiki/Linux) y [macOS](https://www.apple.com/macos/).
