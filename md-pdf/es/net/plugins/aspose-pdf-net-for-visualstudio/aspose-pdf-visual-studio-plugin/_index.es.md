## Introducción

Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se utiliza para desarrollar programas informáticos para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio utiliza plataformas de desarrollo de software de Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede producir tanto código nativo como código administrado.

### Aspose.PDF para .NET

Aspose.PDF para .NET es un componente de creación y manipulación de documentos PDF que permite que tus aplicaciones .NET lean, escriban y manipulen documentos PDF existentes sin utilizar Adobe Acrobat.

Para más detalles sobre [Aspose.PDF para .NET API](http://www.aspose.com/.net/Pdf-component.aspx)

### Explora ejemplos de Aspose.PDF para .NET utilizando Visual Studio
### Explora los ejemplos de Aspose.PDF para .NET usando Visual Studio

Aspose Visual Studio Plugin - **Explora los ejemplos de Aspose.PDF para .NET** es una herramienta excelente para descargar y explorar rápidamente los ejemplos de la API Aspose.PDF para .NET. Ahorra mucho tiempo y esfuerzo al proporcionar una opción muy simple para seleccionar, descargar y abrir los proyectos de ejemplo más recientes sin tener que salir de Visual Studio.

Esta versión ofrece las siguientes características:

- Compatible con Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de lanzar desde **Archivo > Aspose** en el menú de Visual Studio
- Ahorra tiempo y reduce la curva de aprendizaje.
- Te permite seleccionar y abrir los últimos ejemplos de la API Aspose.PDF para .NET alojados en Github.com.
- Te permite abrir proyectos de ejemplos en **C#** o **VB.NET**
- Un proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias necesarias configuradas para ofrecerte un entorno listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de la API desde **NuGet** y los ejemplos desde **GitHub**.
- Descarga automáticamente las últimas bibliotecas de API de **NuGet** y ejemplos de **GitHub**.

## Usando

Una vez instalado, puedes lanzar y usar el complemento interactivo como se describe a continuación:
**1.** Abre el complemento seleccionando **Archivo > Aspose** seguido de **Nuevo Proyecto de Ejemplo de Aspose.PDF** en Visual Studio.

![New Aspose.PDF Example Project dialog](aspose-pdf-visual-studio-plugin_3.png)

**2.** La primera vez que abras el complemento verás el diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los últimos ejemplos desde **GitHub**.

![Aspose.PDF Example Downloading dialog](aspose-pdf-visual-studio-plugin_4.png)
![Aspose.PDF Ejemplo de diálogo de descarga](aspose-pdf-visual-studio-plugin_4.png)

**3.** En el diálogo Seleccionar una API para ver ejemplos, selecciona o cambia la ubicación donde se crearán los proyectos de ejemplo y también selecciona o cambia el tipo de proyecto de ejemplo como **C#** o **VB.NET** y haz clic en **Continuar** para abrir el proyecto de ejemplo en Visual Studio.

![Aspose.PDF Ejemplo de diálogo para seleccionar carpeta](aspose-pdf-visual-studio-plugin_1.png)

**4.** Nuevos proyectos de ejemplos según la selección del tipo de proyecto de **C#** o **VB.NET** en Visual Studio.
