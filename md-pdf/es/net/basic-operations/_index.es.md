La sección **Operaciones básicas** describe la primera y más simple operación con documentos PDF en Aspose.PDF para .NET:

- [Crear documento PDF](/pdf/net/create-document/) - este artículo describe cómo crear un archivo PDF usando C#.
- [Abrir documento PDF](/pdf/net/open-pdf-document/) - este artículo describe varias formas de cómo abrir un documento PDF en una aplicación .NET.
- [Guardar documento PDF](/pdf/net/save-pdf-document/) - este artículo describe varias formas de cómo guardar un documento PDF en un archivo, flujo, enviar a la Web o guardar como documento PDF/A (o PDF/X).
