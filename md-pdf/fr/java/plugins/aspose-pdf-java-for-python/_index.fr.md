# Introduction à Aspose.PDF Java pour Python

## JPype

JPype est un effort pour permettre aux programmes Python d'accéder pleinement aux bibliothèques de classes Java. Cela est réalisé non pas en ré-implémentant Python, comme l'ont fait Jython/JPython, mais plutôt en s'interfaçant au niveau natif dans les deux machines virtuelles.

Lisez-en plus sur [readthedocs.org](http://jpype.readthedocs.org/en/latest/userguide.html)

## Aspose.PDF pour Java

Aspose.PDF pour Java est une bibliothèque de classes avancée pour Java qui vous permet d'effectuer une grande gamme de tâches de traitement de documents directement dans vos applications Java.

Aspose.PDF pour Java prend en charge le traitement de documents Word (DOC, DOCX, OOXML, RTF), HTML, OpenDocument, PDF, EPUB, XPS, SWF et tous les formats d'image. Avec Aspose.PDF, vous pouvez générer, modifier et convertir des documents sans utiliser Microsoft Word.

Lisez-en plus sur [aspose.com](https://products.aspose.com/words/java)

## Aspose.PDF Java pour Python

Le projet Aspose.PDF pour Python montre comment différentes tâches peuvent être effectuées en utilisant les API Aspose.PDF Java en Python. Ce projet vise à fournir des exemples utiles pour les développeurs Python qui souhaitent utiliser Aspose.PDF pour Java dans leurs projets Python en utilisant [JPype](http://jpype.readthedocs.org/en/latest/userguide.html).

## Exigences Système et Plates-formes Supportées

### Exigences Système

Voici les exigences système pour utiliser Aspose.PDF Java pour Python :

- JPype
- Composant Aspose.PDF téléchargé.

### Plates-formes Supportées

Voici les plates-formes supportées :
