{{% alert color="primary" %}}

La bibliothèque Aspose.PDF pour Java ne peut pas prendre en charge les paramètres régionaux avec une combinaison spécifique langue-pays (comme "en-KR"). Mais une fonctionnalité pour définir des paramètres régionaux classiques pour Aspose.PDF est présente dans l'API et peut être utilisée en appelant la méthode com.aspose.pdf.LocaleOptions.setLocale().
