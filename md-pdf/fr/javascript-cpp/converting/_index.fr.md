Le PDF a été développé pour fournir une norme de présentation des documents et autres documents de référence dans un format indépendant des logiciels applicatifs, du matériel et du système d'exploitation. Le contenu des fichiers PDF n'est pas limité au texte, il peut s'agir de liens hypertexte, d'images, de boutons cliquables et de formulaires, de signatures électroniques, de filigranes, et plus encore. Par conséquent, il est souvent nécessaire de convertir les fichiers PDF dans un autre format afin de modifier ou de changer leur contenu. 
**Notre outil Aspose.PDF pour JavaScript via C++** vous permet de convertir vos documents PDF avec succès, rapidement et facilement vers les formats les plus populaires.
 Pour une liste complète des formats pris en charge, consultez la section [Formats de fichiers pris en charge par Aspose.PDF](https://docs.aspose.com/pdf/javascript-cpp/supported-file-formats/).

Aspose.PDF pour JavaScript via C++* permet de convertir des documents PDF en divers formats. De plus, vous pouvez vérifier la qualité de la conversion Aspose.PDF et voir les résultats en ligne avec l'application de conversion Aspose.PDF. Découvrez les sections de conversion de documents avec des extraits de code.

## Comment utiliser JavaScript via C++ pour la conversion

Les documents Word sont les plus polyvalents et modifiables possibles. Convertir un PDF en DOC manuellement est une tâche très chronophage. Dans cet article, vous apprendrez comment convertir un PDF en DOC et DOCX en Javascript.

- [Convertir un PDF en DOC](/pdf/javascript-cpp/convert-pdf-to-doc/) - vous pouvez convertir votre document PDF en format Word avec Javascript.

Les formats de nombres sont nécessaires non seulement pour rendre les données du tableau plus faciles à lire, mais aussi pour rendre le tableau plus facile à utiliser. Of course, si vous avez besoin de convertir de telles données d'un document PDF au format Excel, utilisez notre Aspose.PDF pour Javascript.

- [Convertir PDF en Microsoft XLSX](/pdf/javascript-cpp/convert-pdf-to-xlsx/) - cette section décrit comment convertir un document PDF en XLSX, et CSV.

Le format PowerPoint est utilisé pour créer diverses présentations. Les fichiers PPT contiennent un grand nombre de diapositives ou de pages contenant diverses informations.

- [Convertir PDF en Microsoft PowerPoint](/pdf/javascript-cpp/convert-pdf-to-powerpoint/) - ici, nous parlons de la conversion de PDF en PowerPoint en suivant le processus de conversion.

Le langage HTML est un langage de description de document hypertexte, un langage standard pour créer des pages web. Avec Aspose.PDF pour Javascript, vous pouvez facilement convertir des documents HTML et vice versa.

Il existe de nombreux formats d'image qui doivent être convertis en PDF pour différentes fins. Aspose.PDF pour JavaScript permet les formats d'images les plus populaires.

- [Convertir PDF en divers formats d'Images](/pdf/javascript-cpp/convert-pdf-to-images-format/) - convertir les pages PDF en images aux formats JPEG, PNG, SVG, et autres formats.
This section comprend des formats tels que : EPUB, XPS, TeX, Texte et PDF en niveaux de gris.

- [Convertir un fichier PDF en d'autres formats](/pdf/javascript-cpp/convert-pdf-to-other-files/) - ce sujet décrit la manière de convertir un document PDF en divers formats

PDF/A est une version du PDF conçue pour l'archivage à long terme de documents électroniques. Honnêtement, extérieurement, il est très difficile de déterminer si c'est un PDF ou un PDF/A. Pour vérifier ce fichier, des validateurs sont utilisés. Consultez les articles suivants pour une conversion de qualité de PDF en PDF/A et vice versa.

- [Convertir PDF en formats PDF/A](/pdf/javascript-cpp/convert-pdf-to-pdfa/) - La boîte à outils Javascript d'Aspose.PDF offre un moyen facile de convertir PDF en PDF/A.

- [Convertir PDF/A en format PDF](/pdf/javascript-cpp/convert-pdfa-to-pdf/) - convertir PDF/A en format PDF avec Javascript facilement, rapidement et de haute qualité.
