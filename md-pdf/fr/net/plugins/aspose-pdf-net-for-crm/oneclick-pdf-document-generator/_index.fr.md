## Introduction

Le générateur de documents PDF OneClick d'Aspose .NET est un module complémentaire open source utilisé pour générer et exporter des PDF à partir du CRM. Vous pouvez concevoir un modèle dans Word, l'attacher dans le CRM et générer le document comme vous le souhaitez. Vous pouvez utiliser le générateur de documents PDF OneClick avec les entités Compte, Contact, Prospect, Opportunité et Devis. Il offre l'option d'attacher le document généré à l'enregistrement lui-même et vous pouvez également télécharger le document. Ce module complémentaire peut également résoudre les champs de jeu de options et de recherche. De plus, il vous permet de générer le PDF depuis le bouton de la barre supérieure ainsi que via un bouton de clic unique sur le formulaire.

Le générateur de documents PDF OneClick d'Aspose .NET peut être utilisé avec CRM on-premises et CRM en ligne.

Les principales caractéristiques de ce module complémentaire sont :

- Créez un modèle dans Word et attachez-le dans CRM.
- Créer un modèle dans Word et l'attacher dans le CRM.
- Permet d'ajouter des Optionset ainsi que des recherches.
- Fonctionne avec Compte, Contact, Prospect, Opportunité et Devis.
- Un bouton d'un clic qui peut générer un document.
- Les documents générés peuvent être téléchargés ainsi qu'attachés avec l'enregistrement de l'entité même.
- Un bouton de ruban est également fourni pour choisir le modèle et générer le document.

Cet add-on fonctionnera avec toutes les versions de **Microsoft Dynamics CRM 2015 et CRM Online**.

## Installation ou Désinstallation

### Comment Installer

#### Importation de la Solution

- Téléchargez le fichier de solution. (Géré si vous avez l'intention de le désinstaller)
- Ouvrez CRM et allez à Importer Solution.
- Cliquez sur Parcourir et sélectionnez le fichier de solution téléchargé puis cliquez sur suivant.
- Cliquez sur Suivant et attendez que la solution soit importée.
- Cliquez sur fermer lorsque la solution est importée avec succès.

#### Configuration de la Page Web

- Téléchargez le code source pour le site web.
- Copiez sur le serveur. (Recommandé sous inetpub)
- Ouvrez IIS. (Sur le serveur où vous avez l'intention de déployer le site web, accessible depuis le serveur CRM)
- Créez un nouveau site Web.
- Choisissez le chemin physique et configurez le site web.
- Parcourez pour vérifier s'il est accessible.
- Ouvrez CRM et ajoutez un nouvel enregistrement sous Aspose>> Entité de Configuration Aspose et spécifiez le Nom et l'URL de la page Web déployée comme ci-dessous.

![Générateur de documents PDF OneClick](oneclick-pdf-document-generator_1.png)
![Générateur de documents PDF OneClick](oneclick-pdf-document-generator_1.png)

### Comment désinstaller

- Supprimez le site web depuis IIS.
- Allez dans la solution et sélectionnez "Générateur de documents PDF OneClick".
- Appuyez sur le bouton Supprimer. Appuyez sur OK pour confirmer.
