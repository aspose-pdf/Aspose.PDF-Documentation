A classe [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form) permite que você exporte dados para um arquivo XML a partir do arquivo PDF usando o método exportXml. Para exportar dados para XML, você precisa criar um objeto da classe [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form), abrir o formulário PDF de origem usando o método bindPDF e então chamar o método exportXml usando o objeto OutputStream. Finalmente, você pode fechar o objeto OutputStream e dispensar o objeto Form. O snippet de código a seguir mostra como exportar dados para um arquivo XML.
