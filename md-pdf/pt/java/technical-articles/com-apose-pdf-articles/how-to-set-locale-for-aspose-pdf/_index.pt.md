{{% alert color="primary" %}}

A biblioteca Aspose.PDF para Java não pode suportar localidades com combinação específica de linguagem-país (como "en-KR"). Mas um recurso para definir uma localidade clássica para Aspose.PDF está presente no API e pode ser usado chamando o método com.aspose.pdf.LocaleOptions.setLocale().
