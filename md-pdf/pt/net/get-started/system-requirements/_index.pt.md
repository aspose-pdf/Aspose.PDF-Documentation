## Visão Geral
Para produzir e manipular documentos PDF, a máquina em que o Aspose.PDF para .NET é executado não precisa ter o Adobe Acrobat instalado: o Aspose.PDF para .NET é um motor de criação de documentos PDF. No entanto, para visualizar os documentos PDF produzidos pelo Aspose.PDF para .NET, a máquina precisa ter pelo menos um Adobe Acrobat Reader instalado. Ele pode ser baixado gratuitamente.

## Ambientes de Desenvolvimento
Você pode usar o Aspose.PDF para .NET para desenvolver aplicações em qualquer ambiente de desenvolvimento que vise a plataforma .NET, mas os seguintes ambientes são explicitamente suportados:

- Microsoft Visual Studio 2010

- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
