## Baixar Bibliotecas Necessárias

Baixe as bibliotecas necessárias mencionadas abaixo. Estas são necessárias para executar exemplos do Aspose.PDF para Python via Java.

- Aspose: [Aspose.PDF for Python via Java Component](https://releases.aspose.com/pdf/pythonjava/)
- ou PyPI: [O Índice de Pacotes Python](https://pypi.org/project/aspose-pdf-for-python-via-java/)
- [JPype](https://pypi.python.org/pypi/JPype1)
- JDK 1.8+

### Como configurar o código-fonte

Por favor, siga estes passos simples para abrir e estender o código-fonte enquanto utiliza:

1. Instale apenas o Python 3.7+ do [Site Oficial](https://www.python.org/downloads)
2. Instale o JPype 1.5.0+ do [Sourceforge](http://sourceforge.net/projects/jpype/files/JPype/)
3. Instale o Java JDK 1.8+ dos [Downloads Java](https://www.oracle.com/java/technologies/downloads/)
4. Baixe Aspose.PDF para Python via Java API: [Aspose.PDF para Python via Componente Java](https://releases.aspose.com/pdf/pythonjava/)
5. Ou instale Aspose.PDF para Python via Java API usando pip: [PyPI](https://pypi.org/project/aspose-pdf-for-python-via-java/)
