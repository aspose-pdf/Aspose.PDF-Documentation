## Отправка PDF в браузер для скачивания

Иногда при разработке приложения ASP.NET возникает необходимость отправить файл(ы) PDF в веб-браузер(ы) для скачивания без их физического сохранения. Для этого можно сохранить документ PDF в объект MemoryStream после его создания и передать байты из этого MemoryStream в объект Response. Это заставит браузер скачать сгенерированный документ PDF.

## Извлечение встроенных файлов из файла PDF

Aspose.PDF выделяется, когда речь идет о расширенных возможностях работы с файлами формата PDF. Он извлекает встроенные файлы гораздо лучше, чем другие инструменты, предлагающие эту функцию.
