## Aspose.PDF for Reporting Services에 오신 것을 환영합니다

Microsoft SQL Server Reporting Services는 많은 조직이 가지고 있는 요구를 충족합니다: 비즈니스 인텔리전스 및 보고 솔루션을 구축해야 하는 필요성. 지금까지 개발자는 보고서를 애플리케이션에 포함시켜야 했거나, 조직은 비싸고 때로는 문제가 있는 외부 보고 솔루션을 구매해야 했습니다. 이제 Microsoft SQL Server Reporting Services는 엔터프라이즈 전반에 걸쳐 보고서를 배포할 수 있는 완전한 솔루션을 제공합니다; 이를 통해 기업은 더 나은 결정을 더 빠르게 내릴 수 있습니다.

{{% /alert %}}
