## نظرة عامة

لإنتاج مستندات PDF، لا تحتاج الآلة لتشغيل [Aspose.PDF لنظام Android عبر Java](https://products.aspose.com/pdf/android-java/) إلى تثبيت [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) حيث أن Aspose.PDF لنظام Java هو محرك إنشاء مستندات PDF بحد ذاته. ومع ذلك، لعرض مستندات PDF المنتجة بواسطة Aspose.PDF لنظام Android عبر Java تحتاج الآلة على الأقل إلى [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). يمكن [تحميله](https://www.adobe.com/acrobat/pdf-reader.html) مجانًا.
