## 概述

为了生成 PDF 文档，运行 [Aspose.PDF for Java](https://products.aspose.com/pdf/java/) 的机器不需要安装 [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html)，因为 Aspose.PDF for Java 本身就是一个 PDF 文档创建引擎。然而，为了查看由 Aspose.PDF for Java 生成的 PDF 文档，机器至少需要安装一个 [Adobe Acrobat Reader](https://www.adobe.com/acrobat/pdf-reader.html)。它可以免费下载安装。

## 支持的操作系统

Aspose.PDF for Java 支持任何运行 Java 运行时的 32 位或 64 位操作系统，包括但不限于：

### Windows

- Microsoft Windows 2000 专业版 (x64, x86)
- Microsoft Windows 2000 服务器版 (x64, x86)
- Microsoft Windows 2003 服务器版 (x64, x86)
- Microsoft Windows 2008 服务器版 (x64, x86)
- Microsoft Windows 2012 服务器版 (x64, x86)
- Microsoft Windows 2012 R2 服务器版 (x64, x86)
- Microsoft Windows 2016 服务器版 (x64, x86)
- Microsoft Windows 2019 服务器版 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Azure

### Linux

- Linux (Ubuntu, CentOS 和其他)

### Mac

- Mac OS X

## 支持的 Java 版本

Aspose.PDF for Java 支持以下 Java 版本：

- J2SE 8.0 (1.8) 或更高版本（例如 Java 10）
- 支持在 IBM i (Iseries 或 As/400) 上的 Aspose.Pdf for Java

您可以使用 Aspose.PDF for Java 在任何面向 Java 平台的开发环境中开发应用程序，但以下环境是明确支持的：

- Microsoft Visual Studio 2003
- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2011
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
