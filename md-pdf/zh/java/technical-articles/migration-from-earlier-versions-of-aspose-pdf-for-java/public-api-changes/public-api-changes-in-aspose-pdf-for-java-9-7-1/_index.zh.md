{{% alert color="primary" %}}

此页面列出了 [Aspose.PDF for Java 9.7.1](http://www.aspose.com/community/files/72/java-components/aspose.pdf-for-java/entry600386.aspx) 中引入的公共 API 更改。它不仅包括新的和已废弃的公共方法，还包括对 Aspose.PDF for Java 后台行为的任何更改的描述，这些更改可能会影响现有代码。任何可能被视为回归并修改现有行为的行为尤为重要，并在此处记录。

{{% /alert %}}

在 **com.aspose.facades.Form** 类中添加了一个名为 **importXml(String inputXml)** 的方法，以支持将 XML 数据导入 PDF 表单。

以下方法在 **com.aspose.pdf.generator.legacyxmlmodel.Heading** 类中更改

- isAutoSequence(boolean) 更改为 setAutoSequence(boolean)

- isInList(boolean) 更改为 setInList(boolean)

## 新功能实现

- [EPUB 转换为 PDF](http://www.aspose.com/docs/display/pdfjava/Convert+EPUB+File+to+PDF+Format)。
- [将不可搜索的 PDF 转换为可搜索的 PDF 文档](http://www.aspose.com/docs/display/pdfjava/Converting+non+searchable+PDF+to+searchable+PDF+document)。
