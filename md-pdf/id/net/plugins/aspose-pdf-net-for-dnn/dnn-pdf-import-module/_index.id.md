## Introduction

Modul Aspose .NET PDF Import untuk DNN memungkinkan pengembang untuk mendapatkan/membaca isi dokumen PDF tanpa memerlukan perangkat lunak lain seperti Adobe Acrobat atau pembaca PDF.

### Module Features

Modul ini menunjukkan fitur impor yang kuat yang disediakan oleh [Aspose.PDF](http://www.aspose.com/pdf-component-suite.aspx). Ini menambahkan kontrol penjelajah file yang sederhana, pemilihan Pane Konten, dan tombol **Import dari PDF** di halaman tempat modul ditambahkan. Saat mengklik tombol, isi dokumen ditampilkan di layar segera.

## Menginstal

Setelah diunduh, silakan ikuti langkah-langkah ini untuk menginstal modul ke situs DNN Anda:

1. Masuk ke situs Anda sebagai akun tingkat **Host** atau Supersuser lainnya.
1. Navigasikan ke menu Host dan pilih **Ekstensi**.
1.
1.
1. Seperti yang diarahkan, jelajahi lokasi file zip yang telah diunduh, pilih file tersebut kemudian klik **Buka**.
1. Klik **Lanjut**, terima lisensi, lanjutkan proses instalasi. Setelah selesai klik tombol **Kembali**.

Silakan periksa [video instalasi modul ini](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) dari DNN untuk detail lebih lanjut

**Catatan:** Jika Anda mendapatkan kesalahan saat mengunggah modul ini karena batasan maxRequestLength di konfigurasi web DNN Anda. Buka web.config dan perbarui maxRequestLength menjadi 20MB dengan mengatur **maxRequestLength=”20480″** dan coba unggah modul lagi.

## Menggunakan

Setelah Anda berhasil menginstal Aspose .NET PDF Import untuk modul DNN, sangat mudah untuk mulai menggunakannya di situs web Anda. Silakan ikuti langkah-langkah sederhana ini untuk memulai

1. Pastikan Anda telah masuk ke DNN sebagai akun tingkat Host atau Admin
1. Navigasi ke halaman di mana Anda ingin menambahkan modul Import
1.
1. Dari daftar pilih **Aspose .NET PDF Import for DNN** dan seret ke tempat yang Anda inginkan di halaman.

Anda telah berhasil menambahkan modul Aspose .NET PDF Import for DNN ke halaman Anda. Sebuah kotak pencarian file dan sebuah tombol bertuliskan **Import from PDF** kini akan muncul di halaman bersama dengan dropdown pemilihan Content Pane. Siapa saja dapat memilih file PDF dan mengklik tombol **Import from PDF** untuk menampilkan isi dokumen yang dipilih di halaman.
