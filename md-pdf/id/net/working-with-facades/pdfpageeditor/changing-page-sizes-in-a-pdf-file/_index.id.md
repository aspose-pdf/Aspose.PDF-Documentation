## Rincian

Kelas [PdfPageEditor](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor) dalam [namespace Aspose.Pdf.Facades](https://docs-qa.aspose.com/display/pdftemp/Aspose.Pdf.Facades+namespace) berisi properti bernama [PageSize](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor/properties/pagesize) yang dapat digunakan untuk mengubah ukuran halaman dari satu halaman individu atau beberapa halaman sekaligus. Properti Pages dapat digunakan untuk menetapkan nomor halaman yang memerlukan ukuran halaman baru. Kelas PageSize berisi daftar berbagai ukuran halaman sebagai anggotanya. Salah satu anggota dari kelas ini dapat ditetapkan ke properti PageSize dari kelas [PdfPageEditor](https://reference.aspose.com/pdf/net/aspose.pdf.facades/pdfpageeditor). Anda juga dapat mendapatkan ukuran halaman dari halaman mana pun menggunakan metode GetPageSize dan memasukkan nomor halaman.
