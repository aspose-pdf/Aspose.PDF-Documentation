{{% alert color="primary" %}}

Anda dapat mendapatkan pengantar tentang PDF/A (PDF yang dapat diarsipkan) Conformance dalam dokumentasi Aspose.PDF.

Jika Anda ingin membuat dokumen PDF/A, tambahkan parameter laporan berikut.

{{% /alert %}}


{{% alert color="primary" %}}

**Nama Parameter**: PdfConformance  
**Tipe Data**: String  
**Nilai yang didukung**: PdfA1A, PdfA1B, PdfA2A, PdfA2B, PdfA2U, PdfA3A, PdfA3B, PdfA3U  

**Contoh**
{{< highlight csharp >}}

<Render>
â€¦
    <Extension Name="APPDF" Type="Aspose.Pdf.ReportingServices.Renderer,Aspose.Pdf.ReportingServices">
    <Configuration>
    <PdfConformance>PdfA1A</PdfConformance>
    </Configuration>
    </Extension>
</Render>
{{< /highlight >}}
