{{% alert color="primary" %}}

Esta página lista as alterações na API pública introduzidas no [Aspose.PDF para Java 10.4.1](http://www.aspose.com/community/files/72/java-components/aspose.pdf-for-java/entry641525.aspx). Inclui não apenas novos métodos públicos e métodos obsoletos, mas também uma descrição de quaisquer alterações no comportamento nos bastidores no Aspose.PDF para Java que podem afetar o código existente. Qualquer comportamento introduzido que possa ser visto como uma regressão e modifique o comportamento existente é especialmente importante e está documentado aqui.

{{% /alert %}}

**Alterações na API Pública e Incompatibilidades Retroativas**

Alterações na classe **com.aspose.pdf.IconFit**:

- public boolean getSpreadOnBorder() -> public boolean isSpreadOnBorder() 

Alterações na classe **com.aspose.pdf.Measure**:

- public boolean getForceDenominator() -> public boolean isForceDenominator()

Alterações na classe **com.aspose.pdf.RedactionAnnotation**:

- public boolean getRepeat() -> public boolean isRepeat()

Alterações na classe **com.aspose.pdf.SaveOptions**:

- public boolean getCloseResponse() -> public boolean isCloseResponse()

Alterações na classe **com.aspose.pdf.Signature**:

- public boolean getShowProperties() -> public boolean isShowProperties()

Alterações na classe **com.aspose.pdf.TextParagraph**:

- public boolean getJustify() -> public boolean isJustify()

Alterações na classe **com.aspose.pdf.TextStamp**:

- public boolean getWordWrap() -> public boolean isWordWrap()
- public boolean getJustify() -> public boolean isJustify()
- public boolean getScale() -> public boolean isScale()

Alterações na classe **com.aspose.pdf.TextState**:

- public boolean getUnderline() -> public  boolean isUnderline()
