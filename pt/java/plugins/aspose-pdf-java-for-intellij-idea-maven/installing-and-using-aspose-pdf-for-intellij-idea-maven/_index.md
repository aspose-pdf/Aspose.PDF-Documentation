## Instalando

### Como Instalar Aspose.PDF Java para IntelliJ IDEA (Maven)

Os passos abaixo demonstram como instalar o plugin **Aspose.PDF Java para IntelliJ IDEA (Maven)**.

O plugin está listado no [site oficial de plugins](https://plugins.jetbrains.com/plugin/7974-aspose-pdf-java-for-intellij-idea-maven/) do IntelliJ IDEA CE - repositório de plugins como **Aspose.PDF Java para IntelliJ IDEA Maven**, portanto, não é necessário baixá-lo manualmente e instalá-lo a partir do disco, mas essa opção extra ainda existe no IntelliJ IDEA.

O plugin **Aspose.PDF Java para IntelliJ IDEA (Maven)** pode ser facilmente instalado (sem download manual) de dentro do IntelliJ IDEA. Para fazê-lo:

1. Selecione **Configurações** no menu **Arquivo**.
2. Clique em **Plugins**, depois navegue pelos repositórios no IntelliJ IDEA.

![todo:image_alt_text](https://i.imgur.com/NDZtoKw.jpg)

## Utilizando

### Assistente de Projeto Maven Aspose.PDF

Para criar um projeto Maven Aspose.PDF para usar o [Aspose.PDF for Java API](https://products.aspose.com/pdf/java/) no IntelliJ IDEA, você precisa ter conexão com a internet.

Depois, siga os passos abaixo:

1. Selecione **Novo Projeto**.
2. Selecione **Projeto Maven Aspose.PDF**.
3. Clique em **Próximo**.

![todo:image_alt_text](https://i.imgur.com/hd2mYnW.jpg)


4. Forneça um **GroupId, ArtifactId** e Versão para o seu projeto maven e clique em Avançar.

![todo:image_alt_text](https://i.imgur.com/aNlx0Qe.jpg)

5. Você pode selecionar a caixa de seleção "Também baixar código-fonte de exemplos" para baixar os exemplos de uso da API (como mostrado na imagem acima)

6. Forneça um **Nome do Projeto** e **Localização**, assim como quando cria qualquer aplicação Java

![todo:image_alt_text](https://i.imgur.com/8VEeFun.jpg)

7. Clique em **Concluir**.

Isso irá recuperar a referência da última dependência maven do [Aspose.PDF para Java API](https://products.aspose.com/pdf/java/) do Repositório Maven da Aspose Cloud e configurá-la no pom.xml Se você selecionou a caixa de seleção "Também baixar código-fonte de exemplos". O download dos exemplos também começará do [Repositório GitHub Aspose.PDF para Java API](https://goo.gl/cPAbJM)

![todo:image_alt_text](https://i.imgur.com/bG2mdam.jpg)

![todo:image_alt_text](https://i.imgur.com/aNYhLl5.jpg)

Parabéns !
 Você criou com sucesso o projeto Maven Aspose.PDF para usar o [Aspose.PDF para Java API](https://products.aspose.com/pdf/java/).

![todo:texto_alt_da_imagem](https://i.imgur.com/UJlSbuM.jpg)

O projeto Maven do Aspose.PDF para Java API criado está pronto para ser modificado de acordo com os requisitos do seu projeto.

Se você optou por baixar [Códigos Fonte de Exemplo](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples).

### Assistente de Exemplos Aspose.PDF

Para importar/copiar [Códigos Fonte de Exemplo](https://goo.gl/cPAbJM) no projeto usando o **Assistente de Exemplos Aspose.PDF**, você precisa primeiro baixar os exemplos optando por "Também Baixar Código Fonte dos Exemplos" no Assistente de Projeto Maven Aspose.PDF.

Em seguida, siga os passos abaixo no IntelliJ IDEA:

1. Selecione **Arquivo** > **Novo -** Como mostrado abaixo:

![todo:texto_alt_da_imagem](https://i.imgur.com/N8tT9Q0.jpg)

-OU- **Clique com o botão direito** (na janela do projeto) => **Novo** - Como mostrado abaixo:

![todo:texto_alt_da_imagem](https://i.imgur.com/aUBWkhp.jpg)
2. Selecione a opção **Exemplo Aspose.PDF** no menu - Conforme mostrado abaixo:

![todo:image_alt_text](https://i.imgur.com/hvH2V0m.jpg)

O assistente irá atualizar/verificar os últimos [Exemplos de API Aspose.PDF para Java](https://goo.gl/cPAbJM) disponíveis

![todo:image_alt_text](https://i.imgur.com/5PZwsuq.jpg)

3. O diálogo do **assistente de Exemplos Aspose.PDF** aparecerá, onde você pode selecionar qualquer categoria de exemplos para criar/copiar os exemplos em seu projeto
Selecione qualquer categoria de exemplos e clique em **Criar**:

![todo:image_alt_text](https://i.imgur.com/l2oGUBW.jpg)

Isso criará/copiará os exemplos da categoria selecionada no projeto sob o pacote **com.aspose.pdf.examples**.
Além disso, quaisquer recursos necessários para a execução dos exemplos serão copiados para a pasta **src/main/resources** - Conforme mostrado abaixo:

![todo:image_alt_text](https://i.imgur.com/8GQL3Yb.jpg)

Parabéns! Você realizou com sucesso o seguinte:

1. Você criou o Projeto Maven Aspose.PDF usando o Assistente de Projeto Maven Aspose.PDF,

2. Você baixou [Aspose.PDF para Java API - Exemplos de Códigos Fonte](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)  
3. Você criou / copiou Exemplos de Códigos Fonte para o seu projeto usando o Assistente de Exemplos Aspose.PDF.
