Para converter um documento PDF para o formato DOC ou DOCX usando **Aspose.PDF Java for Python**, simplesmente invoque o módulo **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "Documento foi convertido com sucesso"
```
