## Como converter arquivo SVG para formato PDF em Python

Para converter arquivo SVG para formato PDF usando **Aspose.PDF Java para Python**, simplesmente invoque o módulo **SvgToPdf**.

Código Python:

```python
options = self.SvgLoadOptions();
doc=self.Document()
pdf = self.Document()
pdf=self.dataDir +'input1.pdf'
Salve a saída no formato XLS
doc.save(self.dataDir + "SVG1.pdf");

```
