Para obter a contagem de páginas de um documento Pdf usando **Aspose.PDF Java para Python**, basta invocar a classe **GetNumberOfPages**.

```Python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "Contagem de Páginas:" . page_count

```
