## Introdução

### Eclipse IDE

Eclipse IDE é um famoso Ambiente de Desenvolvimento Integrado (IDE) para Java. O IDE é definitivamente o produto mais conhecido do projeto Open Source Eclipse. Hoje, é o ambiente de desenvolvimento líder para Java com uma participação de mercado de aproximadamente 60%.

O Eclipse IDE pode ser estendido com componentes de software adicionais. O Eclipse chama esses componentes de software de Plug-ins. Vários projetos de código aberto e empresas estenderam o Eclipse IDE ou criaram aplicações independentes (Eclipse RCP) em cima da estrutura Eclipse.

### Aspose.PDF para Java

[Aspose.PDF para Java](https://products.aspose.com/pdf/java/) é uma API robusta de criação de documentos PDF que habilita suas aplicações Java a ler, escrever e manipular documentos PDF sem usar o Adobe Acrobat.

Aspose.PDF para Java oferece uma incrível gama de funcionalidades, incluindo opções de compressão de PDF, criação e manipulação de tabelas, suporte a gráficos, funções de imagem, ampla funcionalidade de hiperlink, controles de segurança estendidos e manipulação de fontes personalizadas.

### Aspose.PDF Java (Maven) para Eclipse

- Aspose.PDF Java (Maven) para Eclipse é um plugin para **Eclipse IDE** da **Aspose.** Este Plugin é destinado a desenvolvedores que utilizam a plataforma Maven para desenvolvimentos em Java e desejam usar o Aspose.PDF para Java em seus projetos. O Plugin permite criar projetos Maven para usar a API Aspose.PDF para Java e também baixar [Exemplos de Código](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) da API.

**ASSISTENTES**:
O plugin contém dois assistentes

**Projeto Maven Aspose.PDF (assistente)**

- Este assistente de Novo Projeto permite que desenvolvedores criem um projeto **Maven** para usar o Aspose.PDF para Java em Novo -> Projeto -> Maven -> Projeto Maven Aspose.PDF.

- A referência da dependência Maven da API Aspose.PDF para Java é automaticamente buscada do [Repositório Maven da Aspose Cloud](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) e é adicionada no pom.xml.
- O projeto criado sempre conterá a versão mais recente disponível da dependência **Maven** para a API Aspose.PDF para Java.
- As etapas do assistente também apresentam a opção de baixar [Exemplos de Código](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) para usar a API Aspose.PDF para Java.
Exemplo de Código Aspose.PDF (assistente)

- Este assistente de Novo Arquivo permite que você copie os [Exemplos de Código](https://github.com/aspose-pdf/Aspose.Pdf-for-Java) baixados para o seu projeto para usar o Aspose.PDF para Java em Novo -> Outros -> Java -> Exemplo de Código Aspose.PDF.
- Amostras disponíveis são exibidas em formato de árvore, de onde o usuário pode selecioná-las categoricamente.
- Todos os exemplos dentro da categoria selecionada serão copiados para a pasta do pacote do projeto "**com.aspose.pdf.examples**" junto com os recursos necessários dentro da pasta "**src/main/resources**" necessários para executar os exemplos.
- Exemplos de Código da API Aspose.PDF para Java são destinados a demonstrar as várias funções da API.

- O assistente também procurará e atualizará novos Exemplos de Código disponíveis) do repositório de exemplos do Aspose.PDF para Java.

## Requisitos do Sistema e Plataformas Suportadas

### Requisitos do Sistema

- **Memória do Sistema:** 2 GB ou mais (Recomendado)
- **SO:** Qualquer sistema operacional que suporte a JVM (Máquina Virtual Java)
- **Conexão com a Internet:** 2 MB ou mais rápido (Recomendado)

### Plataformas Suportadas

- Eclipse Mars.1 (4.5.1) - Recomendado
- Eclipse Juno ou posterior.

## Baixando

### Baixar Eclipse IDE

Você precisará primeiro instalar o Eclipse IDE antes de baixar o plugin Aspose.PDF Java (Maven) para Eclipse.

Para baixar o Eclipse IDE

1. Vá para [https://eclipse.org](https://eclipse.org/).
1. Baixe e instale o Eclipse IDE recomendado para desenvolvedores Java SE / EE.

### Baixar Aspose.PDF Java (Maven) para Eclipse

A seguir estão três métodos recomendados para o download e instalação bem-sucedidos do plugin Aspose.PDF Java (Maven) para Eclipse:

- Instalação por arrastar e soltar do [Eclipse Marketplace](https://marketplace.eclipse.org/content/asposepdf-java-maven-eclipse) para o seu espaço de trabalho do Eclipse.

- Ou vá para **Ajuda** > **Instalar Novo Software...** > Digite o seguinte URL do site de atualização em **Trabalhar com
Então selecione "Aspose.PDF Java (Maven) for Eclipse" e **Concluir**. Aceite o Contrato de Licença e Instale o plugin.

### Como aplicar a Licença Aspose?

Este Plugin usa uma versão de avaliação do Aspose.PDF. Quando você estiver satisfeito com a sua avaliação, você pode comprar uma licença no [site da Aspose](https://purchase.aspose.com/buy). Para remover a mensagem de avaliação e limitações de recursos, uma licença do produto deve ser aplicada. Você receberá um arquivo de licença após ter comprado o produto. Por favor, siga os passos abaixo para aplicar a licença

- Certifique-se de que o arquivo de licença seja nomeado como Aspose.PDF.Java.lic
- Coloque o arquivo **Aspose.PDF.Java.lic** na pasta que contém o Aspose.PDF.jar
- Use o seguinte código para ativar a licença:

{{< highlight java >}}

License license = new License();

license.setLicense("Aspose.PDF.Java.lic");

{{< /highlight >}}

### Como configurar o código-fonte do Aspose.PDF Java (Maven) para Eclipse

Os passos simples abaixo levarão suavemente à configuração bem-sucedida do código-fonte do plugin **"Aspose.PDF Java (Maven) para Eclipse"** no Eclipse IDE.

1. Baixe / Clone o código fonte.
1. Escolha **Arquivo** > Importar > Geral > Projetos Existentes no Espaço de Trabalho
1. Navegue até o código fonte do projeto mais recente que você baixou
1. Selecione o Projeto Eclipse que você deseja importar
1. Clique em Concluir
1. O código do plugin Aspose.PDF Java para Eclipse agora está pronto para ser aprimorado.
