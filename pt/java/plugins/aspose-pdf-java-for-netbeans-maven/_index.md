## Introdução

### NetBeans IDE

O NetBeans é uma IDE Java completa, escrita totalmente em Java. É principalmente destinada ao desenvolvimento em Java, mas também suporta outras linguagens, em particular, PHP, C/C++ e HTML5.

A IDE NetBeans permite que você desenvolva rápida e facilmente aplicativos Java para desktop, móveis e web, bem como aplicativos HTML5 com HTML, JavaScript e CSS. É gratuita e de código aberto e possui uma grande comunidade de usuários e desenvolvedores em todo o mundo.

A equipe do NetBeans apoia ativamente o produto e busca sugestões de recursos da comunidade em geral. Cada lançamento é precedido por um período para testes e feedback da Comunidade.

### Aspose.PDF para Java

[Aspose.PDF para Java](https://products.aspose.com/pdf/java/) é uma API robusta de criação de documentos PDF que permite que suas aplicações Java leiam, escrevam e manipulem documentos PDF sem usar o Adobe Acrobat.

Aspose.PDF para Java oferece uma incrível riqueza de recursos, incluindo opções de compressão de PDF, criação e manipulação de tabelas, suporte a gráficos, funções de imagem, funcionalidade extensiva de hiperlinks, controles de segurança ampliados e manipulação de fontes personalizadas.

### Aspose.PDF Java para NetBeans (Maven)

- Aspose.PDF Java para NetBeans (Maven) é um Plugin para **NetBeans IDE** pela **Aspose.** O Plugin é destinado a desenvolvedores que usam a plataforma Maven para desenvolvimentos em Java e querem usar o Aspose.PDF para Java em seus projetos. O Plugin permite criar projetos maven para usar a API Aspose.PDF para Java e também baixar [Exemplos de Código](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) da API.

## ASSISTENTES

O plugin contém dois assistentes

### Projeto Aspose.PDF Maven (assistente)

- Este assistente de Novo Projeto permite que os desenvolvedores criem um projeto **Maven** para usar o Aspose.PDF para Java a partir de Novo Projeto -> Maven-> Projeto Aspose.PDF Maven.
- A referência da dependência maven do Aspose.PDF para Java API é automaticamente buscada do [Aspose Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf) e adicionada no pom.xml.
- O projeto criado sempre conterá a versão mais recente disponível da dependência **Maven** para Aspose.PDF para Java API.
- As etapas do assistente também apresentam a opção de baixar Exemplos de Código para usar o Aspose.PDF para Java API.

### Exemplo de Código Aspose.PDF (assistente)

- Este assistente de Novo Arquivo permite que você copie os [Exemplos de Código](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples) baixados para o seu projeto para usar o Aspose.PDF para Java a partir de **Novo Arquivo -> Java -> Exemplo de Código Aspose.PDF.**
- As amostras disponíveis são exibidas em formato de árvore, de onde o usuário pode selecioná-las categoricamente.

- Todos os exemplos dentro da categoria selecionada serão copiados para a pasta de pacotes **com.aspose.pdf.examples** do projeto junto com os recursos necessários dentro da pasta **src/main/resources** necessários para executar os exemplos.
- Exemplos de Código do Aspose.PDF para a API Java são destinados a demonstrar a variedade da API.
- O assistente também buscará e atualizará para novos disponíveis no [repositório de exemplos do Aspose.PDF para Java.](https://github.com/aspose-pdf/Aspose.PDF-for-Java/tree/master/Examples)

## Download

### Baixar NetBeans

Você deve ter o NetBeans IDE pré-instalado antes de baixar o plugin **Aspose.PDF Java para NetBeans (Maven)**.

O plugin **Aspose.PDF Java para NetBeans (Maven)** suporta **NetBeans IDE 8.0 e 8.1**. Recomendamos usar apenas a versão mais recente do NetBeans IDE.

Para baixar o NetBeans IDE,

1. Navegue para: <https://netbeans.apache.org/download/index.html>

1. Clique em Download para baixar o pacote apropriado do NetBeans IDE.

Baixe apenas o pacote de download do NetBeans com suporte para Java EE / SE da versão mais recente atual, ou seja, **NetBeans IDE 8.1**.

### Baixar Aspose.PDF Java para NetBeans (Maven)

- O plugin pode ser baixado diretamente da URL de hospedagem de plugins do NetBeans: <http://plugins.netbeans.org/plugin/63218>
  e posteriormente pode ser instalado usando a opção do IDE:
  - Vá para **Tools**>**Plugins**>**Downloaded**
  - Clique em **Add Plugins...** e procure o arquivo de extensão .nbm baixado para **Aspose.PDF Java for NetBeans (Maven)**.
  - Clique em **"Install"** para instalar o plugin
- Ou pode ser instalado diretamente (sem download manual da URL de hospedagem de plugins do NetBeans) usando as opções do IDE:
  - Vá para **Tools**>**Plugins**>**Available Plugins**
  - Na caixa de pesquisa, digite: "Aspose"
  - Selecione o plugin **Aspose.PDF Java NetBeans (Maven)**
  - Clique em **"Install"**

### Como aplicar a licença Aspose?

Este Plugin usa uma versão de avaliação do Aspose.PDF. Assim que estiver satisfeito com sua avaliação, você pode comprar uma licença no [site da Aspose](https://purchase.aspose.com/buy).  
Para remover a mensagem de avaliação e as limitações de recursos, uma licença do produto deve ser aplicada. Você receberá um arquivo de licença após comprar o produto. Por favor, siga os passos abaixo para aplicar a licença.

- Certifique-se de que o arquivo de licença seja nomeado como **Aspose.PDF.Java.lic
- Coloque o arquivo **Aspose.PDF.Java.lic** na pasta que contém o Aspose.PDF.jar
- Use o seguinte código para ativar a licença:

```java
License license = new License();
license.setLicense("Aspose.PDF.Java.lic");
```
