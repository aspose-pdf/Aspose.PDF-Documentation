## Apache Struts Web Framework

O framework web Apache Struts é uma solução de código aberto gratuita para criar aplicações web Java.

As aplicações web diferem dos sites convencionais, pois as aplicações web podem criar uma resposta dinâmica. Muitos sites entregam apenas páginas estáticas. Uma aplicação web pode interagir com bancos de dados e motores de lógica de negócios para personalizar uma resposta.

Uma maneira de separar preocupações em uma aplicação de software é usar uma arquitetura Model-View-Controller (MVC). O Modelo representa o código de negócio ou de banco de dados, a Visão representa o código de design da página, e o Controlador representa o código de navegação. O framework Struts é projetado para ajudar os desenvolvedores a criar aplicações web que utilizem uma arquitetura MVC.

[Saiba mais](http://struts.apache.org/birdseye.html)

### Aspose.PDF para Java

O Aspose.PDF para Java é um componente de criação de documentos PDF que permite que suas aplicações Java leiam, escrevam e manipulem documentos PDF sem usar o Adobe Acrobat.

Aspose.PDF for Java é um componente com preço acessível que oferece uma incrível riqueza de recursos, incluindo: opções de compressão de PDF, criação e manipulação de tabelas, suporte a gráficos, funções de imagem, funcionalidade extensa de hiperlink, controles de segurança estendidos e manipulação de fontes personalizadas.

Aspose.PDF for Java permite que você crie arquivos PDF diretamente através da API fornecida e de modelos XML. Usar o Aspose.PDF for Java também permitirá que você adicione capacidades de PDF às suas aplicações em pouco tempo.

## Aspose.Words Java para Struts 1.3

Aspose.Words Java para Struts 1.3 é uma aplicação web de livraria simples escrita usando Struts 1.3 e Maven.

- A aplicação web pretende mostrar a integração e uso do [Aspose.Words for Java](https://products.aspose.com/words/java/) dentro do framework Struts 1.3 e Maven.
- Ela é executada em um contêiner web como o Apache Tomcat e requer maven para construir a aplicação web .war para implantação.

- O caso de uso atual na aplicação exporta registros da grade de livros para o formato de documento MS-WORD criando um documento MS-WORD, criando uma tabela nele e escrevendo textos / dados nele.

## Requisitos do Sistema

A seguir estão os requisitos do sistema para o Aspose.PDF Java para a aplicação web Struts 1.3:

- Java 1.6 ou superior instalado.
- Servidor Apache Tomcat 7 ou superior
- Maven 3
- Componente Aspose.PDF.

## Plataformas Suportadas

A seguir estão as plataformas suportadas:

- Aspose.PDF 10.4.0 ou superior.
- IDE Java (Eclipse, NetBeans, IntelliJ, ...).
