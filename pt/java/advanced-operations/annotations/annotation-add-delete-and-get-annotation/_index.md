**O que são anotações em documentos PDF?**

São objetos adicionais que você adiciona ao seu arquivo para expandir o conteúdo do texto, fazer edições, comentários para outros usuários. Também é possível tornar o texto no documento mais legível, destacá-lo, sublinhá-lo ou adicionar texto completamente novo.

Nós combinamos os diferentes tipos de anotações disponíveis para a biblioteca Aspose.PDF para Java em grupos:

- [Anotação de Texto em PDF](/pdf/pt/java/text-annotation/)
- [Anotação de Destaques em PDF](/pdf/pt/java/highlights-annotation/)
- [Anotação de Figuras em PDF](/pdf/pt/java/figures-annotation/)
- [Anotação Multimídia](/pdf/pt/java/multimedia-annotation/)

- [Anotações Adesivas em PDF](/pdf/pt/java/sticky-annotations/)
- [Anotações Extras](/pdf/pt/java/extra-annotations/)
