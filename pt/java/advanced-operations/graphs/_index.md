## O que é um Gráfico

O principal objetivo do gráfico é mostrar fatos numéricos em forma visual para que possam ser entendidos rápida, fácil e claramente. Assim, gráficos são representações visuais de dados coletados. Os dados também podem ser apresentados na forma de uma tabela; no entanto, uma apresentação gráfica é mais fácil de entender. Isso é especialmente verdadeiro quando há uma tendência ou comparação a ser mostrada.

Adicionar gráficos a documentos PDF é uma tarefa muito comum para desenvolvedores ao trabalhar com o Adobe Acrobat Writer ou outras aplicações de processamento de PDF.
 Existem muitos tipos de gráficos que podem ser usados em aplicativos PDF. Os operadores gráficos usados em fluxos de conteúdo PDF descrevem a aparência das páginas que devem ser reproduzidas em um dispositivo de saída rasterizado.

[Aspose.PDF para Java](/pdf/java/) também suporta a adição de gráficos a documentos PDF. Para esse propósito, a classe Graph é fornecida. Graph é um elemento de nível de parágrafo e pode ser adicionado à coleção de Parágrafos em uma instância de Página. Uma instância de Graph contém uma coleção de Shapes.

Os seguintes tipos de formas são suportados pela classe [Graph](https://reference.aspose.com/pdf/java/com.aspose.pdf.drawing/Graph):

- [Arc](/pdf/java/add-arc/) - às vezes também chamado de bandeira é um par ordenado de vértices adjacentes, mas às vezes também chamado de linha dirigida.
- [Circle](/pdf/java/add-circle/) - exibe dados usando um círculo dividido em setores. Usamos um gráfico de círculo (também chamado de gráfico de pizza) para mostrar como os dados representam porções de um todo ou de um grupo.

- [Curve](/pdf/java/add-curve/) - é uma união conectada de linhas projetivas, cada linha encontrando três outras em pontos duplos ordinários.- [Line](/pdf/java/add-line) - gráficos de linha são usados para exibir dados contínuos e podem ser úteis na previsão de eventos futuros quando mostram tendências ao longo do tempo.
- [Rectangle](/pdf/java/add-rectangle/) - é uma das muitas formas fundamentais que você verá em gráficos, pode ser muito útil para ajudá-lo a resolver um problema.
- [Ellipse](/pdf/java/add-ellipse/) - é um conjunto de pontos em um plano, criando uma forma oval e curva.
