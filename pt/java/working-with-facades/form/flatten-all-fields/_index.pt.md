O método flattenAllFields(..) da classe [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form) permite achatar todos os campos do formulário PDF. Primeiro, você precisa criar um objeto [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form), vincular o arquivo PDF de origem e, em seguida, chamar o método flattenAllFields(...) e, finalmente, salvar o PDF atualizado usando o método save(..).
