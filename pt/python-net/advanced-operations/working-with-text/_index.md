Todos nós, às vezes, precisamos adicionar texto a um arquivo PDF. Por exemplo, quando você quer acrescentar uma tradução abaixo do texto principal, colocar uma legenda ao lado de uma imagem ou simplesmente preencher um formulário. Também é útil que todos os elementos de texto possam ser formatados no estilo desejado por você. As manipulações de texto mais populares em seu arquivo PDF são: adicionar texto ao PDF, formatar texto dentro do arquivo PDF, substituir e girar texto em seu documento. **Aspose.PDF para Python via .NET** é a melhor solução que possui tudo o que você precisa para interagir com o conteúdo de PDFs.

Você pode fazer o seguinte:

- [Adicionar Texto ao Arquivo PDF](/pdf/python-net/add-text-to-pdf-file/) - adicione texto ao seu PDF, use fontes de stream e arquivos, adicione string HTML, adicione um hyperlink, etc.
- [Tooltip de PDF](/pdf/python-net/pdf-tooltip/) - você pode adicionar um tooltip ao texto pesquisado adicionando um botão invisível usando Python.
- [Formatação de Texto dentro do PDF](/pdf/python-net/text-formatting-inside-pdf/) - Muitos recursos que você pode adicionar ao seu documento ao formatar o texto nele. Adicione recuo de linha, adicione borda ao texto, adicione texto sublinhado, adicione quebra de linha com a biblioteca Aspose.PDF.
- [Substituir Texto no PDF](/pdf/python-net/replace-text-in-pdf/) - para substituir texto em todas as páginas de um documento PDF. Primeiro, você precisa usar o TextFragmentAbsorber.
- [Girar Texto Dentro do PDF](/pdf/python-net/rotate-text-inside-pdf/) - gire texto dentro do PDF usando a propriedade de rotação da classe TextFragment.
- [Pesquisar e Obter Texto das Páginas de um Documento PDF](/pdf/python-net/search-and-get-text-from-pdf/) - você pode usar a classe TextFragmentAbsorber para pesquisar e obter texto das páginas.
- [Usando FloatingBox](/pdf/python-net/floating-box/) - esta página explica como formatar texto dentro de uma caixa flutuante.
