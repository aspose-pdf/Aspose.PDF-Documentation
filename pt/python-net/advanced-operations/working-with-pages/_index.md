**Aspose.PDF for Python via .NET** permite inserir uma página em um documento PDF em qualquer localização no arquivo, bem como adicionar páginas ao final de um arquivo PDF. Esta seção mostra como adicionar páginas a um PDF sem o Acrobat Reader.
Você pode adicionar texto ou imagens nos cabeçalhos e rodapés do seu arquivo PDF e escolher diferentes cabeçalhos no seu documento com a biblioteca Python da Aspose.
Além disso, experimente recortar páginas em documentos PDF programaticamente usando Python.

Esta seção ensina como adicionar marcas d'água ao seu arquivo PDF usando a classe Artifact. Você verá o exemplo de programação para esta tarefa.
Adicione número de página usando a classe [PageNumberStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/pagenumberstamp/). Para adicionar um carimbo no seu documento use as classes [ImageStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/imagestamp/) e [TextStamp](https://reference.aspose.com/pdf/python-net/aspose.pdf/textstamp/). Use a adição de marca d'água para criar imagens de fundo no seu arquivo PDF com **Aspose.PDF for Python via .NET**.

- [Deleting Pages](/pdf/python-net/delete-pages/) - exclua páginas do seu arquivo PDF usando a coleção PageCollection.
- [Extracting Pages](/pdf/python-net/extract-pages/) - extraia páginas do seu arquivo PDF usando o Aspose.PDF para Python.
- [Rotating Pages](/pdf/python-net/rotate-pages/) - você pode mudar a orientação das páginas em um arquivo PDF existente.
- [Changing Page Size](/pdf/python-net/change-page-size/) - alter o tamanho da página no seu documento PDF.
- [Adding Headers and Footers](/pdf/python-net/add-headers-and-footers-of-pdf-file/) - adicione texto ou imagens nos cabeçalhos e rodapés do seu arquivo PDF.
- [Cropping PDF Pages](/pdf/python-net/crop-pages/) - você pode recortar páginas em documentos PDF programaticamente com diferentes propriedades de página.
