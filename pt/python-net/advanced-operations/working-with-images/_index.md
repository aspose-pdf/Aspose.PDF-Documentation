Existem muitos métodos e ferramentas para editar e manipular imagens em documentos PDF.
Além das funções habituais de adicionar e remover imagens do arquivo PDF, a biblioteca Aspose.PDF para Python também permite extrair imagens.
**Aspose.PDF for Python** é uma ferramenta inteligente e eficiente para trabalhar com imagens em um PDF existente.

Você pode fazer o seguinte:

- [Add Image to Existing PDF File](/pdf/python-net/add-image-to-existing-pdf-file/) - adicione imagens e referências de uma única imagem no documento PDF, depois controle a qualidade.
- [Delete Images from PDF File](/pdf/python-net/delete-images-from-pdf-file/) - verifique o trecho de código para excluir imagens do arquivo PDF.
- [Extract Images from PDF File](/pdf/python-net/extract-images-from-pdf-file/) - o próximo artigo mostra como extrair imagens de um arquivo PDF usando a biblioteca Python.
- [Search and Get Images from PDF Document](/pdf/python-net/search-and-get-images-from-pdf-document/) - você pode obter uma imagem de uma página individual e pesquisar entre imagens em todas as páginas com Python.
- [Replace Image in Existing PDF File](/pdf/python-net/replace-image-in-existing-pdf-file/) - verifique nosso trecho de código, ele mostra como substituir uma imagem em um arquivo PDF.
