**Parse PDF** documentos é um termo relacionado à extração de vários tipos de informações de arquivos PDF. Esta seção aborda como:

- [Extrair texto de PDF](/pdf/python-net/extract-text-from-pdf/). A análise ou extração de texto é a operação mais popular em PDFs prontos. Você aprenderá sobre a extração de texto de um documento inteiro, de uma página específica ou de uma região específica em uma página.
- [Extrair imagens de PDF](/pdf/python-net/extract-images-from-the-pdf-file/). A extração de imagens faz o mesmo para imagens como a operação acima para texto.
- [Extrair fontes de PDF](/pdf/python-net/extract-fonts-from-pdf/). A extração de fontes é uma operação específica com fontes em PDFs.
- [Extrair dados do formulário](/pdf/python-net/extract-data-from-acroform/). Se você tem vários documentos PDF com formulários, provavelmente precisa obter os dados desses formulários. Este artigo ajudará a entender como extrair dados de AcroForms com Aspose.PDF para Python via .NET.
- [Extrair dados da tabela](/pdf/python-net/extract-data-from-table-in-pdf/). Obtenha dados de tabelas em documentos PDF.
- [Extrair dados vetoriais de PDF](/pdf/python-net/extract-vector-data-from-pdf/). Você pode obter os dados vetoriais (caminho, polígono, polilinha), como posição, cor, espessura da linha, etc.
