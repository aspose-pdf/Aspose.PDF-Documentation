AlternativeHeadline: Carregar e Salvar formatos de arquivo com Aspose.PDF para Python
Abstract: Este artigo fornece uma tabela abrangente detalhando os formatos de arquivo que o Aspose.PDF para Python pode manipular. Ele descreve as capacidades da biblioteca de carregar e salvar vários tipos de documentos, oferecendo insights sobre sua versatilidade no gerenciamento de diferentes formatos de arquivo. Os formatos abrangidos incluem tipos bem conhecidos como PDF, EPUB, HTML e XML, todos os quais podem ser tanto carregados quanto salvos pela biblioteca. Também suporta formatos de imagem como JPEG, PNG e TIFF para ambas as operações. No entanto, alguns formatos como CGM, MHT, PCL, PS, XSLFO e MD estão disponíveis apenas para carregamento, enquanto outros como XLS, XLSX, PPTX, DOC, DOCX e MobiXML estão disponíveis somente para salvamento. Esta tabela serve como um guia prático para usuários que desejam entender a compatibilidade de formatos do Aspose.PDF para Python, destacando tanto seus pontos fortes quanto suas limitações no suporte a formatos de documentos e imagens.
