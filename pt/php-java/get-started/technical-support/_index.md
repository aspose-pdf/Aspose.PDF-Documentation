Aspose.PDF permite que você utilize suporte técnico gratuito para todos os seus produtos. Se você tiver dúvidas sobre o Aspose.PDF, verifique o próximo artigo:

- A fonte de suporte mais popular é [Aspose.Forums](https://forum.aspose.com/). Na seção [Aspose.PDF forum](https://forum.aspose.com/c/pdf/10), você pode fazer sua pergunta e obter uma resposta o mais rápido possível.

- Você deve saber que a Aspose não fornece suporte técnico por telefone. Você pode ligar apenas para questões de vendas e compras.

- Lembre-se de que temos diferenças de fuso horário, o que pode afetar a velocidade das respostas para você.

Se você precisar de ajuda com um problema no Aspose.PDF, siga as próximas recomendações para garantir que ele seja resolvido da maneira mais eficiente:

- A Aspose hospeda todas as APIs Java no [Repositório Aspose](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf). Você pode usar facilmente o Aspose.PDF para PHP via API Java diretamente em seus projetos Maven com configurações simples.

- Talvez a solução para o seu problema já tenha sido discutida e tenha uma solução, para isso, estude o fórum e a documentação, verifique a [Referência da API](https://reference.aspose.com/pdf/java) antes de relatar o problema.

- Quando você formular sua pergunta, inclua o documento original e possivelmente um fragmento do seu código que causa o problema. Você pode compactar esses arquivos em um documento se houver vários.

- Não se preocupe, apenas você e os desenvolvedores da Aspose têm acesso aos arquivos confidenciais.
