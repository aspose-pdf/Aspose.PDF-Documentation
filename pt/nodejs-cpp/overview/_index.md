_Esta página apresenta uma visão geral dos recursos do Aspose.PDF for Node.js via C++._

**Aspose.PDF for Node.js via C++** permite que desenvolvedores manipulem arquivos PDF diretamente no ambiente Node.js. Esta solução Node.js é construída com o uso da tecnologia WebAssembly e baseia‑se no Aspose.PDF for .NET. Você pode usar facilmente o aspose-pdf para Node.js via C++ a partir de e então seguir as instruções de instalação.

**Sobre as extensões de arquivo CJS e MJS**

Node.js oferece suporte a dois sistemas modulares para organizar e compartilhar código. São eles:

- Módulos CommonJS
- Módulos ECMAScript

Os módulos CommonJS eram o método clássico de processamento de dependências no Node.js, enquanto os módulos ECMAScript são a adição mais recente e seguem o formato padrão usado em navegadores modernos. 'cjs', 'mjs' - são extensões usadas para especificar os formatos de módulo (CommonJS e Módulos ECMAScript) usados em arquivos JavaScript. 

Nos trechos de código, você perceberá uma diferença ao trabalhar com o módulo AsposePDFforNode.js. A distinção principal entre 'require' e 'import' está nos seus casos de uso. **Require** está limitado a importar módulos inteiros, enquanto **Import** tem a flexibilidade de importar tanto módulos inteiros quanto exportações específicas desses módulos.

**Duas abordagens diferentes**

JavaScript oferece duas abordagens, 'then' e 'await', para lidar com operações assíncronas.

O método 'then' funciona com promises para gerenciar tanto os resultados bem‑sucedidos quanto os erros em ordem sequencial. Por outro lado, 'await' faz o JavaScript aguardar, literalmente suspendendo a execução da função até que a promise seja resolvida e retorne seu resultado.

O Aspose.PDF for Node.js suporta uma grande variedade de funções, como:

- Definir, obter e remover informações de arquivo PDF
- Excluir Marcadores
- Excluir Anotações
- Extrair um texto de PDF
- Otimizar documento PDF
- Otimizar recursos 
- Divida o PDF em dois arquivos.
- Mesclar dois arquivos em PDF
- Reparar PDF
- Recursos de conversão

Além disso, o Aspose.PDF for Node.js via C++ pode ser usado para converter facilmente documentos PDF para vários formatos com alto desempenho e qualidade perfeita. Veja a seção [Aspose.PDF Formatos de Arquivo Suportados](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). Também, este kit de ferramentas suporta a criptografia e a descriptografia de documentos PDF.
