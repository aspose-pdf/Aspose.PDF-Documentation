## Recursos de segurança do PDF

A API oferece criptografia de 40 bits ou 128 bits. Também é possível definir senhas de usuário e mestra para maior segurança. A manipulação de conteúdo em arquivos, como copiar conteúdo, imprimir, impressão degradada, modificação de anotações, preenchimento de campos de formulário, bem como a montagem de arquivos PDF, pode ser restrita usando esta biblioteca.

- Criptografar arquivo PDF
- Descriptografar arquivo PDF
- Alterar senha
- Assinar PDF
