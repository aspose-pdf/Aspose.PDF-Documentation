Use o **Aspose.PDF for Node.js via C++** para trabalhar com imagens em documentos PDF. É possível adicionar e excluir imagens no seu arquivo PDF no ambiente Node.js.

Você pode fazer o seguinte:

- [Adicionar Imagem ao Arquivo PDF](/pdf/pt/nodejs-cpp/add-image-to-pdf/) - adicionar imagens ao PDF no ambiente Node.js.
- [Excluir Imagens de Arquivo PDF](/pdf/pt/nodejs-cpp/delete-images-from-pdf-file/) - verifique o trecho de código para excluir imagens de um arquivo PDF.
