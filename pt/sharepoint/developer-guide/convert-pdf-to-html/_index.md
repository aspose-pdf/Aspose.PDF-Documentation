{{% alert color="primary" %}}

O Aspose.PDF for SharePoint oferece recurso para converter documentos PDF da biblioteca de documentos do SharePoint para o formato HTML. Neste artigo, demonstraremos a conversão de PDF para HTML.

{{% /alert %}}

## **Convertendo documento PDF para HTML**

Converta o documento PDF da biblioteca de documentos do SharePoint para HTML da seguinte forma:

1. Clique **Convert to HTML** no menu ECB do documento PDF.

![todo:image_alt_text](convert-pdf-to-html_1.png)

2. Faça o download e salve o arquivo HTML resultante no disco.

![todo:image_alt_text](convert-pdf-to-html_2.png)
