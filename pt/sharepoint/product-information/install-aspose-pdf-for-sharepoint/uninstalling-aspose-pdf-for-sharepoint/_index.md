{{% alert color="primary" %}}

Para desinstalar o Aspose.PDF for SharePoint, basta executar o programa de instalação. Se o Aspose.PDF for SharePoint já estiver instalado, o programa de instalação sugere removê-lo.

![todo:image_alt_text](uninstalling-aspose-pdf-for-sharepoint_1.png)

Durante a desinstalação, o programa de instalação desativa o recurso Aspose.PDF for SharePoint para todas as coleções de sites e retracta a solução da fazenda de servidores.

{{% /alert %}}
