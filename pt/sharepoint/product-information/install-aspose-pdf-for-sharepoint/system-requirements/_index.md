## **Requisitos do Servidor**

Para instalar o Aspose.PDF for SharePoint, você precisa de um computador executando o Microsoft SharePoint Server 2010.

## **Requisitos do Cliente**

Você precisa de um navegador para acessar sites do SharePoint. O Aspose.PDF for SharePoint foi testado com os seguintes navegadores:

- Microsoft Internet Explorer 8
- Mozilla FireFox 3.5
- Google Chrome 3
- Safari 4
