{{% alert color="primary" %}}

Com Aspose.PDF for SharePoint, você pode exportar listas, itens de lista e páginas Wiki do SharePoint de bibliotecas do SharePoint para o formato PDF.

{{% /alert %}}

{{% alert color="primary" %}}

Para gerar documentos, Aspose.PDF for SharePoint depende de uma versão embutida do [Aspose.PDF for .NET](http://www.aspose.com/categories/.net-components/aspose.pdf-for-.net/default.aspx).


{{% /alert %}}
