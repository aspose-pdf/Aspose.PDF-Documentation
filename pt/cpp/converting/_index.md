A biblioteca **Aspose.PDF for C++** oferece a capacidade de converter documentos PDF para vários formatos, bem como converter de outros formatos para PDF.

Nossa biblioteca **Aspose.PDF for C++** permite que você converta com sucesso, rapidez e facilidade seus documentos PDF para os formatos mais populares e vice-versa.

## Como usar Aspose.PDF para converter

Aspose.PDF suporta o maior número de formatos de documentos populares, tanto para carregamento quanto para salvamento.

Preste atenção que a seção atual descreve apenas conversões populares.
Para uma lista completa de formatos suportados, veja [Formatos de Arquivos Suportados pelo Aspose.PDF](https://docs.aspose.com/pdf/cpp/supported-file-formats/).

Explore as seções sobre conversão de documentos com trechos de código.

Documentos Word são tão versáteis quanto possível e editáveis. Converter PDF para Word manualmente é uma tarefa muito demorada. Este artigo mostrará como converter programaticamente PDF para Word em C++.

- [Converter PDF para Microsoft Word](/pdf/pt/cpp/convert-pdf-to-word/) - você pode converter seu documento PDF para o formato Word com C++

Os formatos de número são necessários não apenas para tornar os dados na tabela mais fáceis de ler, mas também para tornar a tabela mais fácil de usar. Claro, se você precisar converter esses dados de um documento PDF para o formato Excel, use nossa biblioteca Aspose.PDF.

- [Converter PDF para Microsoft Excel](/pdf/pt/cpp/convert-pdf-to-excel/) - esta seção descreve como converter documento PDF para XLSX, ODS, CSV e SpreadSheetML

O formato PowerPoint é usado para criar várias apresentações. Arquivos PPT contêm um grande número de slides ou páginas contendo várias informações.

- [Converter PDF para Microsoft PowerPoint](/pdf/pt/cpp/convert-pdf-to-powerpoint/) - aqui estamos falando sobre converter PDF para PowerPoint acompanhando o processo de conversão

HyperText Markup Language é uma linguagem de descrição de documentos de hipertexto, uma linguagem padrão para criar páginas web. Com Aspose.PDF para C++ você pode facilmente converter documentos HTML.

- [Converter arquivo PDF para formato HTML](/pdf/pt/cpp/convert-pdf-to-html/) - converta seus documentos PDF para arquivos HTML como páginas separadas ou como uma única página

Existem muitos formatos de imagem que precisam ser convertidos para PDF para diferentes propósitos. Aspose.PDF permite os formatos de imagem mais populares e vice-versa.

- [Converter formatos de imagens para arquivo PDF](/pdf/pt/cpp/convert-images-format-to-pdf/) - converta diferentes formatos de imagens para arquivo PDF
- [Converter PDF para vários formatos de imagens](/pdf/pt/cpp/convert-pdf-to-images-format/) - converta páginas PDF como imagens em JPEG, PNG e outros formatos

Esta seção inclui formatos como: EPUB, Markdown, PCL, XPS, LATex/TeX, Texto e PostScript.

- [Converter outros formatos de arquivo para PDF](/pdf/pt/cpp/convert-other-files-to-pdf/) - este tópico descreve a conversão com vários formatos como EPUB, XPS, Postscript, texto e outros
- [Converter arquivo PDF para outros formatos](/pdf/pt/cpp/convert-pdf-to-other-files/) - este tópico descreve a maneira de converter um documento PDF para vários formatos

PDF/A é uma versão do PDF projetada para o arquivamento de longo prazo de documentos eletrônicos. Honestamente, externamente, é muito difícil determinar se é PDF ou PDF/A. Para verificar este arquivo, são usados validadores. Verifique os seguintes artigos para uma conversão de qualidade de PDF para PDF/A e vice-versa.

- [Converter PDF para formatos PDF/A](/pdf/pt/cpp/convert-pdf-to-pdfa/) - A biblioteca C++ da Aspose.PDF tem uma maneira fácil de converter PDF para PDF/A
