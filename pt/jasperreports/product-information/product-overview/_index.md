### **Bem-vindo à documentação do Aspose.PDF para JasperReports!**

Aspose.PDF para JasperReports é um componente flexível projetado para exportar relatórios do JasperReports e JasperServer para o formato de documento portátil (PDF). Recursos importantes do relatório, como tabelas, gráficos e imagens, são convertidos para PDF com o mais alto grau de precisão.



{{% /alert %}}

### **Descrição do Produto**

Para criar documentos PDF, o Aspose.PDF para JasperReports depende de uma versão embutida do [Aspose.PDF para Java](https://products.aspose.com/pdf/java/), a biblioteca líder de mercado da Aspose. O Adobe Acrobat Professional não precisa estar instalado no sistema para gerar documentos PDF com o Aspose.PDF para JasperReports.
