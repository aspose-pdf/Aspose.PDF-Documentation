## O que é um arquivo PDF?

PDF significa Portable Document Format, usado para exibir documentos em formato eletrônico independente do software, hardware ou sistema operacional no qual são visualizados. Originalmente desenvolvido pela Adobe® Systems como um formato de arquivo universalmente compatível baseado no formato PostScript, tornou-se um padrão internacional de fato para troca de documentos e informações.

Um arquivo PDF exibe exatamente o mesmo conteúdo e layout, independentemente do sistema operacional, dispositivo ou aplicação de software em que é visualizado.

O formato PDF permite integrar vários tipos de conteúdo – texto, imagens e gráficos vetoriais, vídeos, animações, arquivos de áudio, modelos 3D, campos interativos, hyperlinks, botões, etc.

Os PDFs são fáceis de criar, ler e usar por todos.

Oferece opções para configurar diferentes níveis de acesso para proteger o conteúdo e todo o documento, como marcas d'água, senhas ou assinaturas digitais.

No entanto, não basta apenas abrir seu documento. Ao trabalhar com PDF, você enfrentará a necessidade de criar tal documento novamente, editá‑lo ou convertê‑lo para o formato que precisa. Aspose.PDF for Android via Java ajudará a resolver suas tarefas.

## Por que usar Aspose.PDF for Android via Java?

Usar Aspose.PDF for Android via Java em seu projeto lhe proporciona as seguintes vantagens:

- ampla variedade de funções
- certamente funciona com o Java Oracle JDK
- conveniência e facilidade de uso
- Referência da API no formato JavaDoc

Aspose.PDF for Android via Android via Java pode ser integrado a qualquer aplicação, por exemplo uma aplicação web JSP/JSF ou uma aplicação Windows.

Aspose.PDF for Android via Android via Java é rápido e leve. Ele cria documentos PDF de forma eficiente e ajuda sua aplicação a ter melhor desempenho.

## Grande variedade de funções

- compressão de documentos
- suporte para objetos de gráfico
- integração com fontes de dados
- adicionar, atualizar, excluir anexos e anotações
- suporte fácil para converter PDF para outros formatos e vice‑versa
- dividir, concatenar, extrair ou inserir páginas e muito mais
- manipular estilos de texto e formatação
- adicionar ou atualizar criptografia, senha e restrições de segurança
