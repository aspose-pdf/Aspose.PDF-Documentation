Aspose.PDF permite que você utilize suporte técnico gratuito para todos os seus produtos. Se você tem dúvidas sobre Aspose.PDF, consulte o próximo artigo:

- A fonte de suporte mais popular é [Aspose.Forums](https://forum.aspose.com/). Em [Fórum Aspose.PDF](https://forum.aspose.com/c/pdf/10) seção, você pode fazer sua pergunta e obter uma resposta o mais rápido possível.

- Você deve saber, que a Aspose não fornece suporte técnico por telefone. Você pode ligar está disponível apenas para questões de vendas e compras.

- Por favor, lembre-se, que temos diferenças de fuso horário, isso pode afetar a rapidez das respostas para você.

Se você precisar de ajuda com um problema com o Aspose.PDF, siga as próximas recomendações  para garantir que ele seja resolvido da maneira mais eficiente:

- Aspose hospeda todas as APIs Java em [Aspose Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-pdf). Você pode usar facilmente o Aspose.PDF for Android via Java API diretamente em seus projetos Maven com configurações simples

- Talvez a solução para o seu problema já tenha sido discutida e tenha uma solução, para isso, estude o fórum e a documentação, verifique [Referência da API](https://reference.aspose.com/pdf/java) antes de relatar o problema.

- Ao formular sua pergunta, inclua o documento original e, possivelmente, um fragmento do seu código que cause o problema. Você pode compactar esses arquivos em um único documento se houver vários.

- Não se preocupe, apenas você e os desenvolvedores da Aspose têm acesso aos arquivos temporários.
