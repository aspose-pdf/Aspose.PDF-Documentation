## Overview

Para produzir documentos PDF, a máquina para executar o [Aspose.PDF para Android via Java](https://products.aspose.com/pdf/android-java/) não precisa ter o [Adobe Acrobat](https://www.adobe.com/acrobat/acrobat-pro.html) instalado, pois o Aspose.PDF para Java em si é um motor de criação de documentos PDF. No entanto, para visualizar documentos PDF produzidos pelo Aspose.PDF para Android via Java, a máquina precisa ter pelo menos um [Adobe Acrobat Reader](http://www.adobe.com/products/acrobat/readermain.html). Ele pode ser [baixado](https://www.adobe.com/acrobat/pdf-reader.html) gratuitamente.

## Supported Operating Systems

O Aspose.PDF para Android via Java suporta qualquer dispositivo móvel que execute o runtime Java, incluindo, mas não se limitando a:

### Android

- Android API 32
- Android API 31
- Android 4.0
- Android 5.0
- Android 6.0
- Android 7.0
- Android 8.0
- Android 9.0
- Android 10.0
- Android 11.0


## Ambientes de Desenvolvimento

Você pode usar o Aspose.PDF para Android via Java para desenvolver aplicativos em qualquer ambiente de desenvolvimento que tenha como alvo a plataforma Java no Android, mas os seguintes ambientes são explicitamente suportados:

- Android Studio e ferramentas SDK
