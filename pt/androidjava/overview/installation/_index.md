## Componente Aspose.PDF for Android via Java

{{% alert color="primary" %}}

**Aspose.PDF for Android via Java** é um componente que permite aos desenvolvedores criar e manipular documentos PDF programaticamente, sejam eles simples ou complexos. Aspose.PDF for Android via Java permite que os desenvolvedores insiram tabelas, gráficos, imagens, hyperlinks, fontes personalizadas - e muito mais - em documentos PDF. Também pode ser usado para compactar documentos PDF. Aspose.PDF for Android via Java fornece excelentes recursos de segurança para desenvolver PDFs seguros. O recurso mais distintivo do Aspose.PDF for Android via Java\u0027s é que ele suporta a criação de PDF tanto por meio de uma API quanto a partir de modelos XML.

{{% /alert %}} 

## Descrição do Produto

**Aspose.PDF for Android Java** é uma API de processamento de PDF que permite que aplicativos Android leiam, escrevam, manipulem e exibam documentos PDF. A API suporta uma ampla gama de formatos de arquivo padrão, incluindo HTML, TXT, EPUB, XPS e imagens.

**Aspose.PDF for Android via Java** é uma solução acessível que oferece opções de compressão, suporte a gráficos, criação de documentos do zero, manipulação de estilos de texto \u0026 formatação, adição, modificação, extração e exclusão de anexos, anotações e mais recursos avançados populares.67
