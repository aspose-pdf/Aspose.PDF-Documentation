## Componente Aspose.PDF para Android via Java

{{% alert color="primary" %}}

**Aspose.PDF para Android via Java** é um componente que permite aos desenvolvedores criar e manipular documentos PDF programaticamente, sejam eles simples ou complexos. Aspose.PDF para Android via Java permite que os desenvolvedores insiram tabelas, gráficos, imagens, hyperlinks, fontes personalizadas - e mais - em documentos PDF. Também pode ser usado para comprimir documentos PDF. Aspose.PDF para Android via Java fornece excelentes recursos de segurança para desenvolver PDFs seguros. A característica mais distintiva do Aspose.PDF para Android via Java é que ele suporta a criação de PDFs através de uma API e a partir de modelos XML.

{{% /alert %}}

## Descrição do Produto

**Aspose.PDF for Android Java** é uma API de processamento de PDF que permite que aplicativos Android leiam, escrevam, manipulem e exibam documentos PDF. A API suporta uma ampla gama de formatos de arquivo padrão, incluindo HTML, TXT, EPUB, XPS e imagens.

**Aspose.PDF for Android via Java** é uma solução acessível que oferece opções de compressão, suporte a gráficos, criação de documentos do zero, manipulação de estilos e formatação de texto, adição, modificação, extração e exclusão de anexos, anotações e mais recursos avançados populares.
