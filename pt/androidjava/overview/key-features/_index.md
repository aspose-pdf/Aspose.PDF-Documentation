## Funcionalidades Gerais

- Aspose.PDF para Android via Java suporta qualquer dispositivo móvel que execute o runtime Java, incluindo
- API oferece criptografia de 40 bits ou 128 bits

## Versões de PDF Suportadas

Aspose.PDF para Android via Java suporta as versões PDF 1.2, 1.3, 1.4, 1.5, 1.6 e 1.7.

## Gráfico

- Diferentes formas de gráficos, como linha, círculo, arco e curva são suportadas.
- Gráficos podem ser desenhados em qualquer local da página especificado por coordenadas.

- Gráficos podem ser rotacionados, escalados ou preenchidos, e estilos de linha tracejada e poli-tracejada podem ser definidos.
- Gráficos são desenhados com base em coordenadas e têm posição obsoleta em vez de relativa.
- Espaços de cores CMYK e Cinza são suportados.
