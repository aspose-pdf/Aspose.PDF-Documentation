## Recursos Gerais

- Aspose.PDF for Android via Java suporta qualquer dispositivo móvel que execute o runtime Java, incluindo
- API oferece criptografia de 40 bits ou 128 bits

## Versões de PDF suportadas

Aspose.PDF for Android via Java suporta versões de PDF 1.2, 1.3, 1.4, 1.5, 1.6 e 1.7.

## Gráfico

- São suportadas diferentes formas de gráficos, como linha, círculo, arco e curva.
- Os gráficos podem ser desenhados em qualquer local da página especificado por coordenadas.
- Os gráficos podem ser rotacionados, dimensionados ou preenchidos, e estilos de linha tracejada e poli‑tracejada podem ser definidos.
- Os gráficos são desenhados com base em coordenadas e têm posição obsoleta em vez de relativa.
- Os espaços de cores CMYK e Cinza são suportados.
