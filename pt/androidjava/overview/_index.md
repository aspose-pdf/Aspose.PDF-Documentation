_Esta página é uma visão geral dos recursos do Aspose.PDF for Android via Java._

Aspose.PDF for Android Java é uma API de Processamento de PDF que permite que os aplicativos Android leiam, escrevam, manipulem e renderizem documentos PDF. Aspose.PDF for Android via Java fornece uma Interface de Programação de Aplicações (API) muito simples para desenvolvedores, que é fácil de aprender e usar.

O Aspose.PDF for Android via Java suporta uma ampla variedade de funções, como:

- compactação de documentos;
- criação e manipulação de tabelas;
- suporte para objetos de gráfico;
- funcionalidade extensa de hyperlink;
- controles de segurança estendidos;
- manipulação personalizada de Font;
- integração com fontes de dados;
- adicionar ou remover marcadores;
- criar um índice;
- adicionar, atualizar, excluir anexos e anotações;
- importar ou exportar dados de formulário PDF;
- adicionar, substituir ou remover texto e imagens;
- dividir, concatenar, extrair ou inserir páginas;
- transformar páginas em imagem;
- imprimir documentos PDF e muito mais.

Além disso, Aspose.PDF for Android via Java pode ser aplicado para converter texto, imagens, converter PDF para vários formatos de documento com excelente desempenho e boa qualidade.
