## O que é Microsoft Dynamics CRM?

Microsoft Dynamics CRM é um pacote de software de gerenciamento de relacionamento com o cliente desenvolvido pela Microsoft. Pronto para uso, o produto foca principalmente nos setores de Vendas, Marketing e Serviço (help desk), mas a Microsoft tem promovido o Dynamics CRM como uma plataforma XRM e tem incentivado parceiros a usar seu framework proprietário (baseado em .NET) para personalizá-lo. É parte da família de aplicações empresariais Microsoft Dynamics.

Dynamics CRM é uma aplicação cliente-servidor, que, como o Microsoft SharePoint, é principalmente uma aplicação web baseada em IIS que também suporta interfaces extensas de serviços web. Os clientes acessam o Dynamics CRM usando um navegador ou por meio de um plug-in cliente grosso para o Microsoft Outlook.

## Aspose.PDF for .NET

Aspose.PDF for .NET é um componente de criação e manipulação de documentos PDF que permite que suas aplicações .NET leiam, escrevam e manipulem documentos PDF existentes sem usar o Adobe Acrobat.
