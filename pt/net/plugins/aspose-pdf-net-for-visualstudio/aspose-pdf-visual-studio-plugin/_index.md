## Introdução

Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft. É usado para desenvolver programas de computador para Microsoft Windows, bem como sites, aplicações web e serviços web. O Visual Studio utiliza plataformas de desenvolvimento de software da Microsoft, como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store e Microsoft Silverlight. Ele pode produzir tanto código nativo quanto código gerenciado.

### Aspose.PDF for .NET

Aspose.PDF for .NET é um componente de criação e manipulação de documentos PDF que permite que suas aplicações .NET leiam, escrevam e manipulem documentos PDF existentes sem usar o Adobe Acrobat.

Para mais detalhes sobre a [API Aspose.PDF for .NET](https://products.aspose.com/pdf/net/).

### Explore Exemplos de Aspose.PDF for .NET usando o Visual Studio

O Plugin do Visual Studio Aspose - **Explore Exemplos de Aspose.PDF for .NET** é uma ótima ferramenta para baixar e explorar rapidamente exemplos da API Aspose.PDF for .NET. Ele economiza muito tempo e esforço ao fornecer uma opção muito simples para selecionar, baixar e abrir os projetos de exemplo mais recentes sem precisar sair do seu Visual Studio.

Esta versão oferece os seguintes recursos:

- Suporta Visual Studio 2010, Visual Studio 2012 e Visual Studio 2013.
- Fácil de iniciar a partir do menu **Arquivo > Aspose** no Visual Studio.
- Economiza tempo e reduz a curva de aprendizado.
- Permite que você selecione e abra os exemplos mais recentes da API Aspose.PDF for .NET hospedados no Github.com.
- Permite que você abra projetos de exemplo em **C#** ou **VB.NET**.
- Um projeto selecionado é automaticamente aberto no Visual Studio com todas as referências necessárias configuradas para fornecer um ambiente pronto para executar e explorar.
- Baixa automaticamente as bibliotecas de API mais recentes do **NuGet** e exemplos do **GitHub**.

![Pacote de Exemplos Aspose.PDF for .NET](aspose-pdf-visual-studio-plugin_1.png)

## Instalando

É muito simples e fácil instalar o Plugin do Visual Studio Aspose, siga estes passos simples

- Clique duas vezes no arquivo msi baixado.
- Siga os passos, você pode simplesmente pressionar Avançar em cada tela para instalá-lo usando as configurações padrão.

Você receberá um diálogo de sucesso assim que a instalação for concluída.

![Diálogo de Instalação de Exemplos Aspose.PDF for .NET](aspose-pdf-visual-studio-plugin_2.png)

**Nota:** Certifique-se de reiniciar o Visual Studio para que as alterações tenham efeito.

## Usando

Uma vez instalado, você pode facilmente iniciar e usar o plugin interativo conforme descrito abaixo:
**1.** Abra o plugin selecionando **Arquivo > Aspose** seguido de **Novo Projeto de Exemplo Aspose.PDF** no Visual Studio.

![Diálogo de Novo Projeto de Exemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_3.png)

**2.** Na primeira vez que você abrir o plugin, verá o diálogo de seleção de API conforme mostrado abaixo. Levará alguns minutos para baixar a API mais recente e suas bibliotecas do **NuGet** e o código-fonte dos exemplos mais recentes do **GitHub**.

![Diálogo de Download de Exemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_4.png)

**3.** No diálogo Selecionar uma API para Ver Exemplos, selecione ou altere o local onde os projetos de exemplo serão criados e também selecione ou altere o tipo de projeto de exemplo como **C#** ou **VB.NET** e clique em **Continuar** para abrir o projeto de exemplo no Visual Studio.

![Diálogo de Seleção de Pasta do Projeto de Exemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_1.png)

**4.** Novos projetos de Exemplos conforme a seleção do tipo de projeto de **C#** ou **VB.NET** no Visual Studio.

#### Como configurar o código-fonte

Você precisa ter o seguinte instalado para abrir e estender o código-fonte

- Visual Studio 2010.

Siga estes passos simples para começar

1. Baixe/Clone o código-fonte.
1. Abra o Visual Studio 2010 e escolha **Arquivo** > **Abrir Projeto**.
1. Navegue até o código-fonte mais recente que você baixou e abra **AsposeVisualStudioPlugin.sln**.

#### Visão geral do código-fonte

Existem dois projetos na solução

- **AsposeVisualStudioPlugin** - Contém a integração com o Visual Studio, seleção e download automático das APIs e Exemplos da Aspose.
- **Installer** - Projeto de instalador Wix para gerar o arquivo de instalador msi para o plugin.
