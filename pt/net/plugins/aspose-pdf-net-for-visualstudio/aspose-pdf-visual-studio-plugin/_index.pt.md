## Introdução

O Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft. É usado para desenvolver programas de computador para o Microsoft Windows, assim como sites, aplicações web e serviços web. O Visual Studio utiliza plataformas de desenvolvimento de software da Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store e Microsoft Silverlight. Ele pode produzir tanto código nativo quanto código gerenciado.

### Aspose.PDF para .NET

Aspose.PDF para .NET é um componente de criação e manipulação de documentos PDF que permite que suas aplicações .NET leiam, escrevam e manipulem documentos PDF existentes sem usar o Adobe Acrobat.

Para mais detalhes sobre [API Aspose.PDF para .NET](http://www.aspose.com/.net/Pdf-component.aspx)

### Explore exemplos do Aspose.PDF para .NET usando o Visual Studio

Plugin do Visual Studio da Aspose - **Explore exemplos do Aspose.PDF para .NET** é uma ótima ferramenta para baixar e explorar rapidamente exemplos da API Aspose.PDF para .NET. Ele economiza muito tempo e esforço ao fornecer uma opção muito simples para selecionar, baixar e abrir os projetos de exemplo mais recentes sem precisar sair do seu Visual Studio.

- Suporta Visual Studio 2010, Visual Studio 2012 e Visual Studio 2013.

- Um projeto selecionado é automaticamente aberto no Visual Studio com todas as referências necessárias configuradas para proporcionar um ambiente pronto para rodar e explorar.

## Usando

Uma vez instalado, você pode facilmente lançar e usar o plugin interativo conforme descrito abaixo:
**1.** Abra o plugin selecionando **File > Aspose** seguido de **New Aspose.PDF Example Project** no Visual Studio.

![New Aspose.PDF Example Project dialog](aspose-pdf-visual-studio-plugin_3.png)

**2.** A primeira vez que você abrir o plugin, você verá o diálogo de seleção da API conforme mostrado abaixo. Levará alguns minutos para baixar a API mais recente e suas bibliotecas de **NuGet** e o código fonte dos exemplos mais recentes de **GitHub**.

![Aspose.PDF Example Downloading dialog](aspose-pdf-visual-studio-plugin_4.png)
 ![Aspose.PDF Exemplo de Diálogo de Download](aspose-pdf-visual-studio-plugin_4.png)

**3.** No diálogo Selecionar uma API para Ver Exemplos, selecione ou altere o local onde os projetos de exemplo serão criados e também selecione ou altere o tipo de projeto de exemplo como **C#** ou **VB.NET** e clique em **Continuar** para abrir o projeto de exemplo no Visual Studio.

![Aspose.PDF Exemplo de Seleção de Pasta do Projeto](aspose-pdf-visual-studio-plugin_1.png)

**4.** Novos projetos de exemplos conforme a seleção de tipo de projeto de **C#** ou **VB.NET** no Visual Studio.
