## Introdução

O Aspose .NET PDF Import para Módulo DNN permite que desenvolvedores obtenham/leiam conteúdos de documentos PDF sem a necessidade de qualquer outro software, como Adobe Acrobat ou leitor de PDF.

### Características do Módulo

Este módulo demonstra o poderoso recurso de importação fornecido pelo [Aspose.PDF](http://www.aspose.com/pdf-component-suite.aspx). Ele adiciona um controle de navegador de arquivos simples, seleção de Painel de Conteúdo, e um botão **Importar de PDF** na página onde o módulo é adicionado. Ao clicar no botão, os conteúdos do documento são exibidos na tela imediatamente.

## Instalação

Uma vez baixado, siga estes passos para instalar o módulo em seu site DNN:

1. Faça login no site como **Host** ou outra conta de nível Supersuser.
1. Navegue até o menu Host e selecione **Extensões**.
1.
1.
1. Conforme instruído, navegue até o local do arquivo zip baixado, selecione-o e clique em **Abrir**.
1. Clique em **Próximo**, aceite a licença, continue com a instalação. Quando terminar, clique no botão **Retornar**.

Por favor, confira [este vídeo de instalação do módulo](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) do DNN para mais detalhes

**Nota:** Se você receber um erro ao fazer o upload do módulo, isso se deve ao limite de maxRequestLength na configuração web.config da sua instalação do DNN. Abra o web.config e atualize maxRequestLength para 20MB configurando **maxRequestLength=”20480″** e tente fazer o upload do módulo novamente.

## Usando

Após ter instalado o Aspose .NET PDF Import para o módulo DNN, é realmente simples começar a usá-lo em seu site. Por favor, siga estes passos simples para começar

1. Certifique-se de que você está logado no DNN como conta de nível Host ou Admin
1. Navegue até a página onde você deseja adicionar o módulo de Importação
1.
1. Da lista, selecione **Aspose .NET PDF Import for DNN** e arraste para o local de sua escolha na página.

Você adicionou com sucesso o módulo Aspose .NET PDF Import for DNN à sua página. Uma opção de seleção de arquivo e um botão intitulado **Importar de PDF** agora aparecerão na página junto com o dropdown de seleção de Painel de Conteúdo. Qualquer pessoa pode simplesmente selecionar um arquivo PDF e clicar no botão **Importar de PDF** para listar o conteúdo do documento selecionado na página.
