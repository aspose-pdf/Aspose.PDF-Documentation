## Introdução

Aspose .NET PDF Import para o Módulo DNN permite que os desenvolvedores obtenham/leiam conteúdos de documentos PDF sem a necessidade de qualquer outro software, como Adobe Acrobat ou leitor de PDF.

### Recursos do Módulo

Este módulo demonstra o poderoso recurso de importação fornecido pelo [Aspose.PDF](https://products.aspose.com/pdf/net/). Ele adiciona um controle simples de navegador de arquivos, seleção de Painel de Conteúdo e um botão **Importar do PDF** na página onde o módulo é adicionado. Ao clicar no botão, o conteúdo do documento é exibido na tela imediatamente.

## Requisitos do Sistema e Plataformas Suportadas

### Requisitos do Sistema

Para configurar o Aspose .NET Pdf Import para o Módulo DNN, você precisa atender aos seguintes requisitos:

## Instalando

Uma vez baixado, siga estas etapas para instalar o módulo em seu site DNN:

1. Faça login em seu site como **Host** ou outra conta de nível Supersuser.
1. Navegue até o menu Host e selecione **Extensões**.
1. Clique em **Assistente de Instalação de Extensão**.
1. Conforme solicitado, navegue até a localização do arquivo zip baixado, selecione-o e clique em **Abrir**.
1. Clique em **Próximo**, aceite a licença, continue com a instalação. Quando terminar, clique no botão **Retornar**.

Por favor, verifique [este vídeo de instalação do módulo](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) do DNN para mais detalhes.

**Nota:** Se você receber um erro ao fazer o upload do módulo, isso se deve ao limite maxRequestLength na configuração web.config da sua instalação do DNN. Abra o web.config e atualize maxRequestLength para 20MB definindo **maxRequestLength=”20480″** e tente fazer o upload do módulo novamente.

## Usando

Depois de instalar o Aspose .NET PDF Import para o módulo DNN, é muito simples começar a usá-lo em seu site. Siga estas etapas simples para começar

1. Certifique-se de que você está logado no DNN como conta de Host ou Admin.
1. Navegue até a página onde deseja adicionar o módulo de Importação.
1. Selecione **Módulos** -> **Adicionar Novo Módulo** na faixa superior.

1. Na lista, selecione **Aspose .NET PDF Import para DNN** e arraste-o para um local de sua escolha na página.

Você adicionou com sucesso o módulo Aspose .NET PDF Import para DNN à sua página. Um navegador de arquivos e um botão intitulado **Importar do PDF** agora aparecerão na página junto com o menu suspenso de seleção do Painel de Conteúdo. Qualquer pessoa pode simplesmente selecionar um arquivo PDF e clicar no botão **Importar do PDF** para listar o conteúdo do documento selecionado na página.
