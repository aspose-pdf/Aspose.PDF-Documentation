**O que são anotações em documentos PDF?**

Estes são objetos adicionais que você adiciona ao seu arquivo para expandir o conteúdo do texto, fazer edições, comentários para outros usuários. Também é possível tornar o texto no documento mais legível, destacá-lo, sublinhá-lo ou adicionar um texto completamente novo.

Agrupamos os diferentes tipos de anotação disponíveis para a biblioteca Aspose.PDF for .NET em grupos:

- [Anotação de Texto PDF](/pdf/pt/net/text-annotation/)
- [Anotação de Destaques PDF](/pdf/pt/net/highlights-annotation/)
- [Anotação de Figuras PDF](/pdf/pt/net/figures-annotation/)
- [Anotação Multimídia](/pdf/pt/net/multimedia-annotation/)
- [Anotações Adesivas PDF](/pdf/pt/net/sticky-annotations/)
- [Anotações de Link](/pdf/pt/net/link-annotations/)
- [Anotações Extras](/pdf/pt/net/extra-annotations/)
