## Visão Geral
A API pode ser usada para construir aplicações em qualquer sistema operacional que suporte .NET Standard 2.0 e versões posteriores ou .NET Framework 4.0.

Para produzir e manipular documentos PDF, a máquina em que Aspose.PDF for .NET é executado não precisa ter o Adobe Acrobat instalado: Aspose.PDF for .NET é um mecanismo de criação de documentos PDF. No entanto, para visualizar documentos PDF produzidos por Aspose.PDF for .NET, a máquina precisa ter pelo menos um Adobe Acrobat Reader. Ele pode ser baixado gratuitamente.

## Sistemas Operacionais Suportados

### Windows
- Microsoft Windows 2003 Server (x64, x86) SP2[^1].
- Microsoft Windows 2008 Server (x64, x86)[^1].
- Microsoft Windows 2012 Server (x64, x86)[^1].
- Microsoft Windows 2012 R2 Server (x64, x86)[^1].
- Microsoft Windows 2016 Server (x64, x86)[^1].
- Microsoft Windows 2019 Server (x64, x86)[^1].
- Microsoft Windows 2020 Server (x64, x86).
- Microsoft Windows 2022 Server (x64, x86).
- Microsoft Windows Vista (x64, x86)[^1].
- Microsoft Windows XP (x64, x86)[^1].
- Microsoft Windows 7 (x64, x86)[^1].
- Microsoft Windows 8, 8.1 (x64, x86)[^1].
- Microsoft Windows 10 (x64, x86).
- Microsoft Windows 11 (x64, x86).
- Microsoft Azure.

### macOS
- Mac OS X x64 (10.12+).

### Linux
- Linux x64 (6, 7, 27, 9, 8.7+, 18.04, 16.04, 14.04, 18, 17, 42.3+, 12 SP2+).

### Android
- Android (8.0, 10, 11, 12, 13, 14, 15).

## Ambientes de Desenvolvimento
Você pode usar Aspose.PDF for .NET para desenvolver aplicações em qualquer ambiente de desenvolvimento que tenha como alvo a plataforma .NET, mas os seguintes ambientes são explicitamente suportados:

- Microsoft Visual Studio 2010[^1].
- Microsoft Visual Studio 2011[^1].
- Microsoft Visual Studio 2012[^1].
- Microsoft Visual Studio 2013[^1].
- Microsoft Visual Studio 2015[^1].
- Microsoft Visual Studio 2017[^1].
- Microsoft Visual Studio 2019.
- Microsoft Visual Studio 2022.
- Microsoft Visual Studio Code.

## Frameworks Suportados
As seguintes versões do .NET Framework são suportadas:

- .NET Framework 4.0.
- .NET Framework 4.8.1.
- .NET Standard 2.0 (.NET Core 3.1+).
- .NET 6.0.
- .NET 7.0.
- .NET 8.0.

[^1]: Aspose.PDF for .NET é suportado com .NET Framework 4.0. No entanto, como o .NET Framework 4.0 está obsoleto, esse suporte é fornecido apenas para fins de compatibilidade. Observe que reservamos o direito de descontinuar a publicação do assembly para .NET Framework 4.0 a qualquer momento.
