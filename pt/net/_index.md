## Bem-vindo ao Aspose.PDF for .NET

{{% alert color="primary" %}}

Aspose.PDF é uma biblioteca de classes .NET que permite aos desenvolvedores criar documentos PDF, sejam simples ou complexos, programaticamente. Aspose.PDF for .NET permite que os desenvolvedores insiram tabelas, gráficos, imagens, hyperlinks, fontes personalizadas - e mais - em documentos PDF. Além disso, também é possível comprimir documentos PDF. O produto adere a todos os padrões PDF, incluindo ISO 32000-1 (PDF 1.7), ISO 32000-2 (PDF 2.0) e muitas extensões como PDF/A, PDF/UA, PDF/X, PDF/E.

Aspose.PDF for .NET fornece os seguintes recursos principais:

- Converter um documento de HTML, Doc/Docx, Pptx, Markdown, Json, Jpeg e outros.
- Suporte a vários provedores de IA com a capacidade de carregar dados de documentos PDF.
- Suporte à criação de documentos PDF tanto através de uma API quanto a partir de arquivos XML.
- Excelentes recursos de segurança baseados em certificado e proteção por senha para desenvolver documentos PDF seguros.
- Compatível com assinaturas digitais robustas (incluindo PAdES).
- Criação de Formulários PDF (formulários interativos, AcroForms, formulários dinâmicos XFA).

e muito mais.
