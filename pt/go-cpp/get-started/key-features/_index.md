## Software multiplataforma

Suporta várias plataformas e versões do Go. Veja [Requisitos do Sistema](/pdf/pt/go-cpp/system-requirements/) para detalhes.

## Recursos Gerais

Suporta carregamento e gravação a partir de uma ampla lista de [formatos de arquivo](/pdf/pt/go-cpp/supported-file-formats/).

## Padrões PDF suportados

Aspose.PDF for Go suporta as versões PDF 1.2, 1.3, 1.4, 1.5, 1.6 e 1.7.
