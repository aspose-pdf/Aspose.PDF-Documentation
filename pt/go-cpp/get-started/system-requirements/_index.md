## Plataformas Suportadas do Aspose.PDF for Go via C++

O pacote asposepdf é um kit de ferramentas poderoso que permite que os desenvolvedores manipulem arquivos PDF diretamente e ajudem a realizar várias tarefas com PDF.
Contém recursos exclusivos para converter PDF para outros formatos.

Implementado suporte para as plataformas Linux x64, macOS x86_64, macOS arm64 e Windows x64. Usado [cgo](https://go.dev/wiki/cgo).

A versão específica da plataforma da biblioteca dinâmica da pasta ‘lib’ no diretório raiz do pacote é necessária para distribuir a aplicação resultante:

- **libAsposePDFforGo_linux_amd64.so** para a plataforma Linux x64.
- **libAsposePDFforGo_darwin_arm64.dylib** para a plataforma macOS arm64.
- **libAsposePDFforGo_darwin_amd64.dylib** para a plataforma macOS x86_64.
- **AsposePDFforGo_windows_amd64.dll** para a plataforma Windows x64.

Plataforma Windows x64 requer [MinGW-W64](https://www.mingw-w64.org/) instalado.
