Esta seção descreve como assinar digitalmente documentos PDF de forma segura usando JavaScript. Os termos assinatura eletrônica e assinatura digital são usados de maneira intercambiável, mas essencialmente os dois são diferentes. De forma mais geral, uma assinatura digital vem com um selo aprovado por uma [autoridade certificadora](https://en.wikipedia.org/wiki/Certificate_authority) e é usada para proteger o documento assinado contra adulterações. Em vez disso, uma assinatura eletrônica é frequentemente usada para demonstrar a intenção de assinar um documento.
