## Aspose.PDF for Rust via C++

_Esta página é uma visão geral dos  recursos do Aspose.PDF for Rust._

Rust é uma linguagem de programação moderna, de nível de sistema, projetada para desempenho, confiabilidade e segurança de memória. Desenvolvida pela Mozilla Research e agora mantida pela Rust Foundation, Rust tem como objetivo fornecer o poder e o controle de linguagens como C e C++ enquanto evita armadilhas comuns, como desreferenciação de ponteiro nulo, estouros de buffer e condições de corrida.

Rust destaca-se como uma linguagem de programação poderosa, segura e rápida. É ideal para desenvolvedores que precisam de controle fino sobre memória e desempenho, mas também valorizam práticas modernas de desenvolvimento e garantias de segurança.

Se você está desenvolvendo software de alto desempenho, sistemas seguros ou simplesmente quer explorar uma linguagem de próxima geração, Rust é um forte candidato que vale a pena aprender e adotar.

Usando a biblioteca Aspose.PDF for Rust via C++, você pode implementar recursos avançados para criar arquivos PDF do zero, ou processar completamente documentos PDF existentes sem instalar o Adobe Acrobat. O Aspose.PDF for Rust suporta uma **Ampla Variedade de Funções**.

### Adicionar, excluir ou girar páginas

O [Trabalhando com Páginas PDF](/pdf/pt/rust-cpp/working-with-pages/) A seção explica como você pode adicionar, excluir e girar páginas em PDF.

Você pode adicionar, excluir e girar páginas PDF, obter e definir propriedades da página e definir a cor de fundo, melhorando a organização e a acessibilidade do documento.

### Transformar páginas PDF para outros formatos

Converter páginas PDF em imagens torna o conteúdo mais acessível para usuários que podem ter dificuldade em visualizar ou interagir diretamente com documentos PDF. Aspose.PDF permite converter PDF para diferentes formatos usando Rust. Existem:

- TIFF
- BMP
- JPG
- PNG
- DICOM
- SVG
- DOC
- DOCX
- EPUB
- PPTX
- TEX
- TXT
- XLSX
- XPS

Você pode visitar o [Converter formato PDF](/pdf/pt/rust-cpp/conversion/) página para detalhes.
