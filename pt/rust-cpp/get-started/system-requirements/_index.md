## Plataformas Compatíveis do Aspose.PDF for Rust via C++

O pacote asposepdf é uma poderosa caixa de ferramentas que permite aos desenvolvedores manipular arquivos PDF diretamente e ajuda a realizar várias tarefas relacionadas a PDF.
Contém recursos exclusivos para converter PDF para outros formatos.

Implementado suporte para plataformas Linux x64, macOS x86_64, macOS arm64 e Windows x64.

A versão específica da plataforma da biblioteca dinâmica da pasta ‘lib’ no diretório raiz do pacote é necessária para distribuir o aplicativo resultante: 

- libAsposePDFforRust_linux_amd64.so para a plataforma Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib para a plataforma macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib para a plataforma macOS x86_64 
- AsposePDFforRust_windows_amd64.dll para a plataforma Windows x64.
