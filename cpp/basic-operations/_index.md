**Basic operations** section describes the first and simplest operation with PDF documents in Aspose.PDF for Java:

- [Create PDF Document](/pdf/cpp/create-document/) - this article describes how to create PDF file using C++.
- [Open PDF document](/pdf/cpp/open-pdf-document/) - this article describes various ways how to open a PDF document in Java application.
- [Save PDF document](/pdf/cpp/save-pdf-document/) - this article describes various ways how to save a PDF document to file, stream, send to Web or save as PDF/A (or PDF/X) document.
