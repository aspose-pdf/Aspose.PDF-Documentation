_Esta página es una visión general de las características de Aspose.PDF para Python._

**Aspose.PDF for Python via .NET**, API de procesamiento de PDF que permite a los desarrolladores trabajar con documentos PDF sin necesidad de Microsoft Office® o Adobe Acrobat Automation. Consulte las páginas de destino de [Aspose.PDF for Python](https://products.aspose.com/pdf/python-net/) para una descripción más detallada de las características y posibilidades de la biblioteca.

Aspose.PDF for Python admite una gran variedad de funciones, como:

- Admite la mayoría de los estándares PDF establecidos y especificaciones PDF.
- Capacidad de leer y exportar PDFs en múltiples formatos de imagen, incluidos BMP, GIF, JPEG y PNG.
- Establecer información básica (p. ej., autor, creador) del documento PDF.
- Configurar propiedades de página PDF (p. ej., ancho, alto, cropbox, bleedbox, etc.).
- Establecer numeración de páginas, nivel de marcadores, tamaños de página, etc.
- Capacidad de trabajar con texto, párrafos, encabezados, hipervínculos, gráficos, adjuntos, etc.

Además, Aspose.PDF for Python vía .NET se puede utilizar para convertir fácilmente EPUB, Markdown, Text, XPS, PostScript, XML, LaTex a PDF y convertir PDF a varios formatos de documento con excelente rendimiento y buena calidad.

Pruebe nuestras [Aplicaciones gratuitas en línea](https://products.aspose.app/pdf/applications) que demuestran algunas de las funcionalidades más populares de Aspose.PDF.
