_Esta página ofrece una descripción general de las características de Aspose.PDF para Python._

**Aspose.PDF para Python a través de .NET** es una API de procesamiento de PDF que permite a los desarrolladores trabajar con documentos PDF sin necesidad de Microsoft Office® ni de la automatización de Adobe Acrobat. Consulte las páginas del producto de [Aspose.PDF para Python](https://products.aspose.com/pdf/python-net/) para obtener una descripción más detallada de las características y posibilidades de la biblioteca.

Aspose.PDF para Python admite una amplia variedad de funciones, como:

- Admite la mayoría de los estándares PDF establecidos y las especificaciones PDF.
- Capacidad para leer y exportar PDFs en varios formatos de imagen, incluidos BMP, GIF, JPEG y PNG.
- Establecer información básica (p. ej., autor, creador) del documento PDF.
- Configurar propiedades de página PDF (p. ej., ancho, alto, cropbox, bleedbox, etc.).
- Establecer numeración de páginas, nivel de marcadores, tamaños de página, etc.
- Capacidad para trabajar con texto, párrafos, encabezados, hipervínculos, gráficos, archivos adjuntos, etc.

Además, Aspose.PDF para Python a través de .NET permite convertir fácilmente EPUB, Markdown, Text, XPS, PostScript, XML y LaTex a PDF, así como convertir PDF a varios formatos de documento con un rendimiento excelente y buena calidad.

Pruebe nuestras [aplicaciones en línea gratuitas](https://products.aspose.app/pdf/applications), que muestran algunas de las funcionalidades más populares de Aspose.PDF.
