Aspose.PDF le permite usar soporte técnico gratuito para todos sus productos. Si tiene preguntas sobre Aspose.PDF, consulte el siguiente artículo:

- La fuente de soporte más popular es [Aspose.Forums](https://forum.aspose.com/). En la sección del [foro de Aspose.PDF](https://forum.aspose.com/c/pdf/10) puede hacer su pregunta y obtener una respuesta lo antes posible.

- Debe saber que Aspose no brinda soporte técnico por teléfono. Las llamadas solo están disponibles para consultas de ventas y compras.

- Por favor recuerde que tenemos diferencias horarias, lo que puede afectar la rapidez de las respuestas.

Si necesita ayuda con un problema de Aspose.PDF, siga las siguientes recomendaciones para asegurarse de que se resuelva de la manera más eficiente:

- En primer lugar, verifique que está usando la última versión de Aspose.PDF antes de informar el problema; consulte [Descargas de Aspose.PDF para Python](https://pypi.org/project/aspose-pdf/) para conocer la versión más reciente.

- Es posible que la solución a su problema ya haya sido discutida y exista una solución; para ello, revise el foro y la documentación, y consulte la Referencia de API antes de informar el problema.

- Al formular su pregunta, incluya el documento original y, si es posible, un fragmento de su código que cause el problema. Puede comprimir estos archivos en un solo documento si son varios.

- No se preocupe, solo usted y los desarrolladores de Aspose tienen acceso a los archivos adjuntos.
