El análisis de documentos PDF se refiere a la extracción de distintos tipos de información de un archivo PDF. En esta sección encontrará información para:

- [Extraer texto de PDF](/pdf/es/python-net/extract-text-from-pdf/). La extracción de texto es una de las operaciones más habituales con archivos PDF ya creados. Aquí verá cómo extraer texto de un documento completo, de una página concreta o de una región específica.
- [Extraer imágenes de PDF](/pdf/es/python-net/extract-images-from-the-pdf-file/). Extrae imágenes incrustadas de un PDF para reutilizarlas o procesarlas por separado.
- [Extraer fuentes de PDF](/pdf/es/python-net/extract-fonts-from-pdf/). Inspecciona las fuentes usadas en un documento PDF.
- [Extraer datos del formulario](/pdf/es/python-net/extract-data-from-acroform/). Si trabajas con documentos PDF que contienen formularios, aquí aprenderás a extraer datos de AcroForms con Aspose.PDF for Python via .NET.
- [Extraer datos de la tabla](/pdf/es/python-net/extract-data-from-table-in-pdf/). Obtén datos tabulares desde un documento PDF.
- [Extraer datos vectoriales de PDF](/pdf/es/python-net/extract-vector-data-from-pdf/). Recupera datos vectoriales como rutas, polígonos y polilíneas, junto con propiedades como la posición, el color y el grosor de línea.
