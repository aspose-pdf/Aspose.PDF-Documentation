**Parse PDF** documentos es un término relacionado con la extracción de varios tipos de información de un archivo PDF. Esta sección cubre cómo:

- [Extraer texto de PDF](/pdf/python-net/extract-text-from-pdf/). El análisis o extracción de texto es la operación más popular con PDFs listos. Aprenderás a analizar texto de todo un documento, una página en particular o una región específica en una página.
- [Extraer imágenes de PDF](/pdf/python-net/extract-images-from-the-pdf-file/). La extracción de imágenes hace lo mismo para las imágenes que la operación anterior para el texto.
- [Extraer fuentes de PDF](/pdf/python-net/extract-fonts-from-pdf/). La extracción de fuentes es una operación específica con fuentes en PDFs.
- [Extraer datos del formulario](/pdf/python-net/extract-data-from-acroform/). Si tienes varios documentos PDF con formularios, probablemente necesites obtener los datos de esos formularios. Este artículo te ayudará a entender cómo extraer datos de AcroForms con Aspose.PDF para Python a través de .NET.
- [Extraer datos de la tabla](/pdf/python-net/extract-data-from-table-in-pdf/). Obtén datos de la tabla en un documento PDF.
- [Extraer datos vectoriales de PDF](/pdf/python-net/extract-vector-data-from-pdf/). Puedes obtener los datos vectoriales (ruta, polígono, polilínea), como posición, color, ancho de línea, etc.
