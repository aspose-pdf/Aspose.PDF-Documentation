Las funciones de folleto y N-Up ayudan a preparar documentos PDF para impresión y presentación compacta de páginas. Estos flujos de trabajo reorganizan el orden de las páginas o colocan múltiples páginas de origen en una sola hoja, lo que es útil para folletos, borradores y salida de folleto listo para imprimir.
