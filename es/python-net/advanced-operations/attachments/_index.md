En esta sección, explicaremos cómo trabajar con adjuntos en PDF usando Aspose.PDF para Python a través de .NET.
Un adjunto es un archivo adicional que se adjunta a un documento principal; puede ser de diversos tipos, como pdf, word, imagen u otros archivos.
Aprenderás cómo agregar adjuntos a un pdf, obtener la información de un adjunto y guardarla en un archivo, eliminar el adjunto de un PDF programáticamente con Python.
