En esta seccion, explicaremos como trabajar con archivos adjuntos en PDF utilizando Aspose.PDF para Python a traves de .NET.
Un archivo adjunto es un archivo adicional que se incorpora a un documento principal; puede ser de varios tipos, como PDF, Word, imagenes u otros archivos.
Aprendera a agregar archivos adjuntos a PDF, obtener la informacion de un archivo adjunto y guardarla en un archivo, y eliminar archivos adjuntos de un PDF mediante programacion con Python.

Utilice estos artículos cuando necesite incrustar archivos de soporte dentro de un PDF, eliminar archivos incrustados de un documento existente o agrupar materiales relacionados como una cartera de PDF.
