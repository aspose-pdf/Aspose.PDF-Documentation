Abstract: Este artículo analiza la necesidad y los métodos para agregar y manipular texto dentro de archivos PDF, un requisito común para tareas como insertar traducciones, subtítulos o completar formularios. El artículo destaca Aspose.PDF para Python a través de .NET como una solución integral para interactuar con el contenido PDF. Describe varias capacidades de la biblioteca, incluyendo agregar texto a PDFs con fuentes personalizables e hipervínculos, implementar información emergente y dar formato al texto con funciones como sangrías de línea y bordes. Todos a veces necesitábamos agregar texto al archivo PDF. Por ejemplo, cuando deseas agregar una traducción debajo del texto principal, colocar un subtítulo junto a una imagen o simplemente rellenar un formulario. También es útil que todos los elementos de texto puedan formatearse con el estilo que desees. Las manipulaciones de texto más populares en tu archivo PDF son: agregar texto a PDF, formatear texto dentro del archivo PDF, reemplazar y rotar texto en tu documento. **Aspose.PDF for Python via .NET** es la mejor solución que tiene todo lo que necesitas para interactuar con el contenido PDF.

Puedes hacer lo siguiente:

- [Agregar texto a archivo PDF](/pdf/python-net/add-text-to-pdf-file/) - agrega texto a tu PDF, usa fuentes desde stream y archivos, agrega una cadena HTML, agrega un hipervínculo, etc.
- [Información emergente PDF](/pdf/python-net/pdf-tooltip/) - puedes agregar una información emergente al texto buscado añadiendo un botón invisible usando Python.
- [Formateo de texto dentro de PDF](/pdf/python-net/text-formatting-inside-pdf/) - Muchas funciones que puedes añadir a tu documento al formatear el texto dentro de él. Agrega sangría de línea, agrega borde de texto, agrega texto subrayado, agrega salto de línea con la biblioteca Aspose.PDF.
- [Reemplazar texto en PDF](/pdf/python-net/replace-text-in-pdf/) - para reemplazar texto en todas las páginas de un documento PDF. Primero necesitas usar TextFragmentAbsorber.
- [Rotar texto dentro de PDF](/pdf/python-net/rotate-text-inside-pdf/) - rota el texto dentro del PDF usando la propiedad de rotación de la clase TextFragment.
- [Buscar y obtener texto de páginas de documento PDF](/pdf/python-net/search-and-get-text-from-pdf/) - puedes usar la clase TextFragmentAbsorber para buscar y obtener texto de las páginas.
- [Usando FloatingBox](/pdf/python-net/floating-box/) - esta página explica cómo formatear texto dentro de una caja flotante.
