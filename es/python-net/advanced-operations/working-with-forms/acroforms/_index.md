## Fundamentos de AcroForms

**AcroForms** - tecnologia unica de formularios PDF de Adobe. AcroForms es un formulario orientado a pagina. Aparecio por primera vez en 1998. Acepta entradas en forma de formato de datos o FDF y formato de datos de formulario XML o xFDF. Los proveedores externos admiten AcroForms. Cuando Adobe introdujo AcroForms, los llamo "PDF form", que es el formato creado con Adobe Acrobat Pro/Standard y no un tipo especial de formulario XFA estatico o dinamico. Los AcroForms son portatiles y funcionan en todas las plataformas.

Puede utilizar AcroForms para agregar páginas adicionales al documento de formulario PDF. Gracias al concepto de Plantillas, puede usar AcroForms para soportar la población del formulario con múltiples registros de base de datos.

PDF 1.7 admite dos métodos diferentes para integrar datos y formularios PDF.
