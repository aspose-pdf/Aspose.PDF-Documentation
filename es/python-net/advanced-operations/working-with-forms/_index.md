Esta sección describe un enfoque rápido y sencillo para completar programáticamente un documento PDF mediante el uso de la biblioteca Aspose.PDF para Python a través de .NET. La sección también analiza cómo se puede utilizar Aspose.PDF para Python para descubrir y mapear los campos disponibles dentro de un PDF existente con AcroForms.

AcroForms permite a los usuarios rellenar, enviar y almacenar los formularios electrónicamente, lo que los convierte en una forma conveniente y eficiente de recopilar información.

- [AcroForms](/pdf/python-net/acroforms/) - crear formulario, rellenar campo de formulario, extraer datos del formulario, agregar y eliminar campos en su PDF con la biblioteca Python.
- [XFA Forms](/pdf/python-net/xfa-forms/) - convertir XFA, uso de IgnoreNeedsRendering.
