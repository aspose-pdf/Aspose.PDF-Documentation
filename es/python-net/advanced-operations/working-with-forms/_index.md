Esta seccion describe un enfoque rapido y sencillo para completar programaticamente un documento PDF mediante Aspose.PDF para Python a traves de .NET. La seccion tambien analiza como puede utilizarse Aspose.PDF para Python para descubrir y mapear los campos disponibles en un PDF existente con AcroForms.

Los AcroForms permiten a los usuarios rellenar, enviar y almacenar formularios electronicamente, lo que los convierte en una forma comoda y eficiente de recopilar informacion.

- [AcroForms](/pdf/es/python-net/acroforms/) - crear formularios, rellenar campos, extraer datos del formulario y agregar o eliminar campos en PDF con Python.
- [Formularios XFA](/pdf/es/python-net/xfa-forms/) - convertir formularios XFA y usar `IgnoreNeedsRendering`.
