Esta sección describe un enfoque rápido y sencillo para completar programáticamente un documento PDF mediante el uso de la biblioteca Aspose.PDF for Python via .NET. También explica cómo utilizar Aspose.PDF for Python para detectar y mapear los campos disponibles en un PDF existente con AcroForms.

AcroForms permite a los usuarios rellenar, enviar y almacenar los formularios electrónicamente, lo que los convierte en una forma cómoda y eficiente de recopilar información.

- [AcroForms](/pdf/es/python-net/acroforms/) - crear formularios, rellenar campos, extraer datos y agregar o eliminar campos en un PDF con la biblioteca de Python.
- [Formularios XFA](/pdf/es/python-net/xfa-forms/) - convertir XFA y usar `IgnoreNeedsRendering`.
