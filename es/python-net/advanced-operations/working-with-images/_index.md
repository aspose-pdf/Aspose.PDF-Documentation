Aspose.PDF for Python via .NET proporciona un conjunto completo de herramientas para operaciones de imágenes en documentos PDF existentes.

Utilice esta sección para insertar nuevas imágenes, eliminar o reemplazar imágenes incrustadas, extraer el contenido de la imagen y examinar los detalles de la ubicación de la imagen en una página.

## Tareas de Imagen Cubiertas

Se cubren las siguientes tareas:

- [Agregar Imagen a un Archivo PDF Existente](/pdf/es/python-net/add-image-to-existing-pdf-file/) - colocar imágenes en coordenadas fijas e incrustar imágenes con diferentes opciones.
- [Eliminar imágenes del archivo PDF](/pdf/es/python-net/delete-images-from-pdf-file/) - eliminar un recurso de imagen o todas las imágenes de una página seleccionada.
- [Extraer imágenes de un archivo PDF](/pdf/es/python-net/extract-images-from-pdf-file/) - extraer imágenes incrustadas de las páginas del PDF.
- [Reemplazar imagen en un archivo PDF existente](/pdf/es/python-net/replace-image-in-existing-pdf-file/) - reemplazar imágenes incrustadas por índice de recurso o ubicación.
- [Buscar y obtener imágenes de un documento PDF](/pdf/es/python-net/search-and-get-images-from-pdf-document/) - encontrar imágenes e inspeccionar la ubicación de las imágenes en un documento.
