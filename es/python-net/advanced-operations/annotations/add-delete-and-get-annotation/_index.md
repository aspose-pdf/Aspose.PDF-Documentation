**¿Qué son las anotaciones en documentos PDF?**

Las anotaciones en documentos PDF son elementos que permiten a los usuarios asignar contenido de una manera determinada en el documento. Las anotaciones pueden usarse para agregar diferentes tipos de notas, resaltar texto, dibujar formas, adjuntar archivos y realizar otras acciones que mejoran la visualización, la interactividad y la interacción dentro de un documento.

Hemos agrupado los diferentes tipos de anotación disponibles para la biblioteca Aspose.PDF for Python en grupos:

- [Anotación de texto PDF](/pdf/python-net/text-annotation/)
- [Anotación de resaltado PDF](/pdf/python-net/highlights-annotation/)
- [Anotación de figuras PDF](/pdf/python-net/figures-annotation/)
- [Anotaciones adhesivas PDF](/pdf/python-net/sticky-annotations/)
- [Anotaciones de enlace](/pdf/python-net/link-annotations/)
- [Anotaciones extra](/pdf/python-net/extra-annotations/)
