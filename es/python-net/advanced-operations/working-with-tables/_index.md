**Aspose.PDF for Python via .NET** permite trabajar con tablas en archivos PDF de forma avanzada. Esta herramienta perfecta ayuda a superar la simplicidad de los PDFs al extraer tablas. Con el recurso de la biblioteca Python puedes crear o añadir fácilmente una tabla en un documento PDF existente, determinar si la tabla se romperá en la página actual, extraer la tabla y eliminar tablas de PDFs existentes.

Puedes hacer lo siguiente:

- [Adding Tables](/pdf/python-net/adding-tables/) - crea tu tabla en un archivo pdf combinando columnas o filas, considerando bordes, márgenes y relleno.
- [Extracting Tables](/pdf/python-net/extracting-table/) - puedes extraer una tabla de un archivo PDF.
- [Integrating Tables with Data Sources](/pdf/python-net/integrate-table/) - integra la tabla con una base de datos, integra la tabla con una fuente de DataFrame de Pandas usando la biblioteca Python.
- [Removing Tables](/pdf/python-net/removing-tables/) - elimina una tabla o varias tablas de un documento PDF.
- [Manipulating Tables in existing PDF](/pdf/python-net/manipulating-tables/) - manipula tablas en tu PDF usando TableAbsorber.
