{{% alert color="primary" %}}

Durante la instalación, Aspose.PDF for SharePoint se activa para todas las colecciones de sitios seleccionadas. Después de la instalación, puede usar el menú Site Actions en el sitio web raíz de una colección de sitios para activar y desactivar Aspose.PDF for SharePoint.

{{% /alert %}}
