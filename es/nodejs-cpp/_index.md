## Bienvenido a Aspose.PDF for Node.js via C++

Node.js es un entorno de ejecución multiplataforma para aplicaciones del lado del servidor y de redes escritas en JavaScript. Esencialmente, le permite ejecutar aplicaciones JavaScript fuera de un contexto de navegador.

Aspose.PDF for Node.js via C++ es un potente conjunto de herramientas que permite ejecutar JavaScript y ayuda a los desarrolladores a realizar diversas tareas con PDF en el entorno Node.js.

Aspose.PDF for Node.js permite a los desarrolladores manipular archivos PDF directamente. Aspose.PDF for Node.js via C++ está construido sobre el uso de la tecnología WebAssembly y se basa en Aspose.PDF for .NET.
