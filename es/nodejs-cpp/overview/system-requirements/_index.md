## Descripción general

Para procesar documentos PDF usando el [Aspose.PDF for Node.js vía C\u002B\u002B](https://products.aspose.com/pdf/nodejs-cpp/), no necesita bibliotecas y programas de terceros.
Requiere instalado [Node.js](https://nodejs.org/). Debes tener Node.js versión 16 o superior. Recomendamos encarecidamente usar versiones LTS.
Node.js puede ser [instalado](https://nodejs.org/en/learn/getting-started/how-to-install-nodejs) de diferentes maneras.

## Sistemas operativos compatibles

Todas las versiones compatibles de [Linux](https://en.wikipedia.org/wiki/Linux) y [macOS](https://www.apple.com/macos/).
