_Esta página es una descripción general de las características de Aspose.PDF for Node.js via C++._

**Aspose.PDF for Node.js via C++** permite a los desarrolladores manipular archivos PDF directamente en el entorno Node.js. Esta solución Node.js está construida sobre el uso de la tecnología WebAssembly y se basa en Aspose.PDF for .NET. Puede usar fácilmente aspose-pdf para Node.js via C++ desde y luego seguir las instrucciones de instalación.

**Acerca de las extensiones de archivo CJS y MJS**

Node.js admite dos sistemas modulares para organizar y compartir código. Estos son:

- Módulos CommonJS
- Módulos ECMAScript

Los módulos CommonJS eran el método clásico de procesamiento de dependencias en Node.js, mientras que los módulos ECMAScript son la incorporación más reciente y siguen el formato estándar utilizado en navegadores modernos. 'cjs', 'mjs' son extensiones usadas para especificar los formatos de módulo (CommonJS y Módulos ECMAScript) utilizados en archivos JavaScript. 

En los fragmentos de código, observará una diferencia al trabajar con el módulo AsposePDFforNode.js. La principal distinción entre 'require' e 'import' radica en sus casos de uso. **Require** está limitado a importar módulos completos, mientras que **Import** tiene la flexibilidad de importar tanto módulos completos como exportaciones específicas de esos módulos.

**Dos enfoques diferentes**

JavaScript ofrece dos enfoques, 'then' y 'await', para manejar operaciones asíncronas.

El método 'then' funciona con promesas para gestionar tanto resultados exitosos como errores en un orden secuencial. Por el contrario, 'await' hace que JavaScript espere, literalmente suspendiendo la ejecución de la función hasta que esa promesa se resuelva y devuelva su resultado.

El Aspose.PDF for Node.js soporta una amplia variedad de funciones, como:

- Establecer, Obtener y Eliminar información del archivo PDF
- Eliminar marcadores
- Eliminar anotaciones
- Extraer un texto de PDF
- Optimizar documento PDF
- Optimizar recursos 
- Dividir PDF en dos archivos.
- Combinar dos archivos en PDF
- Reparar PDF
- Funciones de conversión

Además, Aspose.PDF for Node.js via C\u002B\u002B puede usarse para convertir fácilmente documentos PDF a varios formatos con alto rendimiento y calidad perfecta. Ver la sección [Formatos de archivo compatibles con Aspose.PDF](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/). Además, este conjunto de herramientas soporta el cifrado y descifrado de documentos PDF.
