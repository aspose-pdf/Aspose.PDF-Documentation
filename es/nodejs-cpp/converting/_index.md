El PDF se desarrolló para proporcionar un estándar para presentar documentos y otros materiales de referencia en un formato que es independiente del software de aplicación, el hardware y el sistema operativo. El contenido de los archivos PDF no se limita al texto; puede incluir hipervínculos, imágenes, botones y formularios clicables, firmas electrónicas, marcas de agua y más. Por lo tanto, a menudo es necesario convertir archivos PDF a otro formato para editar o cambiar su contenido. 
**Nuestro Aspose.PDF for Node.js via C++** permite convertir sus documentos PDF de manera exitosa, rápida y sencilla a los formatos más populares. Para una lista completa de los formatos compatibles, consulte la sección [Formatos de archivo compatibles con Aspose.PDF](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/).

Aspose.PDF for Node.js via C++ permite convertir documentos PDF a varios formatos en el entorno Node.js. Además, puede comprobar la calidad de la conversión de Aspose.PDF y ver los resultados en línea con la aplicación convertidora de Aspose.PDF. Aprenda las secciones de conversión de documentos con fragmentos de código.

## Cómo usar Node.js via C++ para la conversión

Los documentos de Microsoft Word son los más versátiles y editables posibles. En este artículo, aprenderá cómo convertir PDF a Word en el entorno Node.js de Aspose.PDF.

- [Convertir PDF a DOC](/pdf/es/nodejs-cpp/convert-pdf-to-doc/) - puedes convertir tu documento PDF al formato Word con Node.js.

Los formatos de número son necesarios no solo para que los datos de la tabla sean más fáciles de leer, sino también para que la tabla sea más fácil de usar. Por supuesto, si necesitas convertir esos datos de un documento PDF al formato Excel, utiliza nuestro Aspose.PDF for Node.js.

- [Convertir PDF a Microsoft XLSX](/pdf/es/nodejs-cpp/convert-pdf-to-xlsx/) - esta sección describe cómo convertir un documento PDF a XLSX.

El formato PowerPoint se utiliza para crear diversas presentaciones. Los archivos PPT contienen una gran cantidad de diapositivas o páginas que incluyen diversa información.

- [Convertir PDF a Microsoft PowerPoint](/pdf/es/nodejs-cpp/convert-pdf-to-powerpoint/) - aquí estamos hablando de convertir PDF a PowerPoint siguiendo el proceso de conversión

HyperText Markup Language es un lenguaje de descripción de documentos hipertexto, un lenguaje estándar para crear páginas web. Con Aspose.PDF for Node.js puedes convertir fácilmente documentos HTML y viceversa.

Existen muchos formatos de imagen que deben convertirse a PDF para diferentes propósitos. Aspose.PDF for Node.js permite los formatos de imagen más populares.

- [Convertir PDF a varios formatos de imagen](/pdf/es/nodejs-cpp/convert-pdf-to-images-format/) - convertir páginas PDF como imágenes en JPEG, PNG, SVG y otros formatos

Esta sección incluye formatos como: EPUB, XPS, TeX, Texto y PDF en escala de grises.

- [Convertir archivo PDF a otros formatos](/pdf/es/nodejs-cpp/convert-pdf-to-other-files/) - este tema describe la forma de convertir documentos PDF a varios formatos

PDF/A es una versión de PDF diseñada para el archivo a largo plazo de documentos electrónicos.
Si sinceramente, externamente, es muy difícil determinar si es PDF o PDF/A. Para verificar este archivo, se utilizan validadores. Consulte los siguientes artículos para una conversión de calidad de PDF a PDF/A y viceversa.

- [Convertir PDF a formatos PDF/A](/pdf/es/nodejs-cpp/convert-pdf-to-pdfa/) - Node.js toolkit by Aspose.PDF tiene una forma fácil de convertir PDF a PDF/A.

- [Convertir PDF/A a formato PDF](/pdf/es/nodejs-cpp/convert-pdfa-to-pdf/) - convertir PDF/A a formato PDF con Node.js fácil, rápido y de alta calidad.
