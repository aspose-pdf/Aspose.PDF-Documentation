El PDF fue desarrollado para proporcionar un estándar para presentar documentos y otros materiales de referencia en un formato que es independiente del software de aplicación, hardware y sistema operativo. El contenido de los archivos PDF no se limita al texto; puede ser hipervínculos, imágenes, botones y formularios clicables, firmas electrónicas, marcas de agua y más. Por lo tanto, a menudo es necesario convertir archivos PDF a algún otro formato para editar o cambiar su contenido.  
**Nuestra herramienta Aspose.PDF para Node.js a través de C++** le permite convertir sus documentos PDF a los formatos más populares de manera exitosa, rápida y sencilla.
 Para obtener una lista completa de formatos compatibles, consulte la sección [Formatos de archivo compatibles con Aspose.PDF](https://docs.aspose.com/pdf/nodejs-cpp/supported-file-formats/).

Aspose.PDF para Node.js a través de C++ permite convertir documentos PDF a varios formatos en el entorno de Node.js. Además, puede verificar la calidad de la conversión de Aspose.PDF y ver los resultados en línea con la aplicación convertidora de Aspose.PDF. Aprenda las secciones de conversión de documentos con fragmentos de código.

## Cómo usar Node.js a través de C++ para la conversión

Los documentos de Microsoft Word son los más versátiles y editables posibles. En este artículo, aprenderá cómo convertir PDF a Word en el entorno de Node.js mediante Aspose.PDF.

- [Convertir PDF a DOC](/pdf/es/nodejs-cpp/convert-pdf-to-doc/) - puede convertir su documento PDF a formato Word con Node.js.

Los formatos numéricos son necesarios no solo para facilitar la lectura de los datos en la tabla, sino también para facilitar el uso de la tabla. Por supuesto, si necesitas convertir dichos datos de un documento PDF a formato Excel, utiliza nuestro Aspose.PDF para Node.js.

- [Convertir PDF a Microsoft XLSX](/pdf/es/nodejs-cpp/convert-pdf-to-xlsx/) - esta sección describe cómo convertir un documento PDF a XLSX.

El formato PowerPoint se utiliza para crear varias presentaciones. Los archivos PPT contienen una gran cantidad de diapositivas o páginas que contienen diversa información.

- [Convertir PDF a Microsoft PowerPoint](/pdf/es/nodejs-cpp/convert-pdf-to-powerpoint/) - aquí hablamos sobre cómo convertir PDF a PowerPoint rastreando el proceso de conversión.

El Lenguaje de Marcado de Hipertexto es un lenguaje de descripción de documentos de hipertexto, un lenguaje estándar para crear páginas web. Con Aspose.PDF para Node.js puedes convertir fácilmente documentos HTML y viceversa.

Existen muchos formatos de imágenes que necesitan ser convertidos a PDF para diferentes propósitos. Aspose.PDF para Node.js permite los formatos de imágenes más populares.

- [Convertir PDF a varios formatos de Imágenes](/pdf/es/nodejs-cpp/convert-pdf-to-images-format/) - convierte páginas de PDF como imágenes en formatos JPEG, PNG, SVG y otros formatos.
This section includes such formats as: EPUB, XPS, TeX, Texto y PDF en escala de grises.

- [Convertir archivo PDF a otros formatos](/pdf/es/nodejs-cpp/convert-pdf-to-other-files/) - este tema describe la forma de convertir un documento PDF a varios formatos

PDF/A es una versión de PDF diseñada para el archivo a largo plazo de documentos electrónicos. Honestamente, externamente, es muy difícil determinar si es PDF o PDF/A. Para verificar este archivo, se utilizan validadores. Consulte los siguientes artículos para una conversión de calidad de PDF a PDF/A y viceversa.

- [Convertir PDF a formatos PDF/A](/pdf/es/nodejs-cpp/convert-pdf-to-pdfa/) - El kit de herramientas de Node.js de Aspose.PDF tiene una forma fácil de convertir PDF a PDF/A.

- [Convertir PDF/A a formato PDF](/pdf/es/nodejs-cpp/convert-pdfa-to-pdf/) - convierta PDF/A a formato PDF con Node.js de forma fácil, rápida y de alta calidad.
