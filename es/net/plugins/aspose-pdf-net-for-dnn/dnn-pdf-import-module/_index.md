## Introducción

Aspose .NET PDF Import para el Módulo DNN permite a los desarrolladores obtener/leer contenidos de documentos PDF sin requerir ningún otro software como Adobe Acrobat o lector de PDF.

### Características del Módulo

Este módulo demuestra la poderosa función de importación proporcionada por [Aspose.PDF](https://products.aspose.com/pdf/net/). Agrega un control de navegador de archivos simple, selección de Panel de Contenido y un botón de **Importar desde PDF** en la página donde se agrega el módulo. Al hacer clic en el botón, el contenido del documento se muestra en la pantalla de inmediato.

## Requisitos del Sistema y Plataformas Soportadas

### Requisitos del Sistema

Para configurar Aspose .NET Pdf Import para el Módulo DNN, necesitas cumplir con los siguientes requisitos:

- DNN 7.0+.

No dudes en contactarnos si deseas instalar este módulo en otras versiones de DNN.

## Instalación

Una vez descargado, sigue estos pasos para instalar el módulo en tu sitio web DNN:

1. Inicia sesión en tu sitio como **Host** o con otra cuenta de nivel Superusuario.
1. Navega al menú Host y selecciona **Extensiones**.
1. Haz clic en **Asistente para Instalar Extensiones**.
1. Como se indica, navega a la ubicación del archivo zip descargado, selecciónalo y haz clic en **Abrir**.
1. Haz clic en **Siguiente**, acepta la licencia, continúa con la instalación. Cuando termines, haz clic en el botón **Regresar**.

Por favor, consulta [este video de instalación del módulo](http://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN para más detalles.

**Nota:** Si obtienes un error al subir el módulo, esto se debe al límite maxRequestLength en el web.config de tu instalación de DNN. Abre web.config y actualiza maxRequestLength a 20MB configurando **maxRequestLength=”20480″** y vuelve a intentar subir el módulo.

## Uso

Después de haber instalado Aspose .NET PDF Import para el módulo DNN, es realmente simple comenzar a usarlo en tu sitio web. Por favor, sigue estos simples pasos para comenzar

1. Asegúrate de haber iniciado sesión en DNN como cuenta de Host o Admin.
1. Navega a la página donde deseas agregar el módulo de Importación.
1. Selecciona **Módulos** -> **Agregar Nuevo Módulo** desde la cinta superior.

1. Desde la lista, selecciona **Aspose .NET PDF Import para DNN** y arrástralo a un lugar de tu elección en la página.

Has agregado exitosamente el módulo Aspose .NET PDF Import para DNN a tu página. Un navegador de archivos y un botón titulado **Importar desde PDF** aparecerán ahora en la página junto con un menú desplegable de selección de Panel de Contenido. Cualquiera puede simplemente seleccionar un archivo PDF y hacer clic en el botón **Importar desde PDF** para listar el contenido del documento seleccionado en la página.
