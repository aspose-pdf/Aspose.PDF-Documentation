## Introducción

Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se utiliza para desarrollar programas informáticos para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio utiliza plataformas de desarrollo de software de Microsoft como Windows API, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede producir tanto código nativo como código administrado.

### Aspose.PDF for .NET

Aspose.PDF for .NET es un componente de creación y manipulación de documentos PDF que permite a tus aplicaciones .NET leer, escribir y manipular documentos PDF existentes sin utilizar Adobe Acrobat.

Para más detalles sobre [la API de Aspose.PDF for .NET](https://products.aspose.com/pdf/net/).

### Explora ejemplos de Aspose.PDF for .NET usando Visual Studio

El plugin de Visual Studio de Aspose - **Explora ejemplos de Aspose.PDF for .NET** es una gran herramienta para descargar y explorar rápidamente ejemplos de la API de Aspose.PDF for .NET. Ahorra mucho tiempo y esfuerzo al proporcionar una opción muy simple para seleccionar, descargar y abrir los últimos proyectos de ejemplo sin tener que salir de tu Visual Studio.

Esta versión proporciona las siguientes características:

- Soporta Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de lanzar desde el menú **Archivo > Aspose** en Visual Studio.
- Ahorra tiempo y reduce la curva de aprendizaje.
- Te permite seleccionar y abrir los últimos ejemplos de la API de Aspose.PDF for .NET alojados en Github.com.
- Te permite abrir proyectos de ejemplo en **C#** o **VB.NET**.
- Un proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias necesarias configuradas para darte un entorno listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de API desde **NuGet** y ejemplos desde **GitHub**.

![Paquete de ejemplos de Aspose.PDF for .NET](aspose-pdf-visual-studio-plugin_1.png)

Puedes descargar el plugin de Visual Studio de Aspose desde una de las siguientes ubicaciones

## Instalando

Es muy simple y fácil instalar el plugin de Visual Studio de Aspose, por favor sigue estos simples pasos

- Haz doble clic en el archivo msi descargado.
- Sigue los pasos, puedes simplemente presionar Siguiente en cada pantalla para instalarlo usando la configuración estándar.

Recibirás un diálogo de éxito una vez que la instalación esté completada.

![Diálogo de instalación de ejemplos de Aspose.PDF for .NET](aspose-pdf-visual-studio-plugin_2.png)

**Nota:** Asegúrate de reiniciar Visual Studio para que los cambios surtan efecto.

## Usando

Una vez instalado, puedes lanzar y usar fácilmente el plugin interactivo como se describe a continuación:
**1.** Abre el plugin seleccionando **Archivo > Aspose** seguido de **Nuevo proyecto de ejemplo Aspose.PDF** en Visual Studio.

![Diálogo de nuevo proyecto de ejemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_3.png)

**2.** La primera vez que abras el plugin verás el diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los últimos ejemplos desde **GitHub**.

![Diálogo de descarga de ejemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_4.png)

**3.** En el diálogo Seleccionar una API para ver ejemplos, selecciona o cambia la ubicación donde se crearán los proyectos de ejemplo y también selecciona o cambia el tipo de proyecto de ejemplos como **C#** o **VB.NET** y haz clic en **Continuar** para abrir el proyecto de ejemplo en Visual Studio.

![Diálogo de selección de carpeta del proyecto de ejemplo Aspose.PDF](aspose-pdf-visual-studio-plugin_1.png)

**4.** Nuevos proyectos de ejemplos según la selección del tipo de proyecto de **C#** o **VB.NET** en Visual Studio.

Puedes obtener el código fuente más reciente desde una de las siguientes ubicaciones

- [Github](https://github.com/asposePdf/Aspose.PDF-for-.NET).

#### Cómo configurar el código fuente

Necesitas tener lo siguiente instalado para abrir y extender el código fuente

- Visual Studio 2010.

Por favor, sigue estos simples pasos para comenzar

1. Descarga/Clona el código fuente.
1. Abre Visual Studio 2010 y elige **Archivo** > **Abrir Proyecto**.
1. Navega hasta el código fuente más reciente que has descargado y abre **AsposeVisualStudioPlugin.sln**.

#### Resumen del código fuente

Hay dos proyectos en la solución

- **AsposeVisualStudioPlugin** - Contiene la integración con Visual Studio, selección y descarga automática de APIs y ejemplos de Aspose.
- **Installer** - Proyecto de instalador Wix para generar un archivo de instalador msi para el plugin.
