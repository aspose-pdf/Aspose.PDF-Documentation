## Visión General
Para producir y manipular documentos PDF, la máquina en la que se ejecuta Aspose.PDF para .NET no necesita tener instalado Adobe Acrobat: Aspose.PDF para .NET es un motor de creación de documentos PDF. Sin embargo, para ver los documentos PDF producidos por Aspose.PDF para .NET, la máquina sí necesita al menos un Adobe Acrobat Reader. Se puede descargar gratuitamente.
