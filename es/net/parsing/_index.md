**Analizar documentos PDF** es un término relacionado con la extracción de diversos tipos de información de un archivo PDF. Esta sección cubre cómo:

- [Extraer texto de PDF](/pdf/net/extract-text-from-pdf/). La extracción de texto es la operación más popular con PDFs ya creados. Aprenderás sobre la extracción de texto de un documento completo, una página en particular o una región específica en una página.
- [Extraer imágenes de PDF](/pdf/net/extract-images-from-the-pdf-file/). La extracción de imágenes hace lo mismo para las imágenes que la operación anterior para el texto.
- [Extraer fuentes de PDF](/pdf/net/extract-fonts-from-pdf/). La extracción de fuentes es una operación específica con fuentes en PDFs.
- [Extraer datos del formulario](/pdf/net/extract-data-from-acroform/).
- [Extraer Datos del Formulario](/pdf/net/extract-data-from-acroform/)
- [Extraer Texto de Sellos](/pdf/net/extract-text-from-stamps/)
- [Extraer Datos de una Tabla](/pdf/net/extract-data-from-table-in-pdf/)
