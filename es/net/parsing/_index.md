**Analizar PDF** documentos es un término relacionado con la extracción de varios tipos de información de archivos PDF. Esta sección cubre cómo:

- [Extraer texto de PDF](/pdf/es/net/extract-text-from-pdf/). La extracción o análisis de texto es la operación más popular con PDFs listos. Aprenderás sobre el análisis de texto de un documento completo, una página en particular o una región particular en una página.
- [Extraer imágenes de PDF](/pdf/es/net/extract-images-from-the-pdf-file/). La extracción de imágenes hace lo mismo para las imágenes que la operación anterior para el texto.
- [Extraer fuentes de PDF](/pdf/es/net/extract-fonts-from-pdf/). La extracción de fuentes es una operación específica con fuentes en PDFs.
- [Extraer datos del formulario](/pdf/es/net/extract-data-from-acroform/). Si tienes un montón de documentos PDF con formularios, probablemente necesites obtener los datos de esos formularios. Este artículo ayudará a entender cómo extraer datos de AcroForms con Aspose.PDF for .NET.
- [Extraer texto de sellos](/pdf/es/net/extract-text-from-stamps/) - obtener información de texto de tu documento PDF.
- [Extraer datos de tabla](/pdf/es/net/extract-data-from-table-in-pdf/) - obtener datos de una tabla en el documento PDF.
- [Extraer datos vectoriales de PDF](/pdf/es/net/extract-vector-data-from-pdf/) - puedes obtener los datos vectoriales (ruta, polígono, polilínea), como posición, color, ancho de línea, etc.
