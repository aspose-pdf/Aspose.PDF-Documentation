{{% alert color="primary" %}}

[Aspose.PDF for .NET](/pdf/es/net/) es una biblioteca de clases que proporciona la funcionalidad para generar documentos PDF desde cero, así como también ofrece la capacidad de editar/manipular documentos PDF existentes. Aspose.PDF está disponible en dos productos distintos [Aspose.PDF for .NET](/pdf/es/net/) y [Aspose.PDF for Java](/pdf/es/java/). Podemos decir que Aspose.PDF soporta “nativamente”:

- El Framework .NET y todos los lenguajes de programación dirigidos a la plataforma .NET, como C# o VB.NET.
- La plataforma Java y el lenguaje de programación Java.

{{% /alert %}}

Tenga en cuenta que también es posible usar [Aspose.PDF for .NET](/pdf/es/net/) desde muchos lenguajes de programación, incluyendo: ASP, C++, ColdFusion, Delphi, JavaScript, Perl, PHP, PowerBuilder, Python, VBScript, etc.

**Existen dos versiones de la biblioteca de clases Aspose.PDF:**

- [Aspose.PDF para .NET](/pdf/es/net/) para usar con cualquier lenguaje de programación que apunte a la plataforma .NET.
- [Aspose.PDF para Java](/pdf/es/java/) para usar con el lenguaje de programación Java.

Usar Aspose.PDF desde cualquier otro lenguaje de programación requiere el uso de alguna *tecnología de interoperabilidad*. La tecnología de interoperabilidad debe permitir que tu lenguaje de programación llame a una biblioteca de clases escrita para .NET o Java.
