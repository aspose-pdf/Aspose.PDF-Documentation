Para obtener el conteo de páginas de un documento PDF usando **Aspose.PDF Java para Python**, simplemente invoque la clase **GetNumberOfPages**.

```Python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'input1.pdf'
page_count = pdf.getPages().size()
print "Conteo de páginas:" . page_count

```
