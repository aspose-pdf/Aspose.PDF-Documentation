Para convertir un documento PDF a formato DOC o DOCX usando **Aspose.PDF Java para Python**, simplemente invoca el módulo **PdfToDoc**.

```python
doc= self.Document()
pdf = self.Document()
pdf=self.dataDir + 'Template.pdf'
doc.save(self.dataDir + 'template.docx')
print "El documento ha sido convertido exitosamente"
```
