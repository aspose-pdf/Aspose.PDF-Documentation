{{% alert color="primary" %}}

La biblioteca Aspose.PDF para Java no puede soportar configuraciones regionales con combinaciones específicas de idioma-país (como "en-KR"). Pero una funcionalidad para establecer una configuración regional clásica para Aspose.PDF está presente en la API y se puede usar llamando al método com.aspose.pdf.LocaleOptions.setLocale().
