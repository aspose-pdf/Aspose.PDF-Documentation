Para obtener metadatos XMP de un archivo PDF, necesitas crear un objeto [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) y vincular el archivo PDF usando el método [bindPdf(..)](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Facade#bindPdf-com.aspose.pdf.IDocument-). Puedes pasar propiedades de metadatos XMP específicas al objeto [PdfXmpMetadata](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/PdfXmpMetadata) para obtener sus valores.
