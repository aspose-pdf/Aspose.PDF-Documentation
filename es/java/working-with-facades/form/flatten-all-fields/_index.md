El método flattenAllFields(..) de la clase [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form) permite aplanar todos los campos del formulario PDF. Primero necesitas crear un objeto [Form](https://reference.aspose.com/pdf/java/com.aspose.pdf.facades/Form), vincular el archivo PDF de origen y luego llamar al método flattenAllFields(...) y finalmente guardar el PDF actualizado usando el método save(..).
