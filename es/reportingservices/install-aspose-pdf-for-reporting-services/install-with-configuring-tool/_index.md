La herramienta Configuring Tool de Aspose.Pdf for Reporting Services puede ayudarle a configurar la extensión Aspose.Pdf for Reporting Services para cualquiera de las versiones compatibles del Report Server (RS). Actualmente es compatible con RS2016, RS2017, RS2019, RS2022 y Power BI Report Server. La herramienta Configuring Tool requiere .NET Framework 4.8.

Si desea instalar la extensión y registrarla en el Report Server, seleccione el tipo de acción ‘Register’. Para anular el registro y desinstalar la extensión, seleccione el tipo de acción ‘Unregister’.

![todo:image_alt_text](install-with-configuring-tool_1.png)

**Los siguientes pasos describen cómo usarlo en detalle:**

1. Introduzca o busque la ruta del archivo DLL para la extensión Aspose.Pdf for Reporting Services;
1. Seleccione el tipo de acción correspondiente: Register o Unregister;
1. Seleccione la pestaña correspondiente a la versión del Report Server que desea configurar. Por favor, asegúrese de haber seleccionado el archivo DLL destinado a su versión de RS. Si la versión solicitada del producto no está instalada en la máquina, la herramienta de configuración le informará con sugerencias. Si está configurando la extensión para la instancia con nombre RS2016 (no la predeterminada \u0027MSSQLSERVER\u0027), por favor introduzca el nombre de la instancia personalizada y luego presione el botón \u0027Refresh\u0027.
1. Asegúrese de que las rutas y nombres de los archivos de configuración mostrados en los cuadros de texto inferiores sean correctos. Si no lo son, puede presionar el botón \u0027Refresh\u0027 para intentar encontrar la instancia RS nuevamente, o puede buscarlos manualmente.
1. Presione el botón \u0027Config\u0027. La herramienta intentará ahora realizar la configuración solicitada y le informará si la configuración fue exitosa o no.
