Puede instalar Aspose.Pdf para Reporting Services usando un instalador MSI. Ejecute Aspose.Pdf.ReportingServices.msi y siga los pasos que ofrece el instalador. El instalador copiará el ensamblado y otros archivos al directorio especificado e instalará el producto en la instancia predeterminada de Reporting Services. No necesita copiar o modificar ningún archivo manualmente, a menos que desee añadir parámetros de configuración especiales como se describe en la sección ‘Configure Aspose.Pdf for Reporting Services’.

La instalación automática es la mejor opción que funciona en la mayoría de los casos. Sin embargo, puede que necesite instalar el producto manualmente en algunas situaciones, como:
 
- La instalación automática falla debido a problemas de seguridad de E/S.
- Necesita instalar el producto en una instancia con nombre (no predeterminada) de Reporting Services 2016 o en varias instancias.
- Actualizas a la última versión y solo deseas reemplazar el ensamblado en lugar de desinstalar la versión anterior e instalar la nueva utilizando el instalador MSI.
 
{{% alert color="primary" %}}

Nota: Ten en cuenta que, en este último caso, podrías terminar con otros componentes (como la documentación offline) que no se actualizan a la versión correspondiente.

{{% /alert %}}
