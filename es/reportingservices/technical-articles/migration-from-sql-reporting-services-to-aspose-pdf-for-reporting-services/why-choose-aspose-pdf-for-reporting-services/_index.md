**Aspose.Pdf for Reporting Services** es una solución .NET robusta que le permite generar informes PDF usando Microsoft SQL Server 2016, 2017, 2019 y 2022 Reporting Services, y Microsoft Power BI Report Server. Aspose.Pdf for Reporting Services no funciona de forma independiente, sino que es una extensión de renderizado para SQL Reporting Services. No solo admite los elementos básicos del informe, como tablas, gráficos, imágenes, Encabezados/Pies de página, líneas, etc., sino que también brinda la capacidad de agregar propiedades personalizadas que le otorgan una ventaja adicional sobre SQL Reporting Services. Al usar estas propiedades, puede generar una Lista de Contenidos que no es compatible con Report Designer. Puede incluir notas al pie/ notas finales en el documento PDF resultante, lo cual no es compatible de forma nativa con Report Builder. Otra característica interesante es la de tener flechas de línea, que tampoco son compatibles con SQL Reporting Services, pero Aspose.Pdf for Reporting Services puede proporcionar la capacidad de agregar flechas al comienzo o al final del elemento de línea. Además, Aspose.Pdf for Reporting Services ofrece la funcionalidad de especificar la alineación del texto (Justify o FullJustify) que no es compatible con Report Designer. Estos modos de texto hacen que el documento resultante esté mejor formateado y sea fácil de leer.

Junto a las características mencionadas anteriormente, también puedes establecer otros parámetros de configuración para los informes, para obtener funcionalidades que no son compatibles de manera nativa con SQL Reporting Services. A continuación se muestra un breve resumen de esas funcionalidades que ofrece Aspose.Pdf for Reporting Services

## Configuración de seguridad

A veces puede que desees exportar un documento PDF protegido con una contraseña, con privilegios limitados de copia de texto e impresión. Desafortunadamente, Reporting Services no admite esta posibilidad. Sin embargo, aún puedes implementarlo utilizando Aspose.Pdf for Reporting Services. Simplemente agrega los parámetros de seguridad correspondientes al informe o al servidor de informes para obtener un documento PDF seguro con privilegios limitados. Para obtener más información, consulta la sección 'Security Setting'.

## Incrustación de Font personalizada

El diseñador de Reporting Services no admite la Font incrustada para el texto; con Aspose.Pdf for Reporting Services puedes incrustar fácilmente la información de la Font en tu documento PDF. Para obtener más información, consulta la sección 'IsFontEmbedded'.

## Metadatos XMP

El diseñador de Reporting Services no admite incrustar los metadatos XMP. Aspose.Pdf for Reporting Services proporciona cuatro parámetros (CreationDate, ModifyDate, MetaDataDate y CreatorTool) para establecer los metadatos XMP correspondientes. Para obtener más información, consulte la sección 'XMP Metadata'.

## PDF/A

Al usar SQL Server Reporting Services, solo puede generar el documento PDF en formato simple, mientras que la generación de documentos PDF/A no está soportada. Pero Aspose.Pdf for Reporting Services ofrece la función de crear documentos compatibles con PDF/A mediante la adición de un único parámetro de configuración. Para obtener más información, consulte la sección 'PDF Conformance'.

## Ángulo de rotación de página

Al usar Aspose.Pdf for Reporting Services, puede establecer el número de grados en que la página debe girarse en sentido horario al mostrarse o imprimirse. Para obtener más información, consulte la sección 'Page Rotating Angle'.

## Tamaño del margen de página

El diseñador de Reporting Services no admite establecer el tamaño de los márgenes de página. Aspose.Pdf for Reporting Services, por otro lado, proporciona cuatro parámetros para establecer el tamaño correspondiente del margen de página. Son PageMarginLeft, PageMarginRight, PageMarginTop y PageMarginBottom. Para obtener más información, consulte la sección 'Page Margin Size'.
