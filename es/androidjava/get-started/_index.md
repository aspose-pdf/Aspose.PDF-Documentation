## ¿Qué es un archivo PDF?

PDF significa Portable Document Format, y se utiliza para mostrar documentos en forma electrónica independiente del software, hardware o sistema operativo en el que se visualizan. Desarrollado originalmente por Adobe® Systems como un formato de archivo universalmente compatible basado en el formato PostScript, se ha convertido en un estándar internacional de facto para el intercambio de documentos e información.

Un archivo PDF muestra el mismo contenido y diseño exactos sin importar el sistema operativo, dispositivo o aplicación de software en el que se visualice.

El formato PDF le permite integrar varios tipos de contenido – texto, imágenes y gráficos vectoriales, videos, animaciones, archivos de audio, modelos 3D, campos interactivos, hipervínculos, botones, etc.

Los PDFs son fáciles de crear, leer y usar por todos.

Ofrece opciones para configurar diferentes niveles de acceso para proteger el contenido y todo el documento, como marcas de agua, contraseñas o firmas digitales.

Sin embargo, no basta solo con abrir su documento. Al trabajar con PDF, se enfrentará a la necesidad de crear dicho documento de nuevo, editarlo o convertirlo al formato que necesite. Aspose.PDF for Android via Java le ayudará a resolver sus tareas.

Usar Aspose.PDF for Android via Java en su proyecto le brinda los siguientes beneficios:

- seguramente funciona con Java Oracle JDK

Aspose.PDF for Android via Android via Java se puede integrar con cualquier aplicación, por ejemplo una aplicación web JSP/JSF o una aplicación Windows.

Aspose.PDF for Android via Android via Java es rápido y liviano. Crea documentos PDF de manera eficiente y ayuda a que su aplicación tenga un mejor rendimiento.
