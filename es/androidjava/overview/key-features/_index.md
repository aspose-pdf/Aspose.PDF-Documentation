## Características generales

- Aspose.PDF for Android via Java admite cualquier dispositivo móvil que ejecute el tiempo de ejecución de Java, incluyendo
- La API ofrece cifrado de 40 bits o 128 bits

## Versiones PDF compatibles

Aspose.PDF for Android via Java admite versiones PDF 1.2, 1.3, 1.4, 1.5, 1.6 y 1.7.

## Gráfico

- Se admiten diferentes formas de gráficos, como línea, círculo, arco y curva.
- Los gráficos pueden dibujarse en cualquier ubicación de la página especificada por coordenadas.
- Los gráficos pueden rotarse, escalarse o rellenarse, y se pueden definir estilos de línea de guiones y poli‑guiones.
- Los gráficos se dibujan basándose en coordenadas y tienen una posición obsoleta en lugar de relativa.
- Se admiten los espacios de color CMYK y Gray.
