## Aspose.PDF for Rust via C++

_Esta página es una visión general de  Aspose.PDF for Rust características._

Rust es un lenguaje de programación moderno a nivel de sistemas diseñado para el rendimiento, la fiabilidad y la seguridad de memoria. Desarrollado por Mozilla Research y ahora mantenido por la Rust Foundation, Rust apunta a proporcionar el poder y el control de lenguajes como C y C++ mientras evita trampas comunes como la desreferenciación de punteros nulos, los desbordamientos de búfer y las condiciones de carrera.

Rust se destaca como un lenguaje de programación potente, seguro y rápido. Es ideal para desarrolladores que necesitan un control granular de la memoria y el rendimiento, pero que también valoran las prácticas de desarrollo modernas y las garantías de seguridad.

Ya sea que estés construyendo software de alto rendimiento, sistemas seguros, o simplemente quieras explorar un lenguaje de próxima generación, Rust es un candidato sólido que vale la pena aprender y adoptar.

Usando Aspose.PDF for Rust a través de la biblioteca C++, puedes implementar capacidades avanzadas para crear archivos PDF desde cero, o procesar completamente documentos PDF existentes sin instalar Adobe Acrobat. Aspose.PDF for Rust soporta una **Amplia variedad de funciones**.

### Agregar, eliminar o rotar páginas

La sección [Trabajando con páginas PDF](/pdf/es/rust-cpp/working-with-pages/) explica cómo puedes agregar, eliminar y rotar páginas en un PDF.

Puedes agregar, eliminar y rotar páginas PDF, obtener y establecer propiedades de página, y establecer el color de fondo, mejorando la organización y accesibilidad del documento.

### Transformar páginas PDF a otros formatos

Convertir páginas PDF a imágenes hace que el contenido sea más accesible para usuarios que pueden tener dificultad para ver o interactuar con documentos PDF directamente. Aspose.PDF permite convertir PDF a diferentes formatos usando Rust. Hay:

- TIFF
- BMP
- JPG
- PNG
- DICOM
- SVG
- DOC
- DOCX
- EPUB
- PPTX
- TEX
- TXT
- XLSX
- XPS

Puedes visitar el [Convertir formato PDF](/pdf/es/rust-cpp/conversion/) página para detalles.
