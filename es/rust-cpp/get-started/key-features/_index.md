## Software multiplataforma

Admite varias plataformas y versiones de Rust. Ver [Requisitos del sistema](/pdf/es/rust-cpp/system-requirements/) para obtener más detalles.

## Características generales

Admite la carga y el guardado de una amplia lista de [formatos de archivo](/pdf/es/rust-cpp/supported-file-formats/).

## Estándares PDF compatibles

Aspose.PDF for Rust admite versiones PDF 1.2, 1.3, 1.4, 1.5, 1.6 y 1.7.
