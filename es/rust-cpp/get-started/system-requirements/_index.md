## Plataformas compatibles de Aspose.PDF for Rust vía C++

El paquete asposepdf es un kit de herramientas potente que permite a los desarrolladores manipular archivos PDF directamente y ayuda a realizar diversas tareas con PDF.
Contiene características únicas para convertir PDF a otros formatos.

Se implementó soporte para las plataformas Linux x64, macOS x86_64, macOS arm64 y Windows x64.

Se requiere la versión específica de la plataforma de la biblioteca dinámica de la carpeta ‘lib’ en el directorio raíz del paquete para distribuir la aplicación resultante: 

- libAsposePDFforRust_linux_amd64.so para la plataforma Linux x64 
- libAsposePDFforRust_darwin_arm64.dylib para la plataforma macOS arm64 
- libAsposePDFforRust_darwin_amd64.dylib para la plataforma macOS x86_64 
- AsposePDFforRust_windows_amd64.dll para la plataforma Windows x64.
