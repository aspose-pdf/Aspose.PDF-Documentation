El PDF fue desarrollado para proporcionar un estándar para presentar documentos y otros materiales de referencia en un formato que sea independiente del software de aplicación, hardware y sistema operativo. El contenido de los archivos PDF no se limita al texto; puede incluir hipervínculos, imágenes, botones y formularios clicables, firmas electrónicas, marcas de agua y más. Por lo tanto, a menudo es necesario convertir los archivos PDF a otro formato para editar o cambiar su contenido.
**Nuestro Aspose.PDF for Go via C++** permite convertir sus documentos PDF a los formatos más populares de forma exitosa, rápida y sencilla. Para obtener una lista completa de los formatos compatibles, consulte la sección [Formatos de archivo compatibles con Aspose.PDF](https://docs.aspose.com/pdf/go-cpp/supported-file-formats/).

Aspose.PDF for Go via C++* permite convertir documentos PDF a varios formatos. Además, puede comprobar la calidad de la conversión de Aspose.PDF y ver los resultados en línea con la aplicación convertidora de Aspose.PDF. Aprenda las secciones de conversión de documentos con fragmentos de código.

## Cómo usar Go para la conversión

Los documentos Word son los más versátiles y editables posibles. Convertir PDF a DOC manualmente es una tarea muy laboriosa. En este artículo, aprenderá cómo convertir PDF a DOC y DOCX en Go.

- [Convertir PDF a DOC](/pdf/es/go-cpp/convert-pdf-to-doc/) - puedes convertir tu documento PDF al formato Word con Go.

Los formatos numéricos son necesarios no solo para que los datos en la tabla sean más fáciles de leer, sino también para que la tabla sea más fácil de usar. Por supuesto, si necesitas convertir dichos datos de un documento PDF al formato Excel, usa nuestro Aspose.PDF for Go.

- [Convertir PDF a Microsoft XLSX](/pdf/es/go-cpp/convert-pdf-to-xlsx/) - esta sección describe cómo convertir un documento PDF a XLSX y CSV.

El formato PowerPoint se utiliza para crear diversas presentaciones. Los archivos PPT contienen una gran cantidad de diapositivas o páginas con diversa información.

- [Convertir PDF a Microsoft PowerPoint](/pdf/es/go-cpp/convert-pdf-to-powerpoint/) - aquí hablamos de convertir PDF a PowerPoint siguiendo el proceso de conversión

HyperText Markup Language es un lenguaje de descripción de documentos hipertexto, un lenguaje estándar para crear páginas web. Con Aspose.PDF for Go puedes convertir fácilmente documentos HTML y viceversa.

Existen muchos formatos de imagen que deben convertirse a PDF para diferentes propósitos. Aspose.PDF for Go permite los formatos de imágenes más populares.

- [Convertir PDF a varios formatos de imágenes](/pdf/es/go-cpp/convert-pdf-to-images-format/) - convertir páginas PDF como imágenes en JPEG, PNG, SVG y otros formatos

Esta sección incluye los siguientes formatos: EPUB, XPS, TeX, Texto y PDF en escala de grises.

- [Convertir archivo PDF a otros formatos](/pdf/es/go-cpp/convert-pdf-to-other-files/) - este tema describe la forma de convertir documentos PDF a varios formatos

PDF/A es una versión de PDF diseñada para el archivado a largo plazo de documentos electrónicos.
Si, honestamente, externamente, es muy difícil determinar si es PDF o PDF/A. Para verificar este archivo, se utilizan validadores. Consulte los siguientes artículos para una conversión de calidad de PDF a PDF/A y viceversa.
