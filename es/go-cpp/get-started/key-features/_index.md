## Software multiplataforma

Admite varias plataformas y versiones de Go. Ver [Requisitos del sistema](/pdf/es/go-cpp/system-requirements/) para más detalles.

## Características generales

Admite cargar y guardar desde una amplia lista de [formatos de archivo](/pdf/es/go-cpp/supported-file-formats/).

## Estándares PDF compatibles

Aspose.PDF for Go admite versiones PDF 1.2, 1.3, 1.4, 1.5, 1.6 y 1.7.
