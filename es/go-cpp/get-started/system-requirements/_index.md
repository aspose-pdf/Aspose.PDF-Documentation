## Plataformas compatibles de Aspose.PDF for Go vía C++

El paquete asposepdf es un conjunto de herramientas potente que permite a los desarrolladores manipular archivos PDF directamente y ayuda a realizar diversas tareas con PDF.
Contiene funciones únicas para convertir PDF a otros formatos.

Implementó soporte para plataformas Linux x64, macOS x86_64, macOS arm64 y Windows x64. Usado [cgo](https://go.dev/wiki/cgo).

Se requiere la versión específica de la plataforma de la biblioteca dinámica de la carpeta 'lib' en el directorio raíz del paquete para distribuir la aplicación resultante:

- **libAsposePDFforGo_linux_amd64.so** para la plataforma Linux x64.
- **libAsposePDFforGo_darwin_arm64.dylib** para la plataforma macOS arm64.
- **libAsposePDFforGo_darwin_amd64.dylib** para la plataforma macOS x86_64.
- **AsposePDFforGo_windows_amd64.dll** para la plataforma Windows x64.

La plataforma Windows x64 requiere [MinGW-W64](https://www.mingw-w64.org/) instalado.
