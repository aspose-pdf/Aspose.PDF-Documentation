**Parse PDF** Dokumente ist ein Begriff, der sich auf die Extraktion verschiedener Arten von Informationen aus einer PDF-Datei bezieht. Dieser Abschnitt behandelt, wie man:

- [Text aus PDF extrahieren](/pdf/de/python-net/extract-text-from-pdf/). Textparsen oder -extraktion ist die beliebteste Operation bei fertigen PDFs. Sie lernen das Textparsen eines gesamten Dokuments, einer bestimmten Seite oder einer bestimmten Region auf einer Seite.
- [Bilder aus PDF extrahieren](/pdf/de/python-net/extract-images-from-the-pdf-file/). Die Bildextraktion macht dasselbe für Bilder wie die oben genannte Operation für Text.
- [Schriftarten aus PDF extrahieren](/pdf/de/python-net/extract-fonts-from-pdf/). Font Extraction ist ein spezifischer Vorgang mit Fonts in PDFs.
- [Daten aus dem Formular extrahieren](/pdf/de/python-net/extract-data-from-acroform/). Wenn Sie eine Reihe von PDF-Dokumenten mit Formularen haben, benötigen Sie wahrscheinlich die Daten aus diesen Formularen. Dieser Artikel hilft zu verstehen, wie man AcroForms-Daten mit Aspose.PDF for Python via .NET extrahiert.
- [Daten aus Tabelle extrahieren](/pdf/de/python-net/extract-data-from-table-in-pdf/). Daten aus einer Tabelle im PDF-Dokument abrufen.
- [Vektor-Daten aus PDF extrahieren](/pdf/de/python-net/extract-vector-data-from-pdf/). Sie können die Vektordaten (Pfad, Polygon, Polylinie) erhalten, wie Position, Farbe, Linienbreite usw.
