## Was ist eine PDF-Datei?

PDF wurde in den 1990er-Jahren von Adobe erstellt, um zwei Ziele zu erreichen. Das erste Ziel ist, dass Sie die Dokumente auf jeder Hardware oder jedem Betriebssystem öffnen können, ohne die zum Erstellen verwendete Anwendung zu benötigen – alles, was Sie benötigen, ist ein PDF‑Reader, und heutzutage erfüllen die meisten Webbrowser diese Anforderung. Das zweite Ziel ist, dass das Layout des Dokuments überall gleich aussieht, wenn Sie ein PDF öffnen.

Aspose.PDF for Python via .NET unterstützt Sie dabei, über das bloße Öffnen von PDF-Dateien hinauszugehen. Sie können die Bibliothek verwenden, um PDFs von Grund auf zu erstellen, vorhandene Dokumente zu bearbeiten, Inhalte zu extrahieren, die Einhaltung von Standards zu validieren und PDFs in andere Formate in Python‑Anwendungen zu konvertieren.

## Warum Aspose.PDF for Python via .NET verwenden?

Die Verwendung von Aspose.PDF for Python via .NET in Ihrem Projekt bietet Ihnen die folgenden Vorteile:

- Ein breites Spektrum an PDF-Verarbeitungsfunktionen
- Zugriff auf die bewährte Aspose.PDF for .NET-Engine von Python
- Eine praktische API für gängige Document-Erstellung, -Bearbeitung, Parsing und Konvertierung
