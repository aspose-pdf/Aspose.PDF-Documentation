In diesem Abschnitt erklären wir, wie man mit Anlagen in PDF unter Verwendung von Aspose.PDF for Python via .NET arbeitet.
Ein Anhang ist eine zusätzliche Datei, die an ein übergeordnetes Dokument angehängt wird; er kann verschiedene Dateitypen umfassen, wie PDF, Word, Bild oder andere Dateien.
Sie lernen, wie man Anhänge zu PDF hinzufügt, die Informationen eines Anhangs abruft und sie in einer Datei speichert sowie den Anhang programmgesteuert mit Python aus einem PDF löscht.

Verwenden Sie diese Artikel, wenn Sie unterstützende Dateien in ein PDF einbetten, eingebettete Dateien aus einem bestehenden Dokument entfernen oder verwandte Materialien zusammen als PDF-Portfolio bündeln müssen.
