Verwenden Sie diesen Abschnitt, wenn Sie PDF-Operationen auf Seitenebene in Python durchführen müssen.

Mit **Aspose.PDF for Python via .NET** können Sie Seiten einfügen, löschen, verschieben, extrahieren, drehen, die Größe ändern und zuschneiden, anschließend Header, Footer, Seitenzahlen, Wasserzeichen und Stempel für Layout- und Review-Workflows anwenden.

## Seitenaufgaben abgedeckt

Die folgenden Aufgaben werden abgedeckt:

- [Seiten hinzufügen](/pdf/de/python-net/add-pages/) - Seiten an bestimmten Positionen einfügen oder Seiten am Ende eines Dokuments anhängen.
- [Seiten verschieben](/pdf/de/python-net/move-pages/) - Seiten innerhalb eines Dokuments oder zwischen Dokumenten verschieben.
- [Seiten löschen](/pdf/de/python-net/delete-pages/) - Seiten aus einem PDF-Dokument entfernen.
- [Seiten extrahieren](/pdf/de/python-net/extract-pages/) - Ausgewählte Seiten in eine neue PDF-Datei extrahieren.
- [Seiten drehen](/pdf/de/python-net/rotate-pages/) - Seitenausrichtung in einer bestehenden PDF-Datei ändern.
- [Seitengröße ändern](/pdf/de/python-net/change-page-size/) - Seitenabmessungen in einem PDF-Dokument anpassen.
- [Kopf‑ und Fußzeilen hinzufügen](/pdf/de/python-net/add-headers-and-footers-of-pdf-file/) - wiederholenden Text, Bilder oder strukturierte Kopf‑/Fußzeilen‑Inhalte hinzufügen.
- [Zuschneiden von PDF-Seiten](/pdf/de/python-net/crop-pages/) - Seiteninhalt mithilfe von Seitengometrie‑Einstellungen zuschneiden.
- [Abrufen und Festlegen von Seiteneigenschaften](/pdf/de/python-net/get-and-set-page-properties/) - Auf verschiedene Eigenschaften von PDF‑Seiten zugreifen und diese ändern, wie Größe, Rotation und Farbeigenschaften.
- [Stempeln](/pdf/de/python-net/stamping/) - Text‑, Bild‑, Seiten‑ und Seitenzahl‑Stempel auf PDF‑Seiten anwenden.
