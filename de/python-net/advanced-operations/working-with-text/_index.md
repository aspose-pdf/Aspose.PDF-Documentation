 Wir alle müssen manchmal Text zu einer PDF-Datei hinzufügen. Zum Beispiel möchten Sie vielleicht eine Übersetzung unter dem Haupttext platzieren, eine Beschriftung neben einem Bild hinzufügen oder ein Antragsformular ausfüllen. Es ist auch hilfreich, wenn alle Textelemente in Ihrem bevorzugten Stil formatiert werden können. Häufige Textoperationen umfassen das Hinzufügen von Text, das Formatieren von Text, das Ersetzen von Text und das Drehen von Text in einem Dokument. **Aspose.PDF for Python via .NET** bietet alles, was Sie benötigen, um mit PDF-Textinhalt zu arbeiten.

Verwenden Sie diesen Abschnitt, wenn Sie Text direkt in PDF‑Seiten in Python‑Anwendungen erzeugen, formatieren, inspizieren oder ändern müssen.

 Sie können Folgendes tun:

- [PDF-Tooltip](/pdf/de/python-net/pdf-tooltip/) -  Sie können einem gefundenen Text einen Tooltip hinzufügen, indem Sie mit Python einen unsichtbaren Button hinzufügen.
- [FloatingBox verwenden](/pdf/de/python-net/floating-box/) - Stilisierten Text und anderen Inhalt in einem schwebenden Container platzieren.
- [Text im PDF ersetzen](/pdf/de/python-net/replace-text-in-pdf/) - vorhandenen Text in einer PDF-Datei ersetzen oder entfernen.
- [Text im PDF drehen](/pdf/de/python-net/rotate-text-inside-pdf/) - Textfragmente und Absätze in einer PDF drehen.
- [Suche und hole Text aus PDF](/pdf/de/python-net/search-and-get-text-from-pdf/) - Text suchen und extrahierte Textdetails von Seiten prüfen.
- [Textformatierung im PDF](/pdf/de/python-net/text-formatting-inside-pdf/) - Zeilenabstand, Ränder, Einrückungen und Textformatierung steuern.
