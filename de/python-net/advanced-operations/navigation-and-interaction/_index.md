Verwenden Sie diesen Abschnitt, wenn Sie interaktive PDF-Navigation in Python erstellen müssen, sei es zum Erstellen von Lesezeichen, Aktualisieren von Linkzielen oder Anhängen von Aktionen an Dokumentereignisse, Seiten und Formularelemente.

## Themen in diesem Abschnitt

- [Aktionen](/pdf/de/python-net/actions/) - In diesem Artikel lernen Sie, wie Sie mit verschiedenen Arten von Aktionen arbeiten.
- [Lesezeichen](/pdf/de/python-net/bookmarks/)- Große Publikationen enthalten normalerweise ein Rahmenwerk von Lesezeichen, das im Bookmarks Pane leicht angezeigt und ausgewählt werden kann, sodass Sie auf ein Lesezeichen klicken können, um zur entsprechenden Seite oder Kapitel zu springen. Der Bookmarks Pane ist ein kontextabhängiges Element und ist in der Seitenleiste nur sichtbar, wenn das geöffnete PDF-Dokument eine Lesezeichenstruktur enthält.
- [Links](/pdf/de/python-net/links/)- Arbeiten Sie mit Links in den PDF-Dokumenten unter Verwendung der Python-Bibliothek.
