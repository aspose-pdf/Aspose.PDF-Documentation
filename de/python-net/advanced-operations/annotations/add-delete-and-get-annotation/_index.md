Anmerkungen in PDF-Dokumenten sind Elemente, die es Benutzern ermöglichen, Inhalte im Dokument auf bestimmte Weise zu platzieren. Anmerkungen können verwendet werden, um verschiedene Arten von Notizen hinzuzufügen, Text hervorzuheben, Formen zu zeichnen, Dateien anzuhängen und weitere Aktionen auszuführen, die das Anzeigen, die Interaktivität und die Interaktion innerhalb eines Dokuments verbessern.

Diese Seite gruppiert die wichtigsten Annotationstypen, die in diesem Teil der Dokumentation unterstützt werden, damit Sie schnell zum benötigten Workflow springen können.

Wir haben die verschiedenen Arten von Annotationen, die für die Aspose.PDF for Python Bibliothek verfügbar sind, in Gruppen zusammengefasst:

- [Markup-Anmerkungen](/pdf/de/python-net/markup-annotations/)
- [Medien-Anmerkungen](/pdf/de/python-net/media-annotations/)
- [Sicherheitsanmerkungen](/pdf/de/python-net/security-annotations/)
- [Form‑Annotationen](/pdf/de/python-net/shape-annotations/)
- [Textanmerkungen](/pdf/de/python-net/text-based-Annotations/)
