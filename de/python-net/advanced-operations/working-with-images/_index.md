Aspose.PDF for Python via .NET bietet ein komplettes Set an Werkzeugen für Bildoperationen in bestehenden PDF-Dokumenten.

Verwenden Sie diesen Abschnitt, um neue Bilder einzufügen, eingebettete Bilder zu entfernen oder zu ersetzen, Bildinhalte zu extrahieren und Details zur Bildplatzierung auf einer Seite zu prüfen.

## Abgedeckte Bildaufgaben

Die folgenden Aufgaben werden abgedeckt:

- [Bild zu bestehender PDF-Datei hinzufügen](/pdf/de/python-net/add-image-to-existing-pdf-file/) - Bilder an festen Koordinaten platzieren und Bilder mit verschiedenen Optionen einbetten.
- [Bilder aus PDF-Datei löschen](/pdf/de/python-net/delete-images-from-pdf-file/) - eine Bildressource oder alle Bilder von einer ausgewählten Seite entfernen.
- [Bilder aus PDF-Datei extrahieren](/pdf/de/python-net/extract-images-from-pdf-file/) - eingebettete Bilder aus PDF-Seiten extrahieren.
- [Bild in vorhandener PDF-Datei ersetzen](/pdf/de/python-net/replace-image-in-existing-pdf-file/) - eingebettete Bilder nach Ressourcenindex oder Position ersetzen.
- [Bilder in PDF-Dokument suchen und abrufen](/pdf/de/python-net/search-and-get-images-from-pdf-document/) - Bilder finden und Bildpositionen in einem Dokument prüfen.
