## Was ist Graph

[Aspose.PDF for Python via .NET](/pdf/de/python-net/) stellt bereit [Graph](https://reference.aspose.com/pdf/python-net/aspose.pdf.drawing/graph/) Klasse zum Zeichnen von Vektorgrafiken in PDF-Dokumenten.

`Graph` ist ein Element auf Absatzebene, also fügst du es einer Seite über die Seite hinzu. `paragraphs` Sammlung. Jeder Graph enthält ein `Shapes` Sammlung, in der Sie Zeichenobjekte wie Linien, Bögen, Kreise, Kurven, Rechtecke und Ellipsen hinzufügen können.

Verwenden Sie diesen Abschnitt, wenn Sie Vektorgrafiken direkt in PDF-Seiten in Python zeichnen müssen, sei es für Diagramme, Schaubilder, Illustrationen oder benutzerdefinierte Seitenanmerkungen.

## Abgedeckte Graph-Formen

Die folgenden Formenarten werden unterstützt von [Graph](https://reference.aspose.com/pdf/python-net/aspose.pdf.drawing/graph/) Klasse:

- [Bogen](/pdf/de/python-net/add-arc/) - zeichne Bogenabschnitte für Teilkreise und gekrümmte Diagrammelemente.
- [Kreis](/pdf/de/python-net/add-circle/) - erstelle Kreisumrisse oder gefüllte Kreise für Marker und visuelle Hervorhebungen.
- [Kurve](/pdf/de/python-net/add-curve/) - füge Bezier‑ähnliche Kurven für benutzerdefinierte Pfade und glatte grafische Elemente hinzu.
- [Linie](/pdf/de/python-net/add-line/) - zeichne gerade Linien, einschließlich formatierter und gestrichelter Linien.
- [Rechteck](/pdf/de/python-net/add-rectangle/) - umrandete, ausgefüllte, verlaufende oder transparente Rechteckformen erstellen.
- [Ellipse](/pdf/de/python-net/add-ellipse/) - ovale Formen zeichnen und bei Bedarf Text darin hinzufügen.

Sie können die Platzierung von Formen auch mit einer Begrenzungsprüfung validieren:

- [Formgrenzen in PDF-Grafiken mit Python prüfen](/pdf/de/python-net/aspose-pdf-drawing-graph-shapes-bounds-check/)
